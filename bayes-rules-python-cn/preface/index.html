序言

译注：本页为原教程的中文 Python 改写初稿。统计术语已按“先验分布、似然函数、后验分布、后验预测、后验可信区间、共轭族、分层模型”等常用译法校正；代码块已替换为 Python 生态的转写或骨架。

巴耶斯统计? 曾经是专业产业和研究圈外的模糊名词,巴耶斯方法正在享受复兴. 这本书的题目说明了所有大惊小怪的事: 贝叶斯规则! 贝叶斯方法为标准统计课程中根深蒂固的频率学派方法提供了强大的替代方法. 虽然经常主义者和巴耶斯人的方法有一个共同的目标——从数据中学习——但巴耶斯人实现该目标的方法正变得受欢迎,原因很多:(1) 巴耶斯人的方法使我们能够根据以前的信息解释新数据,正式将两者编成一套更新的信息;(2) 相对于经常主义者分析中使用的置信间隔和p值,巴耶斯人的结果更容易解释;(3) 在经常主义者的“相似”方法崩溃的情况下,巴耶斯人的方法可以闪耀;(4) 应用巴耶斯人技术所需的计算工具越来越容易获得。 不幸的是,巴耶斯统计的普及程度超过了支持统计所需的课程资源。 为此,贝叶斯规则(Bayes Rules!)的首要目标是让现代贝叶斯人的思维,建模,和计算为更广泛的受众所使用.

观众

Bayes Rules! 将Bayes的力量带给高级本科生和比较训练的从业者. 因此,这本书既不是在研究生一级写的,也无意成为统计领域的第一个介绍。 这本书至少假定读者熟悉典型的本科一级入门统计课程的内容. 读者最好也会有一些本科水平概率,微积分,R统计软件的经验. 单单 请不要离开, 我们提供了所有R代码和足够的概率审查,这样,没有这种背景的读者只要热衷于在苍蝇上拾取这些工具,就仍然能够跟随. 此外,尽管某些微积分概念在贝叶斯分析(因此本书)中很重要,但微积分衍生法却不是. 后者仅限于早期章节中的简单模型设置,容易跳过.
