序言

贝叶斯统计？！
曾经在专业行业和研究圈外鲜为人知的贝叶斯方法正迎来复兴。本书的书名道出了这阵热潮的原因：贝叶斯法则！
贝叶斯方法为根植于标准统计课程中的频率学派（frequentist）方法提供了强大的替代方案。尽管频率学派和贝叶斯方法共享一个共同目标——从数据中学习——但贝叶斯方法实现这一目标的方式正因多种原因而日益普及：
(1) 贝叶斯方法允许我们根据先验信息来解读新数据，将两者正式融合为一组更新后的信息；
(2) 相对于频率学派分析中使用的置信区间和 p 值，贝叶斯结果更易于解释；
(3) 在频率学派“似然”方法失效的场合，贝叶斯方法可以大放异彩；
(4) 应用贝叶斯技术所需的计算工具正变得越来越容易获取。
遗憾的是，贝叶斯统计的流行速度超过了支持它的课程资源建设。为此，《贝叶斯法则！》的首要目标是让更广泛的读者能够接触现代贝叶斯思维、建模和计算。

读者对象

贝叶斯法则！将贝叶斯的力量带给高年级本科生和同等水平的从业者。因此，本书既不是研究生级别的教材，也不是统计学的入门导论。本书至少假设读者熟悉典型本科入门统计课程所涵盖的内容。理想情况下，读者还应具备本科水平的概率论、微积分和 Python 编程语言经验。不过，别走！如果你没有完全符合这些条件，也请不要离开。我们会提供所有 Python 代码和足够的概率论复习，只要读者渴望在快速上手的过程中掌握这些工具，就能跟上内容。此外，尽管某些微积分概念在贝叶斯分析中很重要（因此本书有所涉及），但微积分推导并非重点。后者仅限于前面几章的简单模型设定，并且可以轻松跳过。

本书导航

贝叶斯法则！让读者有能力将贝叶斯方法融入统计和数据科学的日常现代实践中。整体基调是应用导向的：本书利用现代计算资源和可复现的分析流程；讨论强调概念性理解；内容由数据驱动的问题所驱动；传授方式则将传统的“讲授”与“活动”融合在一起。尽管着眼于应用，读者仍将打下必要的理论基础，以便更深入地理解贝叶斯方法论，并具备将其推广到新场景的灵活性。以下是一些导航这趟旅程的建议和技巧。

在做中学。
做贝叶斯统计需要实践。也需要软件。
贝叶斯法则！使用 Python 并借助强大的 PyMC 概率编程库。书中包含了所有 Python 代码，为读者提供了复现这些计算技术并将其推广到本书之外的工具。

We also provide R scripts for each chapter which you can import directly into your R work space! We elaborate on the details below.

拥抱成长型思维。
在阅读本书并将贝叶斯方法付诸实践的过程中，你会犯错。很多错误。犯错并从中吸取教训本就是学习的一部分。我们希望你坚持渡过学习的困难期，从而为贝叶斯社区贡献你独特的见解、视角和经验。

在情境中解读贝叶斯。
贝叶斯统计并非存在于真空中。它是我们可以用来从数据中了解周围世界的一种工具。全书之中，我们鼓励读者反思贝叶斯分析的潜在影响（例如，它们是否有偏？是否公平？）。

练习，练习，再练习。
每章末尾都配有一组练习题。有些让你以小分量回顾所学内容的操作细节。另一些则鼓励你使用 Python 在逼真的场景中综合应用各种概念。

环境配置

当你准备好深入《贝叶斯法则！》时，请按以下步骤进行准备。
首先，下载最新版本的以下软件：

  • Python (https://www.python.org/)
  • Jupyter Notebook (https://jupyter.org/install)

接下来，在 Python 环境中安装以下库：

  • 按照 https://www.pymc.io/ 上的说明仔细安装 PyMC 库。

  • 通过在控制台中输入以下命令来安装一组数据处理和贝叶斯库。

    pip install numpy pandas scipy matplotlib seaborn pymc arviz scikit-learn statsmodels

  • bayesrules 库包含一些我们为《贝叶斯法则！》专门构建的数据集和函数。该库中的若干数据集提取自更大的数据源，使用了非随机方法来获得符合我们教学目标的子集。这些可能存在偏倚的数据集不应用于严格的研究目的。如果你的目标是进行研究，请查阅相关数据帮助文档，查明其原始的完整数据源。

最后，查看 https://bayes-rules.github.io/。
该网站包含关于本书的更多背景信息以及将持续增长的配套资源。
