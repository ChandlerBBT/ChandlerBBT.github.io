第 3 章 Beta-二项贝叶斯模型

每隔四年，美国人就会前往投票站为美国总统投下自己的一票。设想以下场景：“Michelle”决定竞选总统，而你是她在明尼苏达州的竞选经理。因此，你在整个选举季中进行了30次不同的民意调查。尽管Michelle的支持率一直在45%左右徘徊，但她在最低迷的日子里支持率约为35%，在竞选活动中最好的日子里约为55%（图3.1（左））。

图3.1：之前30次明尼苏达州选民对Michelle竞选总统支持率的民调结果（左）以及关于她当前竞选支持率\(\pi\)的相应连续先验模型（右）。

选举是动态的，因此Michelle的支持率始终在变化。然而，这些过去的民调提供了关于\(\pi\)（即当前支持Michelle的明尼苏达州选民比例）的先验信息。事实上，我们可以将这些信息重新组织成一个关于\(\pi\)的正式先验概率模型。我们在2.3节中做过一个类似的例子，当时\(\pi\)是Kasparov在国际象棋中击败Deep Blue的概率。在那个例子中，我们为了适应入门级贝叶斯模型的框架，极大地简化了现实。主要是我们假设\(\pi\)只能取0.2、0.5或0.8，其对应的可能性由一个离散概率模型定义。然而，在Michelle的选举支持和Kasparov的国际象棋水平的现实中，\(\pi\)可以是0到1之间的任意值。我们可以通过构建\(\pi\)的连续先验概率模型来反映这一现实，并进行更细致的贝叶斯分析。一个合理的先验由图3.1（右）中的曲线表示。我们将在3.1节中详细探讨连续模型。现在，只需注意这条曲线保留了以往民调的总体信息和变异性——Michelle的支持率\(\pi\)可以在0到1之间变化，但最可能的值在0.45左右。

采用这种更细致、更连续的观点来看待Michelle的支持率\(\pi\)需要一些新工具。但是，贝叶斯分析的精髓保持不变。无论我们的参数\(\pi\)是连续还是离散，\(\pi\)的后验模型都将结合先验和数据的见解。接下来，你将深入细节，构建Michelle的选举模型。随后，你将把这项工作推广到基础的Beta-二项贝叶斯模型。Beta-二项模型的强大之处在于其广泛的应用。Michelle的选举支持率\(\pi\)并非唯一取值在[0,1]上的兴趣变量。你还可以想象在一些贝叶斯分析中，我们感兴趣的是对使用公共交通的人口比例、晚点的火车比例、喜欢猫胜过狗的人口比例等进行建模。Beta-二项模型为我们在这些情境下研究感兴趣的比例\(\pi\)提供了所需的工具。

  • 使用并调整连续先验。你将学习如何解释和调整连续的Beta先验模型，以反映你对\(\pi\)的先验信息。
  • 使用均值、众数和方差等性质来解释和交流先验与后验模型的特征。
  • 构建基础的Beta-二项模型，用于比例\(\pi\)。

3.1 Beta先验模型

在构建米歇尔在明尼苏达人中选举支持率 \(\pi\) 的贝叶斯选举模型时，我们照例从先验分布开始。我们的连续先验概率模型由图 3.1 中的概率密度函数（pdf）指定。虽然它看起来截然不同，但这个连续 pdf 的作用与表 2.7 中的离散概率质量函数（pmf）\(f(\pi)\) 相同：指定 \(\pi\) 的所有可能取值以及每个取值的相对合理性。也就是说，\(f(\pi)\) 回答：\(\pi\) 可以取哪些值，哪些值比另一些更合理？此外，在考虑 \(\pi\) 的所有可能结果时，pdf 的积分或面积等于 1，就像离散 pmf 的总和为 1 一样。

连续概率模型

设 \(\pi\) 是一个连续随机变量，其概率密度函数为 \(f(\pi)\)。那么 \(f(\pi)\) 具有以下性质：

  • \(f(\pi) \ge 0\)；
  • \(\int_\pi f(\pi)d\pi = 1\)，即 \(f(\pi)\) 下的面积为 1；且
  • 当 \(a \le b\) 时，\(P(a < \pi < b) = \int_a^b f(\pi) d\pi\)，即任意两个可能值 \(a\) 和 \(b\) 之间的面积对应于 \(\pi\) 落在此范围内的概率。

解读 \(f(\pi)\)

\(f(\pi) > 1\) 是可能的，因此连续 pdf 不能被解释为概率。相反，\(f(\pi)\) 可用于比较 \(\pi\) 的两个不同取值的合理性：\(f(\pi)\) 越大，相应的 \(\pi\) 取值越合理。

注意：如果你不熟悉积分，不必担心。继续阅读本书，你不需要进行任何积分运算。

3.1.1 Beta 基础

下一步是将图 3.1（右）中先验分布的图像转化为 \(\pi\) 的正式概率模型。也就是说，我们必须为 pdf \(f(\pi)\) 指定一个公式。在概率领域，有许多常见的“有名”模型，它们的 pdf 和性质已被深入研究。在这里，自然地我们会聚焦于Beta 概率模型。与米歇尔的支持率 \(\pi\) 一样，Beta 随机变量是连续的，且取值限制在 [0,1] 区间。在本节中，你将探索 Beta 模型的性质以及如何调节 Beta 模型以反映我们对米歇尔支持率 \(\pi\) 的先验理解。让我们从 Beta 概率模型的一般定义开始。

Beta 模型

设 \(\pi\) 是一个随机变量，它可以取 0 到 1 之间的任何值，即 \(\pi \in [0,1]\)。那么 \(\pi\) 的变异性可以通过具有形状超参数 \(\alpha > 0\) 和 \(\beta > 0\) 的 Beta 模型来很好地建模：

\[\pi \sim \text{Beta}(\alpha, \beta).\]

Beta 模型由连续 pdf 指定

\[\begin{equation} f(\pi) = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha)\Gamma(\beta)} \pi^{\alpha-1} (1-\pi)^{\beta-1} \;\; \text{ for } \pi \in [0,1] \tag{3.1} \end{equation}\] 其中 \(\Gamma(z) = \int_0^\infty x^{z-1}e^{-y}dx\)，且 \(\Gamma(z + 1) = z \Gamma(z)\)。有趣的事实：当 \(z\) 为正整数时，\(\Gamma(z)\) 简化为 \(\Gamma(z) = (z-1)!\)。

超参数

超参数是用于先验模型的参数。

通过实际操作最能理解这个模型。图 3.2 绘制了在不同形状超参数 \(\alpha\) 和 \(\beta\) 下的 Beta pdf \(f(\pi)\)。观察 Beta pdf 可以呈现的各种形状。这种灵活性意味着我们可以通过调整 \(\alpha\) 和 \(\beta\) 来调节 Beta 模型，以反映我们对 \(\pi\) 的先验理解。例如，请注意当设置 \(\alpha = \beta = 1\)（中间左侧图）时，Beta 模型在 0 到 1 之间是平坦的。在此设置下，Beta 模型等同于可能更熟悉的模型——标准均匀分布。

标准均匀分布模型

当 \(\pi\) 取 0 到 1 之间的任何值都具有同等可能性时，我们可以用标准均匀分布建模 \(\pi\)

\[\pi \sim \text{Unif}(0,1)\]

其概率密度函数为 \(f(\pi) = 1\)，其中 \(\pi \in [0,1]\)。Unif(0,1) 模型是 Beta(\(\alpha,\beta\)) 在 \(\alpha = \beta = 1\) 时的特例。

图 3.2：不同形状超参数 \(\alpha\) 和 \(\beta\) 下的 Beta(\(\alpha,\beta\)) 概率密度函数 \(f(\pi)\)（黑色曲线）。均值和众数分别用蓝色实线和虚线表示。

花一分钟看看你是否能找出形状超参数 \(\alpha\) 和 \(\beta\) 如何反映 \(\pi\) 的典型值以及 \(\pi\) 的变异性。24

  1. 当 \(\alpha = \beta\) 时，你会如何描述 Beta(\(\alpha,\beta\)) 变量 \(\pi\) 的典型行为？
    a) 右偏，\(\pi\) 倾向于小于 0.5。
    b) 对称，\(\pi\) 倾向于在 0.5 附近。
    c) 左偏，\(\pi\) 倾向于大于 0.5。

  2. 使用与上面相同的选项，当 \(\alpha > \beta\) 时，你会如何描述 Beta(\(\alpha,\beta\)) 变量 \(\pi\) 的典型行为？

  3. 哪个模型的 \(\pi\) 可能值的变异性更大，Beta(20,20) 还是 Beta(5,5)？

我们可以通过数值测量来支持对 \(\pi\) 行为的观察。 均值（或“期望值”）和众数提供了集中趋势的度量，即典型值所在。 从概念上讲，均值捕捉了 \(\pi\) 的平均值，而众数捕捉了 \(\pi\) 的最可能值，即使得概率密度函数 \(f(\pi)\) 最大化的 \(\pi\) 值。 这些度量分别在图 3.2 中用实线和虚线表示。 注意，当 \(\alpha\) 小于 \(\beta\) 时（上行），Beta 概率密度函数右偏，因此均值超过众数，且两者都低于 0.5。 当 \(\alpha\) 大于 \(\beta\) 时（下行），情况相反。 当 \(\alpha\) 和 \(\beta\) 相等时（中间行），Beta 概率密度函数关于均值和众数 0.5 对称。 这些趋势反映了 Beta(\(\alpha, \beta\)) 变量 \(\pi\) 的均值 \(E(\pi)\) 和众数的公式：25

\[\begin{equation} \begin{split} E(\pi) & = \frac{\alpha}{\alpha + \beta} \\ \text{Mode}(\pi) & = \frac{\alpha - 1}{\alpha + \beta - 2} \;\;\; \text{ when } \; \alpha, \beta > 1. \\ \end{split} \tag{3.2} \end{equation}\]

例如，一个 \(\text{Beta}(5,5)\) 变量 \(\pi\) 的集中趋势可以描述为

\[E(\pi) = \frac{5}{5 + 5} = 0.5 \;\; \text{ and } \;\; \text{Mode}(\pi) = \frac{5 - 1}{5 + 5 - 2} = 0.5 .\]

图 3.2 还揭示了 \(\pi\) 的 变异 模式。 例如，当值更靠近均值 0.5 时，\(\pi\) 在 Beta(20,20) 模型中的变异性小于 Beta(5,5) 模型。 我们可以通过 方差 来度量 Beta(\(\alpha,\beta\)) 随机变量 \(\pi\) 的变异性

\[\begin{equation} \text{Var}(\pi) = \frac{\alpha \beta}{(\alpha + \beta)^2(\alpha + \beta + 1)} . \tag{3.3} \end{equation}\]

大致而言，方差度量 \(\pi\) 值到其均值 \(E(\pi)\) 的典型 平方 距离。 由于方差的单位是平方，通常使用 标准差 更容易，它度量 \(\pi\) 值到 \(E(\pi)\) 的典型 非平方 距离：

\[\text{SD}(\pi) := \sqrt{\text{Var}(\pi)} .\]

例如，一个 \(\text{Beta}(5,5)\) 变量 \(\pi\) 的值通常偏离其均值 0.5 约 0.151，而一个 \(\text{Beta}(20,20)\) 变量的值通常仅偏离 0.5 约 0.078：

\[\sqrt{\frac{5 \cdot 5}{(5 + 5)^2(5 + 5 + 1)}} = 0.151 \;\;\; \text{ and } \;\;\; \sqrt{\frac{20 \cdot 20}{(20 + 20)^2(20 + 20 + 1)}} = 0.078 .\]

上述公式并非凭空出现。
它们是将均值、众数和方差的一般定义应用于 Beta 概率密度函数 (3.1) 得到的结果。
我们在下面给出这些定义，但你可以跳过，不会影响阅读。

测量集中趋势与变异性的理论基础

设 \(\pi\) 是一个连续随机变量，概率密度函数为 \(f(\pi)\)。考虑两种常见的 \(\pi\) 集中趋势的度量。
\(\pi\) 的均值或期望值捕捉了 \(\pi\) 的加权平均，其中每个可能的 \(\pi\) 值以对应的概率密度函数值作为权重：
\[E(\pi) = \int \pi \cdot f(\pi)d\pi.\] \(\pi\) 的众数捕捉了 \(\pi\) 最可能的取值，即概率密度函数取得最大值时的 \(\pi\) 值：

\[\text{Mode}(\pi) = \text{argmax}_\pi f(\pi).\]

接下来，考虑两种常见的 \(\pi\) 变异性度量。
\(\pi\) 的方差大致衡量了各个可能的 \(\pi\) 值与其均值之间的典型或期望平方距离：

\[\text{Var}(\pi) = E((\pi - E(\pi))^2) = \int (\pi - E(\pi))^2 \cdot f(\pi) d\pi.\]

\(\pi\) 的标准差大致衡量了各个可能的 \(\pi\) 值与其均值之间的典型或期望距离：

\[\text{SD}(\pi) := \sqrt{\text{Var}(\pi)} .\]

注意：如果 \(\pi\) 是离散的，其概率质量函数为 \(f(\pi)\)，我们只需将 \(\int\) 替换为 \(\sum\)。

3.1.2 调整 Beta 先验

了解了 Beta(\(\alpha,\beta\)) 模型的工作原理，现在我们来调整形状超参数 \(\alpha\) 和 \(\beta\)，以反映我们关于 Michelle 选举支持率 \(\pi\) 的先验信息。
我们在图 3.1（左）中看到，在以往的 30 次民意调查中，Michelle 的平均支持率约为 45 个百分点，不过她的最低支持率大约为 25 个百分点，最高约为 65 个百分点。
我们的 Beta(\(\alpha,\beta\)) 先验应该具有类似的模式。
例如，我们希望选择 \(\alpha\) 和 \(\beta\)，使得 \(\pi\) 的值倾向于在 0.45 附近，即 \(E(\pi) = \alpha/(\alpha + \beta) \approx 0.45\)。
或者，移项后得到：

\[\alpha \approx \frac{9}{11} \beta .\]

在试错过程中，我们使用 bayesrules Python 库中的 plot_beta() 函数，绘制满足该约束的 \(\alpha\) 和 \(\beta\) 配对下的 Beta 模型（例如 Beta(9,11)、Beta(27,33)、Beta(45,55)）。
在这些模型中，我们发现 Beta(45,55) 紧密地捕捉到了历史民调的典型结果和变异性：

plt.xlabel("pi")
plt.ylabel("密度")
plt.title("Beta(45, 55) 先验分布")
plt.show()

图 3.3：Beta(45,55) 的概率密度函数。

因此，一个合理的 Michelle 选举支持率先验模型是

\[\pi \sim \text{Beta}(45,55)\]

将 45 和 55 代入 (3.1) 式后得到的 先验概率密度函数 \(f(\pi)\)

\[\begin{equation} f(\pi) = \frac{\Gamma(100)}{\Gamma(45)\Gamma(55)}\pi^{44}(1-\pi)^{54} \;\; \text{ for } \pi \in [0,1] . \tag{3.4} \end{equation}\]

根据 (3.2) 式，该模型表明米歇尔的选举支持率最可能在 45 个百分点左右，其 先验均值 和 先验众数 为

\[\begin{equation} E(\pi) = \frac{45}{45 + 55} = 0.4500 \;\; \text{ and } \;\; \text{Mode}(\pi) = \frac{45 - 1}{45 + 55 - 2} = 0.4490 . \tag{3.5} \end{equation}\]

而且，根据 (3.3) 式，\(\pi\) 的潜在变异性由 先验标准差 0.05 来描述。也就是说，我们关于米歇尔可能选举支持率的其他先验假设往往会偏离 45% 的先验均值 5 个百分点：

\[\begin{equation} \begin{split} \text{Var}(\pi) & = \frac{45 \cdot 55}{(45 + 55)^2(45 + 55 + 1)} = 0.0025 \\ \text{SD}(\pi) & = \sqrt{0.0025} = 0.05.\\ \end{split} \tag{3.6} \end{equation}\]

3.2 二项数据模型与似然函数

在对米歇尔选举支持率 \(\pi\) 的贝叶斯分析第二步中，你准备收集一些数据。你计划进行一项新的民意调查，调查 \(n = 50\) 名明尼苏达州居民，并记录 \(Y\)，即支持米歇尔的人数。结果依赖于 \(\pi\)，因此也将提供对 \(\pi\) 的深入认识——米歇尔的实际支持率越高，\(Y\) 往往越大。为了对 \(Y\) 对 \(\pi\) 的依赖关系进行建模，我们可以对这次民意调查做出以下假设：1) 选民们回答调查时相互独立；2) 任何一名受访选民支持你的候选人米歇尔的概率为 \(\pi\)。根据我们在第 2.3.2 节中的工作，在给定 \(\pi\) 的条件下，\(Y\) 服从二项分布。具体来说，

\[Y | \pi \sim \text{Bin}(50, \pi)\]

其条件概率质量函数 \(f(y|\pi)\) 定义于 \(y \in \{0,1,2,...,50\}\)，

\[\begin{equation} f(y|\pi) = P(Y=y | \pi) = \left(\!\!\begin{array}{c} 50 \\ y \end{array}\!\!\right) \pi^y (1-\pi)^{50-y} . \tag{3.7} \end{equation}\]

鉴于其在我们的贝叶斯分析中的重要性，有必要再次强调二项模型的细节。首先，条件概率质量函数 \(f(y|\pi)\) 回答了一个假设性问题：如果米歇尔的支持率是某个给定的 \(\pi\) 值，那么这 \(50\) 名受访选民中，我们可能会预期有多少人支持她，即 \(Y = y\)？图 3.4 绘制了在一系列可能的 \(\pi\) 值下该概率质量函数的图形。这些图形使我们更形式化地理解：如果米歇尔的支持率 \(\pi\) 较低（顶行），那么民意调查结果 \(Y\) 也很可能较低。如果她的支持率较高（底行），\(Y\) 也很可能较高。

图 3.4: Bin(50, \(\pi\)) 的概率质量函数 \(f(y|\pi)\) 在 \(\pi \in \{0.1, 0.2, \ldots, 0.9\}\) 取值下的图形。在观测到的民意调查数据 \(Y = 30\) 处的概率质量函数以黑色突出显示。

在现实中，我们最终观察到这次民意调查取得了巨大成功：\(Y = 30\) 名受访选民（\(n = 50\)，占 60%）支持米歇尔！这一结果在图 3.4 的各概率质量函数中用黑色线条突出显示。为了只关注这些与观测到的民意调查数据相匹配的结果，我们将这些黑色线条提取出来，并在单一图中进行比较（图 3.5）。它们表示观测到的民意调查数据 \(Y = 30\) 在米歇尔支持率 \(\pi\) 的各个可能水平（\(\{0.1,0.2,\ldots,0.9\}\)）处的似然度。实际上，这个离散的情景集合代表了完整的连续似然函数 \(L(\pi | y=30)\) 上的一小部分点，该似然函数定义于 \(\pi\) 在 0 到 1 之间的任意值（黑色曲线）。

图 3.5: 在观测到的民意调查中，\(Y = 30\) 名受访明尼苏达州居民支持米歇尔（总 \(n = 50\)）的条件下，米歇尔选举支持率 \(\pi\) 的似然函数 \(L(\pi | y = 30)\)。竖线表示在 \(\pi \in \{0.1,0.2,\ldots,0.9\}\) 处计算的似然度。

回想一下，似然函数是通过将二项概率质量函数倒过来使用定义的。将 \(Y = 30\) 视为观测数据，\(\pi\) 视为未知，这与我们的实际情况相符。\(\pi\) 的二项似然函数由将 \(y = 30\) 代入二项概率质量函数 (3.7) 得出：

\[\begin{equation} L(\pi | y=30) = \left(\!\!\begin{array}{c} 50 \\ 30 \end{array}\!\!\right) \pi^{30} (1-\pi)^{20} \; \; \text{ for } \pi \in [0,1] . \tag{3.8} \end{equation}\]

例如，与我们在图 3.5 中看到的一致，如果她的真实支持率是 \(\pi = 0.6\)，那么 50 名被调查选民中有 \(Y = 30\) 人支持 Michelle 的概率为 0.115：

\[L(\pi = 0.6 | y = 30) = \left(\!\!\begin{array}{c} 50 \\ 30 \end{array}\!\!\right) 0.6^{30} 0.4^{20} \approx 0.115\]

但如果她的真实支持率是 \(\pi = 0.5\)，则仅为 0.042：

\[L(\pi = 0.5 | y = 30) = \left(\!\!\begin{array}{c} 50 \\ 30 \end{array}\!\!\right) 0.5^{30} 0.5^{20} \approx 0.042 .\]

这里还要记住，\(L(\pi|y = 30)\) 是 \(\pi\) 的函数，它揭示了观测到的民调数据 \(Y = 30\) 与不同 \(\pi \in [0,1]\) 值的相对兼容性。\(L(\pi|y=30)\) 在 \(\pi = 0.6\) 时达到最大，这表明当 Michelle 的真实支持率也是 60% 时，调查中观察到 60% 支持率最有可能。这很合理！假设的 \(\pi\) 值与 0.6 相差越远，我们观察到当前民调结果的可能性就越小——对于 \(\pi\) 值低于 0.3 或高于 0.9 的情况，\(L(\pi | y = 30)\) 实际上降为 0。因此，如果 Michelle 的真实支持率低至 30% 或高达 90%，我们几乎不可能在新民调中观察到 60% 的支持率。

3.3 Beta 后验模型

我们的贝叶斯模型现在已有两个部分——描述米歇尔支持率 \(\pi\) 的 Beta 先验模型和描述民意调查数据 \(Y\) 对 \(\pi\) 依赖关系的二项模型：

\[\begin{split} Y | \pi & \sim \text{Bin}(50, \pi) \\ \pi & \sim \text{Beta}(45, 55). \\ \end{split}\]

图 3.6：\(\pi\) 的先验模型以及根据最新民调结果（在 \(n = 50\) 名受访明尼苏达州居民中 \(Y = 30\) 人支持米歇尔）得到的 \(\pi\) 的（按比例缩放后的）似然函数。

在这一平衡过程中，这里的后验分布略“偏向”先验分布而非似然函数。 （我们将在第 4 章获得关于为什么会如此的直观理解。） 后验分布集中在 \(\pi = 0.5\) 表明，米歇尔的支持率高于或低于赢得明尼苏达州所需的 50% 门槛的可能性相当。 此外，结合先验与数据的信息后，后验可能值的范围缩小了：我们可以相当肯定地说，米歇尔的支持率在 35% 到 65% 之间。

你可能还会注意到一个新特点：与先验分布一样，\(\pi\) 的后验模型也是连续的，且定义在 [0,1] 区间上。也就是说，与先验类似，后验看起来是参数已经更新、结合了先验信息和数据信息的 Beta(\(\alpha,\beta\)) 模型。事实确实如此。以观测到的民调结果（\(Y = 30\)）为条件，米歇尔选举支持率的后验模型为 Beta(75, 75)：

\[\pi | (Y = 30) \sim \text{Beta}(75,75)\]

其对应的概率密度函数由 (3.1) 给出：

\[\begin{equation} f(\pi | y = 30) = \frac{\Gamma(150)}{\Gamma(75)\Gamma(75)}\pi^{74} (1-\pi)^{74} \;\; \text{ for } \pi \in [0,1] . \tag{3.9} \end{equation}\]

在用数学证明这一结论之前，我们先来看看你对米歇尔选举支持率 \(\pi\) 的认识是如何演变的。贝叶斯分析库中的 summarize_beta_binomial() 函数总结了 \(\pi\) 的先验模型和后验模型的典型值与变异性。这些计算直接来自将先验和后验的 Beta 参数代入 (3.2) 和 (3.3) 式：

import pandas as pd
from scipy.stats import beta

# 参数
alpha_prior = 45
beta_prior = 55
y = 30
n = 50

# 先验汇总（Beta 分布无直接 mode 方法，手动计算）
prior_mean = alpha_prior / (alpha_prior + beta_prior)
prior_mode = (alpha_prior - 1) / (alpha_prior + beta_prior - 2) if alpha_prior > 1 and beta_prior > 1 else float('nan')
prior_var = (alpha_prior * beta_prior) / ((alpha_prior + beta_prior)**2 * (alpha_prior + beta_prior + 1))
prior_sd = prior_var**0.5

# 后验参数
alpha_post = alpha_prior + y
beta_post = beta_prior + n - y

posterior_mean = alpha_post / (alpha_post + beta_post)
posterior_mode = (alpha_post - 1) / (alpha_post + beta_post - 2) if alpha_post > 1 and beta_post > 1 else float('nan')
posterior_var = (alpha_post * beta_post) / ((alpha_post + beta_post)**2 * (alpha_post + beta_post + 1))
posterior_sd = posterior_var**0.5

# 构造数据框
summary_df = pd.DataFrame({
    "model": ["prior", "posterior"],
    "alpha": [alpha_prior, alpha_post],
    "beta": [beta_prior, beta_post],
    "mean": [prior_mean, posterior_mean],
    "mode": [prior_mode, posterior_mode],
    "var": [prior_var, posterior_var],
    "sd": [prior_sd, posterior_sd]
})

print(summary_df)

\[E(\pi) = 0.45 \;\; \text{ vs } \;\; E(\pi | Y = 30) = 0.50 .\]

如果你相信我们所说的，即 \(\pi\) 的后验模型是 Beta(75,75)，那么你可以直接跳到第 3.4 节，并且仍然为本书后续内容做好准备。不过，我们强烈建议你思考一下构建后验的奇妙过程。经历这个过程可以帮助你进一步培养贝叶斯建模的直觉。与我们之前的贝叶斯模型一样，\(\pi\) 的后验条件概率密度函数通过贝叶斯法则 (2.12) 在先验概率密度函数 \(f(\pi)\) 和似然函数 \(L(\pi|y = 30)\) 之间取得平衡：

\[f(\pi | y = 30) = \frac{f(\pi)L(\pi|y = 30)}{f(y = 30)}.\]

回顾第 2.3.6 节，\(f(y = 30)\) 是一个归一化常数，即对于 \(\pi\) 而言它是一个常数，它使得后验概率密度函数 \(f(\pi | y = 30)\) 的积分等于 1。我们不需要为了构建后验模型而计算归一化常数。相反，我们可以利用后验概率密度函数正比于先验概率密度函数 (3.4) 和似然函数 (3.8) 的乘积这一事实来简化后验的构建：

\[\begin{split} f(\pi | y = 30) & \propto f(\pi) L(\pi | y=30) \\ & = \frac{\Gamma(100)}{\Gamma(45)\Gamma(55)}\pi^{44}(1-\pi)^{54} \cdot \left(\!\!\begin{array}{c} 50 \\ 30 \end{array}\!\!\right) \pi^{30} (1-\pi)^{20} \\ & = \left[\frac{\Gamma(100)}{\Gamma(45)\Gamma(55)}\left(\!\!\begin{array}{c} 50 \\ 30 \end{array}\!\!\right) \right] \cdot \pi^{74} (1-\pi)^{74} \\ & \propto \pi^{74} (1-\pi)^{74} . \\ \end{split}\]

在我们计算的第三行中，我们将常数和依赖于 \(\pi\) 的部分分成了两块。在最后一行，我们做了一个很大的简化：我们丢弃了所有不依赖于 \(\pi\) 的常数。我们不需要这些。相反，我们关心的是 \(f(\pi | y=30)\) 对 \(\pi\) 的依赖关系：

\[f(\pi | y=30) = c\pi^{74} (1-\pi)^{74} \propto \pi^{74} (1-\pi)^{74} .\]

我们可以通过计算使概率密度函数积分为 1 的归一化常数 \(c\) 来完成这个后验概率密度函数的定义：

\[1 = \int f(\pi | y=30) d\pi = \int c \cdot \pi^{74} (1-\pi)^{74} d\pi \;\; \Rightarrow \; \; c = \frac{1}{\int \pi^{74} (1-\pi)^{74} d\pi}.\] 但同样，我们并不需要真的去计算这个值。 \(\pi\) 的概率密度函数由其结构对 \(\pi\) 的依赖关系——即该密度函数的 核——来定义。 这里请注意，\(f(\pi|y=30)\) 与 (3.9) 式中的归一化 Beta(75,75) 密度函数具有 相同 的核：

\[f(\pi | y=30) = \frac{\Gamma(150)}{\Gamma(75)\Gamma(75)} \pi^{74} (1-\pi)^{74} \propto \pi^{74} (1-\pi)^{74} .\]

未归一化的后验概率密度函数 \(f(\pi | y=30)\) 与未归一化的 Beta(75,75) 密度函数相吻合，这证实了我们的断言：\(\pi | (Y=30) \sim \text{Beta}(75,75)\)。 很神奇吧。 为了进一步练习如何从未归一化的后验密度函数或核中识别 \(\pi\) 的后验模型，请完成下面的小测验。27

3.4 Beta-二项模型

在上一节中，我们为 Michelle 的选举支持率 \(\pi\) 建立了基本的 Beta-二项模型。在建模过程中，我们假设了一个特定的 Beta(45,55) 先验分布以及特定的民调结果（在 \(n=50\) 名受访选民中，\(Y=30\) 人支持你的候选人）。这只是更一般的 Beta-二项模型的一个特例：

\[\begin{split} Y | \pi & \sim \text{Bin}(n, \pi) \\ \pi & \sim \text{Beta}(\alpha, \beta). \\ \end{split}\]

这个一般模型应用广泛，适用于任何兴趣参数 \(\pi\) 取值在 [0,1] 上的场景，你可以使用任意调整的 Beta 先验分布，以及任何数据 \(Y\)，\(Y\) 是在 \(n\) 次固定、独立的试验中“成功”的次数，每次试验成功的概率为 \(\pi\)。例如，\(\pi\) 可能代表一枚硬币正面朝上的倾向，数据 \(Y\) 记录了在 \(n\) 次抛硬币中观测到的正面次数。或者 \(\pi\) 可能代表使用社交媒体的成年人比例，我们通过抽样 \(n\) 名成年人并记录其中使用社交媒体的人数 \(Y\) 来了解 \(\pi\)。无论什么场景，在观测到 \(n\) 次试验中有 \(Y=y\) 次成功之后，\(\pi\) 的后验分布可以用一个 Beta 模型来描述，该模型揭示了先验（通过 \(\alpha\) 和 \(\beta\)）和数据（通过 \(y\) 和 \(n\)）的影响：

\[\begin{equation} \pi | (Y = y) \sim \text{Beta}(\alpha + y, \beta + n - y) . \tag{3.10} \end{equation}\]

后验集中趋势和变异性的度量来自 (3.2) 和 (3.3)：

\[\begin{equation} \begin{split} E(\pi | Y=y) & = \frac{\alpha + y}{\alpha + \beta + n} \\ \text{Mode}(\pi | Y=y) & = \frac{\alpha + y - 1}{\alpha + \beta + n - 2} \\ \text{Var}(\pi | Y=y) & = \frac{(\alpha + y)(\beta + n - y)}{(\alpha + \beta + n)^2(\alpha + \beta + n + 1)}.\\ \end{split} \tag{3.11} \end{equation}\]

重要的是，注意后验与先验是同一概率模型的不同参数化——先验和后验都是调参不同的 Beta 模型。 在本例中，我们称 Beta(\(\alpha, \beta\)) 模型是相应 Bin(\(n,\pi\)) 数据模型的 共轭先验。 后续工作将强调共轭性简化了后验的构造，因此是贝叶斯建模中一个值得拥有的性质。

共轭先验

如果后验 \(f(\pi|y) \propto f(\pi)L(\pi|y)\) 与先验来自同一模型族，我们就称 \(f(\pi)\) 是 \(L(\pi|y)\) 的共轭先验。

一般 Beta-二项模型的后验构造与选举特例非常相似。 首先，Beta 先验 pdf \(f(\pi)\) 由 (3.1) 定义，而似然函数 \(L(\pi|y)\) 由 (2.7) 定义，后者是给定观测数据 \(Y = y\) 条件下 Bin(\(n,\pi\)) 模型的条件 pmf。 对于 \(\pi \in [0,1]\)，

\[\begin{equation} f(\pi) = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha)\Gamma(\beta)}\pi^{\alpha - 1}(1-\pi)^{\beta - 1} \;\; \text{ and } \;\; L(\pi|y) = \left(\!\!\begin{array}{c} n \\ y \end{array}\!\!\right) \pi^{y} (1-\pi)^{n-y} . \tag{3.12} \end{equation}\]

将这两部分结合，后验 pdf 由贝叶斯定理得出：

\[\begin{split} f(\pi | y) & \propto f(\pi)L(\pi|y) \\ & = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha)\Gamma(\beta)}\pi^{\alpha - 1}(1-\pi)^{\beta - 1} \cdot \left(\!\begin{array}{c} n \\ y \end{array}\!\right) \pi^{y} (1-\pi)^{n-y} \\ & \propto \pi^{(\alpha + y) - 1} (1-\pi)^{(\beta + n - y) - 1} .\\ \end{split}\]

同样，我们省略了不依赖于\(\pi\)的归一化常数，得到了未归一化的后验概率密度函数。注意，它与归一化的Beta(\(\alpha + y\), \(\beta + n - y\))的概率密度函数具有相同的结构，

\[f(\pi|y) = \frac{\Gamma(\alpha+\beta+n)}{\Gamma(\alpha+y)\Gamma(\beta+n-y)}\pi^{(\alpha + y) - 1} (1-\pi)^{(\beta + n - y) - 1}.\]

因此，我们已经验证了我们的断言：在观察到\(n\)次试验中有\(Y = y\)次成功时，\(\pi\)的后验模型为\(\text{Beta}(\alpha + y, \beta + n - y)\)。

3.6 示例：米尔格拉姆的服从行为研究

在 1963 年《异常与社会心理学杂志》的一期中，斯坦利·米尔格拉姆描述了一项研究，他调查了人们服从权威人物命令的倾向，即使这些命令可能会伤害他人 (Milgram 1963)。在论文中，米尔格拉姆这样描述该研究：

“包括命令一名不知情的被试对一名受害者施加电击。使用了一个模拟电击发生器，有 30 个清晰标记的电压水平，范围从 IS 到 450 伏特。该仪器带有从轻微电击到危险：严重电击的文字标示。受害者的反应是标准化的，他其实是实验者的一名训练有素的同谋。在所谓旨在研究惩罚对记忆影响的‘学习实验’背景下，向不知情的被试发出施加电击的命令。随着实验的进行，不知情的被试被命令对受害者施加越来越强的电击，甚至达到标有危险：严重电击的水平。”

换句话说，研究参与者被要求测试另一名参与者（实际上是一名训练有素的演员）记忆事实的能力。如果该演员没有记住一个事实，参与者就被命令对演员施加电击，并且每次后续失败都要提高电击水平。参与者不知道的是，电击是假的，演员只是假装因电击而感到疼痛。

3.6.1 一项贝叶斯分析

我们可以将 Milgram 的研究转换到 Beta-二项框架中。这里感兴趣的参数是 \(\pi\)，即一个人会服从权威（在此例中即施加最严重后果的电击）的概率，即使这意味着对他人造成伤害。因为 Milgram 已于 1984 年去世，我们没有机会询问他在进行研究之前对 \(\pi\) 的理解先验。因此，我们将偏离实际研究，并假设另一位心理学家协助完成了这项工作。在收集数据之前，他们表示 Beta(1,10) 模型准确地反映了他们对 \(\pi\) 的先验理解，这是通过之前的工作形成的。接下来，令 \(Y\) 为 40 名研究参与者中会施加最严重电击的人数。假设每位参与者的行为彼此独立，我们可以用二项分布来建模 \(Y\) 对 \(\pi\) 的依赖。总之，我们有以下 Beta-二项贝叶斯模型：

下面总结并绘制了这一后验分布，并与先验概率密度函数和缩放后的似然函数进行对比。注意，心理学家从先验到后验的认识发生了相当大的变化。尽管他们最初认为不到 ~25% 的人会执行最严重的电击，但鉴于研究数据强有力的反证，他们现在认为这一比例大约在 ~30% 到 ~70% 之间。

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy.stats import beta

def summarize_beta_binomial(alpha, beta_param, y, n):
    """
    计算并返回先验和后验的摘要指标：
    alpha, beta, 均值, 众数, 方差, 标准差
    """
    # 先验指标
    prior_mean = alpha / (alpha + beta_param)
    prior_mode = (alpha - 1) / (alpha + beta_param - 2) if (alpha > 1 and beta_param > 1) else 0.0
    prior_var = alpha * beta_param / ((alpha + beta_param)**2 * (alpha + beta_param + 1))
    prior_sd = np.sqrt(prior_var)

    # 后验参数
    alpha_post = alpha + y
    beta_post = beta_param + n - y

    # 后验指标
    post_mean = alpha_post / (alpha_post + beta_post)
    post_mode = (alpha_post - 1) / (alpha_post + beta_post - 2) if (alpha_post > 1 and beta_post > 1) else 0.0
    post_var = alpha_post * beta_post / ((alpha_post + beta_post)**2 * (alpha_post + beta_post + 1))
    post_sd = np.sqrt(post_var)

    # 组装成表格
    df = pd.DataFrame({
        'model': ['prior', 'posterior'],
        'alpha': [alpha, alpha_post],
        'beta': [beta_param, beta_post],
        'mean': [prior_mean, post_mean],
        'mode': [prior_mode, post_mode],
        'var': [prior_var, post_var],
        'sd': [prior_sd, post_sd]
    })
    return df

def plot_beta_binomial(alpha, beta_param, y, n):
    """绘制先验与后验的密度曲线"""
    x = np.linspace(0, 1, 200)
    prior_pdf = beta.pdf(x, alpha, beta_param)
    alpha_post = alpha + y
    beta_post = beta_param + n - y
    posterior_pdf = beta.pdf(x, alpha_post, beta_post)

    plt.plot(x, prior_pdf, 'r-', lw=2, alpha=0.6, label=f'先验 Beta({alpha},{beta_param})')
    plt.plot(x, posterior_pdf, 'b-', lw=2, alpha=0.6, label=f'后验 Beta({alpha_post},{beta_post})')
    plt.xlabel('θ')
    plt.ylabel('密度')
    plt.title('先验与后验分布')
    plt.legend()
    plt.show()

# 显示摘要
summary = summarize_beta_binomial(alpha=1, beta=10, y=26, n=40)
print(summary.to_string(index=False))

# 绘制图形
plot_beta_binomial(alpha=1, beta=10, y=26, n=40)

3.7 章节总结

在第 3 章中，你为 \(\pi\) —— 一个可以取 0 到 1 之间任何值的未知比例 —— 构建了基础的 Beta-二项模型：

\[\begin{split} Y | \pi & \sim \text{Bin}(n, \pi) \\ \pi & \sim \text{Beta}(\alpha, \beta) \\ \end{split} \;\; \Rightarrow \;\; \pi | (Y = y) \sim \text{Beta}(\alpha + y, \beta + n - y) .\]

这个模型反映了每次贝叶斯分析共有的四个组成部分：

  1. 先验模型
    \(\pi\) 的 Beta 先验模型可以进行调整，以反映每个 \(\pi \in [0,1]\) 的相对先验合理性。

\[f(\pi) = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha)\Gamma(\beta)}\pi^{\alpha - 1}(1-\pi)^{\beta - 1}.\]

  1. 数据模型
    为了了解 \(\pi\)，我们收集 数据 \(Y\)，即在 \(n\) 次独立试验中成功的次数，每次试验的成功概率为 \(\pi\)。\(Y\) 对 \(\pi\) 的依赖关系由二项模型 \(\text{Bin}(n,\pi)\) 概括。

  2. 似然函数
    在观察到数据 \(Y = y\) 且 \(y \in \{0,1,\ldots,n\}\) 之后，将 \(y\) 代入二项概率质量函数得到的 \(\pi\) 的似然函数，提供了一种比较数据与不同 \(\pi\) 兼容性的机制：

    \[L(\pi|y) = \left(\!\begin{array}{c} n \\ y \end{array}\!\right) \pi^{y} (1-\pi)^{n-y} \;\; \text{ 对 } \pi \in [0,1].\]

  3. 后验模型
    通过贝叶斯定理，共轭 Beta 先验与二项数据模型结合，产生了关于 \(\pi\) 的 Beta 后验模型。更新后的 Beta 后验参数 \((\alpha + y, \beta + n - y)\) 反映了先验（通过 \(\alpha\) 和 \(\beta\)）以及观测数据（通过 \(y\) 和 \(n\)）的影响。

\[f(\pi | y) \propto f(\pi)L(\pi|y) \propto \pi^{(\alpha + y) - 1} (1-\pi)^{(\beta + n - y) - 1} .\]
