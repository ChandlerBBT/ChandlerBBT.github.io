前言

即便思考了几十年，贝叶斯规则仍然让我惊叹不已。一个简单的公式怎么能有如此广泛的应用？你将在本书中遇到一系列生动的应用实例，从天气预测到LGBTQ+反歧视法，从谁把苏打水叫作“pop”（或把“pop”叫作苏打水）到如何对企鹅物种进行分类。最重要的是，认真学习本书将使你能够对你关心的数据和应用进行深思熟虑的贝叶斯分析。

统计学和数据科学专注于利用数据了解世界并进行预测。贝叶斯方法提供了一种严谨且强大的工具，能够在给定已知信息（数据）的情况下，对我们感兴趣的未知量进行概率推断和预测。它给出的结果易于解释，直接量化了我们的不确定性。遗憾的是，贝叶斯方法在本科阶段很少深入讲授，或许是担心会涉及太多看似吓人的积分运算，或需要编写太多晦涩的代码。

Bayes Rules! 表明，贝叶斯方法实际上可以被具备基础统计知识的学生和自学者所掌握，即使他们不擅长微积分推导或从头编写复杂的算法。本书通过许多读者友好的特色实现了这一目标，例如清晰的语言和图示解释、用来检验理解程度的测验，以及bayesrules 工具包，它包含了方便你尝试贝叶斯方法的数据集和函数。

更棒的是，这种易用性是通过优秀的教学法实现的，而不是通过提供淡而无味、过度简化的内容。例如，书中介绍了名为分层模型（hierarchical models）的模型和一个称为PyMC的 Python 库，并通过极具启发性的例子展示了如何将它们应用于有趣的实际问题。分层模型和PyMC都是现代贝叶斯数据分析中使用的先进技术。

蜘蛛侠漫画中的彼得·帕克原则说：“能力越大，责任越大。”同样，统计学和数据科学的巨大力量也伴随着巨大的责任，需要考虑对社会的利弊、研究中参与者的隐私权、数据集中的偏差以及所提算法是否会放大这些偏差，以及其他伦理问题。Bayes Rules! 强调公平和伦理，而不是忽视这些关键问题。

只要你认真阅读Bayes Rules!（主动地——一定要尝试自测题并练习部分习题！），你极有可能在体验贝叶斯思维乐趣的同时，增强自己的统计问题解决能力。

——Joseph K. Blitzstein，哈佛大学
