前言

译注：本页为原教程的中文 Python 改写初稿。统计术语已按“先验分布、似然函数、后验分布、后验预测、后验可信区间、共轭族、分层模型”等常用译法校正；代码块已替换为 Python 生态的转写或骨架。

即使经过几十年的思考,贝叶斯的统治也从未停止使我惊叹. 一个简单的公式怎么能有如此广泛的应用? 从天气预测到LGBTQQ反歧视法, 从谁称苏打“流行”(或称流行“Soda”)到如何分类企鹅物种, 最重要的是,仔细研究这本书将增强你的能力, 来进行深思熟虑的贝叶斯分析, 研究你关心的数据和应用。

统计学和数据科学侧重于利用数据来了解世界并作出预测. 贝叶斯方法提供了一种有原则的、强有力的工具,以获得关于我们未知的兴趣数量(数据)的概率和预测。 它提供了易于解释的结果,可以直接量化我们的不确定性。 不幸的是,本科一级很少进行深入的教学,也许出于担心会有许多恐怖的外观构件不能做,或者写太多密码。

Bayes Rules! 显示具有基本统计知识的学生和自学者事实上可以使用Bayesian方法,即使他们不擅长微积分衍生或从零开始编码奇异算法。 本书以许多方便读者的功能实现这一点,例如通过文字和图片进行清晰的解释,测试你的理解的测验,以及包含数据集和功能,便于尝试贝叶斯方法的Bayesrules R包.

更好的办法是通过良好的教学方法实现无障碍,而不是通过对这一主题进行过度简化的审视。 例如,引入了称为等级模型的模型和称为rstan的R包,并有极具启发性的例子说明如何将这些模型应用于有趣的应用. 等级模型和rstan是现代贝叶斯数据分析中使用的最先进技术之一.

蜘蛛侠漫画中的彼得·帕克(Peter Parker)原则说 : “ 有了强大的力量,就有了巨大的责任。 ” 同样,统计和数据科学的强大力量也伴随着考虑对社会的益处和风险、研究参与者的隐私权、数据集中的偏见以及拟议算法是否扩大这些偏见以及其他道德问题的巨大责任。 贝叶斯规则!强调公平和道德,而不是忽视这些关键问题.

既然你读过贝叶斯规则! (积极 ) — — 确保用一些练习来尝试自我解答和实践!) , 在体验贝叶斯思想的喜悦的同时,你加强统计解决问题技能的似然很高。

——约瑟夫·K·布莱茨斯坦,哈佛大学
