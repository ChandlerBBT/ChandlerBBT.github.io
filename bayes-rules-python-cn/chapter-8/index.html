第 8 章 后验推断与预测

想象你正站在纽约市现代艺术博物馆（MoMA）里，被眼前的作品深深吸引。虽然明白“现代”艺术并不一定意味着“新”艺术，但一个疑问还是会冒出来：这位现代艺术家属于X世代甚至更年轻（即出生于1965年或之后）的可能性有多大？
本章我们将通过贝叶斯分析来回答这个问题。为此，令\(\pi\)表示美国主要现代艺术博物馆中展出作品的艺术家里属于X世代或更年轻的比例。
对\(\pi\)使用Beta(4,6)先验模型（图8.1）反映了我们非常模糊的先验假设，即主要现代艺术博物馆更多地展出1965年以前出生的艺术家的作品，也就是说\(\pi\)很可能低于0.5。
毕竟，“现代艺术”可以追溯到19世纪80年代，而且在艺术界要获得如此高的认可往往需要一些时间。

为了进一步了解\(\pi\)，我们将考察从MoMA藏品中随机抽取的\(n =\)100位艺术家。
bayesrules包中的moma_sample数据集是MoMA自己提供的(MuseumofModernArt 2020)数据的一个子集。

# 导入必要的库
import pandas as pd
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 加载数据
moma_sample = pd.read_csv('moma_sample.csv')

在这组随机抽取的艺术家中，\(Y =\)14位是X世代或更年轻：

# 按 genx 分组计数
genx_counts = moma_sample.groupby('genx').size().reset_index(name='n')
print(genx_counts)

认识到\(Y\)对\(\pi\)的依赖关系遵循二项模型，我们的分析采用Beta-二项框架。因此，结合观测到的艺术数据后，\(\pi\)的更新后验模型可由(3.10)推导得出：

\[\begin{split} Y | \pi & \sim \text{Bin}(100, \pi) \\ \pi & \sim \text{Beta}(4, 6) \\ \end{split} \;\;\;\; \Rightarrow \;\;\;\; \pi | (Y = 14) \sim \text{Beta}(18, 92)\]

对应的后验概率密度函数为

\[\begin{equation} f(\pi | y = 14) = \frac{\Gamma(18 + 92)}{\Gamma(18)\Gamma(92)}\pi^{18-1} (1-\pi)^{92-1} \;\; \text{ for } \pi \in [0,1] . \tag{8.1} \end{equation}\]

我们对 \(\pi\) 的理解演变如图 8.1 所示。起初我们模糊地认为展出的艺术家中Gen X比例不足一半，而数据让我们逐渐确信这一比例很可能低于25%。

def plot_beta_binomial(alpha, beta, y, n):
    """
    绘制 Beta-Binomial 模型的先验、似然和后验分布。
    """
    from scipy.stats import beta as beta_dist
    import numpy as np
    import matplotlib.pyplot as plt

    # 参数网格
    theta = np.linspace(0, 1, 1000)

    # 先验 Beta(alpha, beta)
    prior_pdf = beta_dist.pdf(theta, alpha, beta)

    # 似然（用 Beta(y+1, n-y+1) 的形状表示，并归一化）
    likelihood_pdf = theta**y * (1-theta)**(n-y)
    normalization = np.trapz(likelihood_pdf, theta)
    likelihood_pdf /= normalization

    # 后验 Beta(alpha+y, beta+n-y)
    posterior_pdf = beta_dist.pdf(theta, alpha+y, beta+n-y)

    # 绘图
    plt.figure(figsize=(8,5))
    plt.plot(theta, prior_pdf, label=f'先验 Beta({alpha},{beta})', linestyle='--')
    plt.plot(theta, likelihood_pdf, label=f'似然 (y={y}, n={n})', linestyle='-.')
    plt.plot(theta, posterior_pdf, label=f'后验 Beta({alpha+y},{beta+n-y})', linestyle='-')
    plt.xlabel('θ')
    plt.ylabel('密度')
    plt.title('Beta-Binomial 模型')
    plt.legend()
    plt.show()

图8.1：现代艺术博物馆艺术家中Gen X或更年轻一代占比 \(\pi\) 的贝叶斯模型。

在庆祝我们成功构建后验分布之后，请意识到前方还有许多工作。我们必须能够利用这个后验分布来对 \(\pi\) 进行严谨的后验分析。后验分析通常涉及三项任务：估计、假设检验和预测。例如，我们对 \(\pi\) 的估计是多少？我们的模型是否支持“博物馆艺术家中Gen X或更年轻一代的比例不足20%”这一说法？如果再抽取20位博物馆艺术家，我们预测其中有多少是Gen X或更年轻一代？

后验估计

重新审视现代艺术博物馆艺术家中Gen X或更年轻一代占比 \(\pi\) 的 Beta(18, 92) 后验模型（图 8.1）。在贝叶斯分析中，我们可以把整个后验模型视为对 \(\pi\) 的估计。毕竟，这个后验合理值模型完整地描绘了 \(\pi\) 的中心趋势与不确定性。然而，在明确和传达我们的后验认识时，计算 \(\pi\) 的一些简单后验摘要也很有用。请凭直觉思考一下，我们该如何处理这项任务。

以下哪一项最准确地描述了你对 \(\pi\) 的后验估计？

  1. 大约16%的博物馆艺术家是Gen X或更年轻一代。
  2. 最可能的情况是大约16%的博物馆艺术家属于X世代或更年轻，但该数字可能在9%到26%之间。

如果你的答案是b，你的思维本质上就是贝叶斯的。要理解原因，请看图8.2，它展示了我们关于π的Beta(18, 92)后验分布（左）与另一位分析师的Beta(4, 16)后验分布（右）。这位分析师从相同的Beta(4, 6)先验出发，但只观察了10位艺术家，其中0位是X世代或更年轻。尽管他们的不同数据导致了不同的后验，但集中趋势与我们的相似。因此，另一位分析师对π的最佳猜测与我们一致：大约16-17%的博物馆艺术家是X世代或更年轻。然而，只报告这个共同的“最佳猜测”会让我们的两个后验看起来具有误导性的相似。事实上，尽管我们非常有信心年轻艺术家的比例在10%到24%之间，但另一位分析师只愿意将这一数字放在6%到40%更宽的范围内。他们的相对不确定性是有道理的——他们只收集了10件作品，而我们收集了100件。

图8.2：我们关于π的Beta(18, 92)后验模型（左）与另一个Beta(4, 16)后验模型（右）并列展示。阴影区域代表π相应的95%后验可信区间。

这里的结论是，后验估计应同时反映π的集中趋势和变异性。π的后验均值与后验众数仅提供集中趋势的快速总结。对于我们Beta(18, 92)后验，这些特征来自通用的Beta性质(3.2)，并且与我们之前的观察一致，即X世代代表比例最可能在16%左右：

\[\begin{split} E(\pi | Y=14) & = \frac{18}{18 + 92} \approx 0.164 \\ \text{Mode}(\pi | Y=14) & = \frac{18 - 1}{18 + 92 - 2} \approx 0.157 . \\ \end{split}\]

更好的是，为了同时捕获π的集中趋势和变异性，我们可以报告一个后验合理π值的范围。这个范围被称为π的后验可信区间（CI）。例如，我们之前注意到博物馆艺术家中X世代或更年轻的比例最可能在10%到24%之间。这个范围捕获了π的更合理取值，同时排除了更极端和不太可能的情况（图8.2）。事实上，0.1和0.24是第2.5和第97.5后验百分位数（即第0.025和第0.975后验分位数），因此标记了后验合理π值中间的95%。我们可以使用 scipy.stats.beta.ppf() 来确认这些 Beta(18,92) 后验分位数的计算结果：

# 后验分布 Beta(18, 92) 的 0.025 和 0.975 分位数
from scipy.stats import beta
quantiles = beta.ppf([0.025, 0.975], 18, 92)
print(quantiles)

由此得到的π的95%可信区间(0.1, 0.24)如图8.2（左）阴影区域所示。整个后验概率密度函数下的面积为1，而该阴影区域的面积，即落入该区域的π值比例，为0.95。这揭示了可信区间的一种直观解释。有95%的后验概率表明，博物馆艺术家中X世代或更年轻的比例在10%到24%之间：

\[P(\pi \in (0.1, 0.24) | Y = 14) = \int_{0.1}^{0.24} f(\pi|y=14) d\pi = 0.95 .\]

请先停一下。这种解释是否感觉自然且直观？是不是有点平淡？如果是的话，我们很高兴你有这种感觉——这说明你已经开始用贝叶斯的方式思考了。在 第 8.5 节，我们将再次讨论这个结果有多么特别。

在构建上述可信区间时，我们采用的是“中间 95%”的方法。这并不是我们 唯一 的选择。我们可以做的第一个调整是改变 可信水平（见图 8.3）。例如，一个中间 50% 的可信区间，范围从第 25 百分位数到第 75 百分位数，会让我们关注一个更小的、包含更可信的 \(\pi\) 值范围。后验概率为 50% 表明，博物馆艺术家中 Generation X 或更年轻的比例介于 14% 和 19% 之间：

# 计算 Beta(18, 92) 后验的 0.25 和 0.75 分位数
from scipy.stats import beta
beta.ppf([0.25, 0.75], 18, 92)

另一方面，一个更宽的中间 99% 可信区间范围是从第 0.5 百分位数到第 99.5 百分位数，从而仅排除极端的 1%。因此，99% 可信区间能为我们提供更全面的 看似可信（尽管在某些情况下 极不可能）的 \(\pi\) 值：

# 计算 Beta(18, 92) 后验的 0.005 和 0.995 分位数
from scipy.stats import beta
beta.ppf([0.005, 0.995], 18, 92)

虽然 95% 的水平是实践者常见的选择，但这多少有些随意，仅仅是几十年传统的惯性。 不存在一个“正确”的可信水平。 在本书中，根据分析背景，我们有时会使用 50%、80% 或 95% 的水平。每种水平都提供了我们后验理解的不同快照。

图 8.3：\(\pi\) 的 50%、95% 和 99% 后验可信区间。

考虑对可信区间的构造做第二种可能的调整：并非一定要报告后验可信值的 中间 95%。事实上，中间 95% 的方法可能会将一些更可信的值排除在可信区间之外。仔细观察图 8.3 中的 50% 和 95% 可信区间，就会发现这一可能性的证据：我们那个略微不对称的后验分布中，阴影区域也稍微有点偏斜。在 95% 可信区间中，区间上端包含的 \(\pi\) 值，其可信度反而不如那些被排除在区间之外、低于 0.1 的 \(\pi\) 值。如果这种偏斜更极端，我们就应考虑使用 最高后验密度 的 95% 后验值来构建 \(\pi\) 的 95% 可信区间，而不是用 中间 的 95%。你可以在习题中探索这个想法，不过我们在这里不再纠结。主要来说，只有当后验分布极度偏斜时，这种方法才会得出与中间 95% 方法有显著差异的结果。

后验可信区间

设参数 \(\pi\) 具有后验概率密度函数 \(f(\pi | y)\)。后验可信区间（CI）提供了 \(\pi\) 的后验可信值范围，从而总结了后验的中心趋势和变异性。例如，一个中间 95% 可信区间由第 2.5 和第 97.5 后验百分位数构成，

\[(\pi_{0.025}, \pi_{0.975}) .\]

因此，\(\pi\) 落在此范围内的后验概率为 95%：

\[P(\pi \in (\pi_{0.025}, \pi_{0.975}) | Y=y) = \int_{\pi_{0.025}}^{\pi_{0.975}} f(\pi|y)d\pi = 0.95 .\]

后验假设检验

单侧检验

假设检验是后验分析中另一项常见任务。例如，假设我们读到一篇文章声称博物馆艺术家中 Gen X 及更年轻世代的比例不足 20%。我们从关于 \(\pi\) 的后验模型中观察到的两条线索表明，这一说法至少部分可信：

  1. 图 8.3 中后验概率密度函数的大部分落在 0.2 以下。
  2. 关于 \(\pi\) 的 95% 可信区间 (0.1, 0.24) 大部分低于 0.2。

这些观察结果是一个很好的开端。但我们还可以更精确。为了精确评估 \(\pi < 0.2\) 的可信程度，我们可以计算这一场景的后验概率（posterior probability），即 \(P(\pi < 0.2 | Y = 14)\)。该后验概率由图 8.4 中后验概率密度函数下的阴影面积表示，从数学上讲，是通过在 0 到 0.2 范围内对后验概率密度函数积分计算得出：

\[P(\pi < 0.2 \; | \; Y = 14) = \int_{0}^{0.2}f(\pi | y=14)d\pi .\]

我们将跳过积分，直接使用 scipy.stats.beta.cdf() 来计算这个 Beta(18,92) 后验概率。结果揭示了支持我们主张的有力证据：Gen X 及更年轻艺术家占现代艺术博物馆艺术家的比例不足 20% 的后验概率约为 84.9%。

# 后验概率 pi < 0.20
from scipy.stats import beta
post_prob = beta.cdf(0.20, 18, 92)
post_prob

图 8.4：\(\pi\) 低于 0.20 的 Beta(18,92) 后验概率由后验概率密度函数下方的阴影区域表示。

我们对这一主张的分析非常直接明了。我们只是计算了感兴趣场景的后验概率。尽管并非总是必要，但实践者通常会将这个过程形式化为一个假设检验框架。例如，我们可以将分析表述为两个对立的假设：零假设 \(H_0\) 主张至少 20% 的博物馆艺术家是 X 一代或更年轻（即现状），而备择假设 \(H_a\)（我们的主张）则主张该比例低于 20%。用数学符号表示：

\[\begin{split} H_0: & \; \; \pi \ge 0.2 \\ H_a: & \; \; \pi < 0.2 \\ \end{split}\]

请注意，\(H_a\) 声称 \(\pi\) 位于 0.2 的一侧（\(\pi < 0.2\)），而非仅仅是不同于 0.2（\(\pi \ne 0.2\)）。因此，我们称之为单侧假设检验。

我们已经计算出备择假设的后验概率为 \(P(H_a \; | \; Y=14) = 0.849\)。因此，零假设的后验概率为 \(P(H_0 \; | \; Y=14) = 0.151\)。综合起来，\(\pi < 0.2\) 的后验优势（posterior odds）约为 5.62。也就是说，我们的后验评估是，\(\pi\) 低于 0.2 的可能性大约是高于 0.2 的可能性的 6 倍：

\[\text{posterior odds } = \frac{P(H_a \; | \; Y=14)}{P(H_0 \; | \; Y=14)} \approx 5.62.\]

# 后验优势 (posterior odds)
post_odds = post_prob / (1 - post_prob)
post_odds

当然，这些后验优势代表了我们在观察到调查数据后对 \(\pi\) 的更新理解，即在 \(n =\) 100 名抽样艺术家中，\(Y =\) 14 人是 X 一代或更年轻。在对这些艺术家进行抽样之前，我们对主要艺术博物馆中 X 一代代表性的评估要高得多（图 8.5）。

图 8.5：\(\pi\) 低于 0.2 的后验概率（右）与该场景的先验概率（左）对比。

具体而言，由 Beta(4,6) 先验概率密度函数 \(f(\pi)\) 下方 0.2 以下的面积计算出的 \(\pi < 0.2\) 的先验概率仅为 0.0856：

\[P(H_a) = \int_0^{0.2} f(\pi) d\pi \approx 0.0856 .\]

import scipy.stats as stats

# 先验概率 pi < 0.2
prior_prob = stats.beta.cdf(0.20, 4, 6)
prior_prob

因此，零假设的先验概率为 \(P(H_0) = 0.914\)。 由此可知，X世代代表性低于0.2的先验优势大约只有十分之一：

\[\text{先验优势 } = \frac{P(H_a)}{P(H_0)} \approx 0.093 .\]

# 先验优势
prior_odds = prior_prob / (1 - prior_prob)
prior_odds

贝叶斯因子（Bayes Factor，BF） 比较后验优势与先验优势，从而揭示我们关于X世代代表性的认知在观测到样本数据后演变了多少：

\[\text{贝叶斯因子} = \frac{\text{后验优势 }}{\text{先验优势 }} .\]

在我们的示例中，贝叶斯因子约为60。 因此，在观测到艺术品数据后，我们关于X世代假设的后验优势大约是先前优势的60倍。 或者说，我们对该假设的信心大幅提升。

# 贝叶斯因子（假设 post_odds 已在前文计算）
BF = post_odds / prior_odds
BF

我们在下方总结贝叶斯因子，并给出一些解释的指导原则。

贝叶斯因子

在比较两个对立假设 \(H_a\) 与 \(H_0\) 的假设检验中，贝叶斯因子是 \(H_a\) 的优势比：

\[\text{贝叶斯因子} = \frac{\text{后验优势}}{\text{先验优势}} = \frac{P(H_a | Y) / P(H_0 | Y)}{P(H_a) / P(H_0)} .\]

作为一个比率，将贝叶斯因子（BF）与1进行比较是有意义的。为此，考虑以下三种情况：

  1. BF = 1：\(H_a\) 的可信性在观测数据下没有改变。
  2. BF > 1：\(H_a\) 的可信性在观测数据下增加了。因此，贝叶斯因子越大，支持 \(H_a\) 的证据越有说服力。
  3. BF < 1：\(H_a\) 的可信性在观测数据下减小了。

综合来看，后验概率（0.85）和贝叶斯因子（60）提供了相当有说服力的证据，支持了“主要现当代艺术博物馆的艺术家中，X 世代或更年轻的比例不到 20%”这一主张。 读到这句话时你是否觉得有些勉强？ “相当有说服力”这个说法可能显得有些模棱两可。 过去你可能学过用特定阈值来区分“统计显著”与“统计不显著”，或者据此“拒绝”或“未能拒绝”某个假设。 但这种做法会带来虚假的安心感。 现实并非如此非黑即白。 正因如此，无论是频率学派还是贝叶斯学派，更广泛的统计社区都反对用一刀切的规则做出刚性结论，而是提倡一种更细致的方法，结合每次假设检验的具体背景和潜在影响来综合判断。 所以，并不存在一个万能的魔法阈值，能根据贝叶斯因子或后验概率的大小将主张简单归为“真”或“假”两类。 事实上，我们拥有的东西比二元决策更强大——我们有了一个对自己不确定性程度的整体衡量。 这种不确定性程度可以指导我们下一步的行动。 在我们的艺术例子中，我们是否有充分证据支持我们的主张？ 我们是信服的。

双侧检验

特别是在新情境下工作时，我们并不总是希望针对某个参数 \(\pi\) 检验单向主张。 比如，假设一位新艺术研究者只想检验是否有30%的主要博物馆艺术家属于 X 世代或更年轻。 这个假设是双侧的：

\[\begin{split} H_0: & \; \; \pi = 0.3 \\ H_a: & \; \; \pi \ne 0.3 \\ \end{split}\]

当我们试图用处理单侧假设检验的那套方法来解决这个双侧检验时，很快就遇到了问题。 由于 \(\pi\) 是连续变量，\(\pi\) 恰好等于 0.3（即 \(H_0\) 为真）的先验概率和后验概率都是零。 例如，\(\pi = 0.3\) 的后验概率由后验概率密度函数在 0.3 处的线段下方的面积给出。 对任何一条线段来说，这个面积都是 0：

\[P(\pi = 0.3 | Y = 14) = \int_{0.3}^{0.3} f(\pi | y = 14)d\pi = 0 .\]

因此，后验优势比、先验优势比，进而贝叶斯因子，都是未定义的：

\[\text{后验优势比 } = \frac{P(H_a \; | \; Y=14)}{P(H_0 \; | \; Y=14)} = \frac{1}{0} = \text{ 完蛋了！}\]

没问题。成功没有固定套路。为此，试试下面的小测验。45

回想一下，\(\pi\) 的 95% 后验可信区间为 (0.1, 0.24)。这个可信区间是否提供了充分证据，表明 \(\pi\) 与 0.3 不同？

如果你的回答是“是”，那么你已经凭直觉想到了一个合理的双边假设检验思路。\(\pi\) 的假设值（这里是 0.3）“大幅”落在后验可信区间之外，因此我们有充分证据支持 \(H_a\)。0.3 远高于 \(\pi\) 合理取值范围的这一事实，让我们相当确信博物馆艺术家中属于 X 世代或更年轻的比例不是 0.3。不过，在一个上下文中“大幅”或清晰的程度，在另一上下文中可能不同。有鉴于此，最佳实践是提前、在看到任何数据之前就定义好什么是“大幅”。例如，在艺术家代表比例的背景下，我们可以将任何落在 0.3 附近 0.05 窗口之外的比例视为与 0.3 有实际意义的差异。这实质上给我们的假设增加了一点缓冲，\(\pi\) 要么约等于 0.3（在 0.25 到 0.35 之间），要么不是：

\[\begin{split} H_0: & \; \; \pi \in (0.25, 0.35) \\ H_a: & \; \; \pi \notin (0.25, 0.35) \\ \end{split}\]

有了这个设定的缓冲带，我们可以更严格地声称相信 \(H_a\)，因为 \(\pi\) 的整个假设范围 (0.25, 0.35) 都位于其 95% 可信区间之上。还要注意，由于 \(H_0\) 不再包含 \(\pi\) 的单一假设值，其对应的后验概率和先验概率也不再是 0。因此，就像我们在单侧假设检验中做的那样，我们可以（但这里不会）用后验概率和贝叶斯因子计算来补充上述后验可信区间分析。

后验预测

除了后验估计和假设检验，后验分析的第三个常见任务是预测新数据 \(Y'\) 的结果。

假设我们获得了博物馆中另外 20 件艺术作品的数据。基于我们在本章中建立的对 \(\pi\) 的后验理解，你预测其中由 X 世代或更年轻艺术家创作的作品数量是多少？

你对这个小测验的可能第一反应是：“我 知道 答案了，是 3！” 这是一个非常合理的起点。毕竟，我们最佳的后验猜测是，博物馆艺术家中约 16% 是 X 世代或更年轻，而 20 位新艺术家中的 16% 大约是 3。然而，这一计算忽略了我们预测中两个潜在变异性来源：

  • 抽样变异性（sampling variability）在数据中
    当我们随机抽取 20 位艺术家时，我们不期望其中 恰好 有 3 位（16%）是 X 世代或更年轻。相反，这个数字会随着我们碰巧得到的随机样本而波动。

  • 后验变异性（posterior variability）在 \(\pi\) 中
    0.16 并不是 \(\pi\) 的 唯一 后验合理值，\(\pi\) 是博物馆艺术家中 X 世代所占的基础比例。相反，我们的 95% 后验可信区间表明 \(\pi\) 可能在约 0.1 到 0.24 之间。因此，在对 20 件新艺术品进行预测时，我们需要考虑在 每个 可能的 \(\pi\) 下我们期望看到的结果，同时考虑到某些 \(\pi\) 值比其他值更合理。

让我们用一些数学来明确这些概念。首先，令 \(Y' = y'\) 表示 20 件新艺术品中由 X 世代或更年轻艺术家创作的数量（尚待观察），其中 \(y'\) 可以是 \(\{0,1,...,20\}\) 中的任意艺术家数量。在给定 \(\pi\) 的条件下，\(Y'\) 中的随机性或 抽样变异性 可以用 \(Y'|\pi \sim \text{Bin}(20,\pi)\) 建模，其概率密度函数为

\[\begin{equation} f(y'|\pi) = P(Y' = y' | \pi) = \binom{20}{ y'} \pi^{y'}(1-\pi)^{20 - y'} . \tag{8.2} \end{equation}\]

因此，\(Y'\) 的随机结果依赖于 \(\pi\)，而 \(\pi\) 本身也会变化——\(\pi\) 可以是 0 到 1 之间的任何值。为此，给定 原始 数据（\(Y =\) 14）的 \(\pi\) 的 Beta(18, 92) 后验模型描述了 \(\pi\) 中潜在的 后验变异性，即哪些 \(\pi\) 值比其他值更合理。

为了全面理解接下来 20 位艺术家中有多少会是 X 世代，我们必须 结合 \(Y'\) 中的抽样变异性和 \(\pi\) 中的后验变异性。为此，用后验概率密度函数 \(f(\pi|y = 14)\) (8.1) 对 \(f(y'|\pi)\) (8.2) 进行加权，捕捉了在给定 \(\pi\) 下观测到 \(Y' = y'\) 名 X 世代艺术家的机会，同时 考虑到了该 \(\pi\) 值的后验合理性：

\[\begin{equation} f(y'|\pi) f(\pi|y=14) . \tag{8.3} \end{equation}\]

图 8.6 展示了这一思想，绘制了 \(Y'\) 的加权行为 (8.3)，只考虑 \(\pi\) 的三个可能取值：后验第 2.5 百分位数（0.1）、后验众数（0.16）和后验第 97.5 百分位数（0.24）。 自然，我们看到 \(\pi\) 越大，\(Y'\) 倾向于越大：当 \(\pi = 0.1\) 时，\(Y'\) 最可能的取值是 2，而当 \(\pi = 0.24\) 时，最可能的取值是 5。 同时注意，由于低至 0.1 或高至 0.24 的 \(\pi\) 值并不十分合理，在这些情景下可能产生的 \(Y'\) 值被赋予的权重（即竖条短得多）要低于在 \(\pi =\) 0.16 —— 最为合理的 \(\pi\) 值 —— 下产生的值。

图 8.6：对 \(\pi \in \{0.10,0.16,0.24\}\) 绘制了可能的 \(Y'\) 结果，并按 \(\pi\) 对应的后验合理性加权。

综合起来，\(Y'\)（即 20 位新艺术家中属于 X 世代的人数）的 后验预测模型 同时考虑了 \(Y'\) 的抽样变异性与 \(\pi\) 的后验变异性。 具体而言，后验预测概率质量函数（pmf）通过在 (8.3) 上求平均，计算了在 所有 可能的 \(\pi\) 从 0 到 1 的取值中观察到 \(Y' = y'\) 的总体概率，其中 (8.3) 给出的是对于给定的 \(\pi\) 观察到 \(Y' = y'\) 的概率：

\[f(y'|y=14) = P(Y'=y' \; | \; Y=y) = \int_0^1 f(y'|\pi) f(\pi|y=14) d\pi .\]

后验预测模型

令 \(Y'\) 表示变量 \(Y\) 的一个新观测值。 再令概率密度函数 \(f(y'|\pi)\) 表示 \(Y'\) 对 \(\pi\) 的依赖，后验概率密度函数 \(f(\pi|y)\) 表示给定原始数据 \(Y = y\) 时 \(\pi\) 的后验合理性。 那么 \(Y'\) 的后验预测模型的概率密度函数为

\[\begin{equation} f(y'|y) = \int f(y'|\pi) f(\pi|y) d\pi . \tag{8.4} \end{equation}\]

换句话说，观察到 \(Y' = y'\) 的整体概率是对任意可能 \(\pi\) 下观察到该结果的概率（\(f(y'|\pi)\)）按 \(\pi\) 的后验合理性（\(f(\pi|y)\)）进行加权的结果。

\(Y'\) 的概率质量函数的精确公式来自一些微积分运算（这里不展示，不过它很有趣，如果你有微积分基础，我们鼓励你尝试推导）：

\[\begin{equation} f(y'|y=14) = \left(\!\begin{array}{c} 20 \\ y' \end{array}\!\right) \frac{\Gamma(110)}{\Gamma(18)\Gamma(92)} \frac{\Gamma(18+y')\Gamma(112-y')} {\Gamma(130)} \;\;\; \text{ for } y' \in \{0,1,\ldots,20\}. \tag{8.5} \end{equation}\]

尽管这个公式与我们见过的任何公式都不同（比如它不是二项、泊松或我们学过的任何其他分布），它仍然明确指出了 \(Y'\) 可能取哪些值以及每个取值对应的概率。 例如，将 \(y' = 3\) 代入此公式，20 位新艺术家中恰好有 3 位是 X 世代的后验预测概率为 0.2217：

\[f(y'=3|y=14) = \left(\!\begin{array}{c} 20 \\ 3 \end{array}\!\right) \frac{\Gamma(110)}{\Gamma(18)\Gamma(92)} \frac{\Gamma(18+3)\Gamma(112-3)} {\Gamma(130)} = 0.2217 .\]

这个概率质量函数（pmf）公式也反映了我们关于 \(\pi\) 的 Beta(18,92) 后验模型（通过参数 18 和 92）以及原始先验和数据对 \(Y'\) 的后验理解的影响。 也就是说，与任何后验操作一样，我们的后验预测平衡了来自先验和数据的信息。

为了更全面地观察，后验预测概率质量函数 \(f(y'|y=14)\) 绘制在图 8.7 中。 虽然它看起来与假设 \(\pi\) 等于后验众数 0.16 时的 \(Y'\) 模型（图 8.6 中间）相似，但它在 \(\pi\) 偏离 0.16 时可能出现的较小和较大的 \(Y'\) 值上赋予了相对更多的权重。 总而言之，尽管接下来的 20 件艺术品中由 X 世代或更年轻艺术家创作的数量最有可能是 3，但它也可能落在 0 到 10 之间的任何地方。

图 8.7：\(Y'\) 的后验预测模型，即接下来的 20 件艺术品中由 X 世代或更年轻艺术家创作的作品数量。

最后，在建立并检查了 \(Y'\)（即接下来的20位艺术家中属于X世代的人数）的后验预测模型后，我们可能会有一些后续问题。 例如，至少5位艺术家是X世代的后验概率 \(P(Y' \ge 5 | Y = 14)\) 是多少？ 我们期望接下来的20位艺术家中有多少是X世代，\(E(Y' | Y = 14)\)？ 我们可以回答这些问题，只是有点繁琐。 由于 \(Y'\) 的后验预测模型并不常见，我们不能像处理 \(\pi\) 的 Beta 后验模型那样使用预置公式或 Python 函数来计算后验特征。 相反，我们必须从头计算这些特征。 例如，我们可以通过将 pmf (8.5) 在该范围内的 16 个 \(y'\) 值上求和，计算出至少5位艺术家是X世代的后验概率。 这个庞大求和的结果（其细节将填满一整页）是 0.233：

\[\begin{split} P(Y' \ge 5 | y = 14) & = \sum_{y' = 5}^{20} f(y' | y = 14) \\ & = f(y' = 5 | y = 14) + f(y' = 6 | y = 14) + \cdots + f(y' = 20 | y = 14)\\ & = 0.233. \\ \end{split}\]

类似地，尽管我们尚未进行此类计算（且不会再做），接下来20位艺术家中期望的X世代人数可以通过可能的 \(Y'\) 值的后验加权平均获得。 也就是说，我们可以将 0 到 20 的每个 \(Y'\) 值乘以其后验概率 \(f(y'|y=14)\) 后相加。 这个庞大求和的结果表明，我们应期望在20位艺术家中有大约3位是X世代：

\[\begin{split} E(Y' | y = 14) & = \sum_{y' = 0}^{20} y' f(y' | y = 14) \\ & = 0\cdot f(y' = 0 | y = 14) + 1 \cdot f(y' = 1 | y = 14) + \cdots + 20\cdot f(y' = 20 | y = 14)\\ & = 3.273. \\ \end{split}\]

但我不想让你被这类计算过度分心。在本书中，我们再也不会需要进行这样的计算。从第 9 章开始，我们的模型会变得足够复杂，以至于即便像这样繁琐的公式也无法解析得出，我们将需要依赖模拟来近似后验特征。

使用 MCMC 的后验分析

知道贝叶斯后验分析背后有一些理论非常好。当我们在使用像 Beta-二项模型这样简单的模型时，可以直接应用这些理论——即可以计算精确的后验可信区间、概率和预测模型。然而，在第 9 章中，我们将离开这个舒适区，进入无法指定后验模型、更不用说计算其特征精确概括的场景。回顾第 6 章，在这些场景中，我们可以使用 MCMC 方法来近似后验分布。在本节中，我们将探索如何使用所得的马尔可夫链样本值来近似特定的后验特征，进而进行后验分析。

后验模拟

下面我们为 \(\pi\) 运行四条并行的马尔可夫链，每条链进行 10,000 次迭代。抛弃每条链的前 5,000 次迭代后，我们得到四个规模为 5,000 的独立马尔可夫链样本，\(\left\lbrace \pi^{(1)}, \pi^{(2)}, \ldots, \pi^{(5000)}\right\rbrace\)，或者合并后规模为 20,000 的马尔可夫链样本。

import pymc as pm
import numpy as np

# 步骤1：定义模型
with pm.Model() as art_model:
    # 先验
    pi = pm.Beta('pi', alpha=4, beta=6)
    # 似然
    Y_obs = pm.Binomial('Y_obs', n=100, p=pi, observed=14)

# 步骤2：模拟后验
with art_model:
    art_sim = pm.sample(draws=5000, tune=5000, chains=4, random_seed=84735)

后验预测

最后，我们可以利用马尔可夫链值来近似关于\(Y'\)的后验预测模型，即接下来抽样的20位艺术家中属于X世代或更年轻的人数。 额外收获：模拟该模型也有助于我们建立对后验预测背后理论的直观理解。 回想一下，后验预测模型反映了两种变异性来源：

  • 数据中的抽样变异性
    \(Y'\)可能是\(\{0,1,\ldots,20\}\)中的任意一个艺术家数量，取决于艺术家中X世代的基础比例\(\pi\)：\(Y'| \pi \sim \text{Bin}(20,\pi)\)。

  • \(\pi\)的后验变异性
    20,000个马尔可夫链\(\pi\)值的集合提供了对合理\(\pi\)值的变异性和范围的近似感知。

为了捕捉后验预测\(Y'\)中的两种变异性来源，我们可以使用scipy.stats.binom.rvs从每一个20,000个\(\pi\)链值中模拟一个\(\text{Bin}(20,\pi)\)结果\(Y'\)。 前三个结果反映了一个总体趋势：较小的\(\pi\)值往往会产生较小的\(Y'\)值。 这是合理的。 博物馆中X世代艺术家的基础比例越低，我们在下一个20件艺术品的样本中期望看到的X世代艺术家就越少。

贝叶斯的优势

在第 1 章，我们曾对比了贝叶斯分析与频率学派分析。现在我们已经过了一些具体例子，让我们重温其中的一些想法。你或许已经体会到，贝叶斯分析最困难的部分往往是构建或模拟后验模型。一旦有了这个模型，利用后验进行估计、假设检验和预测就相当简单。相比之下，建立相应的频率学派计算公式往往不那么直观。

我们也可以享受贝叶斯结果易于解释的好处。通常，一项 贝叶斯分析 会根据观测数据 \(Y\) 来评估未知参数 \(\pi\) 的不确定性。例如，考虑艺术家研究。根据观测到 100 位抽样艺术家中有 \(Y = 14\) 位属于 X 世代或更年轻，我们得出整个博物馆中 X 世代比例 \(\pi\) 低于 0.20 的后验概率为 84.9%：

\[P(\pi < 0.20 \; | \; Y = 14) = 0.849 .\]

这种计算在频率学派分析中没有意义。反过来，一项 频率学派分析 会根据 假定的 \(\pi\) 值来评估观测数据 \(Y\) 的不确定性。例如，与上述贝叶斯后验概率对应的频率学派指标是 p 值，其公式我们不在此展开：

\[P(Y \le 14 \; | \; \pi = 0.20) = 0.08 .\]

此概率中的条件顺序相反（给定 \(\pi\) 的 \(Y\) 而非给定 \(Y\) 的 \(\pi\)），导致了与贝叶斯概率不同的计算和解释：如果 \(\pi\) 仅为 0.20，那么我们观测到至多 \(Y = 14\) 位 X 世代艺术家的概率只有 8%。这里不是我们的表述别扭，而是 p 值本身。尽管它确实提供了某种有趣的信息，但它回答的问题对人类大脑来说稍微不那么自然：因为我们实际 观测 到了数据但 不 知道 \(\pi\)，去解释一个假设相反的结论会让人感到困惑。主要是，在进行假设检验时，问“我的假设有多大概率？”（贝叶斯概率所回答的）比问“如果我的假设不成立，我的数据有多大概率？”（频率学派概率所回答的）更自然。考虑到 p 值经常被误解和误用，它们在整个频率学派和贝叶斯领域都正在被淡化。

本章小结

在第8章，你学会了如何将后验模型转化为答案。也就是说，你利用精确或近似的后验模型，对未知参数 \(\pi\) 执行三项后验分析任务：

  1. 后验估计
    后验可信区间（credible interval）提供了 \(\pi\) 的一个后验合理取值范围，进而让你对 \(\pi\) 的后验典型值及不确定性都有所感知。
  2. 后验假设检验
    后验概率有助于深入理解关于 \(\pi\) 的相应假设。
  3. 后验预测
    新数据点 \(Y\) 的后验预测模型同时考虑了 \(Y\) 的抽样变异性和 \(\pi\) 的后验变异性。
