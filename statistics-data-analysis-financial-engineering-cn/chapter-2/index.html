第2章 收益率

2.1 引言

投资的目的是为了获得收益。投资的收益（或负收益情况下的亏损）取决于价格变动和所持有资产的数量。投资者关心的是相对于初始投资规模较高的收益。收益率正是衡量这一点，因为资产的收益率（例如股票、债券、股票和债券的组合）是价格变动相对于初始价格的分数形式。

2.1.1 净收益率

设 \(P_t\) 为资产在时间 \(t\) 的价格。假设无股息，则从时间 \(t-1\) 到时间 \(t\) 的持有期净收益率为

\[ R_t = \frac{P_t}{P_{t-1}} - 1 = \frac{P_t - P_{t-1}}{P_{t-1}}. \]

分子 \(P_t - P_{t-1}\) 是持有期内的收益或利润，负利润表示亏损。分母 \(P_{t-1}\) 是持有期开始时的初始投资。因此，净收益率可视为相对收益或利润率。

持有资产所获得的收益为

\[ \text{收益} = \text{初始投资} \times \text{净收益率}. \]

例如，初始投资 10,000 美元，净收益率为 6%，则获得收益 600 美元。由于 \(P_t \geq 0\)，

2 收益率

\[R_t \ge -1, \qquad (2.1)\]

因此最差的可能收益率为-1，即100%的损失，当资产变得一文不值时发生。

2.1.2 总收益率

简单的总收益率定义为 \[\frac{P_t}{P_{t-1}} = 1 + R_t.\] 例如，若 \(P_t = 2\) 且 \(P_{t+1} = 2.1\)，则 \(1 + R_{t+1} = 1.05\)，即105%，而 \(R_{t+1} = 0.05\)，即5%。个体在时刻 \(t\) 的最终财富等于其在时刻 \(t-1\) 的初始财富乘以总收益率。换言之，若 \(X_0\) 是时刻 \(t-1\) 的初始财富，则 \(X_0(1 + R_t)\) 是时刻 \(t\) 的财富。

收益率是无量纲的（scale-free），意味着它们不依赖于单位（美元、美分等）。收益率并非没有单位。其单位是时间；它们依赖于 \(t\) 的单位（小时、日等）。在本例中，若 \(t\) 以年为单位，则更精确地说，净收益率为每年5%。

最近 \(k\) 个时期的总收益率是 \(k\) 个单期总收益率的乘积（从时刻 \(t-k\) 到时刻 \(t\)）： \[1 + R_t(k) = \frac{P_t}{P_{t-k}} = \left(\frac{P_t}{P_{t-1}}\right)\left(\frac{P_{t-1}}{P_{t-2}}\right) \cdots \left(\frac{P_{t-k+1}}{P_{t-k}}\right) = (1+R_t) \cdots (1+R_{t-k+1}).\] \(k\) 期净收益率为 \(R_t(k)\)。

2.1.3 对数收益率

对数收益率（log returns），也称为连续复利收益率（continuously compounded returns），用 \(r_t\) 表示，定义为 \[r_t = \log(1+R_t) = \log\left(\frac{P_t}{P_{t-1}}\right) = p_t - p_{t-1},\] 其中 \(p_t = \log(P_t)\) 称为对数价格（log price）。

对数收益率近似等于收益率，因为若 \(x\) 很小，则 \(\log(1+x) \approx x\)，如图2.1所示，图中绘制了 \(\log(1+x)\)。注意图中，若 \(|x| < 0.1\)，例如收益率小于10%，则 \(\log(1+x)\) 非常接近 \(x\)。

例如，5%的收益率等于4.88%的对数收益率，因为 \(\log(1+0.05) = 0.0488\)。同样地，-5%的收益率等于-5.13%的对数收益率，因为 \(\log(1-0.05) = -0.0513\)。两种情形下，\(r_t = \log(1+R_t) \approx R_t\)。此外，\(\log(1+0.01) = 0.00995\)，\(\log(1-0.01) = -0.01005\)，因此±1%的对数收益率非常接近

对应的净收益率。由于收益率在较短周期内幅度较小，我们可以预期日收益率与日对数收益率相近，年收益率则不太相近，而对于更长的周期（如10年）则不一定相似。

收益率和对数收益率具有相同的符号。当两者均为正（负）时，对数收益率的绝对值小于（大于）收益率。两者之间的差异在均为很大的负值时最为显著。接近下限-1的收益率（即完全损失）对应接近负无穷的对数收益率。

使用对数收益率的一个优点是简化了多期收益率。k期对数收益率就是单期对数收益率的和，而不是像总收益率那样是乘积。要理解这一点，注意到k期对数收益率为：

\[\begin{aligned} r_t(k) &= \log\{1+R_t(k)\} \\ &= \log\{(1+R_t)\cdots(1+R_{t-k+1})\} \\ &= \log(1+R_t)+\cdots+\log(1+R_{t-k+1}) \\ &= r_t + r_{t-1} + \cdots + r_{t-k+1}. \end{aligned}\]

2.1.4 股息调整

许多股票，尤其是成熟公司的股票，会支付股息，在计算收益率时必须考虑。类似地，债券会支付利息。如果一个

股息（或利息）\(D_t\) 在时间 \(t\) 之前支付，则时间 \(t\) 的总收益率定义为

\[1+R_t = \frac{P_t + D_t}{P_{t-1}}, \tag{2.2}\]

因此净收益率为 \(R_t = (P_t + D_t)/P_{t-1} - 1\)，对数收益率为

\[r_t = \log(1+R_t) = \log(P_t + D_t) - \log(P_{t-1}).\]

多期总收益率为单期总收益率的乘积，即

\[1+R_t(k) = \left(\frac{P_t + D_t}{P_{t-1}}\right) \left(\frac{P_{t-1} + D_{t-1}}{P_{t-2}}\right) \cdots \left(\frac{P_{t-k+1} + D_{t-k+1}}{P_{t-k}}\right) = (1+R_t)(1+R_{t-1})\cdots (1+R_{t-k+1}), \tag{2.3}\]

其中，对于任意时间，若 \(s-1\) 与 \(s\) 之间没有股息，则 \(D_s = 0\)。类似地，\(k\) 期对数收益率为

\[r_t(k) = \log\{1+R_t(k)\} = \log(1+R_t) + \cdots + \log(1+R_{t-k+1}) = \log\left(\frac{P_t + D_t}{P_{t-1}}\right) + \cdots + \log\left(\frac{P_{t-k+1} + D_{t-k+1}}{P_{t-k}}\right).\]

2.2 随机游走模型

随机游走假设表明，单期对数收益率 \(r_t = \log(1+R_t)\) 是独立的。因为

\[1+R_t(k) = (1+R_t)\cdots (1+R_{t-k+1}) = \exp(r_t)\cdots \exp(r_{t-k+1}) = \exp(r_t + \cdots + r_{t-k+1}),\]

我们有

\[\log\{1+R_t(k)\} = r_t + \cdots + r_{t-k+1}. \tag{2.4}\]

有时进一步假设对数收益率服从均值为 \(\mu\)、方差为 \(\sigma^2\) 的正态分布 \(N(\mu, \sigma^2)\)。由于正态随机变量之和仍是正态的，单期对数收益率的正态性意味着多期对数收益率的正态性。在此假设下，\(\log\{1+R_t(k)\}\) 服从 \(N(k\mu, k\sigma^2)\)。

2.2.1 随机游走

模型 (2.4) 是随机游走模型的一个例子。设 \(Z_1, Z_2, \ldots\) 为独立同分布（i.i.d.），均值为 \(\mu\)，标准差为 \(\sigma\)。设 \(S_0\) 为任意起点，且

\[S_t = S_0 + Z_1 + \cdots + Z_t, \quad t \ge 1. \tag{2.5}\]

2.2 随机游走模型

由 (2.5) 可知，\(S_t\) 是从 \(S_0\) 出发的随机游走经过 \(t\) 步后的位置。过程 \(S_0, S_1, \ldots\) 称为随机游走，而 \(Z_1, Z_2, \ldots\) 为其步长。

若步长服从正态分布，则称该过程为正态随机游走。给定 \(S_0\) 时 \(S_t\) 的条件期望和方差分别为 \(\mathbb{E}(S_t | S_0) = S_0 + \mu t\) 和 \(\operatorname{Var}(S_t | S_0) = \sigma^2 t\)。参数 \(\mu\) 称为漂移，决定了随机游走的总体方向；参数 \(\sigma\) 称为波动率，决定了随机游走围绕条件均值 \(S_0 + \mu t\) 的波动程度。由于给定 \(S_0\) 时 \(S_t\) 的标准差为 \(\sigma \sqrt{t}\)，因此 \((S_0 + \mu t) \pm \sigma \sqrt{t}\) 给出了均值加减一个标准差的范围，对于正态随机游走，该范围包含 68% 的概率。这个范围的宽度随 \(\sqrt{t}\) 成比例增长，如图 2.2 所示，表明在时刻 \(t = 0\) 时，我们对随机游走遥远未来位置的了解远少于对近期位置的了解。

2.2.1 几何随机游走

回忆 \(\log\{1 + R_t(k)\} = r_t + \cdots + r_{t-k+1}\)。因此，

\[ \frac{P_t}{P_{t-k}} = 1 + R_t(k) = \exp(r_t + \cdots + r_{t-k+1}), \tag{2.6} \]

取 \(k = t\)，我们有

\[ P_t = P_0 \exp(r_t + r_{t-1} + \cdots + r_1). \tag{2.7} \]

我们将对数服从随机游走的过程称为几何随机游走或指数随机游走。若 \(r_1, r_2, \ldots\) 为 i.i.d. \(N(\mu, \sigma^2)\)，则 \(P_t\) 对所有 \(t\) 均服从对数正态分布，该过程称为参数为 \((\mu, \sigma^2)\) 的对数正态几何随机游走。如附录 A.9.4 所述，\(\mu\) 称为 \(\exp(r_t)\) 的对数正态分布的对数均值，\(\sigma\) 称为对数标准差。此外，\(\mu\) 有时也称为对数正态几何随机游走的对数漂移。

2.2.2 对数价格是对数正态几何随机游走吗？

金融数学中的大量工作假设价格服从对数正态几何随机游走或其连续时间对应形式——几何布朗运动。因此一个自然的问题是：这一假设通常成立吗？

快速回答是“否”。对数正态几何随机游走有两个假设：(1) 对数收益率服从正态分布；(2) 对数收益率相互独立。

在第 4 章和第 5 章中，我们将研究几组对数收益率序列的边缘分布。结论是：虽然收益率密度具有类似正态密度的钟形形状，但对数收益率分布的尾部通常比正态尾部重得多。通常，一个...

具有小自由度参数（例如4–6）的t分布比正态模型拟合得好得多。然而，对数收益率分布似乎是对称的，或至少近乎对称。

独立性假设也被违反。首先，收益率之间存在一些相关性。然而，相关性通常很小。更严重的是，收益率表现出波动率聚集，这意味着如果我们在当前收益率中看到高波动性，我们可以预期这种高波动性将持续，至少持续一段时间。波动率聚集可以通过检查平方收益率之间的相关性来检测。

在抛弃资产价格是对数正态几何随机游走的假设之前，值得记住Box的箴言：“所有模型都是错误的，但有些模型是有用的。”这种假设有时是有用的，例如用于推导著名的Black–Scholes公式。

关于收益率，Fama (1965, 1970, 1991, 1998) 进行了综述。反对有效市场假说的证据可以在行为金融学领域找到，该领域利用对人类行为的研究来理解市场行为；参见 Shefrin (2000)、Shleifer (2000) 和 Thaler (1993)。市场无效的一个表现是市场价格过度波动；参见 Shiller (1992) 或 Shiller (2000) 的较少技术性讨论。

2.4.1 数据分析

从本书网站获取数据集 Stock_bond.csv，并将其放入你的工作目录。启动 R1，你应该会看到一个控制台窗口打开。使用“文件”菜单中的“更改目录”切换到工作目录。用以下命令读取数据：

dat = read.csv("Stock_bond.csv", header = TRUE)
