第20章 贝叶斯数据分析与MCMC

20.1 引言

贝叶斯统计基于一种与其他统计推断方法不同的哲学。在贝叶斯统计中，所有未知量，特别是未知参数，都被视为随机变量，其概率分布表达了我们对其可能取值的信念。估计、模型选择和不确定性分析通过使用贝叶斯定理随着新数据的观测来更新我们的信念而实现。

非贝叶斯主义者区分两类未知量：参数和潜变量。对于非贝叶斯主义者而言，参数是没有概率分布的固定量，而潜变量是具有概率分布的随机未知量。例如，对于一个非贝叶斯主义者，MA(q)过程的均值μ、移动平均系数θ1, …, θq和白噪声方差σϵ2是固定参数，而观测不到的白噪声过程本身由潜变量组成。相反，对于贝叶斯主义者，参数和白噪声过程都是未知的随机量。由于本章采取贝叶斯视角，无需区分参数和潜变量，因为现在它们可以以相同方式处理。我们将用θ表示所有未知量的向量，并称之为“参数向量”。例如，在时间序列预测的背景下，θ可以包括未观测的白噪声以及被预测序列的未来值。

贝叶斯统计的一个标志是必须从指定参数的先验信念开始。许多统计学家不愿使用贝叶斯分析，因为从先验信念开始似乎过于主观。因此，贝叶斯与非贝叶斯统计学家之间曾就统计学的哲学基础展开激烈辩论。然而，现在主流统计思想大多支持更为实用的观点：我们应使用任何行之有效的方法。

如果研究者对某个参数所知甚少，这种缺乏知识的情况可以通过使用所谓的无信息先验来应对，该先验相对于数据提供的信息而言，只给出关于参数的极少量信息。在实践中，当贝叶斯分析仅使用弱先验信息时，贝叶斯与非贝叶斯分析通常得出相似结论，因为此时参数的知识主要来自数据。

此外，在金融及许多其他应用领域，分析者通常拥有大量先验信息并愿意加以利用。在商业和金融中，并不像科学研究那样必须追求客观性。需要指定先验可以被视为贝叶斯统计观点的优势而非弱点，因为它迫使分析者仔细思考有多少以及哪种先验信息可用。

过去几十年中，贝叶斯统计的使用急剧增加，因为贝叶斯哲学已被更广泛接受，且贝叶斯估计量变得更容易计算。事实上，对于复杂模型，贝叶斯技术通常是最令人满意的估计方法。我们曾听一位研究者说：“我不是贝叶斯学派，但我使用贝叶斯方法”，无疑其他人也会同意。

本章概述：假设我们关注参数向量 \(\theta\)。贝叶斯分析从 \(\theta\) 的先验概率分布开始，该分布总结了关于 \(\theta\) 的所有先验知识；“先验”意味着在观测数据之前。似然函数在非贝叶斯分析中定义相同，但在贝叶斯统计中具有不同解释——似然是给定 \(\theta\) 下数据的条件分布。贝叶斯推断的关键步骤是利用贝叶斯定理将关于 \(\theta\) 的先验知识与数据中的信息结合起来。这通过计算给定数据下 \(\theta\) 的条件分布来实现，该分布称为后验分布。在许多（即便不是大多数）实际问题中，无法解析计算后验，因此使用数值方法。一类非常成功的数值贝叶斯方法是马尔可夫链蒙特卡洛（MCMC），它通过模拟一条马尔可夫链，使得该链的平稳分布等于参数的后验分布。从链中模拟的数据用于计算贝叶斯估计并进行不确定性分析。

20.2 贝叶斯定理

贝叶斯定理既适用于离散事件，也适用于连续分布的随机变量。我们先从离散事件的情况开始。连续情况将在第20.3节中介绍。

假设 \(B_1, \ldots, B_K\) 是样本空间 \(S\)（所有可能结果的集合）的一个划分。所谓“划分”是指当 \(i \neq j\) 时 \(B_i \cap B_j = \emptyset\)，且 \(B_1 \cup B_2 \cup \cdots \cup B_K = S\)。对于任意集合 \(A\)，我们有

\[ A = (A \cap B_1) \cup \cdots \cup (A \cap B_K), \]

因此，由于 \(B_1, \ldots, B_K\) 互不相交，

\[ P(A) = P(A \cap B_1) + \cdots + P(A \cap B_K). \tag{20.1} \]

由式 (20.1) 和条件概率的定义可得

\[ P(B_j | A) = \frac{P(A | B_j) P(B_j)}{P(A)} = \frac{P(A | B_j) P(B_j)}{P(A | B_1) P(B_1) + \cdots + P(A | B_K) P(B_K)}. \tag{20.2} \]

式 (20.2) 称为贝叶斯定理，也称为贝叶斯规则或贝叶斯定律。贝叶斯定理是一个简单、几乎平凡的数学结果，但其含义深远。贝叶斯定理的重要性在于它用于更新概率。下面是一个例子，虽然过于简单而不现实，但说明了如何应用贝叶斯定理。

例 20.1 离散情况下的贝叶斯定理

假设我们对某只股票的先前知识表明，在任何给定一天价格上涨的概率 \(\theta\) 为 0.4 或 0.6。根据过去的数据（例如来自类似股票的数据），我们相信 \(\theta\) 等于 0.4 或 0.6 的可能性相同。因此，我们有以下先验概率

\[ P(\theta = 0.4) = 0.5 \quad \text{且} \quad P(\theta = 0.6) = 0.5. \]

我们连续观察该股票五天，其价格在这五天全部上涨。假设价格变化在日间是独立的，因此连续五天价格上涨的概率为 \(\theta^5\)。根据这一信息，我们可能怀疑 \(\theta\) 是 0.6 而非 0.4。因此，在连续五次价格上涨的条件下，\(\theta = 0.6\) 的概率应大于先验概率 0.5，但大多少呢？用记号表示，令 \(A\) 为连续五天价格上涨的事件。那么，利用贝叶斯定理，我们有

\[\begin{aligned} P(\theta = 0.6 | A) &= \frac{P(A | \theta = 0.6) P(\theta = 0.6)}{P(A | \theta = 0.6) P(\theta = 0.6) + P(A | \theta = 0.4) P(\theta = 0.4)} \\ &= \frac{(0.6)^5 (0.5)}{(0.6)^5 (0.5) + (0.4)^5 (0.5)} \\ &= \frac{(0.6)^5}{(0.6)^5 + (0.4)^5} = \frac{0.07776}{0.07776 + 0.01024} = 0.8836. \end{aligned}\]

因此，在观察到连续五次价格上涨之前，我们赋予\(\theta=0.6\)的概率为0.5，而在观察到这一事件之后，该概率更新为0.8836。观测数据之前的概率称为先验概率，而基于观测数据的条件概率称为后验概率。因此，\(\theta=0.6\)的先验概率为0.5，后验概率为0.8836。□

20.3 先验分布与后验分布

贝叶斯定理极其重要，因为它精确地告诉我们如何根据新信息更新我们的信念。在没有数学帮助的情况下，人类在收到新信息后修正信念的能力很差1。人们往往对新信息重视不足或过度重视，而借助贝叶斯定理的指导可以缓解这一问题。

现在假设\(\theta\)是一个连续分布的参数向量。先验分布（密度为\(\pi(\theta)\)）表达了我们在观测数据之前关于\(\theta\)的信念。似然函数被解释为给定\(\theta\)下数据\(Y\)的条件密度，记为\(f(y|\theta)\)。利用式(A.19)，\(\theta\)和\(Y\)的联合密度是先验与似然的乘积，即

\[ f(y,\theta) = \pi(\theta) f(y|\theta). \tag{20.3} \]

\(Y\)的边缘密度通过对联合密度积分\(\theta\)得到：

\[ f(y) = \int \pi(\theta) f(y|\theta) \, d\theta, \tag{20.4} \]

而给定\(Y\)下\(\theta\)的条件密度为

\[ \pi(\theta|Y) = \frac{\pi(\theta) f(Y|\theta)}{f(Y)} = \frac{\pi(\theta) f(Y|\theta)}{\int \pi(\theta) f(Y|\theta) \, d\theta}. \tag{20.5} \]

式(20.5)是贝叶斯定理的另一种形式。式(20.5)左侧的密度称为后验密度，它给出了观测数据\(Y\)后\(\theta\)的概率分布。

请注意我们使用\(\pi\)表示\(\theta\)的密度，即\(\pi(\theta)\)是先验密度，\(\pi(\theta|Y)\)是后验密度。相比之下，\(f\)用于表示数据的密度，即\(f(y)\)是数据的边缘密度，\(f(y|\theta)\)是给定\(\theta\)的条件密度。

贝叶斯估计和不确定性分析基于后验分布。最常见的贝叶斯估计量是后验分布的众数和均值1。

  1. 参见 Edwards (1982)。 ↩

后验密度的众数称为最大后验估计（MAP估计）。后验的均值为

\[E(\theta|Y) = \int \theta \,\pi(\theta|Y)\,d\theta = \frac{\int \theta \,\pi(\theta)\,f(Y|\theta)\,d\theta}{\int \pi(\theta)\,f(Y|\theta)\,d\theta} \tag{20.6}\]

也称为后验期望。

例20.2. 更新关于股票价格上涨概率的先验信念

我们继续例20.1，但将简单而不现实的先验（即假设\(\theta\)只能是0.4或0.6）改为更合理的先验，其中\(\theta\)可以是区间\([0,1]\)内的任意值，但靠近1/2的值更有可能。具体地，我们使用Beta(2,2)先验，因此

\[\pi(\theta) = 6\,\theta(1-\theta), \quad 0 < \theta < 1.\]

令\(Y\)为连续五个交易日中股票价格上涨的天数。那么\(Y\)服从二项分布\(\text{Binomial}(n, \theta)\)，\(Y\)的密度为

\[f(y|\theta) = \binom{5}{y} \theta^y (1-\theta)^{5-y}, \quad y = 0,1,\dots,5.\]

由于我们观测到\(Y=5\)，因此\(f(Y|\theta) = f(5|\theta) = \theta^5\)，后验密度为

\[\pi(\theta|5) = \frac{6\theta(1-\theta)\theta^5}{\int_0^1 6\theta(1-\theta)\theta^5\,d\theta} = 56\,\theta^6(1-\theta),\]

这是一个Beta(7,2)密度。

先验和后验密度如图20.1所示。由于连续五天价格上涨，后验密度相对于先验向右移动。后验分布的0.05下分位数和0.05上分位数分别为0.529和0.953，并在图中标出。因此，\(\theta\)位于0.529和0.953之间的后验概率为90%。因此，区间\([0.529, 0.953]\)称为90%后验区间，提供了\(\theta\)的可能取值集合。后验区间是置信区间的贝叶斯对应物，将在20.6节进一步讨论。后验区间也称为可信区间。

\[\int_0^1 \theta\,\pi(\theta|5)\,d\theta = \int_0^1 56\,\theta^7(1-\theta)\,d\theta = 56 \times \frac{1}{72} = 0.778. \tag{20.7}\]

20.4 共轭先验

在例20.2中，先验和后验都是beta分布。这是共轭先验族的一个例子。一个分布族被称为统计模型（或等价地，似然函数）的共轭先验族，如果当先验属于该族时，后验也属于该族。共轭族很方便，因为它们使得后验的计算变得直接。我们只需要更新先验中的参数。为了了解如何操作，我们将推广例20.2。

例20.3 计算股票价格上涨概率的后验密度——共轭先验的一般情形

现在假设\(\theta\)的先验为\(\operatorname{Beta}(\alpha, \beta)\)，因此先验密度为

\[ \pi(\theta) = K_1 \theta^{\alpha-1} (1-\theta)^{\beta-1}, \tag{20.8} \]

其中\(K_1\)是一个常数。我们将会看到，知道\(K_1\)的具体数值并不重要，但根据(A.14)我们知道\(K_1 = \frac{\Gamma(\alpha+\beta)}{\Gamma(\alpha)\Gamma(\beta)}\)。参数

必须已知先验密度，因此这里的 \(\alpha\) 和 \(\beta\) 由数据分析者根据关于 \(\theta\) 的先验知识选择。这些参数的选择将在后面讨论。

假设股票价格被观测了 \(n\) 天，其中有 \(Y\) 天上涨（\(n - Y\) 天未上涨）。则似然为

\[ f(Y|\theta) = K_2 \theta^Y (1-\theta)^{n-Y}, \qquad (20.9) \]

其中 \(K_2 = \binom{n}{Y}\) 是另一个常数。\(\theta\) 和 \(Y\) 的联合密度为

\[ \pi(\theta) f(Y|\theta) = K_3 \theta^{\alpha + Y - 1} (1-\theta)^{\beta + n - Y - 1}, \qquad (20.10) \]

其中 \(K_3 = K_1 K_2\)。于是，后验密度为

\[ \pi(\theta|Y) = \frac{\pi(\theta) f(Y|\theta)}{\int_0^1 \pi(\theta) f(Y|\theta) d\theta} = K_4 \theta^{\alpha + Y - 1} (1-\theta)^{\beta + n - Y - 1}, \qquad (20.11) \]

其中

\[ K_4 = \frac{1}{\int_0^1 \theta^{\alpha + Y - 1} (1-\theta)^{\beta + n - Y - 1} d\theta}. \qquad (20.12) \]

后验分布为 \(\text{Beta}(\alpha + Y, \beta + n - Y)\)。

我们不需要跟踪 \(K_1, \dots, K_4\) 的值。由于 (20.11) 正比于 \(\text{Beta}(\alpha + Y, \beta + n - Y)\) 密度，且所有密度积分为1，可以推知比例常数为1，后验即为 \(\text{Beta}(\alpha + Y, \beta + n - Y)\)。由 (A.14) 可得

\[ K_4 = \frac{\Gamma(\alpha + \beta + n)}{\Gamma(\alpha + Y) \Gamma(\beta + n - Y)}. \]

值得注意的是后验的求解非常容易。只需分别将先验参数 \(\alpha\) 和 \(\beta\) 更新为 \(\alpha + Y\) 和 \(\beta + n - Y\)。利用附录 A.9.7 中关于 Beta 分布的均值和方差的结果，后验均值为

\[ \mathrm{E}(\theta|Y) = \frac{\alpha + Y}{\alpha + \beta + n}, \qquad (20.13) \]

后验方差为

\[ \mathrm{var}(\theta|Y) = \frac{(\alpha + Y)(\beta + n - Y)}{(\alpha + \beta + n)^2 (\alpha + \beta + n + 1)} = \frac{\mathrm{E}(\theta|Y) \{1 - \mathrm{E}(\theta|Y)\}}{\alpha + \beta + n + 1}. \qquad (20.14) \]

当 \(\alpha\) 和 \(\beta\) 相对于 \(Y\) 和 \(n\) 较小时，\(\mathrm{E}(\theta|Y)\) 近似等于 MLE，即 \(Y/n\)。如果我们几乎没有先验知识，

例20.4 估计已知方差的正态总体均值时的后验分布

假设 \(Y_1,\ldots,Y_n\) 独立同分布，服从 \(N(\mu,\sigma^2)\)，且 \(\sigma^2\) 已知。之所以做出 \(\sigma^2\) 已知这一不现实的假设，是为了从简单情形入手，稍后将去掉这一假设。

\(\mu\) 的共轭先验族是正态分布族。为说明这一点，假设 \(\mu\) 的先验分布为 \(N(\mu_0,\sigma_0^2)\)，其中 \(\mu_0\) 和 \(\sigma_0^2\) 已知。从例20.3 中我们了解到，没有必要跟踪那些不依赖于未知参数的项（但可能依赖于数据或已知参数），因此我们只跟踪依赖于 \(\mu\) 的项。简单的代数运算表明，似然函数为

\[ f(Y_1,\ldots,Y_n|\mu) = \prod_{i=1}^n \left[ \frac{1}{\sqrt{2\pi}\sigma} \exp\left\{ -\frac{1}{2\sigma^2}(Y_i-\mu)^2 \right\} \right] \propto \exp\left\{ -\frac{1}{2\sigma^2}\left( -2n\bar{Y}\mu + n\mu^2 \right) \right\}. \qquad (20.16) \]

先验密度为

\[ \pi(\mu) = \frac{1}{\sqrt{2\pi}\sigma_0} \exp\left\{ -\frac{1}{2\sigma_0^2}(\mu-\mu_0)^2 \right\} \propto \exp\left\{ -\frac{1}{2\sigma_0^2}(-2\mu\mu_0+\mu^2) \right\}. \qquad (20.17) \]

精度是方差的倒数，记 \(\tau = 1/\sigma^2\) 为总体精度。将 (20.16) 与 (20.17) 相乘，可以看出后验密度为

\[ \pi(\mu|Y_1,\ldots,Y_n) \propto \exp\left\{ \left( \frac{n\bar{Y}}{\sigma^2} + \frac{\mu_0}{\sigma_0^2} \right) \mu - \left( \frac{n}{2\sigma^2} + \frac{1}{2\sigma_0^2} \right) \mu^2 \right\} = \exp\left\{ (\tau_Y \bar{Y} + \tau_0\mu_0)\mu - \frac{1}{2}(\tau_Y + \tau_0)\mu^2 \right\}, \qquad (20.18) \]

其中 \(\tau_Y = n\tau = n/\sigma^2\)，\(\tau_0 = 1/\sigma_0^2\)，因此 \(\tau_Y\) 是 \(\bar{Y}\) 的精度，\(\tau_0\) 是先验分布的精度。

可以看出 \(\log\{\pi(\mu|Y_1,\ldots,Y_n)\}\) 是 \(\mu\) 的二次函数，因此 \(\pi(\mu|Y_1,\ldots,Y_n)\) 是正态密度。因此，为了找到后验分布，我们只需要计算后验均值和方差。后验均值是使后验密度达到最大的 \(\mu\) 值，即后验众数。因此，为了计算后验均值，我们解

0 = \(\frac{\partial}{\partial \mu} \log\{\pi(\mu | Y_1, \ldots, Y_n)\}\) \(\qquad (20.19)\)

并得到均值为

\[ \mathrm{E}(\mu | Y_1, \ldots, Y_n) = \frac{\tau_Y \bar{Y} + \tau_0 \mu_0}{\tau_Y + \tau_0} = \frac{\frac{n \bar{Y}}{\sigma^2} + \frac{\mu_0}{\sigma_0^2}}{\frac{n}{\sigma^2} + \frac{1}{\sigma_0^2}}. \qquad (20.20)\]

由(A.10)可知，正态密度\(f(y)\)的精度等于\(-2\)乘以\(\log\{f(y)\}\)中\(y^2\)的系数。因此，后验精度为

\[ -2 \times \text{(20.18)中}\mu^2 \text{的系数} = \tau_Y + \tau_0 = \frac{n}{\sigma^2} + \frac{1}{\sigma_0^2}, \]

且后验方差为

\[ \mathrm{Var}(\mu | Y_1, \ldots, Y_n) = \frac{1}{\frac{n}{\sigma^2} + \frac{1}{\sigma_0^2}}. \qquad (20.21)\]

总结：后验分布为

\[ \mathrm{N}\left( \frac{\frac{n \bar{Y}}{\sigma^2} + \frac{\mu_0}{\sigma_0^2}}{\frac{n}{\sigma^2} + \frac{1}{\sigma_0^2}}, \frac{1}{\frac{n}{\sigma^2} + \frac{1}{\sigma_0^2}} \right) = \mathrm{N}\left( \frac{\tau_Y \bar{Y} + \tau_0 \mu_0}{\tau_Y + \tau_0}, \frac{1}{\tau_Y + \tau_0} \right). \qquad (20.22)\]

我们看到后验精度\((\tau_Y + \tau_0)\)是\(\bar{Y}\)的精度与先验精度之和；这是合理的，因为后验结合了数据中的信息和先验中的信息。

注意，当\(n \to \infty\)时，后验精度\(\tau_Y\)趋于无穷，后验分布近似为

\[ \mathrm{N}(\bar{Y}, \sigma^2 / n). \qquad (20.23)\]

这一结果告诉我们，随着数据量的增加，先验的影响变得可忽略。当\(\sigma_0 \to \infty\)（固定\(n\)）时，后验密度也收敛于(20.23)，即先验变得可忽略，因为先验精度降至零。

常见的贝叶斯估计量是后验均值，由(20.20)右边给出。许多统计学家既非坚定的贝叶斯派也非坚定的非贝叶斯派，他们倾向于从两个角度审视估计量。非贝叶斯派将通过检查后验均值的偏倚、方差和均方误差来分析它。我们将看到，通常贝叶斯估计量是有偏的，但其变异性小于\(\bar{Y}\)，且偏倚与方差之间的权衡由先验的选择控制。

为简化记号，令\(\hat{\mu}\)表示后验均值。则

\[ \hat{\mu} = \delta \bar{Y} + (1 - \delta) \mu_0, \qquad (20.24)\]

其中\(\delta = \tau_Y / (\tau_Y + \tau_0)\)，且\(\mathrm{E}(\hat{\mu} | \mu) = \delta \mu + (1 - \delta) \mu_0\)，所以\(\hat{\mu}\)的偏倚为

\[ \mathrm{E}(\hat{\mu} | \mu) - \mu = (\delta - 1)(\mu - \mu_0), \]

因此\(\hat{\mu}\)是有偏的，除非\(\delta = 1\)或\(\mu_0 = \mu\)。

只有当先验精度 \(\tau_0\) 收敛到 0 时，我们才有 \(\delta = 1\)。如果 \(\mu_0 = \mu\)，则意味着先验均值恰好等于真实参数，但显然这种有利情况无法安排，因为 \(\mu\) 未知。 \(\hat{\mu}\) 的方差为

\[ \operatorname{Var}(\hat{\mu}|\mu) = \frac{\delta^2 \sigma^2}{n}, \]

除了 \(\delta = 1\) 的极端情况外，该值小于 \(\operatorname{Var}(\bar{Y}) = \sigma^2/n\)。我们可以看到，\(\delta\) 越小，偏差越大，但方差越小。最佳的偏差-方差权衡是使 \(\hat{\mu}\) 的均方误差最小化，即

\[ \operatorname{MSE}(\hat{\mu}) = \operatorname{Bias}^2(\hat{\mu}) + \operatorname{Var}(\hat{\mu}) = (\delta - 1)^2 (\mu - \mu_0)^2 + \frac{\delta^2 \sigma^2}{n}. \tag{20.25} \]

当然，最好能让 \(\mu_0 = \mu\)，但由于 \(\mu\) 未知，这无法实现。已知的是 \(\delta = \tau_Y / (\tau_Y + \tau_0)\)，\(\delta\) 可以通过选择 \(\tau_0\) 来控制。图 20.3 显示了当 \(\sigma^2/n = 1/2\) 时，MSE 作为 \(\delta \in (0,1)\) 的函数，对应三个不同的 \(\mu - \mu_0\) 值，这个差值被称为“先验偏差”，因为它是参数真值与先验均值之差。对于两个较大的先验偏差，存在 \(\delta\) 的一个范围，在该范围内贝叶斯估计量的 MSE 小于 \(\bar{Y}\) 的 MSE，但如果 \(\delta\) 低于这个范围，则贝叶斯估计量的 MSE 大于 \(\bar{Y}\) 的 MSE，并且“良好”\(\delta\) 值的范围随着先验偏差的增大而缩小。如果先验偏差很大且 \(\delta\) 太小，则贝叶斯估计量的 MSE 可能相当大，因为当 \(\delta \to 0\) 时它收敛到先验偏差的平方；参见 (20.25) 或图 20.3。这一结果表明，要么需要对 \(\mu\) 有一个良好的先验猜测，要么保持先验精度较小以使 \(\delta\) 较大。然而，当 \(\delta\) 较大时，贝叶斯估计量相比 \(\bar{Y}\) 不会有太大改进，实际上当 \(\delta \to 1\) 时它收敛到 \(\bar{Y}\)。总之，选择一个能显著优于 \(\bar{Y}\) 的先验可能具有挑战性。一种方法是结合若干相关数据。

例20.5 估计正态精度时的后验分布

现在假设\(Y_1, \ldots, Y_n\)独立同分布，均值\(\mu\)已知，方差\(\sigma^2\)未知，精度\(\tau = 1 / \sigma^2\)。我们将证明\(\tau\)的共轭先验是伽马分布，并求出\(\tau\)的后验分布。定义\(s^2 = n^{-1} \sum_{i=1}^n (Y_i - \mu)^2\)，它是\(\sigma^2\)的MLE。简单的代数运算表明似然函数为

\[ f(Y_1, \ldots, Y_n | \tau) \propto \exp\left(-\frac{1}{2} n \tau s^2\right) \tau^{n/2}. \qquad (20.26) \]

设先验分布为伽马分布，形状参数为\(\alpha\)，尺度参数为\(b\)，其密度为

\[ \pi(\tau) = \frac{\tau^{\alpha-1}}{\Gamma(\alpha) b^\alpha} \exp(-\tau/b) \propto \tau^{\alpha-1} \exp(-\tau/b). \qquad (20.27) \]

将(20.26)与(20.27)相乘，可得\(\tau\)的后验密度为

\[ \pi(\tau | Y_1, \ldots, Y_n) \propto \tau^{n/2 + \alpha - 1} \exp\left\{-\left(ns^2/2 + b^{-1}\right) \tau\right\}, \qquad (20.28) \]

这表明后验分布是伽马分布，形状参数为\(n/2 + \alpha\)，尺度参数为\(\left(ns^2/2 + b^{-1}\right)^{-1}\)，即

\[ \pi(\tau | Y_1, \ldots, Y_n) = \text{Gamma}\left\{ n/2 + \alpha,\; \left(ns^2/2 + b^{-1}\right)^{-1} \right\}. \qquad (20.29) \]

我们已证明伽马分布是正态精度参数的共轭先验。

伽马分布的期望等于形状参数与尺度参数的乘积，因此\(\tau\)的后验均值为

\[ E(\tau | Y_1, \ldots, Y_n) = \frac{\frac{n}{2} + \alpha}{\frac{ns^2}{2} + b^{-1}}. \]

注意到当\(n \to \infty\)时，\(E(\tau | Y_1, \ldots, Y_n)\)收敛于\(s^{-2}\)，这并不意外，因为\(\sigma^2\)的MLE是\(s^2\)，所以\(\tau\)的MLE是\(s^{-2}\)。□

20.6 后验区间

这个定理之所以重要，有以下几个原因。首先，与结果 5.1 的比较表明，贝叶斯估计量和 MLE 具有相同的大样本分布。特别地，我们看到对于大样本，先验的影响变得可以忽略，因为渐近分布不依赖于先验。此外，该定理揭示了置信区间与后验区间之间的联系，这一点将在下一节讨论。

该定理的假设之一是，随着样本量的增加，先验保持不变，因此最终几乎所有的信息都来自数据。先验信息越丰富，后验分布接近其渐近极限所需的样本量就越大。

20.6 后验区间

在例 20.2 中已经提到了贝叶斯后验区间，现在我们将更深入地讨论。

后验区间与置信区间具有不同的概率解释。置信区间理论将参数视为固定的，而区间视为随机的，因为它基于随机样本。因此，当我们说“置信区间包含真实参数的概率是……”时，所考虑的是区间的概率分布，而不是参数的。此外，该概率表达的是收集数据之前关于收集数据后会发生什么的可能性。例如，如果我们使用 95% 的置信度，那么得到其区间覆盖参数的样本的概率为 0.95。在收集数据并得知区间之后，非贝叶斯学派会说要么区间覆盖参数，要么不覆盖，因此区间覆盖参数的概率要么是 1，要么是 0，当然，我们不知道哪个是实际概率。

在贝叶斯后验区间理论中，情况正好相反。样本被认为是固定的，因为我们使用后验概率，即给定数据下的条件概率。因此，后验区间被视为一个固定量。但在贝叶斯统计中，参数被视为随机的。因此，当贝叶斯学派说“后验区间包含真实参数的概率是……”时，所考虑的概率分布是参数的后验分布。随机量是参数，区间是固定的，并且概率是在收集数据之后计算的。

尽管置信区间和后验区间之间存在这些本质的哲学差异，但在许多同时构造了置信区间和后验区间的例子中，我们会发现它们几乎相等。尤其当先验相对于数据而言较为无信息时，这种情况更为常见，例如在例20.3中，如果\(\alpha+\beta\)远小于\(n\)。

基于中心极限定理，有坚实的理论原因说明为何在大样本下置信区间和后验区间几乎相等。根据结论20.6（后验的中心极限定理），\(\theta\)的第\(i\)个分量的大样本后验区间为 \[E(\theta_i|Y) \pm z_{\alpha/2} \sqrt{\operatorname{var}(\theta_i|Y)}. \tag{20.30}\] 根据结论5.1和7.6（关于MLE的一元和多元中心极限定理），基于MLE的大样本置信区间(5.20)和大样本后验区间(20.30)会随着样本量的增加而相互接近。因此，注重实用的非贝叶斯数据分析师通常乐于使用后验区间，并将其解释为置信区间的大样本近似。除了简单问题外，所有置信区间都基于大样本近似。这适用于使用剖面似然、关于MLE和Fisher信息量的中心极限定理或自助法的置信区间，换句话说，适用于构造置信区间的所有主要方法。

后验区间主要有两种类型：最高概率区间和等尾区间。令\(\psi = \psi(\theta)\)为参数向量\(\theta\)的标量函数，\(\pi(\psi|Y)\)为\(\psi\)的后验密度。最高概率区间是形如\(\{\psi: \pi(\psi|Y) > k\}\)的集合，其中\(k\)为某个常数。随着\(k\)从0增加到\(\infty\)，该区间的后验概率从1递减到0，选择\(k\)使得概率为\(1-\alpha\)。如果\(\pi(\psi|Y)\)有多个众数，那么集合\(\{\psi: \pi(\psi|Y) > k\}\)可能不是一个区间，此时应称为后验集或后验区域而不是后验区间。无论如何，该区域被解释为具有\(1-\alpha\)后验概率的最小集合。当最高后验区域是一个区间时，可以通过计算所有从\(\pi(\psi|Y)\)的\(\alpha_1\)下分位数到\(\alpha_2\)上分位数（其中\(\alpha_1+\alpha_2=\alpha\)）的区间，然后选择其中最短的区间来得到。

等尾后验区间的下限和上限等于\(\pi(\psi|Y)\)的\(\alpha/2\)下分位数和\(\alpha/2\)上分位数。当\(\pi(\psi|Y)\)对称且单峰时，这两类区间重合，根据后验的中心极限定理，对于大样本至少近似成立。

使用蒙特卡洛方法时，两种类型的后验区间都很容易计算；参见第20.7.3节。

例20.7. 方差已知时正态均值的后验区间

本示例继续例20.4。由(20.20)和(20.21)，对于μ的(1−α)100%后验区间为 \[ \frac{\tau_Y \bar{Y} + \tau_0 \mu_0}{\tau_Y + \tau_0} \pm z_{\alpha/2} \sqrt{\frac{1}{\sigma^2/n} + \frac{1}{\sigma_0^2}} \] 其中\(z_{\alpha/2}\)为标准正态分布的α/2上分位数。

如果\(n \to \infty\)或\(\sigma_0 \to \infty\)，则先验信息相对于数据信息变得可忽略，因为\(\tau_Y / \tau_0 \to \infty\)，后验区间收敛到 \[ \bar{Y} \pm z_{\alpha/2} \frac{\sigma}{\sqrt{n}}, \] 即通常的非贝叶斯置信区间。□

尽管前面几节简单例子中的贝叶斯计算很直接，但对于实际问题通常并非如此。后验密度(20.5)分母中的积分通常无法解析计算，后验均值(20.6)分子中的积分也是如此。由于计算困难，直到大约1990年，贝叶斯数据分析的使用远不如现在广泛。幸运的是，已经开发出用于近似后验密度和后验期望的蒙特卡洛模拟方法。这些方法取得了巨大进步，不仅使贝叶斯方法变得实用，还解决了许多以前无法处理的应用问题。

20.7 马尔可夫链蒙特卡洛

贝叶斯分析中最广泛适用的蒙特卡洛方法模拟一条马尔可夫链，其平稳分布即为后验分布。该链的样本用于贝叶斯推断。这种技术称为马尔可夫链蒙特卡洛（MCMC）。BUGS语言实现了MCMC。有三个广泛使用的BUGS版本：OpenBUGS、WinBUGS和JAGS。大多数BUGS程序可以在所有三个版本上运行，但也有例外。JAGS在某种程度上是三者中最通用的，因为它是唯一可以在MacOS下运行的版本。2

20.7.1 Gibbs抽样

Gibbs抽样是最简单的MCMC方法。假设参数向量\(\theta\)可以划分为\(M\)个子向量，使得

\[ \theta = \begin{pmatrix} \theta_1 \\ \ldots \\ \theta_M \end{pmatrix}. \]

令\([\theta_j | Y, \theta_k, k \neq j]\)为给定数据\(Y\)和其他子向量值条件下\(\theta_j\)的条件分布；\([\theta_j | Y, \theta_k, k \neq j]\)称为\(\theta_j\)的全条件分布。如果能够从每个全条件分布中抽样，那么Gibbs抽样是可行的。

Gibbs抽样通过重复抽样子向量\(\theta_1, \ldots, \theta_M\)来构造一条马尔可夫链，具体方式如下。链从参数向量\(\theta\)的任意起始值\(\theta^{(0)}\)开始。然后，子向量\(\theta_1^{(1)}\)从全条件分布\([\theta_1 | Y, \theta_k, k \neq 1]\)中抽取，其中其余子向量\(\theta_k, k \neq 1\)设定为其当前值\(\theta_k^{(0)}\)。接下来，\(\theta_2^{(1)}\)从\([\theta_2 | Y, \theta_k, k \neq 2]\)中抽取，其中\(\theta_k, k \neq 2\)设定为其当前值，即对于\(k=1\)为\(\theta_1^{(1)}\)，对于\(k \geq 2\)为\(\theta_k^{(0)}\)。如此继续，直到每个\(\theta_1, \ldots, \theta_M\)都得到更新，得到\(\theta^{(1)} = (\theta_1^{(1)}, \ldots, \theta_M^{(1)})^T\)。

然后，从\(\theta^{(1)}\)出发，以与从\(\theta^{(0)}\)获得\(\theta^{(1)}\)相同的方式得到\(\theta^{(2)}\)。如此继续，我们得到序列\(\theta^{(1)}, \ldots, \theta^{(N)}\)，这是一条马尔可夫链，其显著性质是其平稳分布就是\(\theta\)的后验分布。此外，无论起始值\(\theta^{(0)}\)如何，链都将收敛到平稳分布。在收敛到平稳分布后，马尔可夫链对后验分布进行抽样，MCMC样本用于计算后验期望、分位数以及后验分布的其他特征。

由于Gibbs样本并非从平稳分布开始，因此前\(N_0\)次迭代被丢弃作为burn-in期（预烧期），\(N_0\)的值需适当选择。我们假定已完成此步骤，\(\theta^{(1)}, \ldots, \theta^{(N)}\)是burn-in期之后的链样本。在第20.7.5节中，将讨论选择\(N_0\)的方法。

例20.8. 正态均值与精度的Gibbs抽样

20.7.3 MCMC 输出的分析

MCMC 输出的分析通常检查参数向量 \(\boldsymbol{\theta}\) 的标量值函数。应对每个感兴趣的标量量进行分析。令 \(\psi = \psi(\boldsymbol{\theta})\) 为这样的一个函数。假设 \(\boldsymbol{\theta}_1, \ldots, \boldsymbol{\theta}_N\) 是从 \(\boldsymbol{\theta}\) 的后验分布中抽取的 MCMC 样本，可以是单个马尔可夫链或多个链的组合，并定义 \(\psi_i = \psi(\boldsymbol{\theta}_i)\)。我们假设预热期和链长度足够，使得 \(\psi_1, \ldots, \psi_N\) 是 \(\psi\) 的后验分布的一个代表性样本。用于诊断收敛性和蒙特卡罗样本量充分性的方法在第 20.7.5 节中解释。

MCMC 样本均值 \(\bar{\psi} = N^{-1} \sum_{i=1}^N \psi_i\) 估计后验期望 \(E(\psi|Y)\)，这是最常见的贝叶斯估计量。MCMC 样本标准差 \(s_{\psi} = \{(N-1)^{-1} \sum_{i=1}^N (\psi_i - \bar{\psi})^2\}^{1/2}\) 估计 \(\psi\) 的后验标准差，并称为贝叶斯标准误。如果数据的样本量足够大，则根据结果20.6，后验分布近似正态，并且 \(\psi\) 的近似 \((1-\alpha)\) 后验区间为

\[ \bar{\psi} \pm z_{\alpha/2} s_{\psi}. \tag{20.32} \]

区间 (20.32) 是 (20.30) 的 MCMC 近似。

然而，我们无需使用这种正态近似来寻找后验区间。如果 \(L(\alpha_1)\) 是 \(\psi_1, \ldots, \psi_N\) 的 \(\alpha_1\) 下样本分位数，\(U(\alpha_2)\) 是 \(\alpha_2\) 上样本分位数，那么 \([L(\alpha_1), U(\alpha_2)]\) 是一个 \(1 - (\alpha_1 + \alpha_2)\) 后验区间。对于等尾后验区间，取 \(\alpha_1 = \alpha_2 = \alpha/2\)。对于最高后验密度区间，我们在一个精细网格上选择 \(\alpha_1\) 和 \(\alpha_2\)，使得 \(\alpha_1 + \alpha_2 = \alpha\) 且 \(U(\alpha_2) - L(\alpha_1)\) 最小化。应使用核密度估计检查 \(\psi\) 的后验密度是否为单峰。如果存在多个峰且它们之间的谷足够深，那么最高后验密度后验区域可能是若干区间的并集，而非单个区间。然而，即使在这种不太常见的情况下，\([L(\alpha_1), U(\alpha_2)]\) 仍可能被用作最短的 \(1-\alpha\) 后验区间。

核密度估计可用于可视化后验密度的形状。例如，参见前文示例20.9中讨论的图20.4。大多数核密度估计的自动带宽选择器基于独立样本的假设。当应用于MCMC输出时，它们可能会欠平滑。如果在R中使用 density() 函数，可以通过使用大于默认值1的 adjust 参数值来纠正这种欠平滑。然而，图20.4使用了默认值，平滑量似乎足够；这可能是由于蒙特卡罗样本量 \(N = 10,000\) 较大。

20.7.5 监控MCMC收敛和混合

燃烧期长度 \(N_0\) 必须足够大，以确保马尔可夫链在燃烧期结束时已收敛到平稳分布。燃烧期后链的长度 \(N\) 必须足够大，使得从MCMC样本计算的矩、分位数和其他量能够精确估计后验分布的对应特征。马尔可夫链是相关序列，MCMC中使用的链通常具有正自相关性。由于自相关，要达到精确估计，马尔可夫链样本必须比独立抽样所需的样本大得多，通常大得多。如果链在后验分布中移动缓慢，则称其混合不良。链的混合越慢，精确估计所需的样本量就越大。

原则上，只需要一条长马尔可夫链就能对后验分布进行抽样。然而，如果生成多条链，则可以比较它们以判断燃烧期长度 \(N_0\) 和链长度 \(N\) 是否足够大。如果链均值的链间变异相对于链内变异较大，则链混合不良。因此，收敛和混合的诊断可以基于链间和链内变异。

如果链的起始值相似，则链间变异性会人为地偏低。因此，建议起始值从比后验分布更分散的分布中随机抽样。例如，可以使用均值为MLE、协方差矩阵为Fisher信息逆矩阵的 \(k\) 倍（\(k>1\)，如 \(k=1.5\) 或 \(2\)）的高斯分布或 \(t\) 分布。

我们现在介绍两个广泛使用的诊断指标：Rhat 和 n.eff。假设我们抽取了 \(M\) 条链，每条链在 burn-in 后长度为 \(N\)。令 \(\theta_{i,j}\) 为第 \(j\) 条链的第 \(i\) 次迭代，并令 \(\psi_{i,j} = \psi(\theta_{i,j})\)，其中 \(\psi\) 是某个标量值函数。例如，要提取第 \(k\) 个参数，可使用 \(\psi(x)= x_k\)，或者 \(\psi\) 可能从精度计算标准差或方差。我们还用 \(\psi\) 表示估计量 \(\psi(\theta)\)。

令 \[ \bar{\psi}_{\cdot,j} = N^{-1} \sum_{i=1}^N \psi_{i,j} \qquad (20.33) \] 为第 \(j\) 条链的均值，并令 \[ \bar{\psi}_{\cdot,\cdot} = M^{-1} \sum_{j=1}^M \bar{\psi}_{\cdot,j}. \qquad (20.34) \] \(\bar{\psi}_{\cdot,\cdot}\) 是所有链均值的平均值，并且是 \(E(\psi|Y)\) 的蒙特卡洛近似。然后定义 \[ B = \frac{N}{M-1} \sum_{j=1}^M \left( \bar{\psi}_{\cdot,j} - \bar{\psi}_{\cdot,\cdot} \right)^2. \qquad (20.35) \] \(B/N\) 是链均值的样本方差。定义

设

\[s^2_j = (N-1)^{-1} \sum_{i=1}^N (\psi_{i,j} - \bar{\psi}_{\cdot,j})^2, \qquad (20.36)\]

为第 \(j\) 条链的方差，并定义

\[W = M^{-1} \sum_{j=1}^M s^2_j. \qquad (20.37)\]

\(W\) 是合并的链内方差。两个方差 \(B\) 和 \(W\) 组合成

\[\widehat{\operatorname{var}}^+(\psi|Y) = \frac{N-1}{N} W + \frac{1}{N} B, \qquad (20.38)\]

其中，如前所述，\(Y\) 是数据。

为了评估收敛性，可以使用

\[\hat{R} = \sqrt{\frac{\widehat{\operatorname{var}}^+(\psi|Y)}{W}}. \qquad (20.39)\]

在函数 gelman.diag() 的输出中，\(\hat{R}\) 被称为“潜在尺度缩减因子”（potential scale reduction factor）；见例20.9的输出。\(\hat{R}\) 也称为“收缩因子”或有时称为“Gelman收缩因子”。

当链尚未达到平稳分布时，根号内的分子 \(\widehat{\operatorname{var}}^+(\psi|Y)\) 是 \(\operatorname{var}(\psi|Y)\) 的有偏高估，分母 \(W\) 是该量的有偏低估。随着预热期和蒙特卡洛样本量的增加，两种偏倚都趋于0。因此，\(\hat{R}\) 的值越大，表明未收敛。如果 \(\hat{R}\) 近似等于1，例如最多1.1，则认为链已收敛到平稳分布，并且可以使用 \(\widehat{\operatorname{var}}^+(\psi|Y)\) 作为 \(\operatorname{var}(\psi|Y)\) 的估计。\(\hat{R}\) 的值较大表明需要更长的预热期。\(\hat{R}\) 的值较小是预热期充足的证据，但我们还需要另一个诊断量——有效样本量，以判断采样期是否足够长。

链的有效样本量为

\[N_{\text{eff}} = M N \frac{\widehat{\operatorname{var}}^+(\psi|Y)}{B}. \qquad (20.40)\]

\(N_{\text{eff}}\) 的解释是：马尔可夫链对 \(\psi\) 的后验期望的估计精度，大致相当于从后验分布中抽取一个大小为 \(N_{\text{eff}}\) 的独立样本所能达到的精度。（当然，通常不可能获得真正的独立样本，这正是使用MCMC的原因。）

\(N_{\text{eff}}\) 是通过比较马尔可夫链抽样下 \(\psi_{\cdot,\cdot}\) 的方差与假设独立抽样下 \(\psi_{\cdot,\cdot}\) 的方差推导得出的。由于

\(\psi_{\cdot,\cdot}\) 是 \(M\) 条独立链均值的平均值，且由于 \(B/N\) 是这 \(M\) 条链均值的样本方差，

\[ \frac{M-1}{N} B \tag{20.41} \]

估计了 \(\psi_{\cdot,\cdot}\) 的蒙特卡洛方差。假设不抽取 \(M\) 条长度为 \(N\) 的链，而是从后验中抽取一个大小为 \(N^*\) 的独立样本。该样本均值的蒙特卡洛方差为

\[ \frac{\operatorname{var}(\psi|Y)}{N^*}, \]

可用下式估计：

\[ \frac{\widehat{\operatorname{var}}_+(\psi|Y)}{N^*}. \tag{20.42} \]

根据定义，有效样本量 \(N_{\text{eff}}\) 是使 (20.41) 等于 (20.42) 的 \(N^*\) 值，因此 \(N^*\) 由 (20.40) 给出。由于 \(B/N\) 是 \(M\) 条链的样本方差，且 \(M\) 通常很小（常在 2 到 5 之间），\(B\) 具有相当大的蒙特卡洛变异性。因此，\(N_{\text{eff}}\) 仅是有效样本量的一个粗略估计。

\(\hat{R}\) 和 \(N_{\text{eff}}\) 由 coda 包中的函数 gelman.diag() 和 effectiveSize() 计算。该包中的函数 gelman.plot() 绘制了模拟过程中不同时刻的 \(\hat{R}\) 值。该函数的文档指出：“gelman.diag 的一个潜在问题是，如果收缩因子恰好接近 1，它可能误判收敛。通过计算多个时间点的收缩因子，gelman.plot 可以显示收缩因子是否真正收敛，还是仍在波动。” 图 20.6 显示了例 20.9 的 Gelman 图。红色虚线是 97.5% 的上置信限。尽管 \(\hat{R}\) 在模拟过程中变化，但到模拟结束时似乎已收敛到接近 1 的值。

\(N_{\text{eff}}\) 应该多大？当然，越大意味着蒙特卡洛精度越高，但较大的 \(N_{\text{eff}}\) 需要更多或更长的链，因此我们不希望 \(N_{\text{eff}}\) 不必要地大。\(N_{\text{eff}}\) 对估计误差的影响可以通过将估计误差 \(\psi - \psi_{\cdot,\cdot}\) 分解为两部分（称为 \(E_1\) 和 \(E_2\)）来看出：

\[ \psi - \psi_{\cdot,\cdot} = \{\psi - \operatorname{E}(\psi|Y)\} + \{\operatorname{E}(\psi|Y) - \psi_{\cdot,\cdot}\} = E_1 + E_2. \tag{20.43} \]

如果 \(\operatorname{E}\{\psi|Y\}\) 可以精确计算，从而用它而不是 \(\psi_{\cdot,\cdot}\) 作为 \(\psi\) 的估计量，那么 \(E_1\) 将是唯一的误差。\(E_2\) 是由于用 \(\psi_{\cdot,\cdot}\) 近似 \(\operatorname{E}\{\psi|Y\}\) 而引入的误差。两个误差 \(E_1\) 和 \(E_2\) 不相关，因此

\[ \operatorname{var}\{(\psi - \psi_{\cdot,\cdot})|Y\} = \operatorname{var}(E_1|Y) + \operatorname{var}(E_2|Y). \]

20.7.6 DIC与pD用于模型比较

在本节中，我们介绍两种广泛使用的统计量 DIC 和 pD。DIC 用于比较同一数据集上的多个模型，是 AIC 的贝叶斯类比。pD 是模型中参数数量的贝叶斯类比。

回顾第5.12节，偏差（现在用 \(D(Y,\theta)\) 表示）是负两倍的对数似然，而 AIC 由 (5.29) 定义为 \[ AIC = D(Y,\hat{\theta}_{ML}) + 2p, \qquad (20.45) \] 其中 \(\hat{\theta}_{ML}\) 是 MLE，\(p\) 是 \(\theta\) 的维度。MLE 的贝叶斯类比是后验均值，即常用的贝叶斯估计量，可以通过 MCMC 估计。

第20章 贝叶斯数据分析与MCMC

我们需要一个参数个数\(p\)的贝叶斯类比量。初看似乎奇怪，我们为什么不直接像非贝叶斯分析那样使用\(p\)本身？毕竟，仅仅因为我们有先验并使用贝叶斯估计，参数个数并没有改变。然而，贝叶斯分析中使用的先验信息在一定程度上约束了估计的参数，使得有效参数数目小于\(p\)。为了理解为什么如此，考虑一个例子：有\(d\)只股票的收益率被认为是相似的。假设收益率服从多元正态分布。我们关注\(d\)个预期收益率，记为\(\mu_1, \ldots, \mu_d\)。对于非贝叶斯而言，有两种方式对\(\mu_1, \ldots, \mu_d\)建模。第一种是假设它们均相等，设为\(\mu\)，那么只有一个参数来建模均值。另一种可能性是假设预期收益率不相等，从而有\(d\)个参数。

贝叶斯可以通过指定一个使\(\mu_1, \ldots, \mu_d\)相似但不相同的先验来实现这两种极端之间的折衷。例如，我们可以假设它们是独立同分布的\(N(\mu, \sigma_\mu^2)\)，其中\(\sigma_\mu^2\)将指定相似程度。认识到\(\sigma_\mu^2\)可以从数据中估计是重要的，也就是说，不需要事先指定均值之间的相似程度。使用这种先验信息的结果是，指定\(\mu_1, \ldots, \mu_d\)的有效参数数目大于1但小于\(d\)。

有效参数数目定义为

\[ p_D = \bar{D}_{\mathrm{avg}} - D(Y, \bar{\theta}), \tag{20.46} \]

其中

\[ \bar{\theta} = (NM)^{-1} \sum_{j=1}^{M} \sum_{i=1}^{N} \theta_{i,j} \]

是\(\theta_{i,j}\)的MCMC样本的平均值，并估计\(\theta\)的后验期望，而

\[ \bar{D}_{\mathrm{avg}} = (NM)^{-1} \sum_{j=1}^{M} \sum_{i=1}^{N} D(Y, \theta_{i,j}) \]

是\(D_{\mathrm{avg}}\)的MCMC估计，

\[ D_{\mathrm{avg}} = E\{D(Y, \theta) | Y\}. \tag{20.47} \]

与(20.45)类似，DIC定义为

\[ \text{DIC} = D(Y, \bar{\theta}) + 2 p_D. \]

由(20.46)，也可得

\[ \text{DIC} = \bar{D}_{\mathrm{avg}} + p_D. \]

函数 dic.samples()（在 rjags 包中）将 \(\hat{D}_{\text{avg}}\) 报告为“平均偏差”，\(p_D\) 报告为“惩罚”，DIC 报告为“惩罚偏差”。见例20.9的输出。

如下例所示，\(p_D\) 主要度量 \(\theta\) 的后验变异性，该变异性随 \(p\) 增加或关于 \(\theta\) 的先验信息相对于样本中信息量的减少而增加。

例20.11 当用已知精度估计正态均值时的 \(p_D\)

假设 \(Y = (Y_1, \ldots, Y_n)\) 独立同分布于 \(N(\mu, 1)\)，因此本例中 \(\theta = \mu\)。那么对数似然为

\[ \log L(\mu) = -\frac{1}{2} \sum_{i=1}^n (Y_i - \mu)^2 - \frac{n}{2} \log(2\pi) = -\frac{1}{2} \left\{ \sum_{i=1}^n (Y_i - \bar{Y})^2 + n(\bar{Y} - \mu)^2 \right\} - \frac{n}{2} \log(2\pi), \] 因此 \[ D(Y, \mu) = \sum_{i=1}^n (Y_i - \bar{Y})^2 + n(\bar{Y} - \mu)^2 + n \log(2\pi). \tag{20.48} \] 当计算 \(p_D\) 时，与 \(\mu\) 无关的量会在 (20.46) 的减法中抵消。因此，为计算 \(p_D\)，我们可以使用 \[ D(Y, \mu) = n(\bar{Y} - \mu)^2. \tag{20.49} \] 那么 \[ D\{Y, E(\mu|Y)\} = n\{\bar{Y} - E(\mu|Y)\}^2, \tag{20.50} \] 且 \[ \begin{aligned} D_{\text{avg}} &= n E\{(\bar{Y} - \mu)^2 | Y\} \\ &= n \left( \{\bar{Y} - E(\mu|Y)\}^2 + E[ \{E(\mu|Y) - \mu\}^2 | Y] \right) \\ &= n \left[ \{\bar{Y} - E(\mu|Y)\}^2 + \operatorname{Var}(\mu|Y) \right] \\ &= D\{Y, E(\mu|Y)\} + n \operatorname{Var}(\mu|Y), \tag{20.51} \end{aligned} \] 因为给定 \(Y\) 时，\(\{\bar{Y} - E(\mu|Y)\}\) 与 \(\{E(\mu|Y) - \mu\}\) 条件不相关。因此， \[ p_D = \hat{D}_{\text{avg}} - D\{Y, E(\mu|Y)\} \approx D_{\text{avg}} - D\{Y, E(\mu|Y)\} = n \operatorname{Var}(\mu|Y) = \frac{n}{n + \tau_0}. \tag{20.52} \]

最后一步用到了（20.21），其中 \(\tau_0\) 是 \(\mu\) 的先验精度。（20.52）中的近似（“≈”）在蒙特卡洛样本量 \(N\) 增至无穷时成为等式。

当 \(\tau_0 \to 0\) 时，先验信息量可忽略，（20.52）右侧收敛于 \(p=1\)。相反，当 \(\tau_0 \to \infty\) 时，先验信息量无界增长，（20.52）右侧收敛于 0。这是一个普遍现象的例子——先验信息越多，有效参数越少。□

一般而言，当 \(p\) 较小且先验信息较少时，\(p_D \approx p\)。在其他情况下，例如当 \(d\) 个均值被建模为来自同一个正态分布时，\(p_D\) 可能远小于 \(p\)——见例 20.12。

分层先验的常见情形是存在多个参数，它们被认为具有相似但不完全相同的值。例如，几种股票的期望收益率可能被认为是相似的。在这种情况下，汇集有关参数的信息以改进先验的设定是有用的，因为良好的先验信息会提高估计的精度。一种有效的汇集信息的方法是使用所谓的“分层先验”进行贝叶斯分析，它允许我们将估计值向彼此或向其他目标收缩。后者的一个例子是将收益率的样本协方差矩阵向 CAPM 或其他因子模型的估计值收缩。这种收缩可以在样本协方差矩阵的高变异性与因子模型（当因子模型不完全拟合时）的协方差矩阵估计量的潜在偏差之间取得权衡。

与之前一样，令似然为 \(f(y|\theta)\)。似然是层次结构中的第一层（或阶段）。到目前为止，在本章中，第二层即 \(\theta\) 的先验密度为 \(\pi(\theta|\gamma)\)，其中先验中的参数向量 \(\gamma\) 具有已知值，比如 \(\gamma_0\)。例如，在例 20.3 中，先验是参数固定的 Beta 分布。

在分层或多阶段先验中，\(\gamma\)是未知的，并拥有自己的先验\(\pi(\gamma|\delta)\)（第三层）。通常，\(\delta\)的值是已知的，尽管可以通过让\(\delta\)未知并赋予其自身的先验来增加更多的层次，以此类推。

通过示例可能最容易理解分层先验。

示例20.12. 估计中盘股的期望收益率

此示例使用midcapD.ts数据集。该数据集包含20只中盘股和市场的500个日收益率，曾在示例5.2中使用过。

该数据集将分为“训练数据”，包含前100天的收益率，以及“检验数据”，包含后400天的收益率。仅使用训练数据进行估计。检验数据将用于比较从训练数据得到的估计值。有意选择较大的检验数据样本量，以便我们可以将检验数据的平均收益率视为20只股票的“真实”期望收益率，尽管这当然只是一个近似。我们将使用训练数据来估计“真实”期望收益率。

我们将比较三种可能的真实期望收益率估计量：

  1. 样本均值（前100天中盘股的20个平均收益率）；
  2. 合并估计（完全收缩，每个期望收益率都有相同的估计值）；
  3. 使用分层先验的贝叶斯估计（收缩）。

方法(a)是“常规”的非贝叶斯估计量，其中每个期望收益率由该股票的样本均值估计。在方法(b)中，每个期望收益率都有相同的估计值，即“均值的均值”，也就是从(a)得到的20个均值的平均值。贝叶斯收缩（将在本例中解释）使用分层先验将20个个体均值向均值的均值收缩。贝叶斯收缩是(a)和(b)之间的折中。示例16.10中也使用了收缩，但在该示例中，收缩量是任意选择的，因为尚未介绍贝叶斯方法。

令\(R_{i,t}\)为第\(i\)只股票第\(t\)日的收益率，以百分比表示。对于贝叶斯收缩，第一层将是简单模型：

\[R_{i,t} = \mu_i + \varepsilon_{i,t},\]

其中\(\varepsilon_{i,t}\)独立同分布\(N(0, \sigma^2_\varepsilon)\)。

该模型有几个不现实的方面：(a) 假设\(\varepsilon_{i,t}\)的标准差不依赖于\(i\)；(b) 假设\(\varepsilon_{i,t}\)和\(\varepsilon_{i',t}\)独立（我们知道存在截面相关性）；(c) 假设不存在GARCH效应；(d) 假设\(\varepsilon_{i,t}\)服从正态分布，而不是

具有厚尾。尽管如此，出于估计期望收益率的目的，这个模型应该是足够的。记住，“所有模型都是错的，但有些是有用的”，当然，“有用”取决于分析的目标。

层次先验的第二层为：

\[ \mu_i \stackrel{i.i.d.}{\sim} N(\alpha, \sigma_\mu^2). \]

这里的假设是，这20只中盘股票的期望收益率是从一个大的期望收益率总体中抽样得到的，该总体可能包含所有中盘股，甚至更大的范围。该总体的均值为\(\alpha\)，标准差为\(\sigma_\mu\)。

如果我们使用非层次先验，那么就需要指定\(\alpha\)和\(\sigma_\mu\)的值。这正是例20.4中所做的，只不过在那个例子中\(\sigma_\epsilon^2\)也是已知的。我们可能对\(\alpha\)和\(\sigma_\mu\)的值有粗略了解，但不太可能掌握精确信息。从例20.4中我们看到，需要相当精确的先验指定才能使贝叶斯估计优于样本均值。事实上，如果先验选择不当，贝叶斯估计很容易劣于样本均值。

第三层将是\(\alpha\)和\(\sigma_\mu\)的先验，这将允许我们利用数据来估计这些参数。理解为什么在这个例子中我们可以估计\(\alpha\)和\(\sigma_\mu\)，而在例20.4中却无法做到这一点，非常重要。原因在于现在我们有了20个期望收益率（\(\mu_i\)），它们服从相同的均值\(\alpha\)和标准差\(\sigma_\mu\)。相比之下，例20.4中只有一个\(\mu\)，因此无法估计该\(\mu\)所来自的总体的均值和方差。

由于现在数据中包含关于\(\alpha\)、\(\sigma_\epsilon^2\)和\(\sigma_\mu^2\)的大量信息，我们可以对它们使用相当无信息的先验，以“让数据自己说话”。

20.9 协方差矩阵的贝叶斯估计

在本节中，我们假设\(Y_1,\ldots,Y_n\)是来自\(d\)维\(N(\mu,\Sigma)\)分布或\(d\)维\(t_\nu(\mu,\Lambda)\)分布的i.i.d.样本。我们将专注于多元正态分布的协方差矩阵\(\Sigma\)或多元\(t\)分布的尺度矩阵\(\Lambda\)的估计。精度矩阵定义为高斯分布和\(t\)分布中的\(\Sigma^{-1}\)或\(\Lambda^{-1}\)。这个定义类似于一元情况，其中精度定义为方差或平方尺度参数的倒数。我们将从高斯分布开始。

20.9.1 估计多元高斯协方差矩阵

在多元高斯情况下，精度矩阵\(\Sigma^{-1}\)的共轭先验是Wishart分布。Wishart分布，记为\(\text{Wishart}(\nu,A)\)，有一个一元参数\(\nu\)称为自由度，以及一个矩阵参数\(A\)，可以是任意非奇异协方差矩阵。当\(\nu\)为整数时，\(\text{Wishart}(\nu,A)\)分布有一个简单定义。令\(Z_i,\ldots,Z_n\)为i.i.d. \(N(\mu,A)\)。在这种情况下，\(\sum_{i=1}^n (Z_i-\mu)(Z_i-\mu)^T\)的分布是\(\text{Wishart}(n,A)\)。此外，\(\sum_{i=1}^n (Z_i-\bar{Z})(Z_i-\bar{Z})^T\) (20.53)的分布是\(\text{Wishart}(n-1,A)\)。由于(20.53)中的和是\(n-1\)倍的样本协方差矩阵，因此Wishart分布对于高斯分布协方差矩阵的推断很重要。

对于任意正值 \(\nu\)，Wishart\((\nu, \mathbf{A})\) 分布的密度为 \[ f(\mathbf{W}) = C(\nu, d) |\mathbf{A}|^{-\nu/2} |\mathbf{W}|^{(\nu-d-1)/2} \exp\left\{ -\frac{1}{2} \operatorname{tr}(\mathbf{A}^{-1}\mathbf{W}) \right\} \tag{20.54} \] 其中归一化常数为 \[ C(\nu, d) = \left\{ 2^{\nu d/2} \pi^{d(d-1)/4} \prod_{i=1}^{d} \Gamma\left( \frac{\nu+1-i}{2} \right) \right\}^{-1}. \] 参数 \(\mathbf{W}\) 是一个非奇异协方差矩阵。期望值为 \(\mathbb{E}(\mathbf{W}) = \nu \mathbf{A}\)。在一元情形 \((d=1)\) 下，Wishart 分布退化为 Gamma 分布。

如果 \(\mathbf{W}\) 服从 Wishart\((\nu, \mathbf{A})\) 分布，则 \(\mathbf{W}^{-1}\) 的分布称为逆 Wishart 分布，参数为 \(\nu\) 和 \(\mathbf{A}^{-1}\)，记为 Inv-Wishart\((\nu, \mathbf{A}^{-1})\)。

令 \(\mathbf{Y} = (\mathbf{Y}_1, \ldots, \mathbf{Y}_n)\) 表示数据。为了导出精度矩阵 \(\boldsymbol{\Sigma}^{-1}\) 的完全条件分布，我们假设 \(\boldsymbol{\mu}\) 已知。由 (7.15) 可知，似然函数为 \[ f(\mathbf{Y} | \boldsymbol{\Sigma}^{-1}) = \prod_{i=1}^{n} \left[ \frac{1}{(2\pi)^{d/2} |\boldsymbol{\Sigma}|^{1/2}} \exp\left\{ -\frac{1}{2} (\mathbf{Y}_i - \boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1} (\mathbf{Y}_i - \boldsymbol{\mu}) \right\} \right]. \] 经过一些简化，得到 \[ f(\mathbf{Y} | \boldsymbol{\Sigma}^{-1}) \propto |\boldsymbol{\Sigma}^{-1}|^{n/2} \exp\left\{ -\frac{1}{2} \sum_{i=1}^{n} (\mathbf{Y}_i - \boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1} (\mathbf{Y}_i - \boldsymbol{\mu}) \right\}. \] 定义 \[ \mathbf{S} = \sum_{i=1}^{n} (\mathbf{Y}_i - \boldsymbol{\mu})(\mathbf{Y}_i - \boldsymbol{\mu})^\top. \] 接下来 \[ \sum_{i=1}^{n} (\mathbf{Y}_i - \boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1} (\mathbf{Y}_i - \boldsymbol{\mu}) = \operatorname{tr}\left\{ \sum_{i=1}^{n} (\mathbf{Y}_i - \boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1} (\mathbf{Y}_i - \boldsymbol{\mu}) \right\} = \operatorname{tr}(\boldsymbol{\Sigma}^{-1} \mathbf{S}). \tag{20.55} \] (20.55) 中的第一个等式是平凡的：标量也是 \(1\times 1\) 矩阵，并等于其迹。第二个等式利用了对于任意矩阵 \(\mathbf{B}\) 和 \(\mathbf{A}\)，若乘积 \(\mathbf{B}\mathbf{A}\) 和 \(\mathbf{A}\mathbf{B}\) 有意义，则 \(\operatorname{tr}(\mathbf{A}\mathbf{B}) = \operatorname{tr}(\mathbf{B}\mathbf{A})\) 这一结果。由此可得 \[ f(\mathbf{Y} | \boldsymbol{\Sigma}^{-1}) \propto |\boldsymbol{\Sigma}^{-1}|^{n/2} \exp\left\{ -\frac{1}{2} \operatorname{tr}(\boldsymbol{\Sigma}^{-1} \mathbf{S}) \right\}. \tag{20.56} \]

假设精度矩阵 \(\boldsymbol{\Sigma}^{-1}\) 的先验为 Wishart\((\nu_0, \boldsymbol{\Sigma}_0^{-1})\)。那么先验密度为

\[ \pi(\Sigma^{-1}) \propto |\Sigma^{-1}|^{(\nu_0-d-1)/2} \exp\left\{ -\frac{1}{2} \operatorname{tr}(\Sigma^{-1}\Sigma_0) \right\}. \tag{20.57} \]

由于后验密度正比于先验密度与似然函数的乘积，由 (20.56) 和 (20.57) 可得，后验密度为

\[ \pi(\Sigma^{-1} \mid Y) \propto |\Sigma^{-1}|^{(n+\nu_0-d-1)/2} \exp\left[ -\frac{1}{2} \operatorname{tr}\left\{ \Sigma^{-1}(S+\Sigma_0) \right\} \right]. \tag{20.58} \]

因此，\(\Sigma^{-1}\) 的后验分布为 Wishart \(\{ n+\nu_0,\, (S+\Sigma_0)^{-1} \}\)。

后验期望为

\[ \mathbb{E}(\Sigma^{-1} \mid Y) = (n+\nu_0) \left\{ (S+\Sigma_0)^{-1} \right\}. \tag{20.59} \]

若 \(\nu_0\) 和 \(\Sigma_0\) 均较小，则

\[ \mathbb{E}(\Sigma^{-1} \mid Y) \approx n S^{-1}. \tag{20.60} \]

\(\Sigma\) 的 MLE 为 \(n^{-1}S\)，因此 \(\Sigma^{-1}\) 的 MLE 为 \(n S^{-1}\)。所以，对于较小的 \(\nu_0\) 和 \(\Sigma_0\)，\(\Sigma^{-1}\) 的贝叶斯估计量接近 MLE。

\(\Sigma^{-1}\) 的全条件分布可以与 \(\mu\) 的模型相结合，以估计两个参数。对于资产收益率的应用，可能会使用如例 20.12 中的 \(\mu\) 的层次先验。无论哪种情况，MCMC 分析都将直接进行。

20.9.3 协变量矩阵的非Wishart先验

20.10 随机波动率模型

我们在例20.13中看到，采用非信息性先验参数的Wishart先验基本上重复了极大似然估计。然而，通常我们希望将协方差矩阵向某个目标收缩，也许是来自因子模型的估计。参见例20.16。

20.10 随机波动率模型

随机波动率模型是GARCH模型的一种替代，用于对条件异方差性进行建模。在第14章的ARIMA/GARCH模型中，存在一个单一的白噪声过程驱动条件均值和条件方差。相比之下，随机波动率模型使用一个白噪声过程驱动条件期望，另一个驱动条件方差。因此，随机波动率模型的拟合更具挑战性，因为未观测的波动率过程由其自身的白噪声过程驱动，而该过程当然也是未观测的；参见下面的(20.63)。贝叶斯分析特别善于处理未观测变量，并且似乎是拟合随机波动率模型的最佳方法。

我们将通过以下模型来说明随机波动率模型：

\[Y_t = \mu + \sum_{j=1}^{k} \beta_j X_{j,t} + a_t, \qquad (20.61)\]

其中\(Y_t\)是观测到的过程，例如某项资产的收益率，\(X_{j,t}, j=1,\ldots,k\)是时刻\(t\)的第\(j\)个协变量，可以是\(Y_t\)的滞后值。此外，\(a_t\)是具有条件异方差性的弱白噪声过程。具体地，

\[a_t = \sqrt{h_t} \epsilon_t \qquad (20.62)\]

其中\(\log(h_t)\)服从ARMA(\(p,q\))过程：

\[\log(h_t) = \beta_0 + \sum_{j=1}^{p} \phi_j \log(h_{t-j}) + \sum_{j=1}^{q} \theta_j v_{t-j} + v_t, \qquad (20.63)\]

ϵt 和 vt 是相互独立的独立同分布白噪声，且 Var(ϵt)=1。注意，由 (20.62) 可知，\(\sqrt{h_t}\) 是 Yt 的条件标准差。如前所述，(20.63) 中的变量均不可观测。

\[\mathrm{var}(\bar{Y}) = n^{-2} \sum_{i=1}^n \sum_{j=1}^n \mathrm{Cov}(Y_i, Y_j) = n^{-2} \sum_{i=1}^n \sum_{j=1}^n \gamma(i-j) = n^{-2} \left\{ n\gamma(0) + 2 \sum_{h=1}^{n-1} \gamma(h)(n-h) \right\} = \frac{\gamma(0)}{n} R_n,\]

其中

\[R_n = 1 + 2 \sum_{h=1}^{n-1} \rho(h) \left(1 - \frac{h}{n}\right).\]

如果 \(Y_1, Y_2, \dots, Y_n\) 是一个不相关过程（白噪声），则 \(R_n = 1\)，且 (20.65) 与 (7.13) 一致。

大多数由 MCMC 生成的平稳过程对所有 \(h\) 有 \(\rho(h) \ge 0\)，因此自相关会使 \(R_n\) 膨胀。这种膨胀可能非常严重。考虑平稳 AR(1) 过程 \(Y_n = \phi Y_{n-1} + \epsilon_i\) 的情况。AR(1) 过程通常能相当好地近似 MCMC 过程。对于 AR(1) 过程，我们可以近似 \(R_n\)：

\[R_n \approx 1 + 2 \sum_{h=1}^\infty \rho(h) = \left( \sum_{h=0}^\infty \phi^h - 1 \right) = \left( \frac{2}{1-\phi} - 1 \right) = 1 + \frac{\phi}{1-\phi},\]

其中我们使用了几何级数求和公式 (3.4)（\(T=\infty\)）。注意当 \(\phi \to 1\) 时，(20.66) 的右侧无界增大。

由恒等式 \(\sum_{i=1}^n (Y_i - \mu)^2 = \sum_{i=1}^n (Y_i - \bar{Y})^2 + n(\bar{Y} - \mu)^2\)，我们得到

\[E\left\{ \sum_{i=1}^n (Y_i - \bar{Y})^2 \right\} = \gamma(0)(n - R_n)\]

因为 \(\gamma(0) = E\{ (Y_i - \mu)^2 \}\) 且由定义 \(\gamma(0)R_n = E\{ n(\bar{Y} - \mu)^2 \}\)。因此，过程方差 \(\gamma(0)\) 的一个无偏估计是

\[\hat{\gamma}(0) = \frac{\sum_{i=1}^n (Y_i - \bar{Y})^2}{n - R_n}.\]

当过程不相关时 \(R_n = 1\)，(20.68) 的右侧就是样本方差 (A.7)。对于正自相关过程，\(R_n > 1\)，样本方差（使用 1 代替 \(R_n\)）会低估。要获得 \(\gamma(0)\) 的无偏估计，可以使用

\[\frac{\sum_{i=1}^n (Y_i - \bar{Y})^2 + \hat{\gamma}(0) R_n}{n},\]

其中 \(\hat{\gamma}^{(0)}R_n\) 是 \(\gamma^{(0)}R_n\) 的无偏估计量。估计 \(\gamma^{(0)}R_n\) 有若干方法，最简单的是使用过程的多个独立实现。设 \(Y_1,\ldots,Y_M\) 为 \(M\) 个独立过程实现的均值，并设 \(\bar{Y} = M^{-1}\sum_{j=1}^M Y_j\)。则

\[ \hat{\gamma}^{(0)}R_n = \frac{\sum_{j=1}^M (Y_j - \bar{Y})^2}{M-1} \tag{20.70} \]

是 \(\gamma^{(0)}R_n\) 的无偏估计量。第20.7.5节用于MCMC监测的统计量 \(\hat{\text{var}}^+(\psi|Y)\) 是 (20.68) 和 (20.70) 的一个特例。
