A.1 引言

假设读者已经熟悉本书所需概率、统计、矩阵代数及其他数学主题的基础知识，因此本附录的目标仅为提供快速回顾，并涵盖一些可能不熟悉的高级主题。

A.2 概率分布

A.2.1 累积分布函数

随机变量 \(Y\) 的累积分布函数（CDF）定义为

\[ F_Y(y) = P\{Y \leq y\}. \]

若 \(Y\) 具有概率密度函数 \(f_Y\)，则

\[ F_Y(y) = \int_{-\infty}^{y} f_Y(u)\,du. \]

许多 CDF 和 PDF 可通过计算机软件包计算，例如，R 中的 pnorm()、pt() 和 pbinom() 分别计算正态分布、t 分布和二项分布随机变量的 CDF。类似地，dnorm()、dt() 和 dbinom() 计算这些分布的概率密度函数。

A.2.2 分位数与百分位数

若随机变量\(Y\)的分布函数\(F(y)\)连续且严格递增，则其存在反函数\(F^{-1}\)。对0到1之间的每个\(q\)，\(F^{-1}(q)\)称为 \(q\)分位数或 \(100q\) 百分位数。

中位数是50%百分位数或0.5分位数。25%和75%百分位数（0.25和0.75分位数）称为第一和第三四分位数，中位数是第二四分位数。三个四分位数将连续随机变量的取值范围划分为概率相等的四个组。类似地，20%、40%、60%和80%百分位数称为五分位数，10%、20%、…、90%百分位数称为十分位数。

对于任意分布函数\(F\)（无论是否可逆），其伪逆定义为

\[ F^{-}(x) = \inf\{ y : F(y) \ge x \}. \]

这里的“inf”表示下确界，即集合的最大下界（参见附录A.5）。对于0到1之间的任意\(q\)，第\(q\)分位数定义为\(F^{-}(q)\)。若\(F\)可逆，则\(F^{-1} = F^{-}\)，因此该分位数定义与可逆分布函数的分位数定义一致。\(F^{-}\)常称为分位数函数。

有时将 \((1-\alpha)\) 分位数称为 \(\alpha\) 上分位数，以强调分位数之上的概率大小。类似地，分位数也可称为下分位数。

分位数在以下意义下“保持变换”。若\(Y\)是随机变量，其\(q\)分位数为\(y_q\)，\(g\)是严格递增函数，且\(X = g(Y)\)，则\(g(y_q)\)是\(X\)的\(q\)分位数；参见(A.5)。

A.2.3 对称性与众数

若概率密度函数\(f\)满足对所有\(y\)有\(f(\mu - y) = f(\mu + y)\)，则称其关于\(\mu\)对称。概率密度函数的众数是局部极大值，即存在\(\epsilon > 0\)使得当\(y - \epsilon < x < y\)或\(y < x < y + \epsilon\)时有\(f(y) > f(x)\)的值\(y\)。只有一个众数的概率密度函数称为单峰，两个众数称为双峰，两个或更多众数称为多峰。

A.2.4 分布的支撑集

离散分布的支撑集是所有具有正概率的\(y\)的集合。更一般地，若对任意\(\epsilon > 0\)，区间 \((y - \epsilon, y + \epsilon)\) 具有正概率，则称点\(y\)在分布的支撑集中。例如，正态分布的支撑集是 \((-\infty, \infty)\)，伽马分布或对数正态分布的支撑集是 \([0, \infty)\)，而二项分布 \(\text{Binomial}(n, p)\) 的支撑集是 \(\{0, 1, 2, \dots, n\}\)（假设 \(p \neq 0, 1\)）。1

A.3 期望值和方差何时存在？

随机变量的期望值可能是无穷大，或者根本不存在。另外，随机变量不一定有定义良好且有限的方差。为了理解这些事实，令 \(Y\) 为具有密度 \(f_Y\) 的随机变量。\(Y\) 的期望定义为

\[\int_{-\infty}^{\infty} y f_Y(y)\,dy\]

前提是该积分有定义。如果

\[\int_{-\infty}^0 y f_Y(y)\,dy = -\infty \quad \text{且} \quad \int_0^{\infty} y f_Y(y)\,dy = \infty,\tag{A.1}\]

则期望形式上为 \(-\infty + \infty\)，这是未定义的，因此期望不存在。如果 (A.1) 中的积分都有限，则 \(E(Y)\) 存在且等于这两个积分之和。期望可以存在但为无穷，因为如果

\[\int_{-\infty}^0 y f_Y(y)\,dy = -\infty \quad \text{且} \quad \int_0^{\infty} y f_Y(y)\,dy < \infty,\]

则 \(E(Y) = -\infty\)；如果

\[\int_{-\infty}^0 y f_Y(y)\,dy > -\infty \quad \text{且} \quad \int_0^{\infty} y f_Y(y)\,dy = \infty,\]

则 \(E(Y) = \infty\)。

如果 \(E(Y)\) 未定义或为无穷，则涉及 \(E(Y)\) 的方差也无法定义。如果 \(E(Y)\) 有定义且有限，则方差也有定义。方差有限当且仅当 \(E(Y^2) < \infty\)；否则方差为无穷。

有限期望和方差的不存在性对金融市场的建模具有重要意义，因为例如第14章讨论的流行 GARCH 模型不一定具有有限期望值和方差。此外，如第5章所示，能够很好地拟合股票收益率的 t 分布可能具有不存在的均值或方差。

有人可能会认为，任何来自金融市场的变量 \(Y\) 都是有界的，即存在常数 \(M < \infty\) 使得 \(P(|Y| \leq M) = 1\)。在这种情况下，(A.1) 中的积分都是有限的（实际上最多为 \(M\)），\(E(Y)\) 存在且有限。此外，\(E(Y^2) \leq M^2\)，因此 \(Y\) 的方差有限。那么，我们是否还需要担心期望值和方差是否存在且有限的数学细节呢？答案是应该担心。一个随机变量可能在绝对值上被一个非常大的常数 \(M\) 所界定，但如果 \(M\) 足够大，其行为可能非常类似于一个没有期望值、或期望值为无穷、或期望值有限但方差无穷的随机变量。这可以在模拟中看到。

A.4 单调函数

函数\(g\)是递增的，如果当\(x_1 < x_2\)时，有\(g(x_1) \leq g(x_2)\)；严格递增如果当\(x_1 < x_2\)时，有\(g(x_1) < g(x_2)\)。递减和严格递减的定义类似，而\(g\)是（严格）单调的，如果它是（严格）递增或（严格）递减的。

A.5 集合的最小值、最大值、下确界和上确界

集合的最小值和最大值分别是集合中最小的值和最大的值（如果存在的话）。例如，如果\(A = \{x : 0 \leq x \leq 1\}\)，则\(A\)的最小值和最大值分别为0和1。然而，并非所有集合都有最小值或最大值，例如，\(B = \{x : 0 < x < 1\}\)既没有最小值也没有最大值。每个集合都有下确界（inf）和上确界（sup）。集合\(C\)的下确界是小于等于\(C\)中所有元素的最大数。类似地，\(C\)的上确界是大于等于\(C\)中所有元素的最小数。刚刚定义的集合\(B\)的下确界为0，上确界为1。以下记号是标准的：\(\min(C)\)和\(\max(C)\)表示\(C\)的最小值和最大值（如果存在的话），而\(\inf(C)\)和\(\sup(C)\)表示下确界和上确界。

A.6 随机变量的函数

假设\(X\)是一个随机变量，其PDF为\(f_X(x)\)，并且\(Y = g(X)\)，其中\(g\)是严格递增函数。由于\(g\)严格递增，它存在逆函数，我们用\(h\)表示。那么\(Y\)也是随机变量，其CDF为 \[ F_Y(y) = P(Y \leq y) = P\{g(X) \leq y\} = P\{X \leq h(y)\} = F_X\{h(y)\}. \tag{A.2} \] 对(A.2)求导，我们得到\(Y\)的PDF： \[ f_Y(y) = f_X\{h(y)\} h'(y). \tag{A.3} \] 将类似的论证应用于\(g\)严格递减的情形，可以证明，只要\(g\)是严格单调的，则有 \[ f_Y(y) = f_X\{h(y)\} |h'(y)|. \tag{A.4} \]

A.7 随机样本

同样地，根据(A.2)，当 \(g\) 严格递增时，有

\[F^{-1}_Y(p) = g\{F^{-1}_X(p)\}, \qquad (\text{A.5})\]

因此，\(Y\) 的第 \(p\) 分位数可通过将 \(g\) 应用于 \(X\) 的第 \(p\) 分位数得到。当 \(g\) 严格递减时，它将 \(X\) 的第 \(p\) 分位数映射到 \(Y\) 的第 \((1-p)\) 分位数。

结果 A.1 假设对于某些常数 \(a\) 和 \(b \neq 0\)，有 \(Y = a + bX\)。令 \(g(x) = a + bx\)，则 \(g\) 的反函数为 \(h(y) = (y - a)/b\)，且 \(h'(y) = 1/b\)。那么

\[F_Y(y) = F_X\{b^{-1}(y - a)\}, \quad b > 0,\] \[= 1 - F_X\{b^{-1}(y - a)\}, \quad b < 0,\] \[f_Y(y) = |b|^{-1} f_X\{b^{-1}(y - a)\},\]

并且

\[F^{-1}_Y(p) = a + bF^{-1}_X(p), \quad b > 0,\] \[= a + bF^{-1}_X(1 - p), \quad b < 0.\]

我们称 \(\{Y_1, \ldots, Y_n\}\) 为来自某概率分布的随机样本，若它们各自服从该分布且相互独立。此时，我们称它们是独立同分布的，简称为 i.i.d.。该概率分布通常称为总体，其期望、方差、累积分布函数和分位数分别称为总体均值、总体方差、总体累积分布函数和总体分位数。值得一提的是，总体实际上是无限的。存在一种从有限总体中（通常无放回）抽样的统计理论，但这种抽样方式不在我们讨论范围之内。即使总体是有限的（例如，调查房价时），总体通常也足够大，以至于可以视为无限。

若 \(Y_1, \ldots, Y_n\) 是来自未知概率分布的一个样本，则总体均值可由样本均值估计：

\[\bar{Y} = n^{-1} \sum_{i=1}^n Y_i, \qquad (\text{A.6})\]

总体方差可由样本方差估计：

\[s_Y^2 = \frac{\sum_{i=1}^n (Y_i - \bar{Y})^2}{n-1}. \qquad (\text{A.7})\]

分母为 \(n-1\) 而非 \(n\) 的原因将在第5.9节讨论。样本标准差是 \(s_Y\)，即 \(s_Y^2\) 的平方根。

A.8 二项分布

假设我们进行 \(n\) 次实验，其中 \(n\) 是固定的（非随机）整数。每次实验有两种可能的结果，称为“成功”和“失败”；成功的概率为 \(p\)，失败的概率为 \(q = 1-p\)。假设 \(p\) 和 \(q\) 对所有 \(n\) 次实验都相同。令 \(Y\) 为成功总次数，因此 \(Y\) 可以等于 \(0,1,2,\dots,n\)。如果实验是独立的，则

\[P(Y = k) = \binom{n}{k} p^k q^{n-k}, \quad k = 0,1,2,\dots,n,\]

其中 \(\displaystyle \binom{n}{k} = \frac{n!}{k!(n-k)!}\)。

\(Y\) 的分布称为二项分布，记为 \(\text{Binomial}(n,p)\)。\(Y\) 的期望值为 \(np\)，方差为 \(npq\)。\(\text{Binomial}(1,p)\) 分布也称为伯努利分布，其概率函数为

\[P(Y = y) = p^y (1-p)^{1-y}, \quad y = 0,1. \qquad (A.8)\]

注意，\(p^y\) 等于 \(p\)（当 \(y=1\) 时）或 \(1\)（当 \(y=0\) 时），\((1-p)^{1-y}\) 类似。

函数 pbinom()、dbinom()、qbinom() 和 rbinom() 分别计算二项分布的 CDF、PDF、分位数和随机数。例如，

> pbinom(3,6,0.5)
[1] 0.65625
from scipy.stats import binom
prob = binom.cdf(3, 6, 0.5)
print(prob)

表明公平硬币抛掷 6 次得到 3 次或更少正面的概率为 0.65625。

A.9 一些常见的连续分布

A.9.1 均匀分布

区间 \((a,b)\) 上的均匀分布记为 \(\text{Uniform}(a,b)\)，其 PDF 在 \((a,b)\) 上等于 \(1/(b-a)\)，在此区间外等于 0。容易验证，若 \(Y \sim \text{Uniform}(a,b)\)，则其期望为

\[E(Y) = \frac{1}{b-a} \int_a^b Y \, dY = \frac{a+b}{2},\]

即区间的中点。此外，

\[E(Y^2) = \frac{1}{b-a} \int_a^b Y^2 \, dY = \left. \frac{Y^3}{3} \right|_a^b \cdot \frac{1}{b-a} = \frac{b^2 + ab + a^2}{3}.\]

重新参数化是指用一组等价的参数替换分布的参数。均匀分布可以通过使用\(\mu = (a+b)/2\)和\(\sigma = (b-a)/\sqrt{12}\)作为参数来重新参数化。然后\(\mu\)是位置参数，\(\sigma\)是尺度参数。使用哪种参数化取决于我们希望强调分布的哪些方面。均匀分布的参数化\((a,b)\)指定了其端点，而参数化\((\mu,\sigma)\)给出了均值和标准差。我们可以在两种或多种参数化之间自由切换，使用在给定上下文中最有用的那种。均匀分布没有形状参数，因为其密度函数的形状总是矩形的。

A.9.2 通过分布函数和逆分布函数的变换

如果\(Y\)具有连续分布函数\(F\)，那么\(F(Y)\)服从Uniform(0,1)分布。\(F(Y)\)通常称为\(Y\)的概率变换。如果\(F\)严格递增，则这个事实很容易看出，因为此时\(F^{-1}\)存在，所以

\[ P\{F(Y) \le y\} = P\{Y \le F^{-1}(y)\} = F\{F^{-1}(y)\} = y. \tag{A.9} \]

即使\(F\)不是严格递增的，该结论也成立，但证明稍微复杂一些。只需\(F\)连续即可。

如果\(U\)服从Uniform(0,1)且\(F\)是一个分布函数，那么\(Y = F^{-}(U)\)以\(F\)为其分布函数。这里\(F^{-}\)是\(F\)的伪逆。当\(F\)连续且严格递增时，这很容易证明，因为此时\(F^{-1} = F^{-}\)且

\[ P(Y \le y) = P(F^{-1}(U) \le y) = P(Y \le F(y)) = F(y). \]

实际上，该结论对任意分布函数\(F\)都成立，但在一般情况下证明更困难。\(F^{-}(U)\)通常称为分位数变换，因为\(F^{-}\)是分位数函数。

A.9.3 正态分布

标准正态分布具有熟悉的钟形密度：

\[\phi(y) = \frac{1}{\sqrt{2\pi}} \exp\left( -y^2/2 \right), \quad -\infty < y < \infty.\]

标准正态分布的均值为0，方差为1。如果\(Z\)服从标准正态分布，那么\(\mu + \sigma Z\)的分布称为均值为\(\mu\)、方差为\(\sigma^2\)的正态分布，记为\(N(\mu, \sigma^2)\)。根据结果A.1，\(N(\mu, \sigma^2)\)的密度为：

\[\frac{1}{\sigma} \phi\left( \frac{y - \mu}{\sigma} \right) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left\{ -\frac{(y - \mu)^2}{2\sigma^2} \right\}. \tag{A.10}\]

参数\(\mu\)是位置参数，\(\sigma\)是尺度参数。正态分布没有形状参数，因为其密度始终是相同的钟形曲线2。标准正态分布的CDF为：

\[\Phi(y) = \int_{-\infty}^{y} \phi(u) \, du.\]

可以使用软件（如R的pnorm函数）计算\(\Phi\)。如果\(Y \sim N(\mu, \sigma^2)\)，那么由于\(Y = \mu + \sigma Z\)，其中\(Z\)为标准正态分布，根据结果A.1，

\[F_Y(y) = \Phi\{ (y - \mu) / \sigma \}. \tag{A.11}\]

正态分布也以其伟大德国数学家卡尔·弗里德里希·高斯的名字称为高斯分布。

A.9.5 指数分布与双指数分布

尺度参数为 \(\theta > 0\) 的指数分布（记为 Exponential(\(\theta\))）的分布函数为

\[F(y) = 1 - e^{-y/\theta}, \quad y > 0.\]

Exponential(\(\theta\)) 的密度函数为

\[f(y) = \frac{e^{-y/\theta}}{\theta}, \qquad (A.12)\]

期望值为 \(\theta\)，标准差为 \(\theta\)。逆分布函数为

\[F^{-1}(y) = -\theta \log(1 - y), \quad 0 < y < 1.\]

对于指数分布，可使用函数 pexp()、dexp()、qexp() 和 rexp()。

均值为 \(\mu\)、尺度参数为 \(\theta\) 的双指数分布或 Laplace 分布的密度函数为

\[f(y) = \frac{e^{-|y - \mu|/\theta}}{2\theta}. \qquad (A.13)\]

若 \(Y\) 服从均值为 \(\mu\) 的双指数分布，则 \(|Y - \mu|\) 服从指数分布。双指数分布的标准差为 \(\sqrt{2}\theta\)。均值 \(\mu\) 是位置参数，\(\theta\) 是尺度参数。

A.9.6 伽马分布与逆伽马分布

尺度参数为 \(b > 0\)、形状参数为 \(\alpha > 0\) 的伽马分布的密度函数为

\[\frac{y^{\alpha - 1}}{\Gamma(\alpha) b^{\alpha}} \exp(-y/b),\]

其中 \(\Gamma\) 是第 5.5.2 节定义的伽马函数。该分布的均值、方差和偏度系数分别为 \(b\alpha\)、\(b^{2}\alpha\) 和 \(2\alpha^{-1/2}\)。图 A.2 展示了形状参数分别为 0.75、3/2 和 7/2、均值均为 1 的伽马密度曲线。

伽马分布常用 \(\beta = 1/b\) 进行参数化，此时密度函数为

\[\frac{\beta^{\alpha} y^{\alpha - 1}}{\Gamma(\alpha)} \exp(-\beta y).\]

A.10 正态分布抽样

常见情形是我们拥有来自正态分布的随机样本，并希望得到均值和方差的置信区间，或检验关于这些参数的假设。那么，以下分布非常重要，因为它们是许多常用置信区间和检验的基础。

A.10.1 卡方分布

假设 \(Z_1, \dots, Z_n\) 是独立同分布的标准正态变量 \(N(0,1)\)。则 \(Z_1^2 + \cdots + Z_n^2\) 的分布称为具有 \(n\) 个自由度的卡方分布。该分布的期望值为 \(n\)，方差为 \(2n\)。该分布的 \(\alpha\) 上侧分位数记为 \(\chi^2_{\alpha,n}\)，用于方差的检验和置信区间（见后文）。此外，如第5.11节所述，\(\chi^2_{\alpha,n}\) 也用于似然比检验。例如，\(\chi^2_{0.05,10}\) 为 18.31，可通过两种方式计算：

> qchisq(0.05, 10, lower.tail = FALSE)
[1] 18.31
> qchisq(0.95, 10)
[1] 18.31
from scipy.stats import chi2
print(chi2.ppf(0.95, 10))  # 对应 lower.tail = FALSE
print(chi2.ppf(0.95, 10))

到目前为止，自由度参数为整数值，但可以推广。具有 \(\nu\) 个自由度的卡方分布等于尺度参数为2、形状参数为 \(\nu/2\) 的伽马分布。因此，由于伽马分布的形状参数可为任意正值，卡方分布也可对任意正 \(\nu\) 定义为尺度参数为2、形状参数为 \(\nu/2\) 的伽马分布。

A.10.2 F分布

如果 \(U\) 和 \(W\) 独立且分别服从自由度为 \(n_1\) 和 \(n_2\) 的卡方分布，则

\[ \frac{U/n_1}{W/n_2} \]

的分布称为具有 \(n_1\) 和 \(n_2\) 个自由度的F分布。该分布的 \(\alpha\) 上侧分位数记为 \(F_{\alpha,n_1,n_2}\)。\(F_{\alpha,n_1,n_2}\) 用作回归中F检验的临界值。例如，\(F_{0.95,3,7}=4.347\)：

A.11 大数定律与样本均值的中心极限定理

假设\(\bar{Y}_n\)是独立同分布样本\(Y_1,\ldots,Y_n\)的均值。我们假设它们的共同期望值\(E(Y_1)\)存在且有限，记为\(\mu\)。大数定律指出 \[P(\bar{Y}_n \to \mu \text{当} n \to \infty) = 1.\] 因此，对于足够大的样本量，样本均值将接近总体均值。然而，还有更多结论。著名的中心极限定理（CLT）指出，如果\(Y_1,\ldots,Y_n\)的共同方差\(\sigma^2\)有限，则\(\bar{Y}_n\)的概率分布随着\(n\)趋于无穷而越来越接近正态分布。更精确地说，CLT指出 \[P\{\sqrt{n}(\bar{Y}_n - \mu) \leq y\} \to \Phi(y/\sigma) \text{当} n \to \infty, \quad \forall y. \tag{A.17}\] 换言之，对于大的\(n\)，\(\bar{Y}\)近似服从\(N(\mu, \sigma^2/n)\)。

学生们常常记错或误解CLT。一个常见的错误观念是认为大的总体近似正态分布。CLT对总体的分布没有任何说明；它仅仅是关于样本均值分布的陈述。此外，CLT并不假设总体很大；是样本量趋于无穷。假设抽样是有放回的，总体可以非常小，实际上，可能只有两个元素。

当\(Y_1,\ldots,Y_n\)的方差为无穷时，\(\bar{Y}_n\)的极限分布可能仍然存在，但会是一个非正态的稳定分布。

虽然CLT最初是针对样本均值发现的，但现在已知其他估计量在大样本下也近似服从正态分布。特别地，第5.9节的极大似然估计量和第9章讨论的最小二乘估计量也有中心极限定理。这一点非常重要，因为我们使用的大多数估计量都是极大似然估计量或最小二乘估计量。因此，如果我们有足够大的样本，我们可以假设这些估计量近似服从正态分布，并可以利用正态分布进行检验和构造置信区间。

A.12 二元分布

设\(f_{Y_1,Y_2}(y_1,y_2)\)是随机变量对\((Y_1,Y_2)\)的联合密度。那么，\(Y_1\)的边缘密度通过“积分掉”\(Y_2\)得到：\(f_{Y_1}(y_1) = \int f_{Y_1,Y_2}(y_1,y_2) dy_2\)，类似地，\(f_{Y_2}(y_2) = \int f_{Y_1,Y_2}(y_1,y_2) dy_1\)。

A.13 相关与协方差

给定\(Y_1\)的条件下\(Y_2\)的条件密度为

\[f_{Y_2|Y_1}(y_2|y_1)=\frac{f_{Y_1,Y_2}(y_1,y_2)}{f_{Y_1}(y_1)}. \tag{A.18}\]

方程 (A.18) 可重新排列，将\(Y_1\)和\(Y_2\)的联合密度表示为边缘密度与条件密度的乘积：

\[f_{Y_1,Y_2}(y_1,y_2)=f_{Y_1}(y_1)f_{Y_2|Y_1}(y_2|y_1)=f_{Y_2}(y_2)f_{Y_1|Y_2}(y_1|y_2). \tag{A.19}\]

给定\(Y_1\)时\(Y_2\)的条件期望正是使用\(f_{Y_2|Y_1}(y_2|y_1)\)计算的期望：

\[\E(Y_2|Y_1=y_1)=\int y_2 f_{Y_2|Y_1}(y_2|y_1)\,dy_2,\]

它当然是\(y_1\)的函数。给定\(Y_1\)时\(Y_2\)的条件方差为

\[\Var(Y_2|Y_1=y_1)=\int\{y_2-\E(Y_2|Y_1=y_1)\}^2 f_{Y_2|Y_1}(y_2|y_1)\,dy_2.\]

本书其他地方重要的一个公式是

\[f_{Y_1,\ldots,Y_n}(y_1,\ldots,y_n)=f_{Y_1}(y_1)f_{Y_2|Y_1}(y_2|y_1)\cdots f_{Y_n|Y_1,\ldots,Y_{n-1}}(y_n|y_1,\ldots,y_{n-1}), \tag{A.20}\]

这通过反复使用(A.19)得到。

边缘均值和方差与条件均值和方差通过下式关联：

\[\E(Y)=\E\{\E(Y|X)\} \tag{A.21}\]

和

\[\Var(Y)=\E\{\Var(Y|X)\}+\Var\{\E(Y|X)\}. \tag{A.22}\]

结果(A.21)有多个名称，尤其是迭代期望定律和塔规则。

另一个有用的公式是，如果\(Z\)是\(X\)的函数，则

\[\E(ZY|X)=Z\E(Y|X). \tag{A.23}\]

这里的想法是，给定\(X\)，\(Z\)是常数，可以提到条件期望外面。

A.13 相关与协方差

期望和方差总结了随机变量的个体行为。如果我们有两个随机变量\(X\)和\(Y\)，那么有一种方便的方法来总结它们的联合行为——相关和协方差正是为此而设。

684 附录A 概率、统计与代数基础知识

两个随机变量 \(X\) 和 \(Y\) 之间的协方差为

\[\operatorname{Cov}(X,Y)=\sigma_{XY}=E\big[\{X-E(X)\}\{Y-E(Y)\}\big].\]

记号 \(\operatorname{Cov}(X,Y)\) 和 \(\sigma_{XY}\) 可互换使用。若 \((X,Y)\) 是连续分布，利用 (A.36) 式，有

\[\sigma_{XY}=\int\{x-E(X)\}\{y-E(Y)\}f_{XY}(x,y)\,dx\,dy.\]

以下为有用公式：

\begin{align} \sigma_{XY}&=E(XY)-E(X)E(Y), \tag{A.24}\\ \sigma_{XY}&=E[\{X-E(X)\}Y], \tag{A.25}\\ \sigma_{XY}&=E[\{Y-E(Y)\}X], \tag{A.26}\\ \sigma_{XY}&=E(XY)\quad\text{若 }E(X)=0\text{ 或 }E(Y)=0. \tag{A.27} \end{align}

两个变量间的协方差度量它们之间的线性关联，但也受变异性的影响；在其他条件相同时，标准差较大的随机变量具有较大的协方差。相关性是在消除这种大小影响后的协方差，因此相关性是衡量两个随机变量相关程度（更准确地说，线性相关程度）的纯度量。\(X\) 与 \(Y\) 之间的Pearson 相关系数为

\[\operatorname{Corr}(X,Y)=\rho_{XY}=\sigma_{XY}/(\sigma_X\sigma_Y). \tag{A.28}\]

Pearson 相关系数有时简称为相关系数，但存在其他类型的相关系数；参见第 8.5 节。

给定一个二元样本 \(\{(X_i,Y_i)\}_{i=1}^n\)，样本协方差记为 \(s_{XY}\) 或 \(\hat{\sigma}_{XY}\)，其定义为

\[s_{XY}=\hat{\sigma}_{XY}=(n-1)^{-1}\sum_{i=1}^n (X_i-\bar{X})(Y_i-\bar{Y}), \tag{A.29}\]

其中 \(\bar{X}\) 和 \(\bar{Y}\) 是样本均值。通常因子 \((n-1)^{-1}\) 可替换为 \(n^{-1}\)，但此变化相对于 \(\hat{\sigma}_{XY}\) 的随机变异影响很小。样本相关系数为

\[\hat{\rho}_{XY}=r_{XY}=\frac{s_{XY}}{s_X s_Y}, \tag{A.30}\]

其中 \(s_X\) 和 \(s_Y\) 是样本标准差。

如果 \(X\) 和 \(Y\) 独立，则对于所有函数 \(g\) 和 \(h\)，有 \[ E\{g(X)h(Y)\} = E\{g(X)\}E\{h(Y)\}. \tag{A.31} \] 这一事实可用于证明如果 \(X\) 和 \(Y\) 独立，则 \(\sigma_{XY}=0\)，因此变量不相关。但逆命题不成立。例如，如果 \(X\) 在 \([-1,1]\) 上均匀分布且 \(Y = X^2\)，则简单计算表明 \(\sigma_{XY}=0\)，但这两个随机变量并不独立。关键在于 \(Y\) 与 \(X\) 相关，实际上完全由 \(X\) 决定，但关系高度非线性，而相关性度量的是线性关联。

不相关但相依的随机变量的另一个例子是二元 \(t\) 分布。对于该分布，即使相关系数为 0，两个变量也是相依的；参见第 7.6 节。

如果 \(E(Y|X)=0\)，则 \(Y\) 和 \(X\) 不相关，因为由迭代期望法则有 \(E(Y)=E\{E(Y|X)\}=0\)（公式 (A.32)），然后 \[ \operatorname{Cov}(Y,X)=E(YX)=E\{E(YX|X)\}=E\{XE(Y|X)\}=0 \tag{A.33} \] 这里使用了 (A.27)、再次应用迭代期望法则、(A.23)（取 \(Z=X\)）以及 (A.32)。

结果 (A.22) 有一个重要的解释。如果已知 \(X\) 且需要预测 \(Y\)，则 \(E(Y|X)\) 是最优预测，因为它最小化期望平方预测误差。如果使用最优预测，则预测误差为 \(Y-E(Y|X)\)，且 \(E\{Y-E(Y|X)\}^2\) 是期望平方预测误差。由迭代期望法则，后者等于 \[ E\{Y-E(Y|X)\}^2 = E\left( E\left[ \{Y-E(Y|X)\}^2 | X \right] \right) = E\{ \operatorname{Var}(Y|X) \}, \tag{A.34} \] 这是 (A.22) 右端的第一项。此外，(A.22) 中的第二项 \(\operatorname{Var}\{E(Y|X)\}\) 是最优预测的变异性，度量了 \(E(Y|X)\) 跟踪 \(Y\) 的程度——\(E(Y|X)\) 变化越大，跟踪效果越好。因此，跟踪能力与期望平方预测误差之和为常数 \(\operatorname{Var}(Y)\)——提高跟踪能力会降低期望平方预测误差。

通过考察最坏和最好的情况，我们可以获得一些见解。最坏情况是当 \(X\) 与 \(Y\) 独立时。此时，\(E(Y|X)=E(Y)\)，跟踪能力 \(\operatorname{Var}\{E(Y|X)\}=0\)，期望平方预测误差取最大值 \(\operatorname{Var}(Y)\)。最好情况是当 \(Y\) 是 \(X\) 的函数时，例如存在某个函数 \(g\) 使得 \(Y=g(X)\)。此时，\(E(Y|X)=g(X)=Y\)，预测误差为 0，跟踪能力为 \(\operatorname{Var}(Y)\)，达到其最大可能值。

A.13.1 正态分布：条件期望与方差

对于某些概率分布，条件期望和方差的计算可能很困难，但对于具有双变量正态分布的对 \((Y_1, Y_2)\) 来说，这是相当容易的。

对于双变量正态对，给定 \(Y_1\) 时 \(Y_2\) 的条件期望等于给定 \(Y_1\) 时 \(Y_2\) 的最佳线性预测器4:

\[E(Y_2|Y_1 = y_1) = E(Y_2) + \frac{\sigma_{Y_1, Y_2}}{\sigma_{Y_1}^2} \{y_1 - E(Y_1)\}.\]

因此，对于正态随机变量，最佳线性预测与最佳预测相同。此外，给定 \(Y_1\) 时 \(Y_2\) 的条件方差是期望的平方预测误差：

\[\operatorname{Var}(Y_2|Y_1 = y_1) = \sigma_{Y_2}^2 (1 - \rho_{Y_1, Y_2}^2). \tag{A.35}\]

一般来说，\(\operatorname{Var}(Y_2|Y_1 = y_1)\) 是 \(y_1\) 的函数，但我们在 (A.35) 中看到，在双变量正态分布的特殊情况下，\(\operatorname{Var}(Y_2|Y_1 = y_1)\) 是常数，即与 \(y_1\) 无关。

A.14 多元分布

多元分布是附录 A.12 中双变量分布的推广。随机向量是元素为随机变量的向量。连续分布随机变量的随机向量 \(\mathbf{Y} = (Y_1, \ldots, Y_d)\) 具有多元概率密度函数 \(f_{Y_1, \ldots, Y_d}(y_1, \ldots, y_d)\)，如果对于所有集合 \(A \subset \Re^d\)，

\[P\{(Y_1, \ldots, Y_d) \in A\} = \int_A \!\cdots\!\int f_{Y_1, \ldots, Y_d}(y_1, \ldots, y_d) \,dy_1 \cdots dy_d.\]

\(Y_j\) 的 PDF 是通过对 \(f_{Y_1, \ldots, Y_d}\) 积分其他变量得到：

\[f_{Y_j}(y_j) = \int_{y_1} \!\cdots\! \int_{y_{j-1}} \int_{y_{j+1}} \!\cdots\! \int_{y_d} f_{Y_1, \ldots, Y_d}(y_1, \ldots, y_d) \,dy_1 \cdots dy_{j-1} dy_{j+1} \cdots dy_d.\]

类似地，任何 \((Y_1, \ldots, Y_d)\) 子集的 PDF 是通过对 \(f_{Y_1, \ldots, Y_d}(y_1, \ldots, y_d)\) 积分其他变量得到。

函数 \(g(Y_1, \ldots, Y_d)\) 的期望由公式

\[E\{g(Y_1, \ldots, Y_d)\} = \int_{y_1} \!\cdots\! \int_{y_d} g(y_1, \ldots, y_d) f_{Y_1, \ldots, Y_d}(y_1, \ldots, y_d) \,dy_1 \cdots dy_d. \tag{A.36}\]

A.15 随机过程

离散时间随机过程是一个随机变量序列 \(\{Y_1, Y_2, Y_3, \ldots\}\)。\(Y_n\) 的分布称为其边际分布。如果 \(Y_{n+1}\) 的条件分布仅依赖于 \(Y_n\)（给定过去），则称该过程为 Markov 过程或 Markov 性过程。

A.16 估计

给定\(\{Y_1, Y_2, \dots, Y_n\}\)，\(Y_{n+1}\)的条件分布等于给定\(Y_n\)的条件分布，因此\(Y_{n+1}\)仅依赖于过程的之前值。第12.4节中的AR(1)过程是马尔可夫过程的一个简单例子。计算机模拟生成的过程将是马尔可夫的，如果仅使用\(Y_n\)和独立于\(\{Y_1, Y_2, \dots, Y_{n-1}\}\)的随机数来生成\(Y_{n+1}\)。一个重要的例子是马尔可夫链蒙特卡洛，这是第20.7节的主题。

如果对于所有\(n\)，当\(Y_n\)具有分布\(\pi\)时，\(Y_{n+1}\)也具有分布\(\pi\)，则分布\(\pi\)称为马尔可夫过程的平稳分布。

随机过程也可以具有连续时间参数。例如布朗运动和几何布朗运动，它们分别用于对连续时间中的对数价格和股票价格进行建模。

A.16.1 引言

统计推断的主要领域之一是从数据中估计未知参数，例如总体均值。估计量定义为观测数据的任何函数。关键问题是在许多可能的估计量中应该使用哪一个。如果\(\theta\)是未知参数，\(\hat{\theta}\)是估计量，那么\(E(\hat{\theta}) - \theta\)称为偏差，\(E\{\hat{\theta} - \theta\}^2\)称为均方误差（MSE）。我们寻求有效的估计量，即具有尽可能小的MSE（或其他不准确性度量）的估计量。通过简单的代数可以证明，MSE等于偏差平方加上方差，即 \[E\{\hat{\theta} - \theta\}^2 = \{E(\hat{\theta}) - \theta\}^2 + \text{Var}(\hat{\theta}), \qquad (A.43)\] 因此，一个有效的估计量将同时具有小偏差和小方差。偏差为零的估计量称为无偏估计量。然而，不一定必须使用无偏估计量——我们只希望偏差小，不一定精确为零。如果小偏差能带来方差的显著降低，我们应该愿意接受小偏差。

最流行的估计方法是最小二乘法（第9.2.1节）、极大似然估计（第5.9节和第5.14节）和贝叶斯估计（第20章）。

A.16.2 标准误

当从一个随机样本计算估计量时，它是一个随机变量，但这一事实通常不被初学者所理解。在初次接触统计估计时，学生往往不认为诸如样本均值这样的估计量是随机的。如果我们只有一个样本，那么样本均值看起来并不随机。然而，如果我们意识到观测到的……

由于估计量是一个随机变量，它具有期望和标准差。我们已经看到，其期望与参数之间的差异称为偏倚。估计量的标准差称为其标准误。如果该标准差的公式中存在未知参数，则可以用估计值替换。如果 \(\hat{\theta}\) 是 \(\theta\) 的一个估计量，那么 \(s_{\hat{\theta}}\) 表示其标准误，其中所有未知参数已替换为估计值。

例A.1. 均值的标准误

假设 \(Y_1, \ldots, Y_n\) 是独立同分布的，均值为 \(\mu\)，方差为 \(\sigma^2\)。那么，由(7.13)式可得，样本均值 \(\bar{Y}\) 的标准差为 \(\sigma/\sqrt{n}\)。因此，\(\sigma/\sqrt{n}\)，或者当 \(\sigma\) 未知时 \(s/\sqrt{n}\)（其中 \(s\) 是样本标准差），称为样本均值的标准误。也就是说，标准误是 \(\sigma/\sqrt{n}\) 或 \(s/\sqrt{n}\)，取决于 \(\sigma\) 是否已知。□

A.17 置信区间

与其通过单个数字估计未知参数，通常更好的做法是提供一个数值范围，以体现估计的不确定性。这样的范围称为区间估计。一种区间估计——贝叶斯可信区间——在第20章中介绍。另一种区间估计是置信区间。置信区间定义为满足以下要求：区间包含真实参数的概率是一个指定的值，称为置信系数。因此，例如，如果构造了大量独立的90%置信区间，那么大约90%的区间将包含参数。

A.17.1 均值的置信区间

如果 \(\bar{Y}\) 是来自正态总体的样本均值，那么

\[ \bar{Y} \pm t_{\alpha/2, n-1} s_{\bar{Y}} \qquad \text{(A.44)} \]

是一个具有 \((1-\alpha)\) 置信水平的置信区间。该置信区间在6.3.2节中推导。如果 \(\alpha=0.05\)（即0.95或95%置信水平），且 \(n\) 相当大，则 \(t_{\alpha/2, n-1}\) 近似为2，因此 \(\bar{Y} \pm 2 s_{\bar{Y}}\) 常被用作近似95%置信区间。由于 \(s_{\bar{Y}} = s / \sqrt{n}\)，置信区间也可以写为 \(\bar{Y} \pm 2 s / \sqrt{n}\)。当 \(n\) 相当大（比如20或更多）时，根据中心极限定理，\(\bar{Y}\) 近似服从正态分布，因此可以去掉总体本身为正态的假设。

示例 A.2. 正态均值置信区间

\[ 16.1 \pm (2.797)\frac{\sqrt{2.7}}{\sqrt{25}} = 16.1 \pm 0.919 = [15.18, 17.02]. \]

A.17.2 方差和标准差的置信区间

正态分布方差的 \((1-\alpha)\) 置信区间为

\[ \left[ \frac{(n-1)s_Y^2}{\chi_{\alpha/2,\,n-1}^2},\ \frac{(n-1)s_Y^2}{\chi_{1-\alpha/2,\,n-1}^2} \right], \tag{A.45} \]

其中 \(n\) 是样本量，\(s_Y^2\) 是公式 (A.7) 给出的样本方差，且如附录 A.10.1 所定义，\(\chi_{\gamma,\,n-1}^2\) 是具有 \(n-1\) 个自由度的卡方分布的 \((1-\gamma)\) 分位数。

A.18 假设检验

A.18.1 假设、错误类型与拒绝域

统计假设检验利用数据判断一个陈述（称为原假设）是否为真。原假设的否定称为备择假设。例如，假设 \(Y_1, \ldots, Y_n\) 是来自 \(N(\mu, 1)\) 的独立同分布样本，且 \(\mu\) 未知。原假设可以设定为 \(\mu = 1\)。我们记 \(H_0: \mu = 1\) 和 \(H_1: \mu \neq 1\) 来表示原假设和备择假设。

我们希望能够避免两种错误。如果原假设为真但被我们拒绝，则犯了第一类错误。反之，如果原假设为假却被我们接受，则犯了第二类错误。

拒绝域是导致我们拒绝 \(H_0\) 的可能样本集合。例如，假设 \(\mu_0\) 是 \(\mu\) 的一个假设值，原假设为 \(H_0: \mu = \mu_0\)，备择假设为 \(H_1: \mu \neq \mu_0\)。如果 \(|\bar{Y} - \mu_0|\) 超过一个适当选择的临界值 \(c\)，则拒绝 \(H_0\)。拒绝域的选择使得第一类错误的概率低于一个预先设定的小值，称为显著性水平，通常记为 \(\alpha\)。实践中常用的 \(\alpha\) 值为0.01、0.05或0.1。随着 \(\alpha\) 变小，拒绝域也必须变小。在该例中，由于当 \(|\bar{Y} - \mu_0|\) 超过 \(c\) 时拒绝原假设，临界值 \(c\) 随着 \(\alpha\) 变小而变大。\(c\) 的值很容易确定。假设 \(\sigma\) 已知，则 \(c = z_{\alpha/2} \sigma / \sqrt{n}\)，其中 \(z_{\alpha/2}\) 是标准正态分布的 \(\alpha/2\) 上侧分位数，如附录A.9.3所定义。如果 \(\sigma\) 未知，则用 \(s_X\) 代替 \(\sigma\)，并用 \(t_{\alpha/2, n-1}\) 代替 \(z_{\alpha/2}\)，其中 \(t_{\alpha/2, n-1}\) 是自由度为 \(n-1\) 的 \(t\) 分布的 \(\alpha/2\) 上侧分位数，如第5.5.2节所定义。使用 \(t\) 分位数的检验称为单样本 \(t\) 检验。

A.18.2 p值

与其指定 \(\alpha\) 并在该 \(\alpha\) 下决定接受还是拒绝原假设，我们或许会问：“在哪些 \(\alpha\) 值下我们拒绝原假设？”样本的p值定义为最小的 \(\alpha\) 值，使得

如果某样本的p值为0.033，则当使用α=0.05或0.1时拒绝H₀，但当使用α=0.01时接受H₀。□

p值不仅告诉我们是否应接受或拒绝原假设，还告诉我们接受或拒绝H₀的决策是否属于边缘情况。例如，若使用α=0.05而p值为0.047，则拒绝H₀，但知道决策很接近；若p值为0.001，则决策远非如此。

进行假设检验时，统计软件会常规计算p值。这比要求用户指定α，然后报告在该α下是否拒绝原假设要方便得多。

A.18.3 双样本t检验

双样本t检验用于检验两个总体均值之差的假设。当从两个总体独立抽样时，使用独立样本t检验。令μᵢ、Ȳᵢ、sᵢ和nᵢ分别为第i个总体的均值、样本均值、样本标准差和样本容量，i=1,2。令Δ₀为μ₁−μ₂的假设值。我们假定两个总体具有相同的标准差，并通过合并标准差来估计该参数，合并标准差为

\[ s_{\text{pool}} = \left\{ \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1 + n_2 - 2} \right\}^{1/2}. \tag{A.46} \]

独立样本t统计量为

\[ t = \frac{\bar{Y}_1 - \bar{Y}_2 - \Delta_0}{s_{\text{pool}} \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}. \]

如果原假设为 H0: \(\mu_1 - \mu_2 = \Delta_0\)，备择假设为 H1: \(\mu_1 - \mu_2 \neq \Delta_0\)，则当 \(|t| > t_{\alpha/2,\, n_1+n_2-2}\) 时拒绝 H0。如果原假设为 H0: \(\mu_1 - \mu_2 \leq \Delta_0\)，备择假设为 H1: \(\mu_1 - \mu_2 > \Delta_0\)，则当 \(t > t_{\alpha,\, n_1+n_2-2}\) 时拒绝 H0；如果原假设为 H0: \(\mu_1 - \mu_2 \geq \Delta_0\)，备择假设为 H1: \(\mu_1 - \mu_2 < \Delta_0\)，则当 \(t < -t_{\alpha,\, n_1+n_2-2}\) 时拒绝 H0。

有时样本是配对的而非独立的。例如，假设我们想比较小盘股与大盘股5的收益率，并且对于每一年，我们拥有小盘股投资组合和大盘股投资组合的收益率。对于任意一年，两个投资组合的收益率是相关的，因此独立样本检验无效。令 \(d_i = X_{i,1} - X_{i,2}\) 为第 \(i\) 对来自总体 1 和总体 2 的观测值之差，并令 \(\bar{d}\) 和 \(s_d\) 分别为 \(d_1, \dots, d_n\) 的样本均值和样本标准差。配对样本 \(t\) 统计量为

\[ t = \frac{\bar{d} - \Delta_0}{s_d / \sqrt{n}}. \tag{A.47}\]

拒绝域与独立样本 \(t\) 检验相同，只是 \(t\) 分位数的自由度参数为 \(n-1\) 而不是 \(n_1 + n_2 - 2\)。

检验的功效是当 H1 为真时正确拒绝 H0 的概率。配对样本常用于获得更高的功效。在比较小盘股和大盘股的例子中，两个投资组合的收益率都有较高的年度变化，但 \(d_i\) 将不受这种变化的影响，因此 \(s_d\) 相对于 \(s_1\) 和 \(s_2\) 应该较小。数据的变异性较小意味着 \(\mu_1 - \mu_2\) 可以更精确地估计，并且该参数偏离 \(\Delta_0\) 的情况更有可能被检测到。

由于 \(\bar{d} = \bar{Y}_1 - \bar{Y}_2\)，(A.46) 和 (A.47) 的分子相等。区别在于分母。当配对观测值 \((Y_{i,1}, Y_{i,2})\) 之间的相关性为正时，(A.47) 的分母将小于 (A.46) 的分母。正是 (A.47) 分母较小使得配对 \(t\) 检验具有更高的功效。

假设某人拥有配对样本但错误地使用了独立样本 \(t\) 检验。如果 \(Y_{i,1}\) 和 \(Y_{i,2}\) 之间的相关性为零，则配对样本的行为与独立样本相同，使用错误检验的影响很小。假设相关性为正。使用错误检验的结果是，如果 H0 为假，则真实的 \(p\) 值会被高估，并且与使用配对样本检验相比，拒绝 H0 的可能性更低。然而，如果 \(p\) 值很小，那么我们可以自信地拒绝 H0，因为配对样本检验的 \(p\) 值会更小6。不幸的是，统计方法经常被使用

A.18.4 统计显著性 vs. 实际显著性

当我们拒绝原假设时，通常会说存在统计显著性效应。在此语境下，“显著性”一词容易被误解。它并不意味着存在实际重要性的效应。例如，假设我们检验两个总体均值相等的原假设，对立假设为它们不相等。统计显著性仅仅意味着两个样本均值差异足够大，以至于这种差异不能合理地归因于偶然。统计显著性并不意味着总体均值如此不同以至于它们的差异具有任何实际重要性。当使用大样本时，微小且不重要的效应很可能具有统计显著性。

在判断实际重要性时，置信区间比检验更有用。在比较两个总体均值的情况下，重要的是构建置信区间，并且仅当该区间内的所有差异都足够大以至于具有实际重要性时，才得出存在实际重要性效应的结论。多大算“足够大”不是一个统计问题，而必须由领域专家来回答。例如，假设两个总体均值之间的差异超过0.2被认为是重要的（至少对于所考虑的目的而言）。如果95%置信区间为[0.23,0.26]，那么我们有95%的置信度可以认为存在重要差异。如果区间为[0.13,0.16]，那么我们有95%的置信度可以认为不存在重要差异。如果置信区间为[0.1,0.3]，那么我们无法以95%的置信度说明差异是否重要。

A.19 预测

假设\(Y\)是一个当前未知的随机变量，例如利率或股票价格的未来变化。设\(X\)是一个已知的随机向量，对预测\(Y\)有用。例如，如果\(Y\)是股票价格或宏观经济变量的未来变化，\(X\)可能是该股票价格或宏观经济变量近期变化的向量。

我们希望找到一个关于\(X\)的函数，记为\(\hat{Y}(X)\)，它能最好地预测\(Y\)。这里的意思是使均方误差\(\mathbb{E}[\{Y-\hat{Y}(X)\}^2]\)尽可能小。使均方误差最小的函数\(\hat{Y}(X)\)称为基于\(X\)的\(Y\)的最佳预测。注意\(\hat{Y}(X)\)可以是\(X\)的任意函数，不一定是第11.9.1节中的线性函数。理论上，最佳预测很简单——它就是给定\(X\)时\(Y\)的条件期望。即，\(\mathbb{E}(Y|X)\)是在所有\(\hat{Y}(X)\)（\(X\)的任意函数）中使\(\mathbb{E}[\{Y-\hat{Y}(X)\}^2]\)最小的最佳预测。

如果\(Y\)和\(X\)独立，则\(E(Y|X)=E(Y)\)。如果\(X\)未被观测，则\(E(Y)\)将被用于预测\(Y\)。因此，当\(Y\)和\(X\)独立时，\(Y\)的最优预测值与\(X\)未知时相同，因为\(X\)不包含任何对预测\(Y\)有用的信息。

在实践中，使用\(E(Y|X)\)进行预测并非易事。问题在于\(E(Y|X)\)可能难以估计，而最优线性预测器可以通过第9章所述的线性回归进行估计。然而，较新的非参数回归技术可用于估计\(E(Y|X)\)。非参数回归在第21章中讨论。

A.20 向量与矩阵的基本事实

向量\(\mathbf{x} = (x_1, \dots, x_p)^T\)的范数为\(\|\mathbf{x}\| = \left(\sum_{i=1}^p x_i^2\right)^{1/2}\)。

方阵\(A\)是对角矩阵，如果对所有\(i \neq j\)有\(A_{i,j}=0\)。我们使用记号\(\operatorname{diag}(d_1, \dots, d_p)\)表示\(p \times p\)对角矩阵\(A\)，其中\(A_{i,i}=d_i\)。

若\(O^T = O^{-1}\)，则矩阵\(O\)是正交矩阵。这意味着\(O\)的列相互正交（垂直），且其范数都等于1。

任何对称矩阵\(\Sigma\)都有特征值-特征向量分解，简称为特征分解，即\[ \Sigma = O \operatorname{diag}(\lambda_i) O^T, \tag{A.50} \]其中\(O\)是正交矩阵，其列是\(\Sigma\)的特征向量，\(\lambda_1, \dots, \lambda_p\)是\(\Sigma\)的特征值。此外，如果所有\(\lambda_1, \dots, \lambda_p\)非零，则\(\Sigma\)非奇异，且\[ \Sigma^{-1} = O \operatorname{diag}(1/\lambda_i) O^T. \]

令\(\mathbf{o}_1, \dots, \mathbf{o}_p\)为\(O\)的列。那么，由于\(O\)是正交的，对于任何\(j \neq k\)，有\[ \mathbf{o}_j^T \mathbf{o}_k = 0. \tag{A.51} \]此外，对于\(j \neq k\)，有\[ \mathbf{o}_j^T \Sigma \mathbf{o}_k = 0. \tag{A.52} \]为了说明这一点，令\(\mathbf{e}_j\)为第\(j\)个单位向量，即在第\(j\)个坐标上为1、其余为0的向量。那么，\(\mathbf{o}_j^T O = \mathbf{e}_j^T\)且\(O^T \mathbf{o}_k = \mathbf{e}_k\)，因此对于\(j \neq k\)，\[ \mathbf{o}_j^T \Sigma \mathbf{o}_k = \mathbf{o}_j^T \left\{ O \operatorname{diag}(\lambda_i) O^T \right\} \mathbf{o}_k = \lambda_j \lambda_k \mathbf{e}_j^T \mathbf{e}_k = 0. \]

协方差矩阵的特征值-特征向量分解在第7.8节中用于寻找椭圆轮廓密度的方向。即使密度不是椭圆轮廓，这种分解也很重要，并且是主成分分析（PCA）的基础。
