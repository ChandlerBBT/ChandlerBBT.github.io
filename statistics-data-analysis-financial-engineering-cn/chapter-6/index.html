第6章 重抽样

6.1 引言

仅仅为一组统计模型参数找到单一估计值是不够的。还需要对这些估计值的不确定性进行评估。标准误和置信区间是表达不确定性的常用方法。1在过去，评估不确定性有时很困难，甚至不可能，尤其是对于复杂模型。幸运的是，现代计算机的速度以及受此速度启发的统计方法创新，在很大程度上克服了这个问题。本章我们将应用一种称为“自助法”或“重抽样”的计算机模拟技术来寻找标准误和置信区间。自助法适用范围非常广泛，将在本书余下部分中大量使用。

自助法是现代计算革新统计学的一个例子。马尔可夫链蒙特卡洛（MCMC）是另一个例子；见第20章。

术语“bootstrap”由Bradley Efron（1979）创造，源自短语“自力更生”（pulling oneself up by one's bootstraps）。

当统计量是从随机选择的样本中计算得出时，这些统计量就是随机变量。学生往往没有意识到这一事实。毕竟，关于\(\bar{Y}\)有什么随机可言？我们只是对数据求了平均，那有什么随机？关键在于，样本只是众多可能样本中的一个。每个可能的样本都给出不同的\(\bar{Y}\)值。因此，尽管我们只看到一个\(\bar{Y}\)值，但它是从众多可能值中随机选出的，因此\(\bar{Y}\)是一个随机变量。

统计推断的方法，如置信区间和假设检验，都是以统计量的随机性为前提的。例如，

置信区间的置信系数告诉我们，在随机样本抽取之前，由该样本构造的区间包含参数的概率。因此，根据大数定律，置信系数也是覆盖其参数的区间的长期频率。置信区间通常使用概率论推导。然而，在许多情况下，必要的概率计算是棘手的，此时我们可以用蒙特卡洛模拟替代理论计算。

但如何模拟从未知总体中抽样呢？答案当然是我们无法精确做到。然而，样本是总体的良好代表，我们可以通过从样本中抽样来模拟从总体中抽样，这称为重抽样。每个重样本的样本大小与原始样本相同，均为\(n\)。原因在于我们试图模拟原始抽样过程，因此希望重抽样尽可能与原始抽样相似。通过自助法近似，我们指的是用重抽样来近似抽样过程。

有两种基本的重抽样方法：无模型方法和基于模型的方法，分别也称为非参数方法和参数方法。在本章中，我们假设样本来自某个总体的独立同分布观测。对于相依数据，重抽样需要不同的技术，这将在第13.6节中讨论。

在无模型重抽样中，重样本是从原始样本中有放回抽取的。为什么要有放回？原因在于只有有放回抽样才能得到独立观测，而我们希望重样本与原始样本一样是独立同分布的。实际上，如果重样本是无放回抽取的，那么每个重样本将与原始样本完全相同，从而重样本不显示任何随机变异。这当然不太令人满意。

基于模型的重抽样不从原始样本中抽取样本。相反，它假设原始样本是从参数族密度\(\{f(y|\theta): \theta \in \Theta\}\)中独立同分布抽取的，因此对于未知的\(\theta\)，\(f(y|\theta)\)是总体密度。重样本是从密度\(f(y|\hat{\theta})\)中独立同分布抽取的，其中\(\hat{\theta}\)是参数向量\(\theta\)的某个估计。

重抽样的数量通常应该很大。具体多大取决于上下文，将在后面更详细地讨论。有时使用数千甚至数万个重样本。我们用\(B\)表示重样本的数量。

在阅读以下章节时，请记住：在重抽样中，原始样本扮演总体的角色，因为重样本是从原始样本中抽取的。因此，来自样本的估计扮演真实总体参数的角色。

6.2 Bootstrap 估计偏差、标准差和MSE

设 \(\theta\) 为一维参数，\(\hat{\theta}\) 为基于样本的估计量，\(\hat{\theta}^*_1, \ldots, \hat{\theta}^*_B\) 为基于 \(B\) 个重样本的估计量。令 \(\bar{\hat{\theta}}^*\) 表示 \(\hat{\theta}^*_1, \ldots, \hat{\theta}^*_B\) 的均值。星号表示从重样本中计算的统计量。

估计量 \(\hat{\theta}\) 的偏差定义为 \(\text{BIAS}(\hat{\theta}) = E(\hat{\theta}) - \theta\)。由于期望是总体均值，通过重样本取平均来估计，因此偏差的 bootstrap 估计量为

\[ \text{BIAS}_{\text{boot}}(\hat{\theta}) = \bar{\hat{\theta}}^* - \hat{\theta}. \tag{6.1} \]

注意，如前节最后一段所述，在偏差的 bootstrap 估计中，未知的总体参数 \(\theta\) 被替换为样本估计量 \(\hat{\theta}\)。\(\hat{\theta}\) 的 bootstrap 标准误是 \(\hat{\theta}^*_1, \ldots, \hat{\theta}^*_B\) 的样本标准差，即

\[ s_{\text{boot}}(\hat{\theta}) = \sqrt{\frac{1}{B-1} \sum_{b=1}^B (\hat{\theta}_b^* - \bar{\hat{\theta}}^*)^2}. \tag{6.2} \]

\(s_{\text{boot}}(\hat{\theta})\) 估计 \(\hat{\theta}\) 的标准差。

\(\hat{\theta}\) 的均方误差 (MSE) 为 \(E[(\hat{\theta} - \theta)^2]\)，其估计量为

\[ \text{MSE}_{\text{boot}}(\hat{\theta}) = \frac{1}{B} \sum_{b=1}^B (\hat{\theta}_b^* - \hat{\theta})^2. \]

与偏差估计类似，在估计 MSE 时，未知的 \(\theta\) 被替换为 \(\hat{\theta}\)。MSE 同时反映了偏差和变异性，事实上，

\[ \text{MSE}_{\text{boot}}(\hat{\theta}) \approx \text{BIAS}_{\text{boot}}^2(\hat{\theta}) + s_{\text{boot}}^2(\hat{\theta}). \tag{6.3} \]

如果在 (6.1) 的分母中使用 \(B\) 而不是 \(B-1\)，那么 (6.3) 将是等式而非近似。由于 \(B\) 通常较大，近似误差通常非常小。

6.2.1 Bootstrapping t 分布 MLE

计算 MLE 的函数（例如 R 中的 fitdistr()）通常同时计算 MLE 的标准误。这些标准误的理论依据是极大似然估计的“渐近”或“大样本”近似，称为 CLT（中心极限定理）2。该近似仅在样本大小趋于无穷时精确成立。

当样本量趋于无穷时，标准误的精度会增加。由于样本量总是有限的，无法确保标准误的准确性。通过自助法计算标准误可以作为大样本近似精度的一种检验，如下例所示。

6.3 Bootstrap 置信区间

令各概率相等。我们所说的置信区间的“准确性”，是指名义覆盖概率与实际覆盖概率之间的一致程度。即使是精确置信区间，如正态均值的 (A.44) 和正态方差的 (A.45)，也仅在数据完全满足假设（例如严格服从正态分布）时才是精确的。

6.3.1 正态近似区间

令 \(\hat{\theta}\) 为 \(\theta\) 的估计，\(s_{\text{boot}}(\hat{\theta})\) 为由 (6.2) 给出的标准误估计。则 \(\theta\) 的正态理论置信区间为

\[ \hat{\theta} \pm s_{\text{boot}}(\hat{\theta}) z_{\alpha/2}, \qquad (6.4) \]

其中 \(z_{\alpha/2}\) 是正态分布的 \(\alpha/2\) 上分位数。当 \(\hat{\theta}\) 为 MLE 时，该区间本质上与 (5.20) 相同，区别仅在于此处使用 Bootstrap 而非 Fisher 信息量来求标准误。

为避免混淆，需强调正态近似并不假定总体服从正态分布，而只假定 \(\hat{\theta}\) 由中心极限定理近似服从正态分布。

6.3.2 Bootstrap-t 区间

通常我们拥有 \(\hat{\theta}\) 的标准误（例如来自 Fisher 信息量）。此时可使用 Bootstrap-t 方法，与正态近似置信区间相比，该方法有望提供更精确的置信区间，即名义覆盖概率更接近实际覆盖概率。我们首先说明 Bootstrap-t 方法如何与正态总体均值的常规 t 置信区间相关联，然后讨论一般理论。

总体均值的置信区间

假设我们想基于随机样本构建总体均值的置信区间。我们首先使用所谓的“t 统计量”3：

\[ t = \frac{\bar{Y} - \mu}{s / \sqrt{n}}. \qquad (6.5) \]

\(t\) 的分母 \(s/\sqrt{n}\) 即为均值的标准误，因此分母估计了分子的标准差。

  1. 实际上，\(t\) 并不完全是统计量，因为它依赖于未知的 \(\mu\)，而统计量按定义仅依赖于样本，与未知参数无关。但“t 统计量”这一术语使用广泛，故此处沿用。 ↩

如果我们从正态分布总体中抽样，那么\(t\)的概率分布是自由度为\(n-1\)的\(t\)分布。使用第5.5.2节的符号，我们用\(t_{\alpha/2, n-1}\)表示\(\alpha/2\)上侧\(t\)值，即该分布的\(\alpha/2\)上分位数。因此，\(t\)在(6.5)中超过\(t_{\alpha/2, n-1}\)的概率为\(\alpha/2\)。由于\(t\)分布的对称性，\(t\)小于\(-t_{\alpha/2, n-1}\)的概率也是\(\alpha/2\)。因此，对于正态分布数据，有\(1-\alpha\)的概率满足

\[-t_{\alpha/2, n-1} \le t \le t_{\alpha/2, n-1}. \qquad (6.6)\]

将(6.5)代入(6.6)，经过一些代数运算，我们得到

\[1-\alpha = P\left\{ \bar{Y} - t_{\alpha/2, n-1} \frac{s}{\sqrt{n}} \le \mu \le \bar{Y} + t_{\alpha/2, n-1} \frac{s}{\sqrt{n}} \right\}, \qquad (6.7)\]

这表明

\[\bar{Y} \pm \frac{s}{\sqrt{n}} t_{\alpha/2, n-1}\]

是\(\mu\)的一个\(1-\alpha\)置信区间，假设数据服从正态分布。这就是式(A.44)给出的置信区间。注意在(6.7)中，随机变量是\(\bar{Y}\)和\(s\)，而\(\mu\)是固定的。

如果我们并非从正态分布总体中抽样，那该怎么办呢？此时，由(6.5)定义的\(t\)的分布并非\(t\)分布，而是某种我们未知的其他分布。这里存在两个问题。首先，我们不知道总体的分布。其次，即使总体分布已知，从总体分布推导\(t\)统计量的分布也是一项困难且通常棘手的概率计算。这一计算仅对正态总体完成过。

考虑到这两个问题的难度，我们还能得到置信区间吗？答案是“可以，通过重抽样。”

我们从原始样本中构造大量（例如\(B\)个）重样本。令\(Y_{\text{boot},b}\)和\(s_{\text{boot},b}\)分别为第\(b\)个重样本的样本均值和样本标准差，\(b = 1, \ldots, B\)，并令\(\bar{Y}\)为原始样本的均值。定义

\[t_{\text{boot},b} = \frac{\bar{Y} - Y_{\text{boot},b}}{s_{\text{boot},b}/\sqrt{n}}. \qquad (6.8)\]

注意\(t_{\text{boot},b}\)的定义方式与\(t\)相同，但有两处改动。第一，\(t\)中的\(\bar{Y}\)和\(s\)分别被替换为\(Y_{\text{boot},b}\)和\(s_{\text{boot},b}\)。第二，\(t\)中的\(\mu\)被替换为\(\bar{Y}\)。最后这一点有些微妙，它运用了第6.1节末尾陈述的原则——重样本是将原始样本作为总体进行抽样得到的。因此，对于重样本而言，总体均值就是\(\bar{Y}\)！

由于重样本之间相互独立，集合\(t_{\text{boot},1}, t_{\text{boot},2}, \ldots\)可以视为来自\(t\)统计量分布的一个随机样本。在计算了\(B\)个\(t_{\text{boot},b}\)值（每个重样本一个）之后，我们找出这些\(t_{\text{boot},b}\)值的\(\alpha/2\)下分位数和上分位数。将这些百分位数记为\(t_L\)和\(t_U\)。

6.3 自助法置信区间

如果原始总体是偏斜的，那么就没有理由认为α/2下分位数是α/2上分位数的相反数，就像t分布这样的对称总体那样。换言之，我们不一定期望\(t_L = -t_U\)，但这不会造成问题，因为自助法允许我们在不假设它们之间任何关系的情况下估计\(t_L\)和\(t_U\)。现在我们将置信区间(6.7)中的\(-t_{\alpha/2,n-1}\)和\(t_{\alpha/2,n-1}\)分别替换为\(t_L\)和\(t_U\)。最后，μ的自助法置信区间为：\[\left(\bar{Y} + t_L \frac{s}{\sqrt{n}},\; \bar{Y} + t_U \frac{s}{\sqrt{n}}\right).\]在(6.9)中，\(\bar{Y}\)和\(s\)是原始样本的均值和标准差，只有\(t_L\)和\(t_U\)是从\(B\)个自助法重样本中计算得到的。

自助法解决了上述两个问题。我们不需要知道总体分布，因为我们可以用样本来估计它。样本本身并不是概率分布。我们所做的是通过赋予样本中每个观测值概率\(1/n\)（其中\(n\)是样本大小）来创建一个概率分布，称为经验分布。此外，我们不需要使用概率论来计算t统计量的分布。相反，我们可以从经验分布进行模拟。

一般参数的置信区间

构造μ的t置信区间的方法可以推广到其他参数。设\(\hat{\theta}\)和\(s(\hat{\theta})\)是从样本计算得到的θ的估计值及其标准误。设\(\hat{\theta}^*_b\)和\(s_b(\hat{\theta})\)是来自第\(b\)个自助法样本的相同量。那么第\(b\)个自助法t统计量为：\[t_{\text{boot},b} = \frac{\hat{\theta} - \hat{\theta}^*_b}{s_b(\hat{\theta})}.\]如同估计总体均值时一样，设\(t_L\)和\(t_U\)是这些t统计量的α/2下和α/2上样本分位数。那么θ的置信区间为：\[\left(\hat{\theta} + t_L s(\hat{\theta}),\; \hat{\theta} + t_U s(\hat{\theta})\right)\]因为\[1 - \alpha \approx P\left\{ t_L \leq \frac{\hat{\theta} - \hat{\theta}^*_b}{s_b(\hat{\theta})} \leq t_U \right\} \approx P\left\{ t_L \leq \frac{\theta - \hat{\theta}}{s(\hat{\theta})} \leq t_U \right\} = P\left\{ \hat{\theta} + t_L s(\hat{\theta}) \leq \theta \leq \hat{\theta} + t_U s(\hat{\theta}) \right\}.\]

近似(6.11)的误差来自蒙特卡洛误差，可以通过选择较大的 \(B\) 来减小。近似(6.12)源于用经验分布对总体分布进行bootstrap近似。第二种近似的误差不依赖于 \(B\)，仅当样本量 \(n\) 较大时才变小。尽管通常无法控制样本量，但幸运的是，在金融工程中样本量往往较大。

6.3.3 基本Bootstrap区间

令 \(q_L\) 和 \(q_U\) 分别为 \(\hat{\theta}^*_1, \dots, \hat{\theta}^*_B\) 的 \(\alpha/2\) 下样本分位数和上样本分位数。满足 \[q_L \leq \hat{\theta}^*_b \leq q_U \tag{6.13}\] 的bootstrap估计的比例为 \(1-\alpha\)。但(6.13)在代数上等价于 \[\hat{\theta} - q_U \leq \hat{\theta} - \hat{\theta}^*_b \leq \hat{\theta} - q_L, \tag{6.14}\] 因此 \(\hat{\theta} - q_U\) 和 \(\hat{\theta} - q_L\) 是 \(\hat{\theta} - \hat{\theta}^*_b\) 分布的下分位数和上分位数。基本bootstrap区间将它们用作 \(\theta - \hat{\theta}\) 分布的下分位数和上分位数。利用bootstrap近似，我们假设 \[\hat{\theta} - q_U \leq \theta - \hat{\theta} \leq \hat{\theta} - q_L \tag{6.15}\] 会在比例为 \(1-\alpha\) 的样本中出现。将 \(\hat{\theta}\) 加到(6.15)的各项中，得到 \[2\hat{\theta} - q_U \leq \theta \leq 2\hat{\theta} - q_L,\] 因此 \[(2\hat{\theta} - q_U, 2\hat{\theta} - q_L) \tag{6.16}\] 是 \(\theta\) 的一个置信区间。区间(6.16)有时被称为基本bootstrap区间。

6.3.4 百分位置信区间

存在几种基于所谓百分位方法的bootstrap置信区间。这里仅详细介绍基本百分位区间。

如6.3.3节所述，令 \(q_L\) 和 \(q_U\) 分别为 \(\hat{\theta}^*_1, \dots, \hat{\theta}^*_B\) 的 \(\alpha/2\) 下样本分位数和上样本分位数。基本百分位置信区间即为 \[(q_L, q_U). \tag{6.17}\] 根据(6.13)，此区间内 \(\hat{\theta}^*_b\) 值的比例为 \(1-\alpha\)。该区间可以基于假设 \(\hat{\theta}^*\) 关于 \(\hat{\theta}\) 对称分布来证明。该假设意味着存在某个 \(C>0\)，使得 \(q_L = \hat{\theta} - C\) 且 \(q_U = \hat{\theta} + C\)。

6.3 自助法置信区间

于是 \(2\hat{\theta} - q_U = q_L\) 且 \(2\hat{\theta} - q_L = q_U\)，因此基本自助法区间 (6.16) 与基本百分位区间 (6.17) 重合。

如果 \(\hat{\theta}^*\) 不关于 \(\hat{\theta}\) 对称分布呢？幸运的是，并非毫无办法。如第4.6节所述，随机变量通常可以通过变换获得对称分布。因此，现在我们仅假设存在某个单调递增函数 \(g\)，使得 \(g(\hat{\theta}^*)\) 关于 \(g(\hat{\theta})\) 对称分布。下面我们将看到，这个较弱的假设足以证明基本百分位区间的合理性。由于 \(g\) 是严格单调递增的，且分位数关于变换保持一致性4，因此 \(g(q_L)\) 和 \(g(q_U)\) 分别是 \(g(\hat{\theta}^*_1), \ldots, g(\hat{\theta}^*_B)\) 的下和上 \(\alpha/2\) 分位数，从而 \(g(\theta)\) 的基本百分位置信区间为

\[\{g(q_L), g(q_U)\}.\]

现在，如果 (6.18) 对 \(g(\theta)\) 具有覆盖概率 \((1-\alpha)\)，那么由于 \(g\) 是单调递增的，(6.17) 对 \(\theta\) 也具有覆盖概率 \((1-\alpha)\)。这证明了百分位区间的合理性，至少在我们愿意假设存在一个对称化变换的前提下。注意，我们仅假设这样的 \(g\) 存在，而非已知。我们不需要知道 \(g\) 的具体形式，因为构造百分位区间时并未用到 \(g\)。

基本百分位方法简单，但精度不高，除非样本量很大。百分位方法存在两个问题。第一个是假设无偏性。基本百分位区间不仅假设 \(g(\hat{\theta}^*)\) 对称分布，还假设其对称中心为 \(g(\hat{\theta})\) 而不是 \(g(\hat{\theta})\) 加上某个偏倚。大多数估计量满足中心极限定理（CLT），例如第4.3.1节的样本分位数 CLT 和第5.10节的 MLE CLT。因此，在样本足够大时偏倚可以忽略，但在实际中样本量可能不够大，偏倚会导致名义覆盖概率与实际覆盖概率产生差异。

第二个问题是 \(\hat{\theta}\) 可能具有非恒定方差，这一问题称为异方差性。如果 \(\hat{\theta}\) 是 MLE，那么 \(\hat{\theta}\) 的方差至少近似为 Fisher 信息量的逆，而 Fisher 信息量通常依赖于 \(\theta\)，并非恒定。

更高级的百分位方法可以校正偏倚和异方差性。BCa 和 ABC（近似自助法置信）百分位区间是常用改进的百分位区间。名称“BCa”中，“BC”表示“偏倚校正”，“a”表示“加速”，指的是方差随 \(\theta\) 变化的速率。BCa 方法自动估计偏倚和方差变化率，并做出适当调整。BCa 和 ABC 区间背后的理论超出了本书范围，但在第6.4节给出的参考文献中有讨论。BCa 和 ABC 方法已在统计软件（如 R）中实现。在 R 的 bootstrap 包中，函数 bcanon()、abcpar() 和 abcnon() 分别实现了非参数 BCa 区间、参数 ABC 区间和非参数 ABC 区间。

例6.3. 基于分位数的尾部权重参数的置信区间

在第5.8节中提到，量化尾部权重的分位数参数可以定义为两个尺度参数的比值：

\[ s(p_1,1-p_1) / s(p_2,1-p_2) \qquad (6.19) \]

其中 \(s(p_1,p_2) = [F^{-1}(p_2) - F^{-1}(p_1)] / a\)，\(a\) 是一个正常数，不影响比值 (6.19)（因此可以忽略），且 \(0 < p_1 < p_2 < 1/2\)。我们将 (6.19) 称为 quKurt。在没有自助法的情况下，为 quKurt 构造置信区间可能是一项艰巨的任务，但借助自助法则非常简单。本示例将使用 BCa 置信区间为 quKurt 进行推断。参数由以下 R 函数从样本 y 计算得到，默认值为 \(p_1 = 0.025\) 和 \(p_2 = 0.25\)：

quKurt = function(y, p1 = 0.025, p2 = 0.25)
{
   Q = quantile(y, c(p1, p2, 1 - p2 , 1 - p1))
   (Q[4] - Q[1]) / (Q[3] - Q[2])
}
def quKurt(y, p1=0.025, p2=0.25):
    # R的quantile默认type=7，numpy.percentile默认线性插值，等效
    Q = np.percentile(y, [p1*100, p2*100, (1-p2)*100, (1-p1)*100])
    return (Q[3] - Q[0]) / (Q[2] - Q[1])

问题1 自由度 \(\nu\) 的极大似然估计是多少？具有该 \(\nu\) 值的 t 分布是否有 限的偏度和峰度？

由于基于四阶中心矩的峰度系数对于某些分布是无限的，如第6.4节所述，我们将定义一种基于分位数的峰度：

\[ \text{quKurt}(F) = \frac{F^{-1}(1-p_1) - F^{-1}(p_1)}{F^{-1}(1-p_2) - F^{-1}(p_2)} \]

其中 \(F\) 是CDF，且 \(0 < p_1 < p_2 < 1/2\)。通常，\(p_1\) 接近零，使得分子对尾部权重敏感，而 \(p_2\) 大得多，用于度量分布中心 的离散程度。由于 quKurt 的分子和分母都是两个分位数之差，它们均无位置参数，因此是尺度参数。此外，由于 quKurt 是两个尺度参数的比值，它 是无标度的，因此是一个形状参数。

一个典型的例子是 \(p_1 = 0.025\) 和 \(p_2 = 0.25\)。通过用样本分位数代替总体分位数来估计 quKurt。

# 定义偏斜t分布的对数似然函数
def skewt_loglik(params, x):
    loc, scale, alpha, nu = params
    if scale <= 0 or nu <= 0:
        return -np.inf
    # Azzalini偏斜t分布的PDF
    t_pdf = stats.t.pdf((x - loc)/scale, nu) / scale
    t_cdf = stats.t.cdf(alpha * (x - loc)/scale * np.sqrt((nu+1)/(nu + ((x - loc)/scale)**2)), nu+1)
    pdf = 2 * t_pdf * t_cdf
    pdf = np.maximum(pdf, 1e-300)  # 避免取0对数
    return np.sum(np.log(pdf))

# 极大似然估计
initial_params = [np.mean(returns), np.std(returns), 0.0, 5.0]
result = minimize(lambda p: -skewt_loglik(p, returns), initial_params, method='Nelder-Mead')
loc_hat, scale_hat, alpha_hat, nu_hat = result.x

# 生成分位数网格
q_grid = np.arange(1, n+1) / (n+1)

# 计算偏斜t分布的分位数
def qsstd(p, loc, scale, alpha, nu):
    # 使用数值求根实现分位数函数
    def func(x, p):
        return skewt_loglik([loc, scale, alpha, nu], np.array([x])) - np.log(p)
    quantiles = np.zeros_like(p)
    for i, prob in enumerate(p):
        # 初值基于t分布分位数
        init = loc + scale * stats.t.ppf(prob, nu)
        from scipy.optimize import root_scalar
        sol = root_scalar(lambda x: func(x, prob), bracket=[init-5*scale, init+5*scale])
        quantiles[i] = sol.root
    return quantiles

# Note: This Python code approximates the R simulation, but some functions like boott and bcanon are not directly available.
# We use scipy's bootstrap for BCa intervals and basic bootstrap.

def kurtosis(x):
    return np.mean(((x - np.mean(x)) / np.std(x, ddof=0))**4)

np.random.seed(3751)
niter = 500
nboot = 400
n = 50  # corrected from "5 0"
nu = 10  # corrected from "n u=1 0"
true_kurtosis = 3 + 6/(nu - 4)

correct = np.zeros((niter, 5))
width = np.zeros((niter, 5))
error = np.zeros(niter)

# Note: boott and bcanon are not implemented; we provide a simplified structure.
# Users should install bootstrap package or use R for exact replication.

for i in range(niter):
    y = t_dist.rvs(df=nu, size=n)
    # Bootstrap resamples
    boot_samples = np.random.choice(y, size=(n, nboot), replace=True)
    theta_star = np.apply_along_axis(kurtosis, 0, boot_samples)

    # Normal interval (int3)
    se_boot = np.std(theta_star, ddof=1)
    int3 = [kurtosis(y) - 1.96*se_boot, kurtosis(y) + 1.96*se_boot]
    width[i,2] = int3[1] - int3[0]
    correct[i,2] = 1 if (int3[0] < true_kurtosis and true_kurtosis < int3[1]) else 0

    # Percentile interval (int4)
    int4 = np.percentile(theta_star, [2.5, 97.5])
    width[i,3] = int4[1] - int4[0]
    correct[i,3] = 1 if (int4[0] < true_kurtosis and true_kurtosis < int4[1]) else 0

    # Basic bootstrap (int5)
    int5 = [2*kurtosis(y) - np.percentile(theta_star, 97.5), 2*kurtosis(y) - np.percentile(theta_star, 2.5)]
    width[i,4] = int5[1] - int5[0]
    correct[i,4] = 1 if (int5[0] < true_kurtosis and true_kurtosis < int5[1]) else 0

    # For int1 (boott) and int2 (bcanon), we leave as NaN as they are not implemented
    width[i,0] = np.nan
    width[i,1] = np.nan
    correct[i,0] = np.nan
    correct[i,1] = np.nan

    error[i] = np.mean(theta_star) - kurtosis(y)

print("Mean width:", np.nanmean(width, axis=0))
