第17章 资本资产定价模型

17.1 CAPM简介

CAPM（资本资产定价模型）有多种用途。它为持有指数基金的被动投资这一广泛实践提供了理论依据1。CAPM可以估算单项投资的预期收益率，并为受监管企业或按成本加成基础运作的企业确定投入资本的“公平”收益率2。

CAPM始于这样一个问题：如果下列假设成立，证券的风险溢价会是多少？

  1. 市场价格处于“均衡状态”。特别地，每种资产的供给等于需求。
  2. 所有人都对预期收益率和风险有相同的预测。
  3. 所有投资者都根据第16章讨论的有效分散化原则，最优地选择投资组合。这意味着每个人都持有风险资产的切点组合以及无风险资产。
  4. 市场奖励承担不可避免风险的人，但不会为由于低效投资组合选择而产生的无谓风险提供奖励。因此，单一证券的风险溢价并非源于其“独立”风险，而是源于其对切点组合风险的贡献。风险的各种组成部分将在第17.4节讨论。
  1. 指数基金持有与某个指数相同的投资组合。例如，S&P 500指数基金按与指数相同的比例持有S&P 500中的所有500只股票。一些基金并非精确复制指数，而是旨在跟踪指数，例如通过与指数协整来实现。↩
  2. 参见Bodie and Merton (2000)。↩

假设3意味着市场组合等于切点组合。因此，一个模拟市场组合的宽基指数基金可以用作切点组合的近似。

CAPM的有效性只有在所有这些假设成立时才能得到保证，当然没有人相信它们完全正确。假设3充其量只是一种理想化。此外，CAPM的一些结论与金融市场的行为相矛盾；参见第18.4.1节的一个例子。尽管存在缺陷，CAPM在金融领域仍被广泛使用，学习金融的学生理解CAPM是必不可少的。它的许多概念，如资产的贝塔、系统性风险和可分散风险，都非常重要，并且CAPM已被推广到第18章介绍的广泛使用的因子模型。

17.2 资本市场线 (CML)

资本市场线将有效投资组合的超额期望收益率与其风险联系起来。超额期望收益率是期望收益率减去无风险利率，也称为风险溢价。资本市场线为 \[ \mu_R = \mu_f + \frac{\mu_M - \mu_f}{\sigma_M} \sigma_R, \tag{17.1} \] 其中\(R\)是给定有效投资组合（市场组合[=切点组合]与无风险资产的混合）的收益率，\(\mu_R = E(R)\)，\(\mu_f\)是无风险利率，\(R_M\)是市场组合的收益率，\(\mu_M = E(R_M)\)，\(\sigma_M\)是\(R_M\)的标准差，而\(\sigma_R\)是\(R\)的标准差。\(R\)的风险溢价为\(\mu_R - \mu_f\)，市场组合的风险溢价为\(\mu_M - \mu_f\)。

在(17.1)中，\(\mu_f\)、\(\mu_M\)和\(\sigma_M\)是常数。变化的是\(\sigma_R\)和\(\mu_R\)。当我们改变有效投资组合\(R\)时，这些值随之变化。可以把资本市场线看作展示了\(\mu_R\)如何依赖于\(\sigma_R\)。

资本市场线的斜率当然是 \[ \frac{\mu_M - \mu_f}{\sigma_M}, \] 它可以解释为市场组合的风险溢价与标准差的比率。这就是夏普著名的“报酬-风险比率”，在金融中广泛使用。方程(17.1)可以重写为 \[ \frac{\mu_R - \mu_f}{\sigma_R} = \frac{\mu_M - \mu_f}{\sigma_M}, \] 这表明任何有效投资组合的报酬-风险比率等于市场组合的该比率——所有有效组合具有与市场组合相同的夏普比率。

17.2 资本市场线（CML）

例17.1. 资本市场线

假设无风险利率为 \(\mu_f = 0.06\)，市场组合的期望收益率为 \(\mu_M = 0.15\)，市场组合的风险为 \(\sigma_M = 0.22\)。则CML的斜率为 \((0.15 - 0.06)/0.22 = 9/22\)。本例中的CML如图17.1所示。□

因此，所有有效投资组合的风险收益比相同，包括市场组合。这一事实由粗线表示，其长度分别为典型有效投资组合的风险和收益。

CML很容易推导。考虑一个有效投资组合，将其资产的比例 \(w\) 投资于市场组合，\((1-w)\) 投资于无风险资产。那么

\[ R = w R_M + (1-w) \mu_f = \mu_f + w (R_M - \mu_f). \tag{17.2} \]

因此，对(17.2)取期望，

\[ \mu_R = \mu_f + w (\mu_M - \mu_f). \tag{17.3} \]

同样，由(17.2)可得

\[ \sigma_R = w \sigma_M, \tag{17.4} \]

或

\[ w = \frac{\sigma_R}{\sigma_M}. \tag{17.5} \]

将(17.5)代入(17.3)即得CML。

CAPM表明，最优投资方式如下：

  1. 确定你能承受的风险\(\sigma_R\)，其中\(0 \le \sigma_R \le \sigma_M\)3；
  2. 计算\(w = \sigma_R / \sigma_M\)；
  3. 将\(w\)比例的资金投资于市场指数基金，即跟踪整个市场的基金；
  4. 将\(1-w\)比例的资金投资于无风险国债或货币市场基金。

或者，

  1. 选择你想要的收益\(\mu_R - \mu_f\)；唯一约束是\(\mu_f \le \mu_R \le \mu_M\)，从而\(0 \le w \le 1\)4；
  2. 计算 \[w = \frac{\mu_R - \mu_f}{\mu_M - \mu_f}\]
  3. 重复上述步骤3和4。

与例16.1不同，我们不是指定期望收益率或收益率标准差，而是可以在保证置信水平\(1-\alpha\)下最大损失不超过某个由企业资本准备金决定的预定界限\(M\)的条件下，寻找具有最高期望收益率的投资组合。如果企业投资金额为\(C\)，那么损失大于\(M\)意味着收益率小于\(-M/C\)。假设收益率服从正态分布，则由(A.11)、(17.3)和(17.4)可得：

\[P\left(R < -\frac{M}{C}\right) = \Phi\left(\frac{-M/C - \{\mu_f + w(\mu_M - \mu_f)\}}{w\sigma_M}\right). \qquad (17.6)\]

因此，我们求解\(w\)的方程：

\[\Phi^{-1}(\alpha) = \frac{-M/C - \{\mu_f + w(\mu_M - \mu_f)\}}{w\sigma_M}.\]

我们可以将\(w = \sigma_R / \sigma_M\)视为投资者风险厌恶程度的指标。\(w\)值越小，投资者越厌恶风险。如果投资者的\(w=0\)，则其100%投资于无风险资产。类似地，\(w=1\)的投资者将全部资金投入风险资产的切点组合5。

  1. 实际上，通过借钱以保证金购买风险资产，\(\sigma_R > \sigma_M\)是可能的。↩
  2. 如果允许以保证金购买资产，这一约束可以放宽。↩
  3. \(w>1\)的投资者是在以保证金购买市场组合，即借钱购买市场组合。↩

17.3 贝塔系数与证券市场线

证券市场线（SML）将资产的超额收益率与其对市场组合回归的斜率联系起来。SML与CML的不同之处在于，SML适用于所有资产，而CML仅适用于有效组合。

假设存在多个证券，以 \(j\) 索引。定义

\[ \sigma_{jM} = \text{第 } j \text{ 个证券收益率与市场组合收益率之间的协方差}. \]

另外，定义

\[ \beta_j = \frac{\sigma_{jM}}{\sigma_M^2}. \tag{17.7} \]

根据第11.9.1节中关于最佳线性预测的理论，\(\beta_j\) 是使用市场组合收益率作为预测变量时，第 \(j\) 个证券收益率的最佳线性预测的斜率。这一结论源自最佳线性预测方程斜率的公式 (11.37)。事实上，基于 \(R_M\) 对 \(R_j\) 的最佳线性预测为

\[ \hat{R}_j = \beta_{0,j} + \beta_j R_M, \tag{17.8} \]

其中 (17.8) 中的 \(\beta_j\) 与 (17.7) 中的相同。此外，\(\beta_{0,j}\) 是截距项，可通过在 (17.8) 中取期望并求解得到 \(\beta_{0,j} = E(R_j) - \beta_j E(R_M)\)。

另一种理解 \(\beta_j\) 重要性的方法是使用线性回归。如第11.9节所述，线性回归是一种基于数据估计最佳线性预测系数的方法。为了应用线性回归，假设我们有一个二元时间序列 \((R_{j,t}, R_{M,t})_{t=1}^n\)，表示第 \(j\) 个资产和市场组合的收益率。那么，\(R_{j,t}\) 对 \(R_{M,t}\) 线性回归的斜率估计为

\[ \hat{\beta}_j = \frac{\sum_{t=1}^n (R_{j,t} - \bar{R}_j)(R_{M,t} - \bar{R}_M)}{\sum_{t=1}^n (R_{M,t} - \bar{R}_M)^2}, \tag{17.9} \]

将分子和分母同时乘以 \(n-1\) 后，该式变为 \(\sigma_{jM}\) 的估计量除以 \(\sigma_M^2\) 的估计量，因此根据 (17.7) 是 \(\beta_j\) 的估计量。

令 \(\mu_j\) 为第 \(j\) 个证券的期望收益率。那么 \(\mu_j - \mu_f\) 是该证券的风险溢价（或风险报酬，或超额期望收益率）。利用CAPM可以证明

\[ \mu_j - \mu_f = \beta_j (\mu_M - \mu_f). \tag{17.10} \]

该方程称为证券市场线（SML），在第17.5.2节中推导。在 (17.10) 中，\(\beta_j\) 是线性方程中的一个变量，而不是斜率；

更精确地说，\(\mu_j\) 是 \(\beta_j\) 的线性函数，斜率为 \(\mu_M - \mu_f\)。这一点值得牢记。否则可能会引起混淆，因为之前我们将 \(\beta_j\) 定义为回归模型的斜率。换言之，\(\beta_j\) 在一个语境中是斜率，但在证券市场线（SML）的另一个语境中却是自变量。我们可以利用 (17.9) 估计 \(\beta_j\)，然后将该估计值代入 (17.10)。

SML 表明，第 \(j\) 个资产的风险溢价等于其贝塔值（\(\beta_j\)）与市场组合风险溢价（\(\mu_M - \mu_f\)）的乘积。因此，\(\beta_j\) 既度量了第 \(j\) 个资产的风险性，也度量了承担该风险所获得的回报。相应地，\(\beta_j\) 是衡量第 \(j\) 个资产“进取性”的指标。根据定义，市场组合的贝塔值为 1，即 \(\beta_M = 1\)。这给出了一个经验法则：

  • \(\beta_j > 1\) 对应“进取型”，
  • \(\beta_j = 1\) 对应“平均风险”，
  • \(\beta_j < 1\) 对应“非进取型”。

图 17.2 展示了 SML 以及一个不在 SML 上的资产 J。该资产与 CAPM 相矛盾，因为根据 CAPM，所有资产都位于 SML 上，因此不存在这样的资产。

考虑如果存在类似 J 的资产会发生什么。投资者不会想要购买它，因为它位于 SML 下方，其风险溢价相对于其贝塔值所对应的风险而言太低。他们会减少对 J 的投资，转而增加对其他证券的投资。因此，J 的价格将下跌，而在价格下跌后，其预期收益率将上升。经过这一上升，资产 J 将回到 SML 上——或者理论是这样预测的。

17.3.2 CML 与 SML 的比较

CML 仅适用于有效组合的收益率 \(R\)。它可以整理为将组合的超额期望收益率与市场组合的超额期望收益率联系起来的形式：

\[ \mu_R - \mu_f = \left(\frac{\sigma_R}{\sigma_M}\right)(\mu_M - \mu_f). \tag{17.11} \]

SML 适用于任何资产，并且与 CML 一样，它将资产的超额期望收益率与市场组合的超额期望收益率联系起来：

\[ \mu_j - \mu_f = \beta_j(\mu_M - \mu_f). \tag{17.12} \]

17.4 证券特征线

令 \(R_{jt}\) 为第 \(j\) 种资产在时间 \(t\) 的收益率。类似地，令 \(R_{M,t}\) 和 \(\mu_{f,t}\) 分别为市场组合在时间 \(t\) 的收益率和无风险收益率。证券特征线（有时简称为特征线）是一个回归模型： \[ R_{j,t} = \mu_{f,t} + \beta_j (R_{M,t} - \mu_{f,t}) + \epsilon_{j,t}, \tag{17.13} \] 其中 \(\epsilon_{j,t} \sim N(0, \sigma^2_{\epsilon,j})\)。通常假设 \(\epsilon_{j,t}\) 在不同资产间不相关， 即对于 \(j \neq j'\)，\(\epsilon_{j,t}\) 与 \(\epsilon_{j',t}\) 不相关。这一假设对通过分散化降低风险具有重要意义； 参见第 17.4.1 节。

如果我们将一个有效组合视为一种资产，那么 \(\mu_R\) 和 \(\mu_j\) 都表示该组合/资产的期望收益率。 (17.11) 和 (17.12) 都表明 \(\sigma_R / \sigma_M = \beta_R\)。

令 \(\mu_{j,t} = \mathbb{E}(R_{j,t})\) 且 \(\mu_{M,t} = \mathbb{E}(R_{M,t})\)。对(17.13)取期望，我们得到：

这就是方程(17.10)，即证券市场线（SML），不过在(17.10)中没有明确显示期望收益率可以依赖于 \(t\)。SML给出了关于期望收益率的信息，但没有给出关于收益率方差的信息。对于后者，我们需要特征线。特征线被称为收益生成过程，因为它给出了收益率的概率模型，而不仅仅是其期望值的模型。

关于SML与特征线之间的区别，可以类比如下：回归线 \(\mathbb{E}(Y|X) = \beta_0 + \beta_1 X\) 给出了给定 \(X\) 时 \(Y\) 的期望值，但没有给出给定 \(X\) 时 \(Y\) 的条件概率分布。而回归模型

\[Y_t = \beta_0 + \beta_1 X_t + \epsilon_t, \quad \epsilon_t \sim N(0, \sigma^2)\]

确实给出了这个条件概率分布。

17.4 特征线

特征线意味着

\[ \sigma_j^2 = \beta_j^2 \sigma_M^2 + \sigma_{\epsilon,j}^2, \]

且对于 \( j \neq j' \)，有

\[ \sigma_{jj'} = \beta_j \beta_{j'} \sigma_M^2 \qquad (17.14) \]

以及

\[ \sigma_{Mj} = \beta_j \sigma_M^2. \]

为使 (17.14) 成立，\( \epsilon_{j,t} \) 和 \( \epsilon_{j',t} \) 必须不相关。第 \( j \) 个资产的总风险为

\[ \sigma_j = \sqrt{ \beta_j^2 \sigma_M^2 + \sigma_{\epsilon,j}^2 }. \]

平方风险包含两个部分：\( \beta_j^2 \sigma_M^2 \) 称为风险的市场成分或系统性成分，而 \( \sigma_{\epsilon,j}^2 \) 称为独特成分、非市场成分或非系统性成分。

17.4.1 通过分散化降低独特风险

风险的市场成分无法通过分散化降低，但独特成分可以通过充分分散化降低甚至消除。

假设有 \( N \) 种资产，其在持有期 \( t \) 的收益率分别为 \( R_{1,t}, \ldots, R_{N,t} \)。如果我们构建一个权重为 \( w_1, \ldots, w_N \) 的投资组合，则该投资组合的收益率为

\[ R_{P,t} = w_1 R_{1,t} + \cdots + w_N R_{N,t}. \]

令 \( R_{M,t} \) 为市场组合的收益率。根据特征线模型 \( R_{j,t} = \mu_{f,t} + \beta_j (R_{M,t} - \mu_{f,t}) + \epsilon_{j,t} \)，因此

\[ R_{P,t} = \mu_{f,t} + \left( \sum_{j=1}^N \beta_j w_j \right) (R_{M,t} - \mu_{f,t}) + \sum_{j=1}^N w_j \epsilon_{j,t}. \]

因此，投资组合的贝塔值为

\[ \beta_P = \sum_{j=1}^N w_j \beta_j, \]

而投资组合的“epsilon”为

\[ \epsilon_{P,t} = \sum_{j=1}^N w_j \epsilon_{j,t}. \]

现在我们假设 \( \epsilon_{1,t}, \ldots, \epsilon_{N,t} \) 不相关。因此，由方程 (7.11) 可得

\[ \sigma_{\epsilon,P}^2 = \sum_{j=1}^N w_j^2 \sigma_{\epsilon,j}^2. \]

示例17.2. 分散化降低风险

假设组合中资产等权重；即对所有\(j\)，\(w_j = 1/N\)。则

\[\beta_P = \frac{\sum_{j=1}^N \beta_j}{N},\] \[\sigma^2_{\epsilon,P} = \frac{N^{-1}\sum_{j=1}^N \sigma^2_{\epsilon,j}}{N} = \frac{\bar{\sigma}^2_\epsilon}{N},\]

其中\(\bar{\sigma}^2_\epsilon\)是\(\sigma^2_{\epsilon,j}\)的平均值。

举例说明，假设简单情形下\(\sigma^2_{\epsilon,j}\)是常数，记为\(\sigma^2_\epsilon\)，对所有\(j\)，则

\[\sigma_{\epsilon,P} = \frac{\sigma_\epsilon}{\sqrt{N}}. \qquad (17.15)\]

例如，假设\(\sigma_\epsilon = 5\%\)。若\(N=20\)，则由(17.15) \(\sigma_{\epsilon,P}=1.12\%\)。若\(N=100\)，则\(\sigma_{\epsilon,P}=0.5\%\)。纽约证券交易所约有1600只股票；若\(N=1600\)，则\(\sigma_{\epsilon,P}=0.125\%\)，从5\%显著降低。□

17.4.2 这些假设合理吗？

一个使得非市场风险可以通过分散化消除的关键假设是\(\epsilon_{1,t},\ldots,\epsilon_{N,t}\)不相关。这个假设意味着资产收益率的截面6中的所有相关性都归因于单一原因，且该原因由市场指数衡量。因此，特征线是一个以\(R_{M,t}\)为“因子”的“单因子”或“单指数”模型。

如果两只能源股票之间的相关性超过它们因市场指数而产生的相关性，那么\(\epsilon_{jt}\)不相关的假设就不成立。在这种情况下，持有所有能源股票的大组合无法消除特有风险。然而，如果存在许多市场板块且板块之间不相关，那么一个人可以通过在所有板块间分散化来消除非市场风险。所需要的只是将板块本身视为基础资产，然后应用CAPM理论。

市场板块内股票之间的相关性可以用因子模型建模；见第18章。

在本节中，我们使用投资组合理论来证明\(\sigma_{j,M}\)量化了第\(j\)项资产对市场组合风险的贡献。同时，我们推导出SML。

17.5 更多投资组合理论

17.5.1 对市场组合风险的贡献

假设市场由\(N\)种风险资产组成，且\(w_{1,M},\ldots,w_{N,M}\)是这些资产在市场组合中的权重。那么，

\[R_{M,t} = \sum_{i=1}^{N} w_{i,M} R_{i,t},\]

这意味着第\(j\)种资产收益率与市场组合收益率之间的协方差为

\[\sigma_{j,M} = \operatorname{Cov}\left(R_{j,t}, \sum_{i=1}^{N} w_{i,M} R_{i,t}\right) = \sum_{i=1}^{N} w_{i,M} \sigma_{i,j}. \tag{17.16}\]

因此，

\[\sigma_M^2 = \sum_{j=1}^{N} \sum_{i=1}^{N} w_{j,M} w_{i,M} \sigma_{i,j} = \sum_{j=1}^{N} w_{j,M} \left( \sum_{i=1}^{N} w_{i,M} \sigma_{i,j} \right) = \sum_{j=1}^{N} w_{j,M} \sigma_{j,M}. \tag{17.17}\]

方程(17.17)表明，第\(j\)种资产对市场组合风险的贡献为\(w_{j,M} \sigma_{j,M}\)，其中\(w_{j,M}\)是第\(j\)种资产在市场组合中的权重，\(\sigma_{j,M}\)是第\(j\)种资产收益率与市场组合收益率之间的协方差。

17.5.2 SML的推导

SML的推导是投资组合理论、微积分和几何推理的一个巧妙应用。它基于一个聪明的想法：构建一个包含两种资产（市场组合和第\(i\)种风险资产）的投资组合，然后观察当分配给第\(i\)种风险资产的组合权重变化时，在风险-收益空间中的轨迹。

考虑一个组合\(P\)，其分配给第\(i\)种风险资产的权重为\(w_i\)，分配给市场（切点）组合的权重为\((1-w_i)\)。该组合的收益率为

\[R_{P,t} = w_i R_{i,t} + (1-w_i) R_{M,t}.\]

期望收益率为

\[\mu_P = w_i \mu_i + (1-w_i) \mu_M, \tag{17.18}\]

风险为

\[\sigma_P = \sqrt{w_i^2 \sigma_i^2 + (1-w_i)^2 \sigma_M^2 + 2 w_i (1-w_i) \sigma_{i,M}}. \tag{17.19}\]

随着\(w_i\)的变化，我们得到在\((\sigma,\mu)\)空间中的轨迹，如图17.3中的虚线曲线所示，该图使用了与图16.3相同的收益率，并以Mobil股票作为资产\(i\)。

从几何上容易看出，这个点轨迹在切点组合（即\(w_i=0\)的点）处的导数等于CML的斜率。我们可以计算这个导数，并令其等于CML的斜率，看看会得到什么。我们会发现结果就是SML。

由(17.18)和(17.19)可得：

\[\frac{d\mu_P}{dw_i} = \mu_i - \mu_M,\]

\[\frac{d\sigma_P}{dw_i} = \frac{1}{2}\sigma_P^{-1}\left\{2w_i\sigma_i^2 - 2(1-w_i)\sigma_M^2 + 2(1-2w_i)\sigma_{i,M}\right\}.\]

因此，

\[\frac{d\mu_P}{d\sigma_P} = \frac{d\mu_P/dw_i}{d\sigma_P/dw_i} = \frac{(\mu_i - \mu_M)\sigma_P}{w_i\sigma_i^2 - \sigma_M^2 + w_i\sigma_M^2 + \sigma_{i,M} - 2w_i\sigma_{i,M}}.\]

接下来，

\[\left.\frac{d\mu_P}{d\sigma_P}\right|_{w_i=0} = \frac{(\mu_i - \mu_M)\sigma_M}{\sigma_{i,M} - \sigma_M^2}.\]

17.6 贝塔的估计与CAPM检验

回忆 \(w_i = 0\) 是切点组合，即图17.3中虚线曲线与CML相切的那一点。因此，

\[ \frac{d\mu_P}{d\sigma_P} \Big|_{w_i=0} \]

必须等于CML的斜率，即 \((\mu_M - \mu_f)/\sigma_M\)。因此，

\[ \frac{(\mu_i - \mu_M)\sigma_M}{\sigma_{i,M} - \sigma_M^2} = \frac{\mu_M - \mu_f}{\sigma_M}, \]

经过代数运算，得到

\[ \mu_i - \mu_f = \frac{\sigma_{i,M}}{\sigma_M^2}(\mu_M - \mu_f) = \beta_i (\mu_M - \mu_f), \]

即方程(17.10)给出的SML。

17.6.1 利用回归进行估计

回顾证券特征线

\[ R_{j,t} = \mu_{f,t} + \beta_j (R_{M,t} - \mu_{f,t}) + \epsilon_{j,t}. \tag{17.20} \]

令 \(R_{j,t}^* = R_{j,t} - \mu_{f,t}\) 为第 \(j\) 种证券的超额收益率，\(R_{M,t}^* = R_{M,t} - \mu_{f,t}\) 为市场组合的超额收益率。则(17.20)可以写为

\[ R_{j,t}^* = \beta_j R_{M,t}^* + \epsilon_{j,t}. \tag{17.21} \]

方程(17.21)是一个无截距项的回归模型，\(\beta_j\) 是斜率。更精细的模型是

\[ R_{j,t}^* = \alpha_j + \beta_j R_{M,t}^* + \epsilon_{j,t}, \tag{17.22} \]

其中包含截距项。CAPM声称 \(\alpha_j = 0\)，但允许 \(\alpha_j \neq 0\) 可以识别错误定价的可能性。

给定时间序列 \(R_{j,t}\)、\(R_{M,t}\) 和 \(\mu_{f,t}\)（\(t = 1, \dots, n\)），我们可以计算 \(R_{j,t}^*\) 和 \(R_{M,t}^*\)，并将 \(R_{j,t}^*\) 对 \(R_{M,t}^*\) 进行回归，以估计 \(\alpha_j\)、\(\beta_j\) 和 \(\sigma_{\epsilon,j}^2\)。通过检验原假设 \(\alpha_j = 0\)，我们检验第 \(j\) 种资产是否根据CAPM存在错误定价。

如第9.2.2节所述，在拟合模型(17.21)或(17.22)时，如果可能，应使用日度数据，而不是周度或月度数据。更难回答的问题是使用多长时间序列。较长的时间序列当然提供更多数据，但模型(17.21)和(17.22)假设 \(\beta_j\) 是常数，这在长时间内可能不成立。

17.6.2 检验CAPM

检验\\(\\alpha=0\\)仅仅检验CAPM的结论之一。接受这一原假设仅意味着CAPM通过了一项检验，并不表明我们现在应该接受它。8要完全检验CAPM，还应检验其其他结论。第18.3节中的因子模型已被用于检验CAPM，并发现了相当有力的证据反对CAPM。幸运的是，这些因子模型确实提供了CAPM的一种推广，很可能对金融决策有用。

通常，作为使用超额收益回归的替代方案，可以将资产的收益率对市场收益率进行回归。当这样做时，应使用带截距模型。在微软数据中，当使用收益率而非超额收益率时，贝塔的估计几乎不变。

17.6.3 阿尔法的解释

如果\\(\\alpha\\)不为零，那么根据CAPM，该证券定价有误。如果\\(\\alpha>0\\)，则证券被低估；平均而言收益率过高。

17.7 使用CAPM进行投资组合分析

假设我们已估计出投资组合中每个资产的贝塔值 \(\beta_j\) 和 \(\sigma_{\epsilon,j}^2\)，以及市场的 \(\sigma_M^2\) 和 \(\mu_M\)。由于无风险利率 \(\mu_f\) 也已知，我们可以通过以下公式计算所有资产收益率的期望、方差和协方差：

\[ \mu_j = \mu_f + \beta_j (\mu_M - \mu_f), \] \[ \sigma_j^2 = \beta_j^2 \sigma_M^2 + \sigma_{\epsilon,j}^2, \] \[ \sigma_{jj'} = \beta_j \beta_{j'} \sigma_M^2 \quad \text{对于 } j \neq j'. \]

这里有一个值得注意的风险：这些估计严重依赖于CAPM假设的有效性。贝塔值、\(\sigma_{\epsilon}^2\)、\(\sigma_M^2\)、\(\mu_M\) 和 \(\mu_f\) 中的任何一个或全部可能随时间 \(t\) 变化。然而，通常假设资产的贝塔值和 \(\sigma_{\epsilon}^2\) 以及市场的 \(\sigma_M^2\) 和 \(\mu_M\) 与 \(t\) 无关，因此可以在收益率时间序列平稳的假设下估计这些参数。

17.8 文献注释

CAPM由 Sharpe (1964)、Lintner (1965a,b) 和 Mossin (1966) 提出。关于CAPM的介绍可在 Bodie、Kane 和 Marcus (1999)、Bodie 和 Merton (2000) 以及 Sharpe、Alexander 和 Bailey (1999) 中找到。我最初是从这三本教材中学习CAPM的。Campbell、Lo 和 MacKinlay (1997) 讨论了CAPM的实证检验。第17.5.2节中SML的推导改编自 Sharpe、Alexander 和 Bailey (1999)。因子模型的讨论可参见 Sharpe、Alexander 和 Bailey (1999)、Bodie、Kane 和 Marcus (1999) 以及 Campbell、Lo 和 MacKinlay (1997)。
