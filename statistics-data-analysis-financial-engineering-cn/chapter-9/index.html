第9章 回归：基础

9.1 引言

回归是所有统计方法中应用最广泛的方法之一。

对于单变量回归，可用的数据包括一个响应变量和\(p\)个预测变量，每个变量在\(n\)个观测上都有测量值。我们用\(Y\)表示响应变量，\(X_1,\ldots,X_p\)表示预测变量或解释变量。此外，\(Y_i\)和\(X_{i,1},\ldots,X_{i,p}\)是第\(i\)个观测的这些变量的值。

回归建模的目标包括研究\(Y\)与\(X_1,\ldots,X_p\)的关系，估计给定\(X_1,\ldots,X_p\)的条件下\(Y\)的条件期望，以及在已知\(X_1,\ldots,X_p\)对应值时预测未来的\(Y\)值。这些目标紧密相连。

将\(Y\)与预测变量或回归变量联系起来的多元线性回归模型为

\[Y_i = \beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p} + \epsilon_i, \tag{9.1}\]

其中\(\epsilon_i\)称为噪声、扰动或误差。形容词“多元”指的是预测变量。具有多个响应变量的多元回归将在第18章中讨论。

\(\epsilon_i\)通常被称为“误差”，因为当用\(\beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p}\)预测\(Y_i\)时，它们就是预测误差。假设

\[\mathrm{E}(\epsilon_i | X_{i,1}, \ldots, X_{i,p}) = 0, \tag{9.2}\]

结合(9.1)可得

\[\mathrm{E}(Y_i | X_{i,1}, \ldots, X_{i,p}) = \beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p}.\]

参数\(\beta_0\)是截距。回归系数\(\beta_1,\ldots,\beta_p\)是斜率。更准确地，\(\beta_j\)是期望响应对第\(j\)个预测变量的偏导数：

直线回归是仅有一个预测变量的线性回归。模型为

\[ Y_i = \beta_0 + \beta_1 X_i + \epsilon_i, \tag{9.4} \]

其中\(\beta_0\)和\(\beta_1\)是未知的截距和斜率，\(\epsilon_i\)称为噪声或误差。

9.2.1 最小二乘估计

回归系数可以通过最小二乘法估计。最小二乘估计是使得下式最小的\(\hat{\beta}_0\)和\(\hat{\beta}_1\)的值：

\[ \sum_{i=1}^n \left\{ Y_i - (\hat{\beta}_0 + \hat{\beta}_1 X_i) \right\}^2. \tag{9.5} \]

9.2.2 \(\hat{\beta}_1\) 的方差

了解估计量方差的公式是有用的，它能说明估计量的精度如何取决于数据的各个方面，例如样本量和预测变量的取值。幸运的是，很容易推导出 \(\hat{\beta}_1\) 的方差公式。由 (9.6)，我们可以将 \(\hat{\beta}_1\) 写成响应的加权平均：

\[ \hat{\beta}_1 = \sum_{i=1}^n w_i Y_i, \]

其中权重 \(w_i\) 为

\[ w_i = \frac{X_i - \bar{X}}{\sum_{i=1}^n (X_i - \bar{X})^2}. \]

我们将 \(X_1, \ldots, X_n\) 视为固定的，因此如果它们是随机的，我们是在给定其值的条件下进行讨论。根据回归模型的假设，有 \(\text{Var}(Y_i \mid X_1, \ldots, X_n) = \sigma_\epsilon^2\) 且 \(Y_1, \ldots, Y_n\) 条件不相关。因此，

\[ \text{Var}(\hat{\beta}_1 \mid X_1, \ldots, X_n) = \sigma_\epsilon^2 \sum_{i=1}^n w_i^2 = \frac{\sigma_\epsilon^2}{\sum_{i=1}^n (X_i - \bar{X})^2} = \frac{\sigma_\epsilon^2}{(n-1)s_X^2}. \qquad (9.8)

值得花些时间审视这个公式。首先，分子 \(\sigma_\epsilon^2\) 就是 \(\epsilon_i\) 的方差。这并不意外。噪声的变异性越大，估计量的变异性就越大。分母表明 \(\hat{\beta}_1\) 的方差与 \((n-1)\) 和 \(s_X^2\) 成反比。因此，\(\hat{\beta}_1\) 的精度随着 \(\sigma_\epsilon^2\) 减小、\(n\) 增大或 \(s_X^2\) 增大而提高。为什么增大 \(s_X^2\) 会减小 \(\text{Var}(\hat{\beta}_1 \mid X_1, \ldots, X_n)\)？原因在于增大 \(s_X^2\) 意味着 \(X_i\) 散布得更开，这使得直线的斜率更容易估计。

例 9.3 回归的最优采样频率

这里是 (9.8) 的一个重要应用。假设我们有两个平稳时间序列 \(X_t\) 和 \(Y_t\)，我们想将 \(Y_t\) 对 \(X_t\) 进行回归。我们刚刚看到了这方面的例子。一个重要的实际问题是我们应该使用日度数据还是周度数据，或者甚至月度或季度数据。采样频率的选择重要吗？答案是“是的”，并且最高可能的采样频率能给出最精确的斜率估计。为了理解原因，我们比较日度和周度数据。假设 \(X_t\) 和 \(Y_t\) 是白噪声序列。由于周度对数收益率就是一周内五个日度对数收益率之和，因此如果从日度对数收益率改为周度对数收益率，\(\sigma^2_\epsilon\) 和 \(s^2_X\) 都将增加五倍，所以比率 \(\sigma^2_\epsilon / s^2_X\) 不会改变。然而，从日度改为周度对数收益率时，\((n-1)\) 大约减少为原来的五分之一。结果是，使用日度对数收益率时 \(\mathrm{Var}(\hat{\beta}_1 | X_1, \ldots, X_n)\) 大约比使用周度对数收益率小五倍。类似地，使用月度收益率比使用周度收益率时 \(\mathrm{Var}(\hat{\beta}_1 | X_1, \ldots, X_n)\) 大约大四倍。

明显的结论是应该使用最高可用的采样频率，通常是日度收益率。为了简化计算，我们假设了 \(X_t\) 和 \(Y_t\) 是白噪声，但这个结论在它们平稳但自相关时仍然成立（自相关将在第12章讨论）。然而，噪声序列，即 (9.4) 中的 \(\epsilon_i, i=1,\ldots,n\) 需要是不相关的。如果噪声是自相关的，并且随着采样频率的增加而变得更高度相关，那么这个结论不一定成立。可能存在边际收益递减的点，其中更频繁的采样并不能提高估计精度。□

9.3 多元线性回归

多元线性回归模型为

\[ Y_i = \beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p} + \epsilon_i. \]

最小二乘估计是使下式最小的 \(\hat{\beta}_0, \hat{\beta}_1, \ldots, \hat{\beta}_p\) 的值：

\[ \sum_{i=1}^n \left\{ Y_i - (\hat{\beta}_0 + \hat{\beta}_1 X_{i,1} + \cdots + \hat{\beta}_p X_{i,p}) \right\}^2. \tag{9.9} \]

最小二乘估计的计算将在 11.1 节讨论。对于应用而言，技术细节并不重要，因为最小二乘估计的软件很容易获得。

第 \(i\) 个拟合值为

\[ \hat{Y}_i = \hat{\beta}_0 + \hat{\beta}_1 X_{i,1} + \cdots + \hat{\beta}_p X_{i,p}. \tag{9.10} \]

并估计 \(E(Y_i \mid X_{i,1}, \ldots, X_{i,p})\)。第 \(i\) 个残差为

\[ \hat{\epsilon}_i = Y_i - \hat{Y}_i = Y_i - (\hat{\beta}_0 + \hat{\beta}_1 X_{i,1} + \cdots + \hat{\beta}_p X_{i,p}) \tag{9.11} \]

并估计 \(\epsilon_i\)。值得注意，(9.11) 可以重写为

\[ Y_i = \hat{Y}_i + \hat{\epsilon}_i. \tag{9.12} \]

\(\sigma^2_\epsilon\) 的无偏估计为

\[ \hat{\sigma}^2_\epsilon = \frac{\sum_{i=1}^n \hat{\epsilon}_i^2}{n-1-p}. \tag{9.13} \]

(9.13) 中的分母是样本量减去估计的回归系数个数。

9.3.1 标准误、t值与p值

在本节中，我们解释回归输出中包含的几个统计量的用法。我们以例9.4中的输出作为说明。

如前所述，估计系数为 \(\hat{\beta}_0 = -9.07 \times 10^{-5}\)，\(\hat{\beta}_1 = 0.355\)，\(\hat{\beta}_2 = 0.300\)，\(\hat{\beta}_3 = 0.00412\)。每个系数都关联着另外三个统计量。

  • 标准误（SE），即最小二乘估计量估计的标准差，告诉了我们估计量的精确度。
  • t值，是检验系数为0的t统计量。t值为估计值与其标准误的比值。例如，对于 cm10_dif，t值为 7.86 = 0.355 / 0.0451。

• p值（lm()输出中的Pr(>|t|)）与检验系数为0的原假设相对于系数不为0的备择假设相关。如果斜率参数的p值很小（如这里的\(\beta_1\)），那么这是相应系数不为0的证据，这意味着预测变量与响应变量存在线性关系。

牢记这一点很重要：p值仅告诉我们是否存在线性关系。\(Y_i\)与\(X_{i,j}\)之间存在线性关系仅意味着\(Y_i\)的线性预测器在\(X_{i,j}\)上的斜率非零，或者等价地，\(X_{i,j}\)与\(Y_i\)之间的偏相关系数不为零。（两个变量之间的偏相关系数是在所有其他变量固定时它们的相关性。）当p值很小（即存在线性关系）时，也可能存在如图A.4g所示的与线性关系的强非线性偏离。此外，当p值很大（即不存在线性关系）时，仍可能存在如图A.4f所示的强非线性关系。由于线性回归分析中可能无法检测到非线性关系，因此数据的图形分析（例如图9.4）和残差分析（见第10章）至关重要。

\(\beta_1\)和\(\beta_2\)的p值非常小，因此我们可以得出结论：这些斜率不为0。\(\beta_0\)的p值很大（0.97），所以我们不会拒绝截距为0的假设。

类似地，我们不会拒绝\(\beta_3\)为零的原假设。换言之，我们可以接受这样的原假设：在给定cm10 dif和cm30 dif的条件下，aaa dif和ff dif不是线性相关的。这一结果不应被解释为aaa dif和ff dif无关，而仅仅意味着当回归模型中包含cm10 dif和cm30 dif时，ff dif对预测aaa dif没有帮助。（实际上，aaa dif和ff dif的相关系数为0.25（这是完全相关系数，而非偏相关系数），并且aaa dif仅对ff dif的线性回归非常显著；检验斜率为0的p值为\(5.158 \times 10^{-14}\)。）

由于联邦基金利率是短期（隔夜）利率，因此ff dif不如10年和30年期国债利率的变化对预测aaa dif有用也就不足为奇了。

对于只有一个预测变量的回归，由(9.8)可知，\(\hat{\beta}_1\)的标准误为\(\hat{\sigma}_\epsilon / \sqrt{\sum_{i=1}^n (X_i - \bar{X})^2}\)。当有两个以上预测变量时，标准误的公式更为复杂，借助矩阵表示可以简化。由于标准误可以使用诸如lm之类的标准软件计算，因此这些公式在应用中不需要，故推迟到第11.1节讨论。

9.4.1 方差分析表

回归拟合的某些结果通常展示在方差分析表中，也称为ANOVA或AOV表。ANOVA表的基本思想是描述在已知\(X_1,\ldots,X_p\)的情况下，\(Y\)的变异中有多少是可预测的。下面给出例9.4中模型的ANOVA表。

anova(lm(aaa_dif ~ cm10_dif + cm30_dif + ff_dif))

\(Y\)的总变异可以分解为两部分：由\(X_1,\ldots,X_p\)可预测的变异和不可预测的变异。可预测的变异由回归平方和度量，即 \[ \text{regression SS} = \sum_{i=1}^n (\hat{Y}_i - \bar{Y})^2. \] 仅使用\(\texttt{cm10\_dif}\)的模型的回归平方和在ANOVA表的第一行，为11.21。第二行的0.15是将\(\texttt{cm30\_dif}\)加入模型后回归平方和的增量。类似地，0.0025是加入\(\texttt{ff\_dif}\)后的增量。因此，四舍五入到两位小数，包含所有三个预测变量的模型回归平方和为\(11.36 = 11.21 + 0.15 + 0.00\)。

\(Y\)中不能被\(X_1,\ldots,X_p\)的线性函数预测的变异量由残差平方和度量，即残差的平方和： \[ \text{residual error SS} = \sum_{i=1}^n (Y_i - \hat{Y}_i)^2. \] 在ANOVA表中，残差平方和位于最后一行，为3.66。总变异由总平方和（total SS）度量，即\(Y\)与其均值的离差平方和： \[ \text{total SS} = \sum_{i=1}^n (Y_i - \bar{Y})^2. \tag{9.14} \]

可以代数证明，

\[ \text{总平方和} = \text{回归平方和} + \text{残差平方和}. \tag{9.15} \]

因此，在例9.4中，总平方和为11.36 + 3.66 = 15.02。

R方，记为\(R^2\)，定义为

\[ R^2 = \frac{\text{回归平方和}}{\text{总平方和}} = 1 - \frac{\text{残差平方和}}{\text{总平方和}}, \]

衡量了\(Y\)的总变异中能被\(X\)线性预测的比例。在本例中，若只含cm10 dif作为模型，\(R^2\)为0.746 = 11.21/15.02；若三个预测变量都纳入模型，则为11.36/15.02 = 0.756。该值可在例9.4的显示输出中找到。

当只有一个\(X\)变量时，\(R^2 = r_{XY}^2 = r_{\hat{Y}Y}^2\)，其中\(r_{XY}\)和\(r_{\hat{Y}Y}\)分别是\(Y\)与\(X\)以及\(Y\)与预测值之间的样本相关系数。换言之，\(R^2\)是\(Y\)与\(X\)之间的相关系数的平方，也是\(Y\)与\(\hat{Y}\)之间的相关系数的平方。当有多个预测变量时，仍然有\(R^2 = r_{\hat{Y}Y}^2\)。由于\(\hat{Y}\)是\(X\)变量的线性组合，\(R\)可视为\(Y\)与多个\(X\)之间的“复”相关系数。残差平方和也称为误差平方和，记作SSE。

9.4.2 自由度 (DF)

每个变异源都有对应的自由度 (DF)。回归的自由度为 \(p\)，即预测变量的个数。总的自由度为 \(n-1\)。残差误差的自由度为 \(n-p-1\)。下面是一种理解自由度的方式：最初，有 \(n\) 个自由度，每个观测值对应一个。然后，一个自由度被分配给截距的估计，剩下 \(n-1\) 个自由度用于估计 \(X\) 变量和 \(\sigma^2_{\epsilon}\) 的效应。每个回归参数在估计中使用一个自由度。因此，剩余 \((n-1)-p\) 个自由度用于利用残差估计 \(\sigma^2_{\epsilon}\)。回归有一个优雅的几何理论，其中响应被视为位于一个 \(n\) 维向量空间中，自由度是各个子空间的维度。然而，这里没有足够的篇幅来深入探讨此话题。

9.4.3 均方和 (MS) 与 \(F\) 检验

如上所述，ANOVA 表中的每个平方和都有一个对应的自由度。平方和与自由度的比值称为均方和：

\[ \text{均方和} = \frac{\text{平方和}}{\text{自由度}} \]

残差均方和是 \(\sigma^2_{\epsilon}\) 的无偏估计，由 (9.13) 给出，即

\[ \hat{\sigma}^2_{\epsilon} = \frac{\sum_{i=1}^n (Y_i - \hat{Y}_i)^2}{n-1-p} \tag{9.16} \]

\[ = \text{残差均方和} = \frac{\text{残差误差 SS}}{\text{残差自由度}} \]

其他均方和用于检验。假设我们有两个模型 I 和 II，且模型 I 中的预测变量是模型 II 中预测变量的子集，则模型 I 是模型 II 的子模型。一个常见的零假设是数据由模型 I 生成。等价地，在模型 II 中，对于不在模型 I 中的变量，其斜率为零。为了检验这个假设，我们使用模型 II 相对于模型 I 的超额回归平方和：

\[ \text{SS(II|I)} = \text{模型 II 的回归 SS} - \text{模型 I 的回归 SS} \]

\[ = \text{模型 I 的残差 SS} - \text{模型 II 的残差 SS} \tag{9.17} \]

等式 (9.17) 成立是因为 (9.15) 对所有模型都成立，特别是对模型 I 和模型 II 都成立。SS(II|I) 的自由度等于模型 II 中特有变量的个数，即

\[ \text{自由度} = \text{模型 II 的回归自由度} - \text{模型 I 的回归自由度} = p_{II} - p_I \]

第9章 回归：基础

在模型II中相比模型I多出的预测变量个数。均方记为MS(II|I)。换句话说，若pᵉ和pᵉᵉ分别为模型I和模型II的参数个数，则df₂₂₂₂₂₂ = pᵉᵉ−pᵉ，而MS(II|I) = SS(II|I)/df₂₂₂₂₂₂。用于检验原假设的F统计量为

\[F = \frac{\text{MS(II|I)}}{\hat{\sigma}_\epsilon^2},\]

其中\hat{\sigma}_\epsilon^2是模型II的残差均方。在原假设下，F统计量服从自由度为df₂₂₂₂₂₂和n−pᵉᵉ−1的F分布，若F统计量超过该F分布的α上分位数，则拒绝原假设。

9.4.4 调整 R²

\(R^2\) 倾向于偏向大模型，因为添加更多预测变量（即使它们与响应变量独立）总会增加 \(R^2\)。回忆：

\[ R^2 = 1 - \frac{\text{残差误差平方和}}{\text{总平方和}} = 1 - \frac{n^{-1} \text{残差误差平方和}}{n^{-1} \text{总平方和}}. \]

通过使用以下“调整”，即用相应的自由度替换两处的 \(n\)，可以减小 \(R^2\) 的偏倚：

\[ \text{调整 } R^2 = 1 - \frac{(n-p-1)^{-1} \text{残差误差平方和}}{(n-1)^{-1} \text{总平方和}} = 1 - \frac{\text{残差均方}}{\text{总均方}}. \]

调整 \(R^2\) 中存在 \(p\)，对预测变量数量进行了惩罚，因此当向模型添加预测变量时，调整 \(R^2\) 可能增加也可能减少。如果添加的变量使残差平方和减少到足以补偿 \(p\) 的增加，则调整 \(R^2\) 会增加。

9.5 模型选择

当存在许多潜在预测变量时，我们通常希望找到一个子集，以建立简约的回归模型。F 检验并不十分适合模型选择。问题之一在于可能存在大量 F 检验，且所有这些检验的联合统计行为是未知的。对于模型选择，使用模型选择准则（如 AIC 或 BIC）更为合适。对于线性回归模型，AIC 为

9.5 模型选择

\(AIC = n\log(\hat{\sigma}^2) + 2(1+p)\)，

其中 \(1+p\) 是包含 \(p\) 个预测变量的模型中的参数个数；截距项给出最后一个参数。BIC 用 \(\log(n)(1+p)\) 替换 AIC 中的 \(2(1+p)\)。第一项 \(n\log(\hat{\sigma}^2)\) 等于（在忽略一个不影响模型比较的加性常数下）\(-2\) 倍在 MLE 处评估的对数似然，假设噪声为高斯分布。

除了 AIC 和 BIC 外，还有两种专门用于回归的模型选择准则。一种是调整 \(R^2\)，我们之前已经见过。另一种是 \(C_p\)。\(C_p\) 与 AIC 相关，通常 \(C_p\) 和 AIC 由同一个模型最小化。使用 \(C_p\) 而非 AIC 的主要原因是某些回归软件只计算 \(C_p\) 而不计算 AIC——例如 R 语言 leaps 包中的 regsubsets() 函数，它将在接下来的例子中使用。

为了定义 \(C_p\)，假设有 \(M\) 个预测变量。令 \(\hat{\sigma}^2_{\epsilon,M}\) 是使用所有预测变量时的 \(\sigma^2_\epsilon\) 的估计量，令 \(SSE(p)\) 是仅使用 \(p \leq M\) 个预测变量的子集模型的残差平方和。如常，\(n\) 是样本量。那么 \(C_p\) 定义为

\[ C_p = \frac{SSE(p)}{\hat{\sigma}^2_{\epsilon,M}} - n + 2(p+1). \qquad (9.18) \]

当然，\(C_p\) 将依赖于所有包含 \(p\) 个预测变量的模型中具体选择哪一个，因此记号“\(C_p\)”可能并不理想。

对于 \(C_p\)、AIC 和 BIC，值越小越好，但对于调整 \(R^2\)，值越大越好。

不应盲目使用模型选择准则。模型选择应依据经济理论和实际考虑，以及模型选择准则。最终模型必须对用户有意义，这一点很重要。领域专业知识可能导致采纳一个并非根据所用准则最优的模型，而是一个略低于最优但更简约或具有更好经济依据的模型。

9.6 共线性与方差膨胀

如果两个或多个预测变量彼此高度相关，那么很难估计它们各自对响应变量的影响。例如，cm10_dif和cm30_dif的相关系数为0.96，散点图9.4显示它们高度相关。如果用aaa_dif对cm10_dif回归，调整\(R^2\)为0.7460，但加入cm30_dif作为第二个预测变量后，调整\(R^2\)仅增加到0.7556。这似乎表明cm30_dif可能与aaa_dif无关，但事实并非如此。实际上，当cm30_dif是唯一预测变量时，调整\(R^2\)为0.7376，说明cm30_dif对aaa_dif有很好的预测能力，与cm10_dif几乎相当。

预测变量之间高度相关的另一个影响是，每个变量的回归系数对另一个变量是否在模型中非常敏感。例如，当cm10_dif是唯一预测变量时，其系数为0.616，但若同时包含cm30_dif，则系数仅为0.360。

第9章 回归：基础

这里的问题在于，cm10_dif 和 cm30_dif 由于高度相关提供了冗余信息。这个问题被称为共线性，当预测变量多于两个时，则称为多重共线性。共线性会增加标准误。当模型中只有 cm10_dif 时，其 \(\beta\) 的标准误为 0.01212，但如果加入 cm30_dif，标准误增至 0.0451，增幅达 372%。

一个变量的方差膨胀因子（VIF）告诉我们，由于模型中包含其他预测变量，该变量的平方标准误（即 \(\hat{\beta}\) 的方差）增加了多少。例如，如果一个变量的 VIF 为 4，那么它的 \(\hat{\beta}\) 的方差是当其他预测变量被删除或与之不相关时的四倍。标准误增大了 2 倍。

假设我们有预测变量 \(X_1, \ldots, X_p\)。那么 \(X_j\) 的 VIF 可通过将 \(X_j\) 对其他 \(p-1\) 个预测变量进行回归得到。令 \(R^2_j\) 为该回归的 \(R^2\) 值，因此 \(R^2_j\) 度量了 \(X_j\) 可以被其他 \(X\) 预测的程度。那么 \(X_j\) 的 VIF 为

\[ \text{VIF}_j = \frac{1}{1 - R^2_j}. \]

\(R^2_j\) 接近 1 意味着大的 VIF。换句话说，\(X_j\) 能越准确地被其他 \(X\) 预测，它就越冗余，其 VIF 越高。VIF\(_j\) 的最小值为 1，当 \(R^2_j = 0\) 时出现。不幸的是，VIF\(_j\) 没有上界。当 \(R^2_j\) 趋近于 1 时，方差膨胀趋于无穷。

在解释 VIF 时，重要的是要记住，VIF\(_j\) 并未告诉我们响应变量与第 \(j\) 个预测变量之间的关系。相反，它只告诉我们第 \(j\) 个预测变量与其他预测变量的相关程度。事实上，VIF 可以在不知道响应变量值的情况下计算。

共线性通常的解决方法是使用 9.5 节讨论的模型选择准则之一来减少预测变量的数量。

9.7 偏残差图

偏残差图用于在去除其他预测变量影响的条件下，可视化某个预测变量对响应变量的效应。第\(j\)个预测变量的偏残差为

\[Y_i - \left( \hat{\beta}_0 + \sum_{j' \neq j} X_{i,j'} \hat{\beta}_{j'} \right) = \hat{Y}_i + \hat{\epsilon}_i - \left( \hat{\beta}_0 + \sum_{j' \neq j} X_{i,j'} \hat{\beta}_{j'} \right) = X_{i,j} \hat{\beta}_j + \hat{\epsilon}_i, \tag{9.19}\]

其中第一个等式利用了(9.12)，第二个等式利用了(9.10)。注意，(9.19)的左边表明，偏残差是减去了除第\(j\)个变量之外所有预测变量效应的响应变量。(9.19)的右边表明，偏残差也等于残差加上第\(j\)个变量的效应。偏残差图就是响应变量对这些偏残差的散点图。

9.8 预测变量的中心化

中心化，更准确地说，均值中心化（mean-centering）一个变量意味着将其表示为与均值的偏差。因此，如果 \(X_{1,k}, \ldots, X_{n,k}\) 是第 \(k\) 个预测变量的值，\(\bar{X}_k\) 是其均值，那么 \((X_{1,k} - \bar{X}_k), \ldots, (X_{n,k} - \bar{X}_k)\) 就是中心化后的预测变量的值。

中心化有两个作用：

  • 中心化可以减少多项式回归中的多重共线性；
  • 如果所有预测变量都经过中心化，那么 \(\beta_0\) 是当每个预测变量都等于其均值时 \(Y\) 的期望值。这赋予了 \(\beta_0\) 可解释的意义。相反，如果变量没有中心化，那么 \(\beta_0\) 是当所有预测变量都等于 0 时 \(Y\) 的期望值。通常，0 在某些预测变量的取值范围之外，这使得除非变量经过中心化，否则对 \(\beta_0\) 的解释几乎没有实际意义。

9.9 正交多项式

如上所述，中心化可以减少多项式回归中的共线性，因为例如，如果 X 为正，则 X 和 X2 会高度相关，但 X − X̄ 和 (X − X̄)2 的相关性会降低。

正交多项式可以完全消除相关性，因为它们被定义为不相关的。这是通过线性代数教材中讨论的 Gram–Schmidt 正交化过程实现的。大多数软件包都可以轻松创建正交多项式，例如使用 R 中的 poly() 函数。正交多项式对于次数高于 2 的多项式回归特别有用，因为中心化在减少共线性方面效果较差。然而，不鼓励使用 4 次及以上的多项式模型，建议改用非参数回归（见第 21 章）。即使三次回归也可能有问题，因为三次多项式的形状范围有限。

def stepAIC_py(model_full, data):
    import numpy as np
    import statsmodels.formula.api as smf
    current_vars = list(model_full.params.index[1:])
    best_model = model_full
    best_aic = model_full.aic
    while True:
        improved = False
        for var in current_vars:
            vars_reduced = [v for v in current_vars if v != var]
            if not vars_reduced:
                formula = 'consumption ~ 1'
            else:
                formula = 'consumption ~ ' + ' + '.join(vars_reduced)
            model_reduced = smf.ols(formula, data=data).fit()
            if model_reduced.aic < best_aic:
                best_aic = model_reduced.aic
                best_model = model_reduced
                current_vars = vars_reduced
                improved = True
                break
        if not improved:
            break
    return best_model
