第11章 回归：高级专题

11.1 线性回归的理论基础

本节提供关于线性最小二乘估计的一些理论结果。使用矩阵可以方便地研究线性回归。方程 (9.1) 可以更简洁地写为

\[Y_i = \mathbf{x}_i^T \boldsymbol{\beta} + \epsilon_i, \quad i = 1, \dots, n \qquad (11.1)\]

其中 \(\mathbf{x}_i = (1, X_{i,1}, \dots, X_{i,p})^T\) 且 \(\boldsymbol{\beta} = (\beta_0, \beta_1, \dots, \beta_p)^T\)。令

\[\mathbf{Y} = \begin{pmatrix} Y_1 \\ \vdots \\ Y_n \end{pmatrix}, \quad \mathbf{X} = \begin{pmatrix} \mathbf{x}_1 \\ \vdots \\ \mathbf{x}_n \end{pmatrix}, \quad \boldsymbol{\epsilon} = \begin{pmatrix} \epsilon_1 \\ \vdots \\ \epsilon_n \end{pmatrix}.\]

则 (11.1) 中的 \(n\) 个方程可表示为

\[\underbrace{\mathbf{Y}}_{n \times 1} = \underbrace{\mathbf{X}}_{n \times (p+1)} \underbrace{\boldsymbol{\beta}}_{(p+1) \times 1} + \underbrace{\boldsymbol{\epsilon}}_{n \times 1}, \qquad (11.2)\]

其中矩阵维度由下括号标出。

\(\boldsymbol{\beta}\) 的最小二乘估计最小化

\[\|\mathbf{Y} - \mathbf{X}\boldsymbol{\beta}\|^2 = (\mathbf{Y} - \mathbf{X}\boldsymbol{\beta})^T (\mathbf{Y} - \mathbf{X}\boldsymbol{\beta}) = \mathbf{Y}^T \mathbf{Y} - 2\boldsymbol{\beta}^T \mathbf{X}^T \mathbf{Y} + \boldsymbol{\beta}^T \mathbf{X}^T \mathbf{X} \boldsymbol{\beta}. \qquad (11.3)\]

将 (11.3) 关于 \(\beta_0, \dots, \beta_p\) 的导数设为 0 并简化所得方程，可得最小二乘估计量为

\[\hat{\boldsymbol{\beta}} = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \mathbf{Y}. \qquad (11.4)\]

利用(7.9)式，可以求出 \(\hat{\boldsymbol{\beta}}\) 的协方差矩阵：

\[ \operatorname{COV}(\hat{\boldsymbol{\beta}} \mid x_1, \ldots, x_n) = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \operatorname{COV}(\mathbf{Y} \mid x_1, \ldots, x_n) \mathbf{X} (\mathbf{X}^T \mathbf{X})^{-1} \] \[ = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T (\sigma_\epsilon^2 \mathbf{I}) \mathbf{X} (\mathbf{X}^T \mathbf{X})^{-1} = \sigma_\epsilon^2 (\mathbf{X}^T \mathbf{X})^{-1}, \]

因为 \(\operatorname{COV}(\mathbf{Y} \mid x_1, \ldots, x_n) = \operatorname{COV}(\boldsymbol{\epsilon}) = \sigma_\epsilon^2 \mathbf{I}\)，其中 \(\mathbf{I}\) 是 \(n \times n\) 单位矩阵。因此，\(\hat{\beta}_j\) 的标准误是 \(\sigma_\epsilon^2 (\mathbf{X}^T \mathbf{X})^{-1}\) 第 \(j\) 个对角线元素的平方根。

拟合值的向量为

\[ \hat{\mathbf{Y}} = \mathbf{X} \hat{\boldsymbol{\beta}} = \{ \mathbf{X} (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \} \mathbf{Y} = \mathbf{H} \mathbf{Y}, \]

其中 \(\mathbf{H} = \mathbf{X} (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T\) 是投影矩阵。第 \(i\) 个观测的杠杆值为 \(H_{ii}\)，即 \(\mathbf{H}\) 的第 \(i\) 个对角线元素。

11.1.1 回归的极大似然估计

在本节中，我们假设线性回归模型的噪声可能不是正态分布且不独立。

例如，考虑独立同分布误差的特殊情况。将尺度参数显式地放入回归模型中是有用的，因此我们假设

\[ Y_i = \mathbf{x}_i^T \boldsymbol{\beta} + \sigma \epsilon_i, \]

其中 \(\{\epsilon_i\}\) 独立同分布，具有已知密度 \(f\)，其方差为 1，而 \(\sigma\) 是未知的噪声标准差。例如，\(f\) 可以是标准化的 t 分布。那么 \(Y_1, \ldots, Y_n\) 的似然为

\[ \prod_{i=1}^n \sigma^{-1} f\left( \frac{Y_i - \mathbf{x}_i^T \boldsymbol{\beta}}{\sigma} \right). \]

极大似然估计最大化对数似然

\[ L(\boldsymbol{\beta}, \sigma) = -n \log(\sigma) + \sum_{i=1}^n \log\left[ f\left( \frac{Y_i - \mathbf{x}_i^T \boldsymbol{\beta}}{\sigma} \right) \right]. \]

对于正态分布误差，有 \(\log\{f(x)\} = -\frac{1}{2} x^2 - \frac{1}{2} \log(2\pi)\)，为了最大化，常数 \(-\frac{1}{2} \log(2\pi)\) 可以忽略。因此，对数似然为

\[ L_{\text{GAUSS}}(\boldsymbol{\beta}, \sigma) = -n \log(\sigma) - \frac{1}{2} \sum_{i=1}^n \left( \frac{Y_i - \mathbf{x}_i^T \boldsymbol{\beta}}{\sigma} \right)^2. \]

显然，最小二乘估计是 \(\boldsymbol{\beta}\) 的 MLE。此外，在 \(\boldsymbol{\beta}\) 已被最小二乘估计替代的条件下，最大化关于 \(\sigma\) 的 \(L_{\text{GAUSS}}(\hat{\boldsymbol{\beta}}, \sigma)\) 是一个标准的微积分练习，结果为

\(\hat{\sigma}^2_{\text{MLE}} = n^{-1} \sum_{i=1}^{n} (Y_i - \mathbf{x}_i^T \hat{\beta})^2\)。

可以证明，\(\hat{\sigma}^2_{\text{MLE}}\) 是有偏的，但是若将 \(n^{-1}\) 替换为 \(\{n - (p+1)\}^{-1}\)，其中 \(p+1\) 是 \(\beta\) 的维数，则偏倚得以消除。这给出了估计量 (9.16)。

现在假设 \(\epsilon\) 的协方差矩阵为 \(\Sigma\)，且对于某个函数 \(f\)，其密度为

\[|\Sigma|^{-1/2} f\bigl\{(\mathbf{Y} - \mathbf{X}\beta)^T \Sigma^{-1} (\mathbf{Y} - \mathbf{X}\beta)\bigr\}.\]

则对数似然为

\[-\frac{1}{2} \log |\Sigma| + \log \Bigl[ f\bigl\{(\mathbf{Y} - \mathbf{X}\beta)^T \Sigma^{-1} (\mathbf{Y} - \mathbf{X}\beta)\bigr\} \Bigr].\]

在重要特例中，若 \(\epsilon\) 服从均值为零的多元正态分布，则 \(\epsilon\) 的密度为

\[\left[ \frac{1}{|\Sigma|^{1/2} (2\pi)^{p/2}} \right] \exp\left\{ -\frac{1}{2} \epsilon^T \Sigma^{-1} \epsilon \right\}, \qquad (11.5)\]

如果 \(\Sigma\) 已知，则 \(\beta\) 的极大似然估计量（MLE）最小化

\[(\mathbf{Y} - \mathbf{X}\beta)^T \Sigma^{-1} (\mathbf{Y} - \mathbf{X}\beta)\]

并被称为广义最小二乘估计量（GLS 估计量）。如果 \(\epsilon_1, \dots, \epsilon_n\) 不相关但方差不相等，则 \(\Sigma\) 是这些方差的对角矩阵，此时广义最小二乘估计量即为加权最小二乘估计量 (10.4)。

GLS 估计量为

\[\hat{\beta}_{\text{GLS}} = (\mathbf{X}^T \Sigma^{-1} \mathbf{X})^{-1} \mathbf{X}^T \Sigma^{-1} \mathbf{Y}. \qquad (11.6)\]

通常，\(\Sigma\) 未知，必须用估计量替代，例如基于误差的 ARMA 模型。

11.2 非线性回归

通常，我们可以从经济理论推导出预测变量与响应变量之间的理论模型，但所推导的模型往往不是线性的。在金融领域，源自经济理论的模型尤为常见，其中许多是非线性的。非线性回归模型为

\[Y_i = f(\mathbf{X}_i; \boldsymbol{\beta}) + \epsilon_i, \qquad (11.7)\]

其中 \(Y_i\) 是第 \(i\) 次观测的响应变量，\(\mathbf{X}_i\) 是第 \(i\) 次观测的预测变量向量，\(f(\cdot; \cdot)\) 是已知函数，

函数，\(\boldsymbol{\beta}\) 是未知参数向量，且 \(\epsilon_1,\ldots,\epsilon_n\) 独立同分布，均值为 0，方差为 \(\sigma_\epsilon^2\)。最小二乘估计 \(\hat{\boldsymbol{\beta}}\) 最小化

\[ \sum_{i=1}^n \{Y_i - f(\mathbf{X}_i; \boldsymbol{\beta})\}^2 . \]

预测值为 \(\hat{Y}_i = f(\mathbf{X}_i; \hat{\boldsymbol{\beta}})\)，残差为 \(\hat{\epsilon}_i = Y_i - \hat{Y}_i\)。由于模型是非线性的，寻找最小二乘估计需要非线性优化。由于非线性回归的重要性，几乎每个统计软件包都包含非线性最小二乘估计的例程。这意味着大部分复杂的编程工作已经为我们完成。然而，我们需要编写一个方程来指定所使用的模型1。相比之下，使用线性回归时只需指定预测变量。

例 11.1. 模拟债券价格

考虑由某个借款人（例如联邦政府或公司）发行的面值 1000 美元的零息债券的价格。假设有多个到期时间，第 \(i\) 个记为 \(T_i\)。同时假设到期收益率为常数，记为 \(r\)。假设对于所有 \(T\) 都有 \(Y_T = r\) 并不现实，仅用于保持本例简单。在第 11.3 节中将使用更现实的模型。

利率 \(r\) 由市场决定，并可以从价格估计。在 \(r\) 为常数的假设下，到期时间为 \(T_i\) 的债券的当前价格为

\[ P_i = 1000 \exp(-r T_i). \tag{11.8} \]

观测到的价格存在一些随机变化。一个原因是债券的价格只能通过债券的出售来确定，因此观测到的价格并不是同时确定的。可能不再反映当前市场价值的价格被称为陈旧价格。每个债券的价格是在该到期期限的债券最后一次交易时确定的，而当时 \(r\) 可能略有不同。仅作为到期时间的函数时 \(r\) 被假定为常数，因此 \(r\) 可能随日历时间变化。因此，我们在模型 (11.8) 中加入噪声项，得到回归模型

\[ P_i = 1000 \exp(-r T_i) + \epsilon_i. \tag{11.9} \]

\(r\) 的估计可以通过最小二乘法确定，即最小化关于 \(r\) 的平方和：

\[ \sum_{i=1}^n \left\{P_i - 1,\!000 \exp(-r T_i)\right\}^2 . \]
  1. 有时甚至可以避免这项工作，因为某些非线性回归软件已经预先编程了许多标准模型。 ↩

最小二乘估计量记作 \(\hat{r}\)。

由于市场数据中的 \(r\) 不太可能是常数，本例使用模拟数据。数据生成时 \(r\) 固定为 0.06，并绘制于图11.1。使用 R 的 nls() 函数求 \(r\) 的非线性最小二乘估计。非线性优化需要参数的初始值，此处 \(r\) 的初始值取 0.04。

bondprices = read.table("bondprices.txt", header = TRUE)
attach(bondprices)
fit = nls(price ~ 1000 * exp(-r * maturity), start = list(r = 0.04))

detach(bondprices)
import pandas as pd
from scipy.optimize import curve_fit
import numpy as np

bondprices = pd.read_table("bondprices.txt")
# Define the model function
def model(maturity, r):
    return 1000 * np.exp(-r * maturity)
# Fit using curve_fit (nonlinear least squares)
popt, pcov = curve_fit(model, bondprices['maturity'], bondprices['price'], p0=[0.04])
# popt contains the parameter estimates
r_hat = popt[0]
# Compute standard errors
se = np.sqrt(np.diag(pcov))
t_stat = r_hat / se
# Print summary
print(f"Estimated r: {r_hat:.5f}")
print(f"Std. Error: {se[0]:.5f}")
print(f"t value: {t_stat[0]:.2f}")
# Residuals
residuals = bondprices['price'] - model(bondprices['maturity'], *popt)
print(f"Residual standard error: {np.std(residuals, ddof=2):.2f}")

如前所述，在非线性回归中，回归函数的形式是非线性的，但已知其形式只依赖于少数未知参数。例如，模型 (11.9) 中的回归函数具有指数形式。因此，非线性回归最好被称为非线性参数回归，以区别于非参数回归，后者回归函数也是非线性的，但未知参数形式未知。非参数回归将在第21章讨论。

多项式回归看起来可能是非线性的，因为多项式是非线性函数。例如，二次回归模型

\[ Y_i = \beta_0 + \beta_1 X_i + \beta_2 X_i^2 + \epsilon_i \tag{11.10} \]

在 \(X_i\) 上是非线性的。然而，通过将 \(X_i^2\) 定义为第二个预测变量，该模型在 \((X_i, X_i^2)\) 上是线性的，因此是多元线性回归的一个例子。使模型 (11.10) 成为线性模型的原因在于，右边是参数 \(\beta_0, \beta_1, \beta_2\) 的线性函数，因此可以通过适当定义变量，将其解释为线性回归。相反，指数模型

\[ Y_i = \beta_0 e^{\beta_1 X_i} + \epsilon_i \]

在参数 \(\beta_1\) 上是非线性的，因此不能通过重新定义预测变量将其转化为线性模型。

\[ \Pr\{\text{default} \mid \text{rating}\} = \exp\{\beta_0 + \beta_1 \text{rating}\}. \tag{11.11} \]

为了使用该模型，他们对违约频率的对数拟合了一个线性函数。这样做的一个难题是，有六个违约频率为零，导致对数变换为负无穷。

Bluhm、Overbeck和Wagner (2003) 通过将等于零的违约频率标记为“未观测到”并在估计过程中不使用它们来解决这一问题。他们方法的缺点在于删除了观测到的最低违约频率的数据，从而高估了违约概率。他们认为，观测到的违约频率为零并不意味真实违约概率为零，这当然正确。然而，即使违约频率为零，它仍是真实违约概率的无偏估计。这里无意批评他们的书——该书写得很好且有用——但我们可以使用模型(11.11)的非线性回归来避免他们方法带来的偏倚。通过非线性回归拟合(11.11)的优点在于避免了使用对数变换，从而允许使用所有数据，包括违约频率为零的数据。Bluhm、Overbeck和Wagner方法与使用模型(11.11)的非线性回归拟合结果如图11.2b所示，纵轴为对数尺度，以使拟合函数呈线性。注意，在信用评级良好时，非线性回归估计的违约概率低于Bluhm、Overbeck和Wagner的有偏方法。两组估计违约概率之间的差异可能相当大。Bluhm、

11.3 从零息债券价格估计远期利率

我们将这些估计称为经验远期利率估计。图11.4显示了即将在例11.3中描述的数据的价格和经验远期利率估计。如图所示，当（11.12）中的分母由于到期期限非常接近而变得很小时，经验远期利率估计可能相当嘈杂。如果到期期限更分散，那么偏倚而非方差将成为主要问题。尽管存在这些困难，经验远期利率估计仍能给出远期利率曲线的总体印象，并且对于与参数模型（接下来讨论）的估计进行比较是有用的。

我们可以使用非线性回归从债券价格估计 \(r(t)\)。在第11.2节中给出了一个估计 \(r(t)\) 的例子，假设 \(r(t)\) 是常数，并使用不同到期期限的零息债券价格作为数据。在本节中，我们在不假设 \(r(t)\) 为常数的情况下估计 \(r(t)\)。

远期利率曲线的参数估计始于一个参数族 \(r(t;\theta)\) 的远期利率和相应的收益率曲线

\[ y_T(\theta) = T^{-1} \int_0^T r(t;\theta)\,dt \]

以及面值1美元债券的价格模型：

\[ P_T(\theta) = \exp\{-T y_T(\theta)\} = \exp\left(-\int_0^T r(t;\theta)\,dt\right). \]

例如，假设 \(r(t;\theta)\) 是一个 \(p\) 次多项式，即

\[ r(t;\theta) = \theta_0 + \theta_1 t + \cdots + \theta_p t^p \]

对于某些未知参数 \(\theta_0, \ldots, \theta_p\)。那么

远期利率曲线的积分形式为

\[ \int_0^T r(t;\theta)\,dt = \theta_0 T + \theta_1 \frac{T^2}{2} + \cdots + \theta_p \frac{T^{p+1}}{p}, \]

收益率曲线为

\[ y_T = T^{-1} \int_0^T r(t;\theta)\,dt = \theta_0 + \theta_1 \frac{T}{2} + \cdots + \theta_p \frac{T^p}{p}. \]

一个流行的模型是Nelson-Siegel族，其远期利率和收益率曲线为

\[ r(t;\theta) = \theta_0 + (\theta_1 + \theta_2 t)\exp(-\theta_3 t), \] \[ y_t(\theta) = \theta_0 + \left( \theta_1 + \frac{\theta_2}{\theta_3} \right) \frac{1-\exp(-\theta_3 t)}{\theta_3 t} - \frac{\theta_2}{\theta_3} \exp(-\theta_3 t). \]

六参数Svensson模型通过添加项 \(\theta_4 t \exp(-\theta_5 t)\) 到远期利率来扩展Nelson-Siegel模型。

用于估计远期利率曲线的非线性回归模型指出，样本中第 \(i\) 只债券的价格（以面值的百分比表示），其到期时间为 \(T_i\)，为

\[ P_i = D(T_i) + \epsilon_i = \exp\left( -\int_0^{T_i} r(t;\theta)\,dt \right) + \epsilon_i, \qquad (11.13) \]

其中 \(D\) 是贴现函数，\(\epsilon_i\) 是由于价格可能稍有滞后以及买卖价差等问题产生的“误差”。2

11.4 变换-两侧回归

Svensson 模型可能比 Nelson–Siegel 模型拟合得更好，但四参数三次多项式和二次样条模型已经拟合得足够好，因此没有必要尝试六参数的 Svensson 模型。□

假设我们有一个理论模型，在没有噪声的情况下， \[ Y_i = f(X_i; \beta). \qquad (11.18) \] 模型 (11.18) 与以下模型相同： \[ h\{Y_i\} = h\{f(X_i; \beta)\}, \qquad (11.19) \] 其中 \(h\) 是任意一对一的函数，例如严格递增函数。在没有噪声的情况下，任意选择 \(h\) 都是一样的，我们不妨继续使用模型 (11.18)，但当存在噪声时，情况就不同了。

当数据带有噪声时，式 (11.19) 可以转化为非线性回归模型 \[ h\{Y_i\} = h\{f(X_i; \beta)\} + \epsilon_i. \qquad (11.20) \] 模型 (11.20) 称为变换-两侧 (TBS) 回归模型，因为式 (11.19) 的两侧都被同一个函数 \(h\) 变换了。通常，\(h\) 是 Box–Cox 变换之一，选择 \(h\) 是为了稳定方差并使误差近似服从正态分布。对于固定的 \(h\)，估计 \(\beta\) 时最小化 \[ \sum_{i=1}^n \left[ h\{Y_i\} - h\{f(X_i; \hat{\beta})\} \right]^2. \qquad (11.21) \] 不同的 \(h\) 选择可以通过残差图进行比较。选择能使残差近似正态分布且方差恒定的 \(h\) 用于最终分析。

11.4.1 TBS的工作原理

TBS本质上是对数据进行加权。为了理解这一点，我们使用泰勒级数线性化4得到：

\[ \sum_{i=1}^n \left[ h(Y_i) - h\{ f(X_i; \hat{\beta}) \} \right]^2 = \sum_{i=1}^n \left[ h^{(1)}\{ f(X_i; \hat{\beta}) \} \right]^2 \left\{ Y_i - f(X_i; \hat{\beta}) \right\}^2. \]

第 \(i\) 个观测的权重为 \(\left[ h^{(1)}\{ f(X_i; \hat{\beta}) \} \right]^2\)。由于最佳权重是方差的倒数，最合适的变换 \(h\) 满足：

\[ \operatorname{Var}(Y_i \mid X_i) \propto \left[ h^{(1)}\{ f(X_i; \hat{\beta}) \} \right]^{-2}. \tag{11.23} \]

例如，若 \(h(y) = \log(y)\)，则 \(h^{(1)}(y) = 1/y\)，(11.23) 变为：

\[ \operatorname{Var}(Y_i \mid X_i) \propto \{ f(X_i; \hat{\beta}) \}^2, \tag{11.24} \]

即响应的条件标准差与其条件均值成正比。这种情况经常出现。例如，若响应服从指数分布，则 (11.24) 必然成立。若响应服从对数正态分布且对数方差为常数，(11.24) 也成立。在这种情况下，对数变换是最优的就不足为奇了，因为对数变换将数据转换为独立同分布的正态噪声。

随机变量的变异系数是其标准差与期望值之比。当 (11.24) 成立时，响应具有常数变异系数。

使条件方差保持恒定的变换称为方差稳定化变换。我们刚刚证明，当变异系数为常数时，方差稳定化变换就是对数变换。

例11.5 泊松响应

假设 \(Y_i \mid X_i\) 服从均值为 \(f(X_i; \beta)\) 的泊松分布，例如 \(Y_i\) 可能是宣告破产的公司数量，

  1. 函数 \(h\) 在点 \(x\) 处的泰勒级数线性化为 \(h(y) \approx h(x) + h^{(1)}(x)(y - x)\)，其中 \(h^{(1)}\) 是 \(h\) 的一阶导数。有关泰勒级数的更多讨论，请参阅任何微积分教材。 ↩

在一年的时间内，以 \(f(\mathbf{X}_i; \boldsymbol{\beta})\) 对该期望数如何依赖于 \(\mathbf{X}_i\) 中的宏观经济变量进行建模。对于泊松分布，方差等于均值，因此 \(\operatorname{Var}(Y_i|\mathbf{X}_i) = f(\mathbf{X}_i; \boldsymbol{\beta})\)。使用与前一个示例相同的推理，可以得出应使用 \(\alpha = 1/2\)；平方根变换是泊松分布响应变量的方差稳定变换。 □

11.5 仅对响应变量进行变换

所谓的 Box–Cox 变换模型为

\[ Y_i^{(\alpha)} = \beta_0 + X_{i,1}\beta_1 + \cdots + X_{i,p}\beta_p + \epsilon_i,\tag{11.25} \]

其中 \(\epsilon_1,\ldots,\epsilon_n\) 为独立同分布 \(N(0,\sigma_\epsilon^2)\)。与 TBS 模型不同，这里仅对响应变量进行变换。变换响应变量的目标是实现以下三个目的：

  1. 简单的模型：\(Y_i^{(\alpha)}\) 在预测变量 \(X_{i,1},\ldots,X_{i,p}\) 和参数 \(\beta_1,\ldots,\beta_p\) 中是线性的；
  2. 常数残差方差；
  3. 高斯噪声。

相比之下，TBS 模型的目标是 2 和 3，但不包括 1。

模型 (11.25) 由 Box 和 Cox (1964) 提出，他们建议通过极大似然估计 \(\alpha\)。R 的 MASS 包中的 \(\texttt{boxcox()}\) 函数可以计算 \(\alpha\) 的轮廓对数似然函数以及置信区间。通常，\(\hat{\alpha}\) 取为置信区间内的某个取整数值，例如 \(-1,\,-1/2,\,0,\,1/2\) 或 \(1\)。选择这些数值之一的原因是，其变换易于解释，即平方根、对数、倒数或其他常见函数。当然，如果不关心变换是否常见，也可以直接取最大化轮廓对数似然函数的 \(\alpha\) 值。通过这种方式选定 \(\hat{\alpha}\) 后，可以对 \(Y_i^{(\hat{\alpha})}\) 关于 \(X_{i,1},\ldots,X_{i,p}\) 进行回归，从而估计 \(\beta_0,\ldots,\beta_p\) 和 \(\sigma_\epsilon^2\)。

11.6 二元回归

二元响应 \(Y\) 只能取两个值，0 或 1，它们编码两种可能的结果，例如，公司贷款违约或不违约。二元回归在给定预测变量 \(X_{i,1},\ldots,X_{i,p}\) 值的条件下，对二元响应为1的条件概率进行建模。由于概率被限制在0和1之间，线性模型不适用于二元响应。然而，线性模型非常方便，因此人们希望有一个具有线性模型许多特征的模型。这推动了广义线性模型（通常称为GLM）的发展。

用于二元响应的广义线性模型的形式为： \[P(Y_i = 1 \mid X_{i,1},\ldots,X_{i,p}) = H(\beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p}) = H(\mathbf{x}_i^{\mathrm{T}} \boldsymbol{\beta}),\] 其中 \(H(x)\) 是随着 \(x\) 从 \(-\infty\) 增加到 \(\infty\) 而从0增加到1的函数，因此 \(H(x)\) 是一个累积分布函数（CDF），最后一个表达式使用了(11.1)的向量记号。最常见的用于二元响应的GLM是probit回归，其中 \(H(x) = \Phi(x)\)（即 \(N(0,1)\) 的CDF）；以及逻辑回归，其中 \(H(x)\) 是逻辑CDF，即 \(H(x) = 1 / \{1 + \exp(-x)\}\)。参数向量 \(\boldsymbol{\beta}\) 可以通过极大似然估计得到。假设在给定 \(x_1,\ldots,x_n\) 的条件下，二元响应 \(Y_1,\ldots,Y_n\) 相互独立。那么，利用(A.8)，似然函数（条件于 \(x_1,\ldots,x_n\)）为： \[\prod_{i=1}^n H(\mathbf{x}_i^{\mathrm{T}} \boldsymbol{\beta})^{Y_i} \left\{1 - H(\mathbf{x}_i^{\mathrm{T}} \boldsymbol{\beta})\right\}^{1-Y_i}. \qquad (11.27)\] MLE可以通过标准软件求得，例如R中的 \(\texttt{glm()}\) 函数。

11.7 非线性模型的线性化

有时，通过对模型和响应变量同时进行变换，可以将非线性模型线性化。在这种情况下，是应该使用线性化变换，还是对原始模型应用非线性回归呢？答案是线性化有时是好事，但并非总是如此。幸运的是，残差分析可以帮助我们判断是否应该使用线性化变换。

例如，考虑模型

\[ Y_i = \beta_1 \exp(\beta_2 X_i). \tag{11.28} \]

该模型与线性模型“等价”

\[ \log(Y_i) = \alpha + \beta_2 X_i, \tag{11.29} \]

其中 \(\alpha = \log(\beta_1)\)。“等价”加了引号，因为在存在噪声时这两个模型不再等价。

假设模型(11.28)具有独立同分布的加性噪声，因此

\[Y_i = \beta_1 \exp(\beta_2 X_i) + \epsilon_i, \quad (11.30)\]

其中\(\epsilon_1,\dots,\epsilon_n\)独立同分布。那么对(11.29)应用对数变换得到模型

\[\log(Y_i) = \log\{\beta_1 \exp(\beta_2 X_i) + \epsilon_i\} \quad (11.31)\]

该模型具有非加性噪声。由于噪声非加性，即使\(\epsilon_1,\dots,\epsilon_n\)为独立同分布的高斯噪声，\(\log(Y_i)\)围绕模型\(\log\{\beta_1 \exp(\beta_2 X_i)\}\)的波动也会表现出非恒定变异和偏度。

例11.8 线性化变换——模拟数据

图11.12a显示了来自模型(11.28)的一个模拟样本，其中\(\beta_1 = 1\)，\(\beta_2 = -1\)，\(\sigma_\epsilon = 0.02\)。\(X_i\)从\(-1\)到\(2.5\)等间距取值，步长为0.025。面板(b)显示了\(\log(Y_i)\)对\(X_i\)的散点图。可以看出，变换使变量间的关系线性化，但引入了非恒定的残差变异。面板(c)和(d)展示了使用线性化模型后的残差图。注意正态概率图的非线性以及严重的非恒定方差。□

线性化并不总是一件坏事。假设噪声是乘性的且服从对数正态分布，那么 (11.28) 变为

\[Y_i = \beta_1 \exp(\beta_2 X_i) \exp(\epsilon_i) = \beta_1 \exp(\beta_2 X_i + \epsilon_i), \tag{11.32}\]

其中 \(\epsilon_1, \dots, \epsilon_n\) 是独立同分布的高斯分布。那么对数变换将 (11.32) 转化为

\[\log(Y_i) = \alpha + \beta_2 X_i + \epsilon_i, \tag{11.33}\]

这是一个满足所有通常假设的线性模型。

11.8 稳健回归

总之，线性化变换是否能使数据更好地符合回归分析的假设是不确定的。残差分析可以帮助判断变换是否恰当。

稳健回归估计量应对两类异常值相对不敏感。第一类是 不良数据，即不属于总体的污染物，例如未检测到的记录错误。第二类是因噪声分布具有厚尾而产生的异常值。存在大量稳健回归估计量，其数量之多已成为使用的障碍。许多数据分析师困惑于哪种稳健估计量最好，因而不愿使用任何方法。我们不介绍可能加剧这一问题的众多估计量，而只提一种：最小修剪平方和估计量，通常称为 LTS。

回忆单变量样本位置参数的稳健估计量——修剪均值。修剪均值简单来说就是移除一定比例的最大观测值和同等比例的最小观测值后样本的均值。这种修剪移除了一些非异常值，在无异常值的理想条件下会损失一些精度，但不会达到不可接受的程度。修剪也移除了异常值，从而使估计量具有稳健性。对于单变量样本，修剪是容易的，因为我们知道要修剪哪些观测值——最大和最小的那些。但在回归中情况并非如此。考虑图 11.13 中的数据。有 26 个观测值紧密沿着一条直线分布，加上两个远离该直线的残差异常值。注意，这些残差异常值既没有极端的 X 值也没有极端的 Y 值。它们仅相对于其他数据的线性回归拟合而言是异常值。

在图 11.13 中残差异常值是明显的，因为只有一个预测变量。当有多个预测变量时，异常值只有在我们有一个模型以及该模型中参数的良好估计时才能被识别。于是困难在于：参数估计需要识别异常值，反之亦然。从图中可以看出，最小二乘直线因将残差异常值包含在用于拟合的数据中而发生了变化

LTS估计量能够同时识别残差异常值并稳健地估计模型参数。设 \(0<\alpha\leq 1/2\) 为修剪比例，并令 \(k\) 等于 \(n\alpha\) 四舍五入后的整数。关于一组回归参数值的修剪平方和定义如下：根据这些参数值计算模型残差，将残差平方，然后对平方残差排序并去掉最大的 \(k\) 个，最后对剩余的平方残差求和。LTS估计就是使修剪平方和最小的那组参数值。LTS估计量可通过R语言的 robust 包中的 ltsReg() 函数计算。

如果噪声分布是厚尾的，那么稳健回归分析的一种替代方法是使用厚尾分布作为噪声模型，然后通过极大似然估计参数。例如，可以假设噪声服从双指数分布或 \(t\) 分布。在后一种情况下，既可以估计自由度，也可以简单地将自由度固定为一个较小的值，这意味着更厚的尾部；参见 Lange, Little, and Taylor (1989)。这种策略被称为稳健建模，而非稳健估计。区别在于：稳健估计假设一个相当严格的模型（如正态噪声分布），但寻找极大似然的稳健替代方案；而稳健建模则使用更灵活的模型，使得极大似然估计本身变得稳健。当存在单个严重残差异常值（尤其是在高杠杆点处）时，稳健回归比带有厚尾噪声分布的MLE更可取；见下一示例。

另一种可能性是残差异常值源于非恒定标准差，异常值主要出现在噪声标准差较高的数据中。解决此问题的方法是应用方差稳定变换，或者对非恒定标准差进行建模，例如使用第14章讨论的某个GARCH模型。

11.9.1 最佳线性预测

通常我们观察到一个随机变量 \(X\)，并希望预测与之相关的未观测随机变量 \(Y\)。例如，\(Y\) 可能是某项资产的未来价格，而 \(X\) 可能是该资产价格的最新变动。预测有许多实际用途，在理论研究中也十分重要。

最小化期望平方预测误差的 \(Y\) 的预测变量是 \(\mathrm{E}(Y|X)\)（参见附录A.19），但 \(\mathrm{E}(Y|X)\) 通常是 \(X\) 的非线性函数且难以计算。对此困难的一个常见解决办法是只考虑 \(X\) 的线性函数作为可能的预测变量，这称为线性预测。在本节中，我们将展示线性预测与线性回归的密切关系。

11.9 回归与最佳线性预测

基于 \(X\) 的 \(Y\) 的线性预测变量是形式为 \(\beta_0 + \beta_1 X\) 的函数，其中 \(\beta_0\) 和 \(\beta_1\) 是我们可选的参数。最佳线性预测指找到 \(\beta_0\) 和 \(\beta_1\)，使得期望平方预测误差

\[\mathrm{E}\{Y - (\beta_0 + \beta_1 X)\}^2, \qquad (11.34)\]

最小化。这样做使得预测变量平均而言尽可能接近 \(Y\)。期望平方预测误差可以重写为

\[\mathrm{E}\{Y - (\beta_0 + \beta_1 X)\}^2 = \mathrm{E}(Y^2) - 2\beta_0 \mathrm{E}(Y) - 2\beta_1 \mathrm{E}(XY) + \beta_0^2 + 2\beta_0 \beta_1 \mathrm{E}(X) + \beta_1^2 \mathrm{E}(X^2).\]

为了找到极小化点，我们将该表达式的偏导数设为零，得到

\[ 0 = -\mathrm{E}(Y) + \beta_0 + \beta_1 \mathrm{E}(X), \tag{11.35} \] \[ 0 = -\mathrm{E}(XY) + \beta_0 \mathrm{E}(X) + \beta_1 \mathrm{E}(X^2). \tag{11.36} \]

经过一些代数运算，我们得到

\[ \beta_1 = \frac{\sigma_{XY}}{\sigma_X^2} \tag{11.37} \]

以及

\[ \beta_0 = \mathrm{E}(Y) - \beta_1 \mathrm{E}(X) = \mathrm{E}(Y) - \frac{\sigma_{XY}}{\sigma_X^2} \mathrm{E}(X). \tag{11.38} \]

可以验证，(11.34) 的二阶导数矩阵是正定的，因此满足 (11.35) 和 (11.36) 的解 (β0, β1) 最小化 (11.34)。因此，Y 的最佳线性预测为

\[ \hat{Y}_{\text{Lin}}(X) = \beta_0 + \beta_1 X = \mathbb{E}(Y) + \frac{\sigma_{XY}}{\sigma_X^2} \{ X - \mathbb{E}(X) \}. \tag{11.39} \]

在实际中，(11.39) 不能直接使用，除非 \(\mathbb{E}(X)\)、\(\mathbb{E}(Y)\)、\(\sigma_{XY}\) 和 \(\sigma_X^2\) 已知，但这通常不成立。线性回归分析本质上是用这些未知参数的最小二乘估计替换后使用 (11.39)——参见第 11.9.3 节。

11.9.2 最佳线性预测的预测误差

在本节中，假设 \(\hat{Y}\) 是 Y 的最佳线性预测。预测误差为 \(Y - \hat{Y}\)。容易证明 \(\mathbb{E}\{Y - \hat{Y}\} = 0\)，因此预测是无偏的。通过简单的代数运算，我们可以证明期望平方预测误差为

\[ \mathbb{E}\{Y - \hat{Y}\}^2 = \sigma_Y^2 - \frac{\sigma_{XY}^2}{\sigma_X^2} = \sigma_Y^2 (1 - \rho_{XY}^2). \tag{11.40} \]

X 在多大程度上有助于我们预测 Y？为回答此问题，首先注意，如果我们未观测到 X，那么我们必须用常数 c 来预测 Y。容易证明，最佳预测器的 c 等于 \(\mathbb{E}(Y)\)。首先注意到期望平方预测误差为 \(\mathbb{E}(Y - c)^2\)。一些代数运算表明

\[ \mathbb{E}(Y - c)^2 = \operatorname{Var}(Y) + \{ c - \mathbb{E}(Y) \}^2, \tag{11.41} \]

由于 \(\operatorname{Var}(Y)\) 不依赖于 c，因此期望平方预测误差在 \(c = \mathbb{E}(Y)\) 时最小。因此，当 X 未观测到时，Y 的最佳预测器是 \(\mathbb{E}(Y)\)，期望平方预测误差为 \(\sigma_Y^2\)；但当 X 观测到时，期望平方预测误差更小，为 \(\sigma_Y^2 (1 - \rho_{XY}^2)\)。因此，\(\rho_{XY}^2\) 是已知 X 时预测误差减少的比例。这是一个重要的事实，我们将会再次见到。

结果 11.1 当 Y 独立于所有可用信息时的预测：如果 Y 独立于当前所有可用信息，即 Y 独立于所有已观测到的随机变量，那么 Y 的最佳预测器是 \(\mathbb{E}(Y)\)，且平方预测误差的期望值为 \(\sigma_Y^2\)。我们说 Y“无法预测”，是指不存在优于其期望值的预测器。

11.9.3 回归是经验最优线性预测

对于单一预测变量的情形，注意最优线性预测 \[ \hat{Y} = E(Y) + \frac{\sigma_{XY}}{\sigma_X^2}\{X - E(X)\}, \] 与最小二乘直线 \[ \hat{Y} = \bar{Y} + \frac{s_{XY}}{s_X^2}(X - \bar{X}) \] 之间的相似性。最小二乘直线是最优线性预测的样本版本。此外，\(\rho_{XY}^2\)，即 \(X\) 和 \(Y\) 之间相关系数的平方，是 \(Y\) 中可用线性预测器预测的变异性比例，而 \(\rho_{XY}^2\) 的样本版本为 \(R^2 = r_{XY}^2 = r_{\hat{Y}Y}^2\)。

11.9.4 多元线性预测

到目前为止，我们假设只有单个随机变量 \(X\) 可用于预测 \(Y\)。更常见的情况是，使用一组观测到的随机变量 \(X_1, \dots, X_n\) 来预测 \(Y\)。

设 \(Y\) 和 \(X\) 分别为 \(p \times 1\) 和 \(q \times 1\) 的随机向量。如 7.3.1 节所述，定义 \[ \Sigma_{Y,X} = E\{Y - E(Y)\}\{X - E(X)\}^T, \] 因此 \(\Sigma_{Y,X}\) 的第 \(i,j\) 个元素是 \(Y_i\) 与 \(X_j\) 的协方差。那么，给定 \(X\) 时 \(Y\) 的最优线性预测为 \[ \hat{Y} = E(Y) + \Sigma_{Y,X} \Sigma_X^{-1} \{X - E(X)\}. \tag{11.42} \] 注意单变量与多元情形下的最优线性预测 (11.39) 和 (11.42) 之间的相似性。

多元线性预测的样本类比即为多元回归。

11.10 回归对冲

通过承担另一种可能方向相反的风险来对冲这种风险。为了对冲所持有债券的利率风险，做市商可以卖空其他流动性更强的债券。假设做市商决定卖空一种流动性更强的30年期国债。

回归对冲确定了用于对冲刚购买债券风险的最佳卖空30年期国债数量。目标是使得做多第一只债券并做空国债的组合价格尽可能不受收益率变化的影响。假设第一只债券的期限为25年。我们可以利用第3.8节的结果确定价格对收益率变化的敏感性。令\(y_{30}\)表示30年期债券的收益率，\(P_{30}\)表示面值1美元的30年期债券的价格，\(DUR_{30}\)表示久期。对于较小的\(\Delta y_{30}\)，价格变化\(\Delta P_{30}\)和收益率变化\(\Delta y_{30}\)的关系为：\(\Delta P_{30} \approx -P_{30} DUR_{30} \Delta y_{30}\)。对于25年期债券也有类似的结果。

考虑一个持有面值\(F_{25}\)的25年期债券并卖空面值\(F_{30}\)的30年期债券的组合。组合的价值为\(F_{25}P_{25} - F_{30}P_{30}\)。设\(\Delta y_{25}\)和\(\Delta y_{30}\)分别为收益率的变化，则组合价值的变化近似为： \[\{F_{30}P_{30}DUR_{30}\Delta y_{30} - F_{25}P_{25}DUR_{25}\Delta y_{25}\}. \tag{11.43}\]

假设\(\Delta y_{30}\)对\(\Delta y_{25}\)的回归为：\(\Delta y_{30} = \hat{\beta}_0 + \hat{\beta}_1 \Delta y_{25}\) \tag{11.44}，并且\(\hat{\beta}_0 \approx 0\)，这通常是对利率变化进行回归的情况，如例9.1所示。将(11.44)代入(11.43)，组合价格的变化近似为： \[\{F_{30}P_{30}DUR_{30}\hat{\beta}_1 - F_{25}P_{25}DUR_{25}\}\Delta y_{25}. \tag{11.45}\] 如果\(F_{30} = F_{25} \frac{P_{25}DUR_{25}}{P_{30}DUR_{30}\hat{\beta}_1}\) \tag{11.46}，则对于所有\(\Delta y_{25}\)的值，这一变化近似为零。

方程(11.46)告诉我们，为了对冲面值\(F_{25}\)的25年期债券，需要卖空多少面值的30年期债券。(11.46)右侧的所有量都是已知或容易计算的：\(F_{25}\)是当前25年期债券的头寸，\(P_{25}\)和\(P_{30}\)是已知的债券价格，\(DUR_{25}\)和\(DUR_{30}\)的计算在第3章中讨论，\(\hat{\beta}_1\)是\(\Delta y_{30}\)对\(\Delta y_{25}\)回归的斜率。

回归的\(R^2\)越高，对冲效果越好。使用两种或更多流动性债券（例如30年期和10年期）进行对冲可以通过多元回归实现，并且可能产生更好的对冲效果。

在本节中，你将拟合短期利率模型。令 \(r_t\) 为时刻 \(t\) 的短期利率（短期借贷的无风险利率）。假设短期利率满足随机微分方程

\[ dr_t = \mu(t, r_t) dt + \sigma(t, r_t) dW_t, \tag{11.47} \]

其中 \(\mu(t, r_t)\) 是漂移函数，\(\sigma(t, r_t)\) 是波动率函数，\(W_t\) 是标准布朗运动。我们使用 (11.47) 的离散近似：

\[ (r_t - r_{t-1}) = \mu(t-1, r_{t-1}) + \sigma(t-1, r_{t-1}) \epsilon_{t-1} \tag{11.48} \]

其中 \(\epsilon_1, \ldots, \epsilon_{n-1}\) 独立同分布，服从 \(N(0, 1)\)。

我们从 Chan, Karolyi, Longstaff 和 Sanders (1992) (CKLS) 模型开始，该模型假设

\[ \mu(t, r) = \mu(r) = a(\theta - r) \tag{11.49} \]

对于某些未知参数 \(a\) 和 \(\theta\)，以及

\[ \sigma(t, r) = \sigma r^\gamma \tag{11.50} \]

对于某些 \(\sigma\) 和 \(\gamma\)。注意区分波动率函数 \(\sigma(t, r)\) 和常数波动率参数 \(\sigma\)。

def backward_selection(X, y, threshold=0):
    cols = list(X.columns)
    while len(cols) > 0:
        model = sm.Logit(y, X[cols]).fit(disp=0)
        aic = model.aic
        aic_without = []
        for col in cols:
            if col == 'const':
                continue
            cols_temp = [c for c in cols if c != col]
            model_temp = sm.Logit(y, X[cols_temp]).fit(disp=0)
            aic_without.append((col, model_temp.aic))
        best = min(aic_without, key=lambda x: x[1])
        if best[1] < aic - threshold:
            cols.remove(best[0])
        else:
            break
    return cols

selected = backward_selection(X, y)
print("Selected variables:", selected)
fit2 = sm.Logit(y, X[selected]).fit()
print(fit2.summary())
