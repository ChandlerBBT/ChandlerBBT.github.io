第5章 单变量分布建模

5.1 引言

如第4章所见，金融时间序列的边缘分布通常不能很好地由正态分布拟合。幸运的是，有许多合适的替代模型，例如t分布、广义误差分布，以及t分布和广义误差分布的偏斜版本。所有这些都将在本章中介绍。通常，这些分布中的参数通过极大似然估计。第5.9和5.14节介绍了极大似然估计量（MLE），第5.18节提供了进一步学习该主题的参考文献。

极大似然的软件对于标准模型是现成的，并且仅对数据分析和建模感兴趣的读者通常不需要过于关注极大似然的技术细节。然而，在进行统计分析时，理解基础理论（至少是概念层面）总是有价值的，因为这可以防止误用。此外，当使用非标准模型时，通常没有自动计算MLE的软件，因此需要理解足够的理论来编写计算MLE的程序。

5.3 位置、尺度与形状参数

参数通常根据它们决定分布的哪些性质而被分类为位置参数、尺度参数或形状参数。位置参数是指在不改变分布形状或变异性情况下，将分布左右平移的参数。尺度参数量化了离差。对于单变量样本，如果当数据乘以 \(a\) 时参数增加 \(|a|\)，则该参数是尺度参数。因此，若 \(\sigma(X)\) 是随机变量 \(X\) 的尺度参数，则 \(\sigma(aX) = |a|\sigma(X)\)。只要标准差有限，尺度参数就是标准差的常数倍。在后续章节中可以找到许多位置参数和尺度参数的例子。如果 \(\lambda\) 是尺度参数，则 \(\lambda^{-1}\) 称为逆尺度参数。由于尺度参数量化离差，逆尺度参数则量化精度。

若 \(f(y)\) 为任意固定密度，则 \(f(y-\mu)\) 是以 \(\mu\) 为位置参数的分布族；\(\theta^{-1}f(y/\theta),\ \theta>0\)，是以 \(\theta\) 为尺度参数的分布族；而 \(\theta^{-1}f\{\theta^{-1}(y-\mu)\}\) 是以 \(\mu\) 为位置参数、\(\theta\) 为尺度参数的分布族。这些事实可通过如下推导得出：若 \(Y\) 具有密度 \(f(y)\) 且 \(\theta>0\)，则根据结果 A.1，\(Y+\mu\) 的密度为 \(f(y-\mu)\)，\(\theta Y\) 的密度为 \(\theta^{-1}f(\theta^{-1}y)\)，而 \(\theta Y+\mu\) 的密度为 \(\theta^{-1}f\{\theta^{-1}(y-\mu)\}\)。

形状参数定义为不会因位置和尺度变化而改变的任何参数。更精确地说，对于任意 \(f(y)\)、\(\mu\) 和 \(\theta>0\)，密度 \(f(y)\) 的形状参数值等于密度 \(\theta^{-1}f\{\theta^{-1}(y-\mu)\}\) 的同一形状参数值。\(t\) 分布的自由度参数和对数正态分布的对数标准差都是形状参数。本章后面还会遇到其他形状参数。形状参数常用于指定分布的偏度或尾部权重。

峰度指概率集中在分布中心尤其尾部而非“肩部”（中心与尾部之间的区域）的程度。

在第4.3.2节中，左尾定义为从 \(-\infty\) 到 \(\mu - 2\sigma\) 的区域，右尾定义为从 \(\mu + 2\sigma\) 到 \(+\infty\) 的区域。这里的 \(\mu\) 和 \(\sigma\) 可以是均值与标准差，或中位数与MAD。诚然，这些定义有些武断。关于中心和肩部的合理定义是：中心为 \(\mu - \sigma\) 到 \(\mu + \sigma\) 的区域，左...

左肩部是从\(\mu - 2\sigma\)到\(\mu - \sigma\)，右肩部是从\(\mu + \sigma\)到\(\mu + 2\sigma\)。参见图5.2上方的图。由于偏度和峰度衡量形状，它们不依赖于位置和尺度参数的值。

随机变量\(Y\)的偏度为

\[ \operatorname{Sk} = E\left[\left(\frac{Y - E(Y)}{\sigma}\right)^3\right] = \frac{E\{Y - E(Y)\}^3}{\sigma^3}. \]

为了理解偏度的含义，观察一个例子是有帮助的；二项分布适合此目的。二项分布\(\operatorname{Binomial}(n,p)\)的偏度为

\[ \operatorname{Sk}(n,p) = \frac{1 - 2p}{\sqrt{np(1-p)}}, \qquad 0 < p < 1. \]

图5.3展示了\(n = 10\)和四个\(p\)值下的二项概率分布及其偏度。注意：

  1. 若\(p < 0.5\)，偏度为正；若\(p > 0.5\)，偏度为负；若\(p = 0.5\)，偏度为0；
  2. 当\(p\)趋近于0或1且\(n\)固定时，偏度的绝对值增大；
  3. 当\(n\)增大到无穷且\(p\)固定时，偏度的绝对值减小到0。
5.4 偏度、峰度与矩

正偏度也称为右偏，负偏度称为左偏。一个分布关于点 \(\theta\) 对称，如果对所有 \(y>0\)，有 \(P(Y>\theta+y)=P(Y<\theta-y)\)。在这种情况下，\(\theta\) 是位置参数，并且等于 \(E(Y)\)，假设 \(E(Y)\) 存在。任何对称分布的偏度为0。性质3在中心极限定理的背景下并不令人惊讶。我们知道，当 \(n\to\infty\) 且 \(p\) 固定且不等于0或1时，二项分布收敛于对称的正态分布。

随机变量 \(Y\) 的峰度为

\[ \mathrm{Kur} = E\left\{\frac{Y-E(Y)}{\sigma}\right\}^4 = \frac{E\{Y-E(Y)\}^4}{\sigma^4}. \]

正态随机变量的峰度为3。峰度的最小可能值为1，由任何取值恰好为两个

不同值，每个概率为 \(1/2\)。二项分布 \(\mathrm{Binomial}(n,p)\) 的峰度为

\[ \mathrm{KurBin}(n,p) = 3 + \frac{1 - 6p(1-p)}{n p (1-p)}. \]

注意到当 \(n \to \infty\) 且 \(p\) 固定时，\(\mathrm{KurBin}(n,p) \to 3\)，即正态分布的值，这又是中心极限定理作用的标志。图 5.3 也给出了该图中分布的峰度。当 \(n = 1\) 且 \(p = 1/2\) 时，\(\mathrm{KurBin}(n,p)\) 等于 1，即峰度的最小值。

对于不对称分布，解释其峰度是困难的，因为对于这类分布，峰度可能同时度量不对称性和尾部重量，因此二项分布并非理解峰度的特别好的例子。为此，我们将转而考察 \(t\) 分布，因为它们是对称的。图 5.2 将正态密度与方差缩放为 1 的 \(t_5\) 密度进行了比较。两者均值均为 0，标准差均为 1。均值和标准差分别是位置参数和尺度参数，不影响峰度。\(t\) 分布的参数 \(\nu\) 是形状参数。若 \(\nu > 4\)，则 \(t_\nu\) 分布的峰度有限，且为

\[ \mathrm{Kurt}(\nu) = 3 + \frac{6}{\nu - 4}. \tag{5.1} \]

例如，\(t_5\) 分布的峰度为 9。由于图 5.2 中的密度具有相同的均值和标准差，因此它们也具有相同的尾部、中心和肩部区域——至少根据我们对这些区域略显武断的定义——这些区域在顶部图中标出。底部图放大了右尾部。注意 \(t_5\) 密度在尾部和中心比 \(\mathrm{N}(0,1)\) 密度有更多的概率。这种 \(t_5\) 的行为是高峰度对称分布的典型特征。

每个正态分布的偏度系数均为 0，峰度为 3。所有正态分布的偏度和峰度必须相同，因为正态分布只有位置参数和尺度参数，没有形状参数。峰度 3 与公式 (5.1) 一致，因为正态分布是 \(\nu = \infty\) 的 \(t\) 分布。分布的“超值峰度”为 \(\mathrm{Kur} - 3\)，度量该分布峰度与正态分布峰度的偏差。由 (5.1) 可知，\(t_\nu\) 分布的超值峰度为 \(6/(\nu - 4)\)。

指数分布2的偏度等于 2，峰度为 9。双指数分布的偏度为 0，峰度为 6。由于指数分布仅有一个尺度参数，而双指数分布仅有一个位置参数和一个尺度参数，它们的偏度和峰度必定是常数，因为偏度和峰度仅依赖于形状参数。

5.4 偏度、峰度和矩

对数正态分布 Lognormal(μ, σ²)（附录 A.9.4 讨论）以对数均值 μ 为尺度参数，以对数标准差 σ 为形状参数——尽管 μ 和 σ 是正态分布本身的位置参数和尺度参数，但对于对数正态分布来说，它们分别是尺度参数和形状参数。σ 对对数正态形状的影响见图 4.11 和图 A.1。对数正态分布 Lognormal(μ, σ²) 的偏度系数为

\[\{\exp(\sigma^2)+2\}\sqrt{\exp(\sigma^2)-1}\tag{5.2}\]

由于 μ 是尺度参数，它不影响偏度。偏度总是正的，并且随着 σ 从 0 增加到 ∞，偏度从 0 增加到 ∞。

如果我们有来自某个分布的样本 \(Y_1,\ldots,Y_n\)，那么该分布的偏度和峰度的估计相对直接。设样本均值和标准差分别为 \(\bar{Y}\) 和 \(s\)。那么样本偏度（记为 \(\hat{Sk}\)）为

\[\hat{Sk}=\frac{1}{n}\sum_{i=1}^n\left(\frac{Y_i-\bar{Y}}{s}\right)^3\tag{5.3}\]

样本峰度（记为 \(\hat{Kur}\)）为

\[\hat{Kur}=\frac{1}{n}\sum_{i=1}^n\left(\frac{Y_i-\bar{Y}}{s}\right)^4\tag{5.4}\]

通常 (5.3) 和 (5.4) 中的因子 \(1/n\) 会被替换为 \(1/(n-1)\)，类似于样本方差。如果样本来自正态分布，那么样本偏度和超额峰度都应当接近 0。样本偏度和峰度对这些值的偏离是非正态性的指示。

需要谨慎对待。偏度和峰度对异常值高度敏感。有时异常值是由于污染物造成的，即不属于被抽样总体的坏数据。例如，数据记录错误。即使只有一个足够离群的污染物，来自正态分布的样本也会根据样本偏度和峰度显得高度非正态。在这种情况下，正态概率图看起来是线性的，但那个孤立污染物会凸显出来。见图 5.4，它是 999 个 N(0,1) 数据点加上一个等于 30 的污染物的正态概率图。该图清楚地显示样本接近正态但有一个异常值。然而，样本偏度和峰度分别为 10.85 和 243.04，这可能错误地给人样本远离正态的印象。此外，即使没有污染物，一个分布也可能在极端尾部之外极度接近正态分布，但偏度或超额峰度却与 0 相差很大。

5.5 厚尾分布

如前几章所述，相较于正态分布具有更高尾部概率的分布称为厚尾分布。由于峰度对尾部权重特别敏感，高峰度几乎等同于具有厚尾分布。厚尾分布在金融中是重要的模型，因为股票收益率和其他市场价格的变化通常具有厚尾。在金融应用中，人们尤其关注收益率分布具有厚尾的情况，因为可能出现极大的负收益率，例如，这可能完全耗尽一家公司的资本储备。如果卖空3，那么极大的正收益率也令人担忧。

5.5.1 指数尾与多项式尾

双指数分布的尾部比正态分布略重。这一点可以通过比较它们的密度来理解。尺度参数为 \(\theta\) 的双指数密度正比于 \(\exp(-|y/\theta|)\)，而 \(N(0,\sigma^2)\) 分布的密度正比于 \(\exp\{-0.5(y/\sigma)^2\}\)。当 \(|y| \to \infty\) 时，项 \(-y^2\) 趋于 \(-\infty\) 的速度远快于 \(-|y|\)。因此，当 \(|y| \to \infty\) 时，正态密度趋于 0 的速度比双指数密度快得多。第 5.6 节将讨论的广义误差分布的密度正比于

\[ \exp(-|y/\theta|^\alpha), \tag{5.8} \]

其中 \(\alpha > 0\) 是形状参数，\(\theta\) 是尺度参数。特殊情况 \(\alpha = 1\) 和 \(\alpha = 2\) 当然对应双指数密度和正态密度。如果 \(\alpha < 2\)，则广义误差分布的尾部比正态分布更重，且 \(\alpha\) 越小，尾部越重。特别地，\(\alpha < 1\) 意味着尾部比双指数分布更重。

然而，形如 (5.8) 的密度都不会具有真正的厚尾，特别地，\(E(|Y|^k) < \infty\) 对所有 \(k\) 成立，因此所有矩都有限。要实现非常厚的右尾，密度必须使得

\[ f(y) \sim A y^{-(a+1)} \quad \text{当 } y \to \infty \tag{5.9} \]

其中 \(A > 0\) 和 \(a > 0\) 是常数。

对于某个 \(A>0\) 和 \(a>0\)，这称为右多项式尾（right polynomial tail），与之相对的是 \[f(y) \sim A \exp(-y/\theta) \quad \text{当 } y \to \infty \tag{5.10}\] 其中 \(A>0\) 且 \(\theta>0\)，这称为指数右尾（exponential right tail）。左多项式尾和左指数尾的定义类似。

多项式尾也称为帕累托尾（Pareto tail），以附录A.9.8中定义的帕累托分布命名。多项式尾的参数 \(a\) 称为尾指数（tail index）。\(a\) 的值越小，尾部越重。\(a\) 必须大于0，因为如果 \(a \le 0\)，密度积分会发散至无穷，而不是1。

如 (5.8) 中的指数尾比任何多项式尾更轻，因为 \[\frac{\exp(-|y/\theta|^\alpha)}{|y|^{-(a+1)}} \to 0 \quad \text{当 } |y| \to \infty\] 对所有 \(\theta>0, \alpha>0\) 和 \(a>0\) 成立。

当然，左右尾的行为也可能完全不同。例如，一个可以是多项式尾而另一个是指数尾，或者两者都是多项式但具有不同的指数。

如果密度两尾都是多项式，则只有当两个尾指数中较小的那个大于 \(k\) 时，\(k\) 阶绝对矩才有限。如果两尾都是指数尾，则所有矩都存在。

5.6 广义误差分布

连续混合

第14章研究的GARCH过程的边际分布也是正态尺度混合，但具有无限多个分量和条件方差的连续分布。尽管GARCH过程比本节中的简单混合模型更复杂，但相同的主题适用——混合分布的非恒定条件方差会导致厚尾的边际分布，即使条件分布是正态分布且具有相对轻的尾部。

正态尺度混合的一般定义是：它是随机变量

\[ \mu + \sqrt{U} Z \tag{5.14} \]

的分布，其中 \(\mu\) 是等于均值的常数，\(Z \sim N(0,1)\)，\(U\) 是给出每个分量方差的正随机变量，且 \(Z\) 与 \(U\) 独立。如果 \(U\) 只能取有限个值，则 (5.14) 是一个离散（或有限）尺度混合分布。如果 \(U\) 是连续分布的，则我们得到一个连续尺度混合分布。\(U\) 的分布称为混合分布。根据 (5.11)，\(t\) 分布是一个连续正态尺度混合，其中 \(\mu = 0\) 且 \(U = \nu/W\)，这里 \(\nu\) 和 \(W\) 的定义与式 (5.11) 之前相同。

尽管有限正态混合有看似厚尾的尾部，但其尾部是指数型的，而非多项式型的。如果混合分布的尾部足够重，则连续正态混合可以具有多项式型尾部，例如 \(t\) 分布就是如此。

5.6 广义误差分布

第5.5.1节简要提到的广义误差分布具有指数型尾部。本节提供关于它们的更详细信息。带有形状参数 \(\nu\) 的标准化广义误差分布（或称GED）的密度为

\[ f_{\text{ged}}^{\text{std}}(y \mid \nu) = \kappa(\nu) \exp\left\{ -\left| \frac{y}{\lambda_\nu} \right|^\nu \right\}, \quad -\infty < y < \infty, \]

其中 \(\kappa(\nu)\) 和 \(\lambda_\nu\) 是由下式给出的常数

\[ \lambda_\nu = \left\{ \frac{2^{-2/\nu} \Gamma(\nu^{-1})}{\Gamma(3/\nu)} \right\}^{1/2} \quad \text{且} \quad \kappa(\nu) = \frac{\nu}{\lambda_\nu 2^{1+1/\nu} \Gamma(\nu^{-1})}, \]

\(f_{\text{std\_ged}}(y|\nu)\) 密度关于0对称，0是其均值、中位数和众数，且方差等于1。然而，它可以通过平移和缩放生成位置-尺度族。均值为\(\mu\)、方差为\(\sigma^2\)、形状参数为\(\nu\)的GED分布的密度为 \[ f_{\text{std\_ged}}(y|\mu,\sigma^2,\nu) := f_{\text{std\_ged}}((y-\mu)/\sigma|\nu)/\sigma. \]

收益率及其他金融市场数据通常没有自然的上下界，因此人们希望使用支撑集为\((-\infty,\infty)\)的模型。如果数据是对称的，则可以使用正态分布、\(t\)分布或广义误差分布作为模型。但如果数据是偏斜的呢？不幸的是，许多著名的偏态分布，如伽马分布和对数正态分布，其支撑集为\([0,\infty)\)，因此不适用于建模许多类型的金融市场数据的变化。本节描述该问题的一种解决方法。

Fernandez和Steel(1998)提出了一种巧妙的方法，用于在对称分布（如正态分布和\(t\)分布）中引入偏度。R中的fGarch包实现了他们的想法。令\(\xi\)为正的常数，\(f\)为关于0对称的密度。定义 \[ f^*(y|\xi) = \begin{cases} f(y\xi) & \text{if } y < 0, \\ f(y/\xi) & \text{if } y \ge 0. \end{cases} \tag{5.15} \] 由于\(f^*(y|\xi)\)的积分为\((\xi^{-1}+\xi)/2\)，因此将\(f^*(y|\xi)\)除以该常数以创建概率密度。归一化后，密度变为

5.9 极大似然估计

极大似然估计是最重要且最广泛使用的估计方法。许多著名的估计量，如样本均值以及回归中的最小二乘估计量，在数据服从正态分布时就是极大似然估计量。通常，极大似然估计能够提供比其他估计方法更有效（方差更小）的估计量。例如，对于\(t\)分布，均值的极大似然估计量比样本均值更有效。

设\(Y = (Y_1, \ldots, Y_n)^\mathsf{T}\)为数据向量，\(\theta = (\theta_1, \ldots, \theta_p)^\mathsf{T}\)为参数向量。令\(f(Y \mid \theta)\)表示\(Y\)的密度函数，它依赖于参数。

将\(L(\theta) = f(Y \mid \theta)\)视为\(\theta\)的函数（其中\(Y\)固定为观测数据），称为似然函数。它反映了实际观测样本的可能性。极大似然估计量（MLE）是使似然函数达到最大的\(\theta\)值。换句话说，MLE是使得观测数据似然最大的\(\theta\)值。记MLE为\(\hat{\theta}_{\text{ML}}\)。通常，最大化\(\log\{L(\theta)\}\)在数学上更简便，该函数称为对数似然。如果数据独立，则似然函数是边缘密度的乘积，而乘积的微分较为繁琐。取对数可将乘积转化为易于微分的和。此外，在数值计算中，使用对数似然可以减少下溢或上溢的可能性。由于对数函数是单调递增的，最大化\(\log\{L(\theta)\}\)等价于最大化\(L(\theta)\)。

在入门统计学教材的例子中，通常可以找到MLE的显式公式。但对于更复杂的模型（比如我们主要使用的模型），MLE没有显式公式。相反，需要编写程序计算任意\(\theta\)下的\(\log\{L(\theta)\}\)，然后使用优化软件数值最大化该函数；见例5.3。R函数optim()和nlminb()用于最小化函数，可应用于\(-L(\theta)\)。

对于许多重要模型，例如第5.14节中的例子以及第12章讨论的ARIMA和GARCH时间序列模型，R和其他软件包包含用于求这些模型MLE的函数。

5.10 Fisher信息量与MLE的中心极限定理

标准误差对于衡量估计量的精度至关重要。我们已经有了简单估计量（如 \(\bar{Y}\)）的标准误差公式，但其他估计量的标准误差呢？幸运的是，有一种简单的方法可以计算最大似然估计量的标准误差。

我们暂时假设 \(\theta\) 是一维的。Fisher信息量 定义为对数似然函数二阶期望的负值。因此，若记 \(I(\theta)\) 为Fisher信息量，则

\[ I(\theta) = -\mathbb{E}\left[ \frac{d^2}{d\theta^2} \log\{L(\theta)\} \right]. \tag{5.18} \]

估计量 \(\hat{\theta}\) 的标准误差 \(s_{\hat{\theta}}\) 就是Fisher信息量倒数的平方根，其中的未知参数 \(\theta\) 替换为 \(\hat{\theta}\)：

\[ s_{\hat{\theta}} = \frac{1}{\sqrt{I(\hat{\theta})}}. \tag{5.19} \]

例5.1 正态模型均值的Fisher信息量

设 \(Y_1, \ldots, Y_n\) 独立同分布于 \(N(\mu, \sigma^2)\)，其中 \(\sigma^2\) 已知。未知参数 \(\mu\) 的对数似然函数为

\[ \log\{L(\mu)\} = -\frac{n}{2}\{\log(\sigma^2) + \log(2\pi)\} - \frac{1}{2\sigma^2} \sum_{i=1}^n (Y_i - \mu)^2. \]

因此，

\[ \frac{d}{d\mu} \log\{L(\mu)\} = \frac{1}{\sigma^2} \sum_{i=1}^n (Y_i - \mu), \]

所以 \(\bar{Y}\) 是 \(\mu\) 的MLE，且

\[ \frac{d^2}{d\mu^2} \log\{L(\mu)\} = -\frac{\sum_{i=1}^n 1}{\sigma^2} = -\frac{n}{\sigma^2}. \]

由此可得 \(I(\hat{\mu}) = n/\sigma^2\) 以及 \(s_{\hat{\mu}} = \sigma/\sqrt{n}\)。由于 \(\mu\) 的MLE是 \(\bar{Y}\)，这个结果就是熟知的事实：当 \(\sigma\) 已知时，\(s_{\bar{Y}} = \sigma/\sqrt{n}\)；当 \(\sigma\) 未知时，\(s_{\bar{Y}} = s/\sqrt{n}\)。□

极大似然估计的偏倚与标准差

在许多例子中，MLE具有较小的偏倚，当样本容量\(n\)增至无穷时，偏倚以速率\(n^{-1}\)趋于0。更精确地，

\[ \text{BIAS}(\hat{\theta}_{\text{ML}}) = E(\hat{\theta}_{\text{ML}}) - \theta \sim \frac{A}{n}, \quad \text{as } n \to \infty, \tag{5.23} \]

其中\(A\)为某常数。正态方差MLE的偏倚便是一例，此时\(A = -\sigma^2\)。

尽管在某些特殊问题中可以校正该偏倚，例如正态方差的估计，但通常偏倚被忽略。这有两个充分理由。第一，对数似然通常是\(n\)项之和，因此以速率\(n\)增长。Fisher信息量也是如此。因此，MLE的方差以速率\(n^{-1}\)减小，即，

\[ \text{Var}(\hat{\theta}_{\text{ML}}) \sim \frac{B}{n}, \quad \text{as } n \to \infty, \tag{5.24} \]

其中\(B > 0\)。变异性应以标准差而非方差来衡量，由(5.24)可得，

\[ \text{SD}(\hat{\theta}_{\text{ML}}) \sim \frac{\sqrt{B}}{\sqrt{n}}, \quad \text{as } n \to \infty. \tag{5.25} \]

收敛速率(5.25)也可由MLE的CLT得到。比较(5.23)和(5.25)可知，随着\(n\)增大，MLE的偏倚相对于标准差变得可以忽略。这对金融市场数据尤为重要，因其样本容量往往很大。

第二，即使参数\(\theta\)的MLE无偏，对\(\theta\)的非线性函数则并非如此。例如，即使\(\hat{\sigma}^2\)是\(\sigma^2\)的无偏估计，\(\hat{\sigma}\)对\(\sigma\)是有偏的。原因在于，对于非线性函数\(g\)，通常有\(E\{g(\hat{\theta})\} \neq g\{E(\hat{\theta})\}\)。因此，不可能校正所有偏倚。

假设 \(\theta\) 是一个参数向量，原假设对 \(\theta\) 施加了 \(m\) 个等式约束。更精确地说，存在 \(m\) 个函数 \(g_1,\dots,g_m\)，原假设为 \(g_i(\theta)=0\)，\(i=1,\dots,m\)。无约束的模型和有约束的模型分别称为全模型和简化模型。此外，还假设这些约束中没有冗余的，即不被其他约束隐含的。为了说明冗余，假设 \(\theta=(\theta_1,\theta_2,\theta_3)\)，约束为 \(\theta_1 = \theta_2\)，\(\theta_2 = \theta_3\) 和 \(\theta_1 = \theta_3\)。那么这些约束存在冗余，因为其中任意两个均蕴含第三个。因此 \(m=2\)，而非3。当然，冗余并非总是如此容易检测。一种检查方法是看 \(m \times \dim(\theta)\) 矩阵

\[ \begin{pmatrix} \nabla g_1(\theta) \\ \vdots \\ \nabla g_m(\theta) \end{pmatrix} \tag{5.26} \]

的秩必须为 \(m\)。这里 \(\nabla g_i(\theta)\) 是 \(g_i\) 的梯度。

\[ \left[ \log\{L(\hat{\theta}_{ML})\} - \log\{L(\hat{\theta}_{0,ML})\} \right] \geq c, \tag{5.27} \]

5.12 AIC和BIC

一个重要的实际问题是在两个或多个可能适用于数据集的统计模型之间进行选择。对数似然的最大值（这里记为\(\log\{L(\hat{\theta}_{\mathrm{ML}})\}\)）可用于衡量模型对数据的拟合程度，或比较两个或多个模型的拟合效果。然而，仅仅通过向模型添加参数就可以增加\(\log\{L(\hat{\theta}_{\mathrm{ML}})\}\)。额外的参数并不一定意味着模型能更好地描述数据生成机制，因为由添加参数带来的额外模型复杂性可能仅仅是在拟合数据中的随机噪声，这个问题称为过拟合（overfitting）。因此，模型应该同时根据数据拟合度和模型复杂性进行比较。为了找到一个简约的模型，需要在最大化拟合和最小化模型复杂性之间取得良好的权衡。

AIC（赤池信息准则）和BIC（贝叶斯信息准则）是在拟合与复杂性之间取得良好权衡的两种方法。它们略有不同，BIC倾向于比AIC更简单的模型。它们的定义为：

\[\mathrm{AIC} = -2\log\{L(\hat{\theta}_{\mathrm{ML}})\} + 2p \tag{5.29}\] \[\mathrm{BIC} = -2\log\{L(\hat{\theta}_{\mathrm{ML}})\} + \log(n)p \tag{5.30}\]

其中\(p\)是模型中的参数个数，\(n\)是样本量。对于这两个准则，“越小越好”，因为较小的值倾向于最大化\(L(\hat{\theta}_{\mathrm{ML}})\)（即最小化\(-\log\{L(\hat{\theta}_{\mathrm{ML}})\}\)）并最小化衡量模型复杂性的\(p\)。项\(2p\)和\(\log(n)p\)被称为“复杂性惩罚”，因为它们惩罚较大的模型。

偏差（deviance）一词通常用于表示负两倍的对数似然，因此 \(\text{AIC} = \text{偏差} + 2p\) 和 \(\text{BIC} = \text{偏差} + \log(n)p\)。偏差衡量模型拟合优度，值越小表示拟合越好。

通常，从一组候选模型中，选择使所使用的准则（AIC 或 BIC）最小的模型。然而，任何与最小值相差在 2 或 3 以内的模型都是良好的候选，可能会被选中，例如因为它比达到绝对最小值的模型更简单或更方便使用。由于 \(\log(n) > 2\) 在通常情况下（即 \(n > 8\)）成立，BIC 对模型复杂度的惩罚比 AIC 更重，因此 BIC 倾向于选择比 AIC 更简单的模型。不过，这两个准则经常选择相同或几乎相同的模型。当然，如果多个候选模型具有相同的 \(p\) 值，那么 AIC、BIC 和 \(-2\log\{L(\hat{\theta}_{ML})\}\) 将由同一个模型最小化。

5.13 验证数据与交叉验证

当同一数据既用于参数估计又用于拟合评估时，会强烈倾向于过拟合（overfitting）。数据包含信号和噪声。信号包含总体中存在的特征，因此也存在于总体的每个样本中，但噪声是随机的，随样本不同而变化。过拟合是指选择一个不必要的复杂模型来拟合噪声。过拟合的明显解决方法是使用独立于参数估计数据的数据来诊断模型拟合。我们将用于估计的数据称为训练数据（training data），将用于评估拟合的数据称为验证数据（validation data）或测试数据（test data）。

例 5.2. 估计中盘股票的期望收益率

本例使用本书网站上的文件 midcapD.ts.csv 中 20 只中盘股票的 500 个日收益率。数据最初来自 R 的 fEcofin 包中的 midcapD.ts 数据集。数据时间为 1991 年 2 月 28 日至 1995 年 12 月 29 日。假设我们需要估计这 20 个期望收益率。考虑两种估计量。第一种称为“分离均值”（separate-means），即 20 个样本均值。第二种称为“共同均值”（common-mean），使用 20 个样本均值的平均值作为所有 20 个期望收益率的共同估计量。

共同均值估计量的背后逻辑是，中盘股票应当具有相似的期望收益率。共同均值估计量通过合并数据大大降低了估计量的方差。共同均值估计量存在一些偏差，因为真实的期望收益率不会完全相同，而这是共同均值估计量无偏的要求。分离均值估计量是无偏的，但代价是方差较大。这是一个经典的偏差-方差权衡（bias–variance tradeoff）的例子。

5.13 验证数据与交叉验证

哪个估计量能达到最佳权衡？为了回答这个问题，我们将数据分为前250天的收益率（训练数据）和后250天（验证数据）。拟合优度的评价标准是平方误差和，即

\[ \sum_{k=1}^{20} \left( \hat{\mu}_k^{\text{train}} - \bar{Y}_k^{\text{val}} \right)^2, \]

其中 \(\hat{\mu}_k^{\text{train}}\) 是基于训练数据的第 \(k\) 个期望收益率的估计量，\(\bar{Y}_k^{\text{val}}\) 是验证数据中第 \(k\) 只股票收益率的样本均值。对于单独均值估计量和公共均值估计量，平方误差和分别为 3.262 和 0.898。当然，结论是：在此例中，公共均值估计量比使用单独均值准确得多。

假设我们同时使用训练数据进行验证？那么拟合优度准则将变为

\[ \sum_{k=1}^{20} \left( \hat{\mu}_k^{\text{train}} - \bar{Y}_k^{\text{train}} \right)^2, \]

其中 \(\bar{Y}_k^{\text{train}}\) 是训练数据中第 \(k\) 只股票的样本均值，同时也是该股票的单独均值估计量。结果会如何？显然，单独均值估计量的平方误差和为 0——每个均值都被完美地估计了！公共均值估计量的平方误差和为 0.920。不恰当地使用训练数据进行验证会导致错误的结论，即单独均值估计量更准确。

在公共均值和单独均值这两个极端之间，还存在一些折衷估计量。这些折衷估计量将单独均值向公共均值收缩。第20章讨论的贝叶斯估计是一种选择收缩量的有效方法；见例20.12，其中对这组收益率进行了进一步分析。□

判断拟合优度的常用准则是偏差（deviance），其定义为 \(-2\) 乘以对数似然。验证数据的偏差为

\[ -2 \log f\left( \mathbf{Y}^{\text{val}} \mid \hat{\theta}^{\text{train}} \right), \tag{5.31} \]

其中 \(\hat{\theta}^{\text{train}}\) 是训练数据的极大似然估计，\(\mathbf{Y}^{\text{val}}\) 是验证数据，而 \(f(\mathbf{y}^{\text{val}} \mid \theta)\) 是验证数据的密度。

当样本量较小时，将数据一次划分为训练数据和验证数据会造成浪费。更好的技术是交叉验证（cross-validation），通常简称为 CV，其中每个观测值扮演训练和验证两个角色。K折交叉验证将数据集划分为大致相等的 K 个子集。

等大小的分组。验证重复K次。在第k次验证中，k = 1, …, K，第k个子集作为验证数据，其余K−1个子集合并形成训练数据。将K个拟合优度估计值结合起来，例如，通过取平均值。一种常见的选择是n折交叉验证，也称为留一交叉验证。使用留一交叉验证时，每个观测值依次充当验证数据集，其余n−1个观测值作为训练数据。

另一种不使用实际验证数据的方法，是计算如果可以获取新数据并用于验证时会发生什么。这就是AIC的推导思路。AIC是对与真实数据独立的一个假设新样本的期望偏差的近似。更精确地说，AIC近似于

\(E\left[-2\log f\{Y^{\text{new}}|\hat{\theta}(Y^{\text{obs}})\}\right]\), (5.32)

其中\(Y^{\text{obs}}\)是观测数据，\(\hat{\theta}(Y^{\text{obs}})\)是由\(Y^{\text{obs}}\)计算的MLE，\(Y^{\text{new}}\)是一个假设的新数据集，且\(Y^{\text{obs}}\)和\(Y^{\text{new}}\)独立同分布。换句话说，\(Y^{\text{new}}\)是\(Y^{\text{obs}}\)的一个未观测的独立复制。由于\(Y^{\text{new}}\)未被观测但与\(Y^{\text{obs}}\)具有相同分布，为了得到AIC，我们将(5.32)中的\(Y^{\text{new}}\)替换为\(Y^{\text{obs}}\)，并省略(5.32)中的期望。然后计算这种替换的影响。近似影响是使(5.32)减少了两倍参数个数。因此，AIC通过在偏差上加2p来补偿，从而

\(\text{AIC} = -2\log f\{Y^{\text{obs}}|\hat{\theta}(Y^{\text{obs}})\} + 2p\), (5.33)

这是(5.29)的重新表述。

AIC中使用的近似在样本量增加时变得更准确。对AIC的小样本修正是

\(\text{AICc} = \text{AIC} + \frac{2p(p+1)}{n-p-1}\). (5.34)

金融市场数据集通常足够大，修正项\(2p(p+1)/(n-p-1)\)很小，因此AIC足够使用，不需要AICc。例如，若\(n = 200\)，则当\(p = 3, 4, 5, 6\)时，\(2p(p+1)/(n-p-1)\)分别为0.12, 0.21, 0.31, 0.44。由于AIC值相差小于1通常被认为无关紧要，因此当n至少为200时，修正对比较具有3到6个参数的模型影响很小。更显著地，当n=500时，对3, 4, 5, 6个参数的修正仅为0.05, 0.08, 0.12, 0.17。

交易者经常使用一组历史数据制定交易策略，然后在新数据上测试策略。这被称为回测，是一种验证形式。

如前所述，仅有少数“教科书”示例能导出MLE的显式公式。在大多数情况下，MLE必须通过数值方法求解。例如，假设\(Y_1,\dots,Y_n\)是来自t分布的i.i.d.样本。令 \[ f^{\text{std}}_{t,\nu}(y|\mu,\sigma) \tag{5.35} \] 表示具有\(\nu\)自由度、均值\(\mu\)和标准差\(\sigma\)的标准化t分布的密度。则参数\(\nu,\mu\)和\(\sigma\)通过最大化 \[ \sum_{i=1}^{n}\log\left\{f^{\text{std}}_{t,\nu}(Y_i|\mu,\sigma)\right\} \tag{5.36} \] 来估计，可使用任意方便的优化软件。其他模型的估计类似。

# 定义负对数似然函数
def neg_log_lik(params):
    loc, scale, df = params
    if scale <= 0 or df <= 0:
        return 1e10
    return -np.sum(stats.t.logpdf(x, df, loc, scale))

# 初始值

5.15 轮廓似然

轮廓似然是一种基于第5.11节引入的似然比检验的技术。轮廓似然用于构造置信区间，并且通常是寻找极大似然估计量的便捷方法。假设参数向量为 \(\boldsymbol{\theta} = (\theta_1, \boldsymbol{\theta}_2)\)，其中 \(\theta_1\) 是一个标量参数，向量 \(\boldsymbol{\theta}_2\) 包含模型中的其他参数。关于 \(\theta_1\) 的轮廓对数似然函数为

\[ L_{\max}(\theta_1) = \max_{\boldsymbol{\theta}_2} \, L(\theta_1, \boldsymbol{\theta}_2). \tag{5.37} \]

(5.37) 的右侧表示在固定 \(\theta_1\) 的情况下，关于 \(\boldsymbol{\theta}_2\) 最大化 \(L(\theta_1, \boldsymbol{\theta}_2)\)，从而得到仅关于 \(\theta_1\) 的函数。定义 \(\hat{\boldsymbol{\theta}}_2(\theta_1)\) 为最大化 (5.37) 右侧的 \(\boldsymbol{\theta}_2\) 值。

\(\theta_1\) 的 MLE 是最大化 \(L_{\max}(\theta_1)\) 的值 \(\hat{\theta}_1\)，而 \(\boldsymbol{\theta}_2\) 的 MLE 是 \(\hat{\boldsymbol{\theta}}_2(\hat{\theta}_1)\)。令 \(\theta_{0,1}\) 为 \(\theta_1\) 的一个假设值。根据第5.11节的似然比检验理论，如果

\[ L_{\max}(\theta_{0,1}) > L_{\max}(\hat{\theta}_1) - \frac{1}{2} \chi^2_{\alpha,1}, \tag{5.38} \]

则接受原假设 \(H_0: \theta_1 = \theta_{0,1}\)。这里 \(\chi^2_{\alpha,1}\) 是自由度为1的卡方分布的 \(\alpha\) 上侧分位数。\(\theta_1\) 的轮廓似然置信区间（更准确地说，置信区域，因为它不一定是一个区间）是所有被接受的原假设值的集合，即

\[ \left\{ \theta_1 : L_{\max}(\theta_1) > L_{\max}(\hat{\theta}_1) - \frac{1}{2} \chi^2_{\alpha,1} \right\}. \tag{5.39} \]

轮廓似然可以针对参数的一个子集（而非单个参数）定义，但此处不展开讨论。

5.16 稳健估计

尽管极大似然估计具有许多吸引人的性质，但它们有一个严重的缺陷，使用它们的人应该意识到。极大似然估计可能对统计模型的假设非常敏感。例如，正态总体均值的MLE是样本均值，\(\sigma^2\)的MLE是样本方差，除分母为\(n\)而不是\(n-1\)这一微小变化外。当总体确实是正态分布时，样本均值和方差是有效的估计量，但这些估计量对异常值非常敏感，尤其是样本标准差。因为这些估计量是数据的平均值

样本均值的一个稳健替代是截尾均值。α-截尾均值的计算方法是：将样本从小到大排序，去掉最小的一部分α和最大的一部分α观测值，然后计算剩余观测值的均值。截尾背后的思想简单明了：在计算均值之前先剔除极端值。α-截尾均值有一个数学公式。设\(k = n\alpha\)四舍五入14为整数；\(k\)是从样本两端去掉的观测值个数。则α-截尾均值为 \[\bar{X}_\alpha = \frac{\sum_{i=k+1}^{n-k} Y_{(i)}}{n-2k},\] 其中\(Y_{(i)}\)是第\(i\)个顺序统计量。α的典型取值为0.1、0.15、0.2和0.25。当α趋近于0.5时，α-截尾均值趋近于样本中位数，即0.5样本分位数。

离散度指的是分布或样本中的变异性。样本标准差是最常见的离散度估计量，但如前所述，它是不稳健的。实际上，样本标准差甚至比样本均值更不稳健，因为平方使异常值更极端。一个稳健的离散度估计量是MAD（中位数绝对偏差）估计量，定义为 \[\hat{\sigma}_{\text{MAD}} = 1.4826 \times \text{median}\{|Y_i - \text{median}(Y_i)|\}. \tag{5.40}\] 该公式解释如下：“\(\text{median}(Y_i)\)”是样本中位数，\(|Y_i - \text{median}(Y_i)|\)是观测值距中位数的绝对偏差，而\(\text{median}\{|Y_i - \text{median}(Y_i)|\}\)是这些绝对偏差的中位数。对于正态分布数据，\(\text{median}\{|Y_i - \text{median}(Y_i)|\}\)估计的不是\(\sigma\)，而是\(\Phi^{-1}(0.75)\sigma = \sigma/1.4826\)，因为对于正态分布数据，随着样本量增加，\(\text{median}\{|Y_i - \text{median}(Y_i)|\}\)将收敛于\(\sigma/1.4826\)。因此，式(5.40)中的因子1.4826校准了\(\hat{\sigma}_{\text{MAD}}\)，使其在应用于正态分布数据时估计\(\sigma\)。

\(\hat{\sigma}_{\text{MAD}}\) 并不估计非正态总体的 \(\sigma\)。它衡量的是离散程度，但不是以标准差衡量的离散程度。而这正是关键所在：对于非正态总体，标准差对分布的尾部非常敏感，并不能告诉我们太多关于分布中心范围的离散信息，而只反映了尾部的信息。

在 R 中，mad() 计算 (5.40)。有些作者将 MAD 定义为 \(\text{median}\{|Y_i - \text{median}(Y_i)|\}\)，即不带 1.4826。这里使用记号 \(\hat{\sigma}_{\text{MAD}}\) 是为了强调通过 1.4826 进行标准化，以估计正态标准差。

另一种替代稳健估计的方法是假设一个离群值更可能出现的模型。此时 MLE 会自动降低离群值的权重。例如，t 分布参数的 MLE 比正态分布参数的 MLE 对离群值稳健得多。

5.17 基于参数变换的变换核密度估计

我们在 4.8 节中看到，变换核密度估计（TKDE）可以避免普通 KDE 应用于偏斜数据时出现的凸包。当尾部都较长（金融数据常见）时，KDE 在尾部也可能出现凸包。例如变量 diffrf 的 KDE 如图 5.9 所示。对于此类数据，TKDE 需要一个变换，该变换在众数右侧是凸的，左侧是凹的。有许多这样的变换，本节将利用概率论的一些事实以及极大似然估计来选择适当的变换。

这里使用的关键思想是：(1) 正态分布数据具有轻尾部，适合用 KDE 估计；(2) 如果知道 CDF，就可以很容易地将数据变换为正态；(3) CDF 可以通过假设一个参数模型并使用极大似然估计来估计。如果一个随机变量具有连续分布 \(F\)，则 \(F(X)\) 服从均匀分布，\(\Phi^{-1}(F(X))\) 服从 N(0,1) 分布；这里 \(\Phi\) 是标准正态 CDF。当然，在实践中 \(F\) 未知，但我们可以参数化地估计 \(F\)，例如假设 \(F\) 是某个 t 分布。并不需要 \(F\) 实际上是 t 分布，只需要 t 分布能够在尾部对 \(F\) 提供足够合理的拟合，从而选择适当的变换。如果已知 \(F\) 是 t 分布，那么当然就不需要使用 KDE 或 TKDE 来估计其密度了。TKDE 中使用的变换是 \(g(y) = \Phi^{-1}(F(y))\)，其逆变换为 \(g^{-1}(x) = F^{-1}(\Phi(x))\)。计算 TKDE 需要 \(g\) 的导数，如下：

\[ g'(y) = \frac{f(y)}{\phi[\Phi^{-1}(F(y))]}. \tag{5.41} \]

def skew_t_loglik(params, data):
    xi, omega, alpha, nu = params
    if omega <= 0 or nu <= 0:
        return -np.inf
    z = (data - xi) / omega
    # Azzalini's skew-t density

def loglik_std(x):
    mu, sigma, nu = x
    # scipy.stats.t：参数为df, loc, scale
    neg_log_lik = -np.sum(t.logpdf(Y, df=nu, loc=mu, scale=sigma))
