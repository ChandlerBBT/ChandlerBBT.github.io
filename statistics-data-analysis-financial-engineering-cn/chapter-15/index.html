第15章 协整

15.1 引言

协整分析是计量经济学中频繁使用的一种技术。在金融领域，它可用于寻找基于均值回归的交易策略。

假设我们可以找到一只股票，其价格（或对数价格）序列是平稳的，因此具有均值回归特性。这将是一个极好的投资机会。每当价格低于均值时，我们可以买入该股票，并在价格回归均值时实现利润。类似地，当价格高于均值时，我们可以通过卖空来获利。可惜的是，收益率是平稳的，但价格不是。我们已经看到对数价格是积分的。然而，并非毫无希望。有时我们可以找到两个或多个资产，其价格联系如此紧密，以至于它们的线性组合是平稳的。那么，由协整向量（即该线性组合的系数向量）赋予权重的投资组合将具有平稳的价格。协整分析是寻找协整向量的一种手段。

两个时间序列 \(Y_{1,t}\) 和 \(Y_{2,t}\) 是协整的，如果每个序列都是 \(I(1)\)，但存在一个 \(\lambda\) 使得 \(Y_{1,t} - \lambda Y_{2,t}\) 是平稳的。例如，共同趋势模型为：

\[ Y_{1,t} = \beta_1 W_t + \epsilon_{1,t}, \] \[ Y_{2,t} = \beta_2 W_t + \epsilon_{2,t}, \]

其中 \(\beta_1\) 和 \(\beta_2\) 非零，两个序列共同的趋势 \(W_t\) 是 \(I(1)\)，而噪声过程 \(\epsilon_{1,t}\) 和 \(\epsilon_{2,t}\) 是 \(I(0)\)。由于共同趋势的存在，

如果 \(Y_{1,t}\) 和 \(Y_{2,t}\) 是非平稳的，但存在这两个序列的某个线性组合可以消除趋势，则它们是协整的。为了说明这一点，注意到若 \(\lambda = \beta_1 / \beta_2\)，则 \[ \beta_2 (Y_{1,t} - \lambda Y_{2,t}) = \beta_2 Y_{1,t} - \beta_1 Y_{2,t} = \beta_2 \epsilon_{1,t} - \beta_1 \epsilon_{2,t} \tag{15.1} \] 中不含趋势 \(W_t\)，因此它是 \(I(0)\) 的。

在第13.3.1节中，我们看到了当残差为单整时伪回归的危险。这一问题使我们在对非平稳时间序列进行回归时应保持谨慎。然而，如果将 \(Y_t\) 对 \(X_t\) 回归且两个序列是协整的，那么残差将是 \(I(0)\) 的，从而最小二乘估计量是一致的。

Phillips-Ouliaris协整检验将一个单整序列对其他序列进行回归，并对残差应用Phillips-Perron单位根检验。原假设为残差是单位根非平稳的，这意味着序列不具有协整关系。因此，小的 \(p\) 值表明序列是协整的，因而适合进行回归分析。但残差仍可能相关，因此应对其进行建模；详见第13.3.3节。

向量误差修正模型是协整的一种回归方法，但这种方法不太令人满意，因为必须选择一个序列作为因变量，并且这个选择必须是...

有些随意。在例15.1中，按到期排序的中间收益率被使用，但并无令人信服的理由。此外，回归只会找到一个协整向量，但可能存在多个。

一种对称处理时间序列的协整替代方法是使用向量误差修正模型（VECM）。在这些模型中，偏离均值的部分称为“误差”，每当平稳线性组合偏离其均值时，它随后会被推回其均值（误差被“修正”）。

当只有两个序列 \(Y_{1,t}\) 和 \(Y_{2,t}\) 时，误差修正的思想最为简单。在这种情况下，误差修正模型为：

\[ \Delta Y_{1,t} = \phi_1 (Y_{1,t-1} - \lambda Y_{2,t-1}) + \epsilon_{1,t}, \tag{15.2} \] \[ \Delta Y_{2,t} = \phi_2 (Y_{1,t-1} - \lambda Y_{2,t-1}) + \epsilon_{2,t}, \tag{15.3} \]

其中 \(\epsilon_{1,t}\) 和 \(\epsilon_{2,t}\) 是白噪声。将 (15.2) 减去 \(\lambda\) 倍的 (15.3) 得到：

\[ \Delta (Y_{1,t} - \lambda Y_{2,t}) = (\phi_1 - \lambda \phi_2)(Y_{1,t-1} - \lambda Y_{2,t-1}) + (\epsilon_{1,t} - \lambda \epsilon_{2,t}). \tag{15.4} \]

令 \(F_t\) 表示 \(t\) 时刻的信息集。如果 \((\phi_1 - \lambda \phi_2) < 0\)，则 \(\mathbb{E}\{\Delta (Y_{1,t} - \lambda Y_{2,t}) \mid F_{t-1}\}\) 与 \(Y_{1,t-1} - \lambda Y_{2,t-1}\) 的符号相反。这导致误差修正，因为每当 \(Y_{1,t-1} - \lambda Y_{2,t-1}\) 为正时，其期望变化为负，反之亦然。

重新排列 (15.4) 表明 \(Y_{1,t-1} - \lambda Y_{2,t-1}\) 是一个系数为 \(1+\phi_1-\lambda\phi_2\) 的 AR(1) 过程。因此，序列 \(Y_{1,t} - \lambda Y_{2,t}\) 在 \(|1+\phi_1-\lambda\phi_2|\) 小于 1、等于 1 和大于 1 的情况下分别对应 I(0)、单位根非平稳或爆炸性序列。

  • 如果 \(\phi_1 - \lambda \phi_2 > 0\)，则 \(1+\phi_1-\lambda\phi_2 > 1\)，\(Y_{1,t} - \lambda Y_{2,t}\) 是爆炸性的。
  • 如果 \(\phi_1 - \lambda \phi_2 = 0\)，则 \(1+\phi_1-\lambda\phi_2 = 1\)，\(Y_{1,t} - \lambda Y_{2,t}\) 是随机游走。
  • 如果 \(\phi_1 - \lambda \phi_2 < 0\)，则 \(1+\phi_1-\lambda\phi_2 < 1\)，\(Y_{1,t} - \lambda Y_{2,t}\) 是平稳的，除非 \(\phi_1 - \lambda \phi_2 \leq -2\)，使得 \(1+\phi_1-\lambda\phi_2 \leq -1\)。

情况 \(\phi_1 - \lambda \phi_2 \leq -2\) 称为“过度修正”。\(Y_{1,t} - \lambda Y_{2,t}\) 的变化方向正确但幅度过大，因此序列在符号上振荡但幅度趋于无穷大。

15.2 向量误差修正模型

为了理解如何将误差修正推广到两个以上序列，将方程(15.2)和(15.3)写成向量形式是有用的。令 \(\mathbf{Y}_t = (Y_{1,t}, Y_{2,t})'\) 和 \(\mathbf{\epsilon}_t = (\epsilon_{1,t}, \epsilon_{2,t})'\)。那么

\[\Delta \mathbf{Y}_t = \alpha \beta' \mathbf{Y}_{t-1} + \mathbf{\epsilon}_t, \tag{15.5}\]

其中

\[\alpha = \begin{pmatrix} \phi_1 \\ \phi_2 \end{pmatrix} \quad \text{且} \quad \beta = \begin{pmatrix} 1 \\ -\lambda \end{pmatrix}, \tag{15.6}\]

因此 \(\beta\) 是协整向量，而 \(\alpha\) 表示均值回归的速度，称为载荷矩阵或调整矩阵。

模型(15.5)也适用于存在 \(d\) 个序列的情况，此时 \(\mathbf{Y}_t\) 和 \(\mathbf{\epsilon}_t\) 是 \(d\) 维的。在这种情况下，\(\beta\) 和 \(\alpha\) 各是一个满秩的 \(d \times r\) 矩阵，其中 \(r \leq d\) 是线性无关协整向量的个数。\(\beta\) 的列是协整向量。

模型(15.5)是向量AR(1)[即VAR(1)]模型，但为了增加灵活性，可以扩展为VAR(\(p\))模型，有几种方法可以实现这一点。我们将使用R语言urca包中ca.jo()函数的符号和VECM的第二种形式。该VECM是

}
import numpy as np
n = 5000
a1 = 0.2
a2 = 0.55
lam = 1.0
y1 = np.zeros(n)
y2 = np.zeros(n)
e1 = np.random.normal(size=n)
e2 = np.random.normal(size=n)
for i in range(1, n):
    y1[i] = y1[i-1] + a1 * (y1[i-1] - lam * y2[i-1]) + e1[i]
    y2[i] = y2[i-1] + a2 * (y1[i-1] - lam * y2[i-1]) + e2[i]

\[ \Delta Y_t = \Gamma_1 \Delta Y_{t-1} + \cdots + \Gamma_{p-1} \Delta Y_{t-p+1} + \Pi Y_{t-1} + \mu + \Phi D_t + \epsilon_t, \tag{15.7} \] 其中 \(\mu\) 是均值向量，\(D_t\) 是非随机回归变量向量，且 \[ \Pi = \alpha \beta'. \tag{15.8} \] 和之前一样，\(\beta\) 和 \(\alpha\) 都是满秩的 \(d \times r\) 矩阵，\(\alpha\) 被称为载荷矩阵。

容易证明，\(\beta\) 的列是协整向量。由于 \(Y_t\) 是 I(1)，(15.7) 左边的 \(\Delta Y_t\) 是 I(0)，因此 (15.7) 右边的 \(\Pi Y_{t-1} = \alpha \beta' Y_{t-1}\) 也是 I(0)。由此可得 \(\beta' Y_{t-1}\) 的 \(r\) 个分量都是 I(0)。

如前所述，协整的价格序列可用于统计套利。与纯套利不同，统计套利意味着一种仅可能盈利而非保证盈利的机会。配对交易利用协整的资产价格对，已成为一种流行的统计套利技术。配对交易要求交易者找到协整的资产对，从中选择那些在考虑交易成本后仍可盈利的对，并最终设计包含买入和卖出信号的交易策略。关于统计套利的完整讨论超出了本书范围，但更多阅读参见第15.4节。

尽管许多公司通过统计套利取得了巨大成功，但仍需注意风险。其一是模型风险：误差修正模型可能不正确。即使模型正确，也必须基于历史数据进行估计，而参数可能发生变化，甚至可能是迅速的。如果统计套利机会存在，其他交易者可能已经发现它们，他们的交易活动就是预期参数变化的一个原因。另一个风险是，在平稳过程回归均值之前，你可能已经破产。这一风险尤其大，因为从事统计套利的公司很可能使用高杠杆。高杠杆会放大由于过程在回归前进一步偏离均值所造成的小损失。参见第2.4.2节和第15.5.4节。

15.5.1 中型股价格的协整分析

数据集 midcapD.ts.csv 包含 20 只中型股的对数日收益率，位于第 2 列至第 21 列。第 1 列和第 22 列分别包含日期和市场收益率。本节中，我们将使用前 10 只股票的收益率。为了从收益率中得到股票价格，我们利用关系式

\[P_t = P_0 \exp(r_1 + \dots + r_t)\]

其中 \(P_t\) 和 \(r_t\) 分别表示时刻 \(t\) 的价格和对数收益率。收益率将用作对数收益率的近似值。时刻 0 的价格未知，因此我们对每只股票取 \(P_0 = 1\)。这意味着我们使用的价格序列存在乘法因子偏差。但这不影响协整向量的数量。如果发现存在协整关系，则需要获取价格数据以研究交易策略。

我们将使用 urca 包中的 ca.jo() 函数对价格进行 Johansen 协整分析。运行

library(urca)
midcapD.ts = read.csv("midcapD.ts.csv", header = T)
x = midcapD.ts[, 2:11]
prices = exp(apply(x, 2, cumsum))

15.5.4 模拟

在本节中，您将运行与 2.4.2 节类似的模拟。不同之处在于，现在的价格过程是均值回归的。

假设一个对冲基金持有一个价值 1,000,000 美元的投资组合，它使用了 50,000 美元的自有资金和 950,000 美元的借款来购买该组合。如果组合价值在任何交易日结束时低于 950,000 美元，则对冲基金必须清算并偿还贷款。

该组合是通过协整分析选出的，其价格是一个 AR(1) 过程：

\[ (P_t - \mu) = \varphi (P_{t-1} - \mu) + \epsilon_t \]

其中 \(P_t\) 是交易日 \(t\) 结束时的组合价格，\(\mu = 1,030,000\) 美元，\(\varphi = 0.99\)，\(\epsilon_t\) 的标准差为 5,000 美元。对冲基金知道价格最终会回归到 1,030,000 美元（假设模型正确，当然这是一个很大的假设）。它决定在 \(P_t \geq 1,020,000\) 美元时清算头寸。这将产生至少 20,000 美元的利润。然而，如果价格跌破 950,000 美元，则必须清算并损失全部 50,000 美元投资，外加 950,000 美元与清算价格之间的差额。

总结：对冲基金将在价格首次高于 1,020,000 美元或低于 950,000 美元的那一天结束时清算。在第一种情况下，它将获得至少 20,000 美元的利润；在第二种情况下，它将遭受至少 50,000 美元的损失。可以推测，亏损的概率很小，我们将通过模拟来观察这个概率有多小。
