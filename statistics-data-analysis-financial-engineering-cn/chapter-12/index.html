第12章 时间序列模型：基础

12.1 时间序列数据

时间序列（time series）是按时间顺序排列的观测值序列，例如，股票的每日对数收益率或消费者价格指数（CPI）的月度值。一个常见的简化假设是数据等间距分布且具有离散时间观测指标；然而，这仅能近似成立。例如，股票的每日对数收益率可能仅在工作日可获取，在节假日有额外空缺，而CPI的月度值按月份等间距，但按天数则不等间距。在这两种情况下，为简化起见，通常将连续观测视作等间距的。在本章中，我们研究时间序列的统计模型。这些模型广泛应用于计量经济学、商业预测以及许多科学领域。

随机过程（stochastic process）是一个随机变量序列，可视为时间序列的“理论”或“总体”对应物——反之，时间序列可视为来自随机过程的样本。“随机”是随机的同义词。在时间序列模型中实现简约性的最有用的方法之一是假设某种随时间推移的分布不变性，即平稳性（stationarity），这是一个接下来要讨论的性质。

性质通常年复一年是相似的。1 类似地，许多消费品的需求，如防晒霜、冬衣和电力，既有随机变化也有季节性变化，但每个夏天与过去的夏天相似，每个冬天与过去的冬天相似，至少在较短的时间段内如此。平稳随机过程是时间序列的概率模型，其行为不随时间变化。

如果一个过程的所有行为方面都不随时间平移而改变，则称该过程为严格平稳的。数学上，平稳性定义为要求对于每个 \(m\) 和 \(n\)，\((Y_1,\dots,Y_n)\) 与 \((Y_{1+m},\dots,Y_{n+m})\) 的分布相同；即 \(n\) 个观测值序列的概率分布不依赖于它们的时间起点。严格平稳是一个非常强的假设，因为它要求随机行为的“所有方面”在时间上都是常数。通常，假设较弱的条件就足够了，即弱平稳。如果一个过程的均值、方差和协方差不随时间平移而改变，则该过程是弱平稳的。更精确地说，\(Y_1,Y_2,\dots\) 是一个弱平稳过程，如果：

  • \(E(Y_t) = \mu\)（有限常数）对所有 \(t\) 成立；
  • \(\Var(Y_t) = \sigma^2\)（正的有限常数）对所有 \(t\) 成立；以及
  • \(\Cov(Y_t, Y_s) = \gamma(|t-s|)\) 对所有 \(t\) 和 \(s\) 成立，其中 \(\gamma(h)\) 是某个函数。

因此，均值和方差不随时间变化，两个观测值之间的协方差仅取决于滞后，即它们之间的时间距离 \(|t-s|\)，而不是直接依赖于指标 \(t\) 或 \(s\)。例如，如果过程是弱平稳的，那么 \(Y_2\) 与 \(Y_5\) 之间的协方差与 \(Y_7\) 与 \(Y_{10}\) 之间的协方差相同，因为每对观测值间隔三个时间单位。“弱平稳”中的“弱”一词指的是我们仅假设均值、方差和协方差是平稳的，而不是其他分布特征，如分位数、偏度和峰度。弱平稳有时也称为协方差平稳。术语“平稳”有时用作严格平稳的简称。

函数 \(\gamma\) 称为过程的自协方差函数。注意 \(\gamma(h) = \gamma(-h)\)。为什么？假设弱平稳，\(Y_t\) 与 \(Y_{t+h}\) 之间的相关性记为 \(\rho(h)\)。函数 \(\rho\) 称为自相关函数。注意 \(\gamma(0) = \sigma^2\)，且 \(\gamma(h) = \sigma^2 \rho(h)\)。另外，\(\rho(h) = \gamma(h)/\sigma^2 = \gamma(h)/\gamma(0)\)。

12.2 平稳过程

统计建模中，重要的是时间序列本身还是其变化量是否平稳并不关键，因为无论哪种情况我们都能得到简洁的模型。

平稳过程的优美之处在于它可以用相对较少的参数来建模。例如，我们不需要对每个 \(Y_t\) 设定不同的期望；相反，它们都有一个共同的期望 \(\mu\)。这意味着 \(\mu\) 可以通过 \(\bar Y\) 精确估计。反之，如果我们不假定平稳性，每个 \(Y_t\) 都有自己的期望 \(\mu_t\)，那么就无法精确估计 \(\mu_t\) —— \(\mu_t\) 只能由单个观测 \(Y_t\) 本身来估计。

当观测到一个时间序列时，一个自然的问题是它是否看起来平稳。这不是一个容易回答的问题，我们永远无法绝对确定答案。然而，目视检查时间序列及其变化可能有所帮助。时间序列图是按时间顺序绘制的序列图。这个非常基本的图有助于评估平稳行为，尽管还可以用其他图来补充，比如后面将要介绍的样本自相关函数图。此外，还有一些平稳性的统计检验——这些将在第12.10节讨论。

平稳序列的时间序列图应显示出围绕某个固定水平的随机波动，这种现象称为均值回归。如果序列在没有固定水平的情况下漂移，则不应将其建模为平稳过程。

12.2.1 白噪声

白噪声是平稳过程最简单的例子。我们将定义几种白噪声，其假设条件逐渐严格。

序列 \(Y_1, Y_2, \ldots\) 是一个均值为 \(\mu\)、方差为 \(\sigma^2\) 的弱白噪声过程，简记为“weak WN(\(\mu, \sigma^2\))”，如果满足以下条件：

  • 对所有 \(t\)，\(E(Y_t) = \mu\)（有限常数）；
  • 对所有 \(t\)，\(\operatorname{Var}(Y_t) = \sigma^2\)（正有限常数）；
  • 对所有 \(t \neq s\)，\(\operatorname{Cov}(Y_t, Y_s) = 0\)。

如果均值未指定，则假定 \(\mu = 0\)。弱白噪声过程是弱平稳的，且有 \(\gamma(0) = \sigma^2\)，当 \(h \neq 0\) 时 \(\gamma(h) = 0\)。

12.3 估计平稳过程的参数

假设我们从一个弱平稳过程中观测到 \(Y_1, \ldots, Y_n\)。为估计过程的均值 \(\mu\) 和方差 \(\sigma^2\)，我们可以使用样本均值 \(\bar Y\) 和样本方差 \(s^2\)。为估计自协方差函数，我们使用样本自协方差函数

\[ \hat\gamma(h) = n^{-1} \sum_{t=1}^{n-h} (Y_{t+h} - \bar Y)(Y_t - \bar Y) = n^{-1} \sum_{t=h+1}^{n} (Y_t - \bar Y)(Y_{t-h} - \bar Y). \tag{12.2} \]

式 (12.2) 是平稳性假设带来简约性的一个例子。由于 \(Y_t\) 与 \(Y_{t+h}\) 之间的协方差不依赖于 \(t\)，所有间隔 \(h\) 个时间单位的 \(n-h\) 对数据点都可以用来估计 \(\gamma(h)\)。有些作者定义 \(\hat\gamma(h)\) 时，将 (12.2) 中的因子 \(n^{-1}\) 替换为 \((n-h)^{-1}\)，但若 \(n\) 相当大且 \(h\) 相对于 \(n\) 较小（典型情况），这一改变影响甚微。为估计 \(\rho(\cdot)\)，我们使用样本自相关函数（样本 ACF），定义为

\[ \hat\rho(h) = \frac{\hat\gamma(h)}{\hat\gamma(0)}. \]

12.3.1 ACF 图与 Ljung–Box 检验

大多数统计软件会绘制带有检验边界的样本 ACF 图。这些边界用于检验自相关系数为 0 的原假设。若样本自相关超出边界，则拒绝原假设。通常检验水平为 0.05，因此可以预期大约每 20 个样本自相关中有一个会偶然超出边界。另一种方法是使用联合检验，同时检验一组原假设是否全部为真。Ljung–Box 检验的原假设为

\[ H_0 : \rho(1) = \rho(2) = \cdots = \rho(K) = 0 \]

其中 \(K\) 通常取 5 或 10。若 Ljung–Box 检验拒绝原假设，则我们得出结论，\(\rho(1), \rho(2), \ldots, \rho(K)\) 中至少有一个非零。2

  1. 最佳线性预测在第 11.9.1 节中讨论。 ↩

如果事实上自相关1到K全部为零，那么在显著性水平0.05的检验下，错误地认为它们不全为零的概率仅为1/20。相比之下，如果逐个检验自相关，那么认为至少有一个非零的概率会高得多。

Ljung–Box检验有时简称为Box检验，但前者更可取，因为该检验基于Ljung和Box的合著论文。

尽管平稳过程在参数上相对节俭（至少与一般的非平稳过程相比），但对于大多数目的而言，平稳过程仍然不够节俭。问题在于仍然有无限多个参数：ρ(1), ρ(2), …。我们需要一类具有有限（且尽可能少）参数个数的平稳时间序列模型。本章的ARIMA模型正是这样一类模型。最简单的ARIMA模型是自回归（AR）模型，我们首先讨论这些模型。

具有相关性的时间序列模型可以由白噪声构造而成。最简单的相关平稳过程是自回归过程，其中Yt被建模为过去观测值的加权平均加上一个白噪声“误差”（也称为“噪声”或“扰动”）。我们从AR(1)过程开始，这是最简单的自回归过程。

设ε1, ε2, …是弱WN(0, σε2)。如果对于某个常数参数μ和φ，对所有t有

\[ Y_t - \mu = \varphi (Y_{t-1} - \mu) + \varepsilon_t \qquad (12.3) \]

则称Y1, Y2, …是AR(1)过程。参数μ是过程的均值，因此(Yt − μ)对所有t均值为零。我们可以将项φ(Yt−1 − μ)解释为过程过去值对当前值的“记忆”或“反馈”。过程{Yt}t=−∞+∞具有相关性，因为Yt−1对其均值的偏差被反馈到了Yt中。参数φ决定了反馈的程度，φ的绝对值越大，反馈越强；φ = 0意味着Yt = μ + εt，因此Yt是弱WN(μ, σε2)。在金融应用中，我们可以将εt视为“新信息”的影响。例如，如果Yt是资产在时刻t的对数收益率，那么εt代表在时刻t揭露的商业和经济信息对资产价格的影响。真正的新信息是无法预料的，因此今天新信息的影响应当与昨天新信息的影响相互独立。这就是我们把新信息建模为白噪声的原因。

12.4 AR(1) 过程

如果 \(Y_1, Y_2, \dots\) 是一个弱平稳过程，那么 \(|\phi| < 1\)。要理解这一点，注意平稳性意味着式 (12.3) 中 \((Y_t - \mu)\) 和 \((Y_{t-1} - \mu)\) 的方差相等，记为 \(\sigma_Y^2\)。因此，\(\sigma_Y^2 = \phi^2 \sigma_Y^2 + \sigma_\epsilon^2\)，这要求 \(|\phi| < 1\)。该过程的均值为 \(\mu\)。简单代数运算表明 (12.3) 可重写为

\[Y_t = (1-\phi)\mu + \phi Y_{t-1} + \epsilon_t.\tag{12.4}\]

回想你在统计课程中学过的线性回归模型 \(Y_t = \beta_0 + \beta_1 X_t + \epsilon_t\)，或参见第9章回归分析介绍。式 (12.4) 只是一个线性回归模型，截距 \(\beta_0 = (1-\phi)\mu\)，斜率 \(\beta_1 = \phi\)。术语“自回归”指的是该过程对其自身过去值的回归。

如果 \(|\phi| < 1\)，反复代入式 (12.3) 可得

\[Y_t = \mu + \epsilon_t + \phi \epsilon_{t-1} + \phi^2 \epsilon_{t-2} + \cdots = \mu + \sum_{h=0}^{\infty} \phi^h \epsilon_{t-h},\tag{12.5}\]

这里假设时间指标 \(t\) 可扩展到负值，使得白噪声过程为 \(\{\epsilon_t\}_{t=-\infty}^{+\infty}\) 且式 (12.3) 对所有整数 \(t\) 成立。式 (12.5) 称为该过程的无限移动平均 [MA(\(\infty\))] 表示。该式表明 \(Y_t\) 是白噪声过程所有过去值的加权平均。这一表示应与 AR(1) 表示（即 \(Y_t\) 仅依赖于 \(Y_{t-1}\) 和 \(\epsilon_t\)）进行对比。

由于 \(|\phi| < 1\)，当滞后阶数 \(h \to \infty\) 时 \(\phi^h \to 0\)。因此，赋予遥远过去的权重很小。事实上，权重相当小。例如，若 \(\phi = 0.5\)，则 \(\phi^{10} = 0.00098\)，所以 \(\epsilon_{t-10}\) 对 \(Y_t\) 几乎无影响。为此，式 (12.5) 中的求和可以截断为有限项，无需实际假设该过程存在于无限过去。

12.4.1 平稳 AR(1) 过程的性质

当 AR(1) 过程弱平稳时（这意味着 \(|\phi| < 1\)），有

\[\mathbb{E}(Y_t) = \mu \quad \forall t,\tag{12.6}\] \[\operatorname{Var}(Y_t) = \gamma(0) = \sigma_Y^2 = \frac{\sigma_\epsilon^2}{1-\phi^2} \quad \forall t,\tag{12.7}\] \[\operatorname{Cov}(Y_t, Y_{t+h}) = \gamma(h) = \phi^{|h|} \frac{\sigma_\epsilon^2}{1-\phi^2} \quad \forall t \text{ 和 } \forall h,\tag{12.8}\] \[\operatorname{Corr}(Y_t, Y_{t+h}) = \rho(h) = \phi^{|h|} \quad \forall t \text{ 和 } \forall h.\tag{12.9}\]

务必记住，公式 (12.6) 至 (12.9) 仅当 \(|\phi| < 1\) 且仅对 AR(1) 过程成立。此外，为使 \(Y_t\) 平稳，\(Y_0\) 必须从平稳分布开始，使得 \(\mathbb{E}(Y_0) = \mu\) 且 \(\operatorname{Var}(Y_0) = \sigma_\epsilon^2/(1-\phi^2)\)。否则，\(Y_t\) 不是平稳的，尽管它最终会收敛到平稳。

这些公式可以使用 (12.5) 来证明。例如，利用第7.3.2节中的 (7.11)，

\[ \text{Var}(Y_t) = \text{Var}\left(\sum_{h=0}^{\infty} \phi^h \epsilon_{t-h}\right) = \sigma_\epsilon^2 \sum_{h=0}^{\infty} \phi^{2h} = \frac{\sigma_\epsilon^2}{1-\phi^2}, \]

这证明了 (12.7)。在 (12.10) 中，我们使用了几何级数的求和公式。该公式为

\[ \sum_{i=0}^{\infty} r^i = \frac{1}{1-r} \quad \text{若 } |r| < 1. \]

此外，对于 \(h > 0\)，

\[ \text{Cov}\left(\sum_{i=0}^{\infty} \epsilon_{t-i} \phi^i, \sum_{j=0}^{\infty} \epsilon_{t+h-j} \phi^j\right) = \frac{\phi^{|h|} \sigma_\epsilon^2}{1-\phi^2}, \]

从而验证了 (12.8)。然后，(12.9) 通过 (12.8) 除以 (12.7) 得到。

务必区分 \(\sigma_\epsilon^2\)（白噪声过程 \(\epsilon_1, \epsilon_2, \ldots\) 的方差）和 \(\gamma(0)\)（平稳 AR(1) 过程 \(Y_1, Y_2, \ldots\) 的方差 \(\sigma_Y^2\)）。从 (12.7) 可以看出，除非 \(\phi = 0\)（此时 \(Y_t = \mu + \epsilon_t\)，使得 \(Y_t\) 和 \(\epsilon_t\) 具有相同方差），否则 \(\gamma(0)\) 大于 \(\sigma_\epsilon^2\)。

AR(1) 过程的 ACF（自相关函数）仅依赖于一个参数 \(\phi\)。这是极大的简洁性，但也付出了代价。如图12.4所示，AR(1) 过程的 ACF 形态范围有限。其 ACF 的幅度以几何级数衰减至零，或缓慢（如 \(\phi = 0.95\)），或中等缓慢（\(\phi = 0.75\)），或迅速（\(\phi = 0.2\)）。若 \(\phi < 0\)，则 ACF 的符号随幅度几何衰减而交替变化。如果数据的样本 ACF 并非以这些方式之一表现，那么 AR(1) 模型就不适用。解决方法是使用更多的 AR 参数，或转而采用其他模型类，如移动平均 (MA) 模型或自回归移动平均 (ARMA) 模型。我们将在本章中研究这些替代方案。

12.4.3 非平稳AR(1)过程

若\(|\phi| \ge 1\)，则AR(1)过程是非平稳的，均值、方差、协方差和相关系数都不是常数。

随机游走（\(\phi=1\)）

若\(\phi=1\)，则

\[Y_t = Y_{t-1} + \epsilon_t\]

且该过程是非平稳的。这就是我们在第2章中看到的随机游走过程。

12.5 AR(1)过程的估计

假设我们从任意点\(Y_0\)开始这个过程。不难看出

\[Y_t = Y_0 + \epsilon_1 + \cdots + \epsilon_t.\]

那么对于所有\(t\)，有\(\mathbb{E}(Y_t|Y_0) = Y_0\)，这是一个常数，但完全依赖于任意起始点。此外，\(\operatorname{Var}(Y_t|Y_0) = t\sigma_\epsilon^2\)，这不是平稳的，而是随时间线性增加。增大的方差使得随机游走发生“漂移”，即\(Y_t\)在条件均值\(Y_0\)附近做越来越长的徘徊，因此不具有均值回复性。

当\(|\varphi| > 1\)时的AR(1)过程

当\(|\varphi| > 1\)时，AR(1)过程表现出爆炸性行为。这可以从图12.5中看出。该图展示了来自不同\(\varphi\)值的AR(1)过程的200个观测值的模拟结果。其中\(\varphi = 1.01\)的爆炸情形明显不同于\(|\varphi| \leq 1\)的其他情形。然而，\(\varphi = 1\)的情形与\(\varphi = 0.98\)的情形差别并不大，尽管前者是非平稳的而后者是平稳的。更长的时间序列将有助于区分\(\varphi = 0.98\)和\(\varphi = 1\)。

R语言中有函数arima()用于拟合AR及其他时间序列模型。函数arima()以及其他软件包中的类似函数有两种主要的估计方法：条件最小二乘法和极大似然估计。这两种方法将在第12.5.2节中解释。它们类似，通常给出几乎相同的估计值。本书中，我们使用R中arima()的默认方法，即使用条件最小二乘估计作为起始值，通过迭代非线性优化计算极大似然估计（MLE）。

12.5.1 残差与模型检验

一旦估计出\(\mu\)和\(\varphi\)，就可以估计白噪声过程\(\epsilon_1, \ldots, \epsilon_n\)。重新排列式(12.3)，我们有

\[\epsilon_t = (Y_t - \mu) - \varphi(Y_{t-1} - \mu). \qquad (12.14)\]

类似于(12.14)，残差\(\hat{\epsilon}_2, \hat{\epsilon}_3, \ldots, \hat{\epsilon}_n\)定义为

\[\hat{\epsilon}_t = (Y_t - \hat{\mu}) - \hat{\varphi}(Y_{t-1} - \hat{\mu}), \quad t \ge 2, \qquad (12.15)\]

它们估计了\(\epsilon_2, \ldots, \epsilon_n\)。第一个噪声项\(\epsilon_1\)无法直接估计，因为假定观测值从\(Y_1\)开始，因此\(Y_0\)不可用。

12.5 AR(1)过程的估计

图12.5展示了来自AR(1)过程的200个观测值的模拟，其中φ取不同值且μ=0。白噪声过程ϵ1, ϵ2, …, ϵ200对所有四个AR(1)过程都是相同的。

残差可用于检验Y1, Y2, …, Yn是AR(1)过程的假设；残差中的任何自相关都是反对AR(1)过程假设的证据。

为了理解为什么残差自相关表明模型可能存在问题，假设我们拟合一个AR(1)模型，Yt = μ + φ(Yt−1 − μ) + ϵt，但真实模型是如下的AR(2)过程4：

\[ Y_t - \mu = \phi_1 (Y_{t-1} - \mu) + \phi_2 (Y_{t-2} - \mu) + \epsilon_t. \]

由于我们拟合了错误的AR(1)模型，无法估计φ2，因为它不在模型中。而且，由于模型设定偏误导致的偏倚，\(\hat{\phi}\)不一定估计φ1。令φ*为\(\hat{\phi}\)的期望值。为了说明，假设\(\hat{\mu} \approx \mu\) 且 \(\hat{\phi} \approx \phi^*\)。当样本量n足够大时，这是一个合理的近似。那么

\[ \hat{\epsilon}_t \approx (Y_t - \mu) - \phi^* (Y_{t-1} - \mu) \] \[ = \phi_1 (Y_{t-1} - \mu) + \phi_2 (Y_{t-2} - \mu) + \epsilon_t - \phi^* (Y_{t-1} - \mu) \] \[ = (\phi_1 - \phi^*)(Y_{t-1} - \mu) + \phi_2 (Y_{t-2} - \mu) + \epsilon_t. \]

我们很快会讨论更高阶的AR模型。

因此，残差并不像使用正确AR(2)模型时那样估计白噪声过程。即使\(\hat{\phi}\)对\(\phi_1\)的估计无偏，使得\(\phi_1 = \phi^*\)，且项\((\phi_1 - \phi^*)(Y_{t-1} - \mu)\)消失，残差中存在的\(\phi_2(Y_{t-2} - \mu)\)仍会导致它们自相关。

为检查残差自相关，可使用ACF图的检验界。任何超出检验界的残差ACF值在0.05水平下显著不为0。

如前所述，风险在于部分样本ACF值可能仅因偶然因素而显著。为防范此风险，可使用Ljung-Box检验，该检验同时检验直至指定滞后阶的所有自相关是否为零。

当Ljung-Box检验应用于残差时，需要进行校正以考虑用\(\hat{\phi}\)代替未知\(\phi\)。部分软件自动完成此校正。在R中，校正并非自动，而是通过将Box.test()中的fitdf参数设为估计的自回归系数参数个数来实现，因此对于AR(1)模型，fitdf应设为1。

12.5.2 AR过程的极大似然估计与条件最小二乘估计

大多数统计软件包可以自动计算AR过程的估计量，用户无需知道软件“引擎盖下”的内容。尽管如此，本小节为对估计方法感兴趣的读者而设。

为了求 \(Y_1,\ldots,Y_n\) 的联合密度，我们使用(A.41)以及

\[f_{Y_t|Y_1,\ldots,Y_{t-1}}(y_t|y_1,\ldots,y_{t-1}) = f_{Y_t|Y_{t-1}}(y_t|y_{t-1}) \qquad (12.16)\]

（公式待续）

对于\(t=2,3,\ldots,n\)。具有性质(12.16)的随机过程称为马尔可夫过程。由(A.41)和(12.16)，我们有

\[ f_{Y_1,\ldots,Y_n}(y_1,\ldots,y_n)= f_{Y_1}(y_1)\prod_{t=2}^{n} f_{Y_t|Y_{t-1}}(y_t|y_{t-1}). \tag{12.17} \]

如果我们假设误差来自高斯白噪声过程，那么由(12.6)和(12.7)，对于平稳AR(1)过程，我们知道\(Y_1\)服从\(\mathcal{N}\{\mu,\sigma_\epsilon^2/(1-\varphi^2)\}\)。给定\(Y_{t-1}\)，\(Y_t\)的唯一随机成分是\(\epsilon_t\)，所以给定\(Y_{t-1}\)时\(Y_t\)服从\(\mathcal{N}\{\mu+\varphi(Y_{t-1}-\mu),\sigma_\epsilon^2\}\)。于是，\(Y_1,\ldots,Y_n\)的联合密度为

\[ \left(\frac{1}{\sqrt{2\pi}\sigma_\epsilon}\right)^n \sqrt{1-\varphi^2}\,\exp\left\{-\frac{(Y_1-\mu)^2}{2\sigma_\epsilon^2/(1-\varphi^2)}\right\} \prod_{i=2}^{n}\exp\left(-\frac{\bigl[Y_i-\{\mu+\varphi(Y_{i-1}-\mu)\}\bigr]^2}{2\sigma_\epsilon^2}\right). \tag{12.18} \]

极大似然估计通过最大化(12.18)关于\((\mu,\varphi,\sigma_\epsilon)\)的对数来得到。一个更简单的估计量是从似然中删除\(Y_1\)的边际密度，然后最大化

\[ \left(\frac{1}{\sqrt{2\pi}\sigma_\epsilon}\right)^{n-1}\prod_{t=2}^{n}\exp\left(-\frac{\bigl[Y_t-\{\mu+\varphi(Y_{t-1}-\mu)\}\bigr]^2}{2\sigma_\epsilon^2}\right) \tag{12.19} \]

的对数。这个估计量称为条件最小二乘估计。它是“条件”的，因为它使用了给定\(Y_1\)时\(Y_2,\ldots,Y_n\)的条件密度。它是最小二乘估计，因为\(\mu\)和\(\varphi\)的估计最小化

12.6 AR(p) 模型

我们已经看到，当 \(|\phi| < 1\) 时，AR(1) 过程的 ACF 呈几何衰减至零，且当 \(\phi < 0\) 时，ACF 会交替符号。这种行为的范围有限，许多时间序列并不以此方式表现。为了获得更灵活但仍简洁的模型类别，我们可以使用一种模型，将过程的当前值对其过去多个近期值（而不仅仅是最新值）进行回归。因此，我们让过程的最后 \(p\) 个值 \(Y_{t-1}, \ldots, Y_{t-p}\) 反馈到当前值 \(Y_t\) 中。

以下是正式定义。随机过程 \(Y_t\) 称为 AR(p) 过程，如果

\[ Y_t - \mu = \phi_1 (Y_{t-1} - \mu) + \phi_2 (Y_{t-2} - \mu) + \cdots + \phi_p (Y_{t-p} - \mu) + \epsilon_t, \]

其中 \(\epsilon_1, \epsilon_2, \ldots\) 是弱白噪声 WN\((0, \sigma_\epsilon^2)\)。

这是一个以时间序列的滞后值作为“x 变量”的多元线性回归5模型。该模型也可以表示为

\[ Y_t = \beta_0 + \phi_1 Y_{t-1} + \cdots + \phi_p Y_{t-p} + \epsilon_t, \]

其中 \(\beta_0 = \{1 - (\phi_1 + \cdots + \phi_p)\} \mu\)。参数 \(\beta_0\) 被称为“常数”或“截距”，类似于 AR(1) 模型。可以证明，对于平稳过程，\(\{1 - (\phi_1 + \cdots + \phi_p)\} > 0\)，因此 \(\mu = 0\) 当且仅当 \(\beta_0 = 0\)。

对于 \(p > 1\) 的 AR(p) 过程，其 ACF 的公式比 AR(1) 过程更为复杂，可在“参考文献”部分列出的时间序列教材中找到。然而，有软件可以计算和绘制任何 AR 过程的 ACF，以及即将介绍的 MA 和 ARMA 过程的 ACF。图 12.9 展示了三个 AR(2) 过程的 ACF 图。这些 ACF 是使用 R 的 ARMAacf() 函数计算的。注意，通过两个 AR 参数可以得到多种多样的 ACF 形态。

我们讨论的关于 AR(1) 模型的大多数概念很容易推广到 AR(p) 模型。条件最小二乘估计或极大似然估计可以使用诸如 R 的 arima() 函数等软件计算。残差定义为

\[ \hat{\epsilon}_t = Y_t - \{\hat{\beta}_0 + \hat{\phi}_1 Y_{t-1} + \cdots + \hat{\phi}_{t-p} Y_{t-p}\}, \quad t \geq p+1. \]
  1. 参见第9章对多元回归的介绍。 ↩

如果AR(p)模型对时间序列拟合良好，那么残差应表现为白噪声。可以通过检查残差样本ACF和Ljung-Box检验来检测残差自相关。任何显著的残差自相关都表明AR(p)模型拟合不佳。

AR模型的一个问题是它们通常需要较大的p值来拟合数据集。以下两个例子说明了这个问题。

12.7.1 MA(1) 过程

AR过程的基本思想是将过去的数据反馈到当前值中，从而在现在和过去之间建立相关性，其效果是所有滞后项至少存在一定的相关性。而有时数据仅表现出短滞后的相关性，例如仅在滞后1阶或滞后1阶和2阶有相关性。以图12.3b为例，通货膨胀率变化的样本ACF在滞后1阶约为−0.4，但在1阶之后的值大约为0.1或更小。AR过程并非如此，如示例12.6所示，它无法提供简洁的拟合。在这种情况下，MA模型是AR模型的一种有用替代。若过程 Yt 可表示为白噪声过程 {ϵt} 过去值的加权平均（移动平均），则 Yt 称为移动平均过程。

12.7 移动平均(MA)过程

MA(1)（一阶移动平均）过程定义为

\[ Y_t - \mu = \epsilon_t + \theta \epsilon_{t-1}, \tag{12.21} \]

其中 \(\epsilon_t\) 如前所述是弱白噪声 WN(\(0, \sigma^2_\epsilon\))。6

可以证明：

\[ \begin{aligned} \mathrm{E}(Y_t) &= \mu, \\ \mathrm{Var}(Y_t) &= \sigma^2_\epsilon (1 + \theta^2), \\ \gamma(1) &= \theta \sigma^2_\epsilon, \\ \gamma(h) &= 0 \quad \text{若 } |h| > 1, \\ \rho(1) &= \frac{\theta}{1 + \theta^2}, \tag{12.22} \\ \rho(h) &= 0 \quad \text{若 } |h| > 1. \tag{12.23} \end{aligned} \]

注意 (12.22) 和 (12.23) 的含义——MA(1) 模型在所有滞后阶数处（除滞后1和滞后0）都具有零自相关。这些公式的推导较为简单，留给读者作为练习。

  1. 一些教科书和软件在编写MA模型时符号是相反的，因此模型 (12.21) 写为 \(Y_t - \mu = \epsilon_t - \theta \epsilon_{t-1}\)。我们采用与R的 arima() 函数相同的MA模型形式。这些说明同样适用于由方程 (12.24) 和 (12.25) 给出的通用MA和ARMA模型。↩

12.7.2 一般MA过程

MA(q)过程可以表示为

\[ Y_t = \mu + \epsilon_t + \theta_1 \epsilon_{t-1} + \cdots + \theta_q \epsilon_{t-q}. \tag{12.24} \]

可以证明，当\(|h| > q\)时，\(\gamma(h) = 0\)且\(\rho(h) = 0\)。当\(|h| \leq q\)时，\(\gamma(h)\)和\(\rho(h)\)的公式可在时间序列教材中找到，并且这些函数可以在R中通过函数ARMAacf()计算。

与AR(p)模型中模型的“常数项”不等于均值不同，在MA(q)模型中，过程均值\(\mu\)与模型中的“常数项”\(\beta_0\)相同。通过观察式(12.24)的右侧可以理解这一点，其中\(\mu\)是模型中的“截距”或“常数项”，同时也是\(Y_t\)的均值，因为\(\epsilon_t, \ldots, \epsilon_{t-q}\)的均值为零。

MA(q)模型可以方便地使用R中的arima()函数进行拟合。

12.8 ARMA过程

具有复杂自相关行为的平稳时间序列通常可以通过混合自回归滑动平均（ARMA）过程来更简约地建模，而不是纯AR或纯MA过程。例如，有时具有一个AR参数和一个MA参数的模型（称为ARMA(1,1)模型）会比纯AR或纯MA模型提供更简约的拟合。本节介绍ARMA过程。

12.8.1 滞后算子

滞后算子\(B\)是一个名称花哨的简单记号。它对于描述ARMA（以及ARIMA）模型很有用。滞后算子定义为

\(B Y_t = Y_{t-1}\)

且更一般地，

\(B^h Y_t = Y_{t-h}\)。

因此，\(B\)将时间后退一个单位，而\(B^h\)则重复此操作，使时间后退\(h\)个单位。注意，对于任意常数\(c\)，有\(Bc = c\)，因为常数不随时间变化。滞后算子有时也称为滞后算子（lag operator）。

12.8.2 ARMA模型

ARMA(p, q)模型同时包含AR项和MA项，由以下方程定义：

\[ (Y_t - \mu) = \phi_1 (Y_{t-1} - \mu) + \cdots + \phi_p (Y_{t-p} - \mu) + \epsilon_t + \theta_1 \epsilon_{t-1} + \cdots + \theta_q \epsilon_{t-q}, \tag{12.25} \]

这显示了Y_t如何依赖于自身的滞后值以及白噪声过程的滞后值。方程(12.25)可以用后移算子更简洁地写为

\[ (1 - \phi_1 B - \cdots - \phi_p B^p)(Y_t - \mu) = (1 + \theta_1 B + \cdots + \theta_q B^q) \epsilon_t. \tag{12.26} \]

白噪声过程是ARMA(0, 0)，因为若p = q = 0，则(12.26)简化为

\[ (Y_t - \mu) = \epsilon_t. \]

12.8.3 ARMA(1,1)过程

ARMA(1,1)模型在实践中常用，且足够简单，可以进行理论研究。本节将推导其方差和ACF的公式。不失一般性，在计算方差和ACF时可以假设μ = 0。将模型

\[ Y_t = \phi Y_{t-1} + \theta \epsilon_{t-1} + \epsilon_t \tag{12.27} \]

乘以ε_t并取期望，得到

\[ \text{Cov}(Y_t, \epsilon_t) = E(Y_t \epsilon_t) = \sigma^2_{\epsilon}, \tag{12.28} \]

因为ε_t与ε_{t-1}和Y_{t-1}独立。由(12.27)和(12.28)，

\[ \gamma(0) = \phi^2 \gamma(0) + (1 + \theta^2)\sigma^2_{\epsilon} + 2 \phi \theta \sigma^2_{\epsilon}, \tag{12.29} \]

然后解(12.29)得到γ(0)的公式

\[ \gamma(0) = \frac{(1 + \theta^2 + 2\phi\theta)\sigma^2_{\epsilon}}{1 - \phi^2}. \tag{12.30} \]

通过类似计算，将(12.27)乘以Y_{t-1}并取期望，得到γ(1)的公式。将该公式除以(12.29)的右边，得到

\[ \rho(1) = \frac{(1 + \phi\theta)(\phi + \theta)}{1 + \theta^2 + 2\phi\theta}. \tag{12.31} \]

对于h ≥ 2，将(12.27)乘以Y_{t-h}并取期望，得到公式

\[ \rho(h) = \phi \rho(h-1), \quad h \geq 2. \tag{12.32} \]

由(12.32)，经过一个滞后，ARMA(1,1)过程的ACF与具有相同φ的AR(1)过程的ACF以相同方式衰减。

12.8.4 ARMA参数的估计

12.8.5 差分算子

差分算子是另一种有用的记号，定义为\(\Delta = 1 - B\)，其中\(B\)是后向算子，因此

\[\Delta Y_t = Y_t - B Y_t = Y_t - Y_{t-1}.\]

12.9 ARIMA过程

例如，如果 \(p_t = \log(P_t)\) 是对数价格，则对数收益率为

\[ r_t = \Delta p_t. \]

差分可以迭代。例如，

\[ \Delta^2 Y_t = \Delta(\Delta Y_t) = \Delta(Y_t - Y_{t-1}) = (Y_t - Y_{t-1}) - (Y_{t-1} - Y_{t-2}) = Y_t - 2Y_{t-1} + Y_{t-2}. \]

\(\Delta^k\) 称为 \(k\) 阶差分算子。 \(\Delta^k\) 的一般公式可由二项式展开得到：

\[ \Delta^k Y_t = (1 - B)^k Y_t = \sum_{\ell=0}^k \binom{k}{\ell} (-1)^\ell Y_{t-\ell}. \qquad (12.33) \]

非平稳时间序列的一阶或二阶差分通常是平稳的。例如，随机游走（非平稳）的一阶差分是白噪声（平稳）。本节介绍自回归滑动平均（ARIMA）过程，它涵盖平稳和非平稳过程。

12.9 ARIMA过程

一个时间序列 \(Y_t\) 被称为 ARIMA(p,d,q) 过程，如果 \(\Delta^d Y_t\) 是 ARMA(p,q) 过程。例如，如果某项资产的对数收益率服从 ARMA(p,q) 模型，那么对数价格服从 ARIMA(p,1,q) 模型。ARIMA(p,d,q) 过程只有在 \(d=0\) 时才是平稳的。否则，只有其 \(d\) 阶或更高阶差分是平稳的。

注意，ARIMA(p,0,q) 模型与 ARMA(p,q) 模型相同。ARIMA(p,0,0)、ARMA(p,0) 和 AR(p) 模型相同。类似地，ARIMA(0,0,q)、ARMA(0,q) 和 MA(q) 模型相同。随机游走是 ARIMA(0,1,0) 模型，白噪声是 ARIMA(0,0,0) 模型。

差分的逆运算是“积分”。过程 \(Y_t\) 的积分是过程 \(w_t\)，其中 \[ w_t = w_{t_0} + Y_{t_0+1} + \cdots + Y_t. \tag{12.34} \] 这里 \(t_0\) 是任意起始时间点，\(w_{t_0}\) 是 \(w_t\) 过程的起始值。容易验证 \[ \Delta w_t = Y_t, \tag{12.35} \] 所以积分和差分互为逆运算7。

如果一个过程经过 \(d\) 次差分后变为平稳，则称该过程为 I(d)。例如，平稳过程是 I(0)。ARIMA(p,d,q) 过程是 I(d)。I(d) 过程被称为“\(d\) 阶单整”。

图12.14展示了一个 AR(1) 过程、其积分以及其二次积分（即积分的积分）。这三个过程分别是 I(0)、I(1) 和 I(2)。它们的行为截然不同。AR(1) 过程是平稳的，并围绕其均值（0）随机波动；通常说该过程均值回归。其积分的行为非常类似于随机游走，没有固定的均值水平。二次积分具有动量：一旦过程开始向上或向下的运动，它往往会持续这一方向。如果数据表现出这种动量，则表明 \(d=2\)。该 AR(1) 过程由 R 函数 arima.sim() 生成。该过程使用 R 的 cumsum() 函数进行了两次积分。

12.9.1 ARIMA过程中的漂移

如果一个非平稳过程具有恒定均值，那么该过程的一阶差分均值应为零。因此，通常假设差分后的过程均值为零。R中的arima()函数采用了这一假设。

12.10 单位根检验

有时，非平稳过程的均值不是常数，而是带有确定性线性趋势，例如 \(E(Y_t) = \beta_0 + \beta_1 t\)。此时，\(\beta_1\) 称为 \(Y_t\) 的漂移。注意 \(E(\Delta Y_t) = \beta_1\)，因此如果 \(Y_t\) 有非零漂移，则 \(\Delta Y_t\) 有非零均值。第12.11节讨论的R函数 auto.arima() 允许差分过程具有非零均值，在输出中称之为漂移（drift）。

这些思想可以推广到更高阶的多项式趋势和更高阶差分。如果 \(E(Y_t)\) 具有 \(m\) 次多项式趋势，则对于 \(d \leq m\)，\(E(\Delta^d Y_t)\) 的均值具有 \(m-d\) 次趋势。当 \(d > m\) 时，\(E(\Delta^d Y_t) = 0\)。

我们已经看到，判断一个时间序列最适合用平稳模型还是非平稳模型建模有时是困难的。为帮助在这两种可能性之间做出决定，使用假设检验可能会有帮助。

什么是单位根？回忆一下，一个 ARMA\((p,q)\) 过程可以写为 \[ (Y_t - \mu) = \phi_1 (Y_{t-1} - \mu) + \cdots + \phi_p (Y_{t-p} - \mu) + \epsilon_t + \theta_1 \epsilon_{t-1} + \cdots + \theta_q \epsilon_{t-q}. \tag{12.36} \]

12.10 单位根检验

图12.17绘制了(a) log(IP)和(b) Δlog(IP)的时间序列图，以及(c) Δlog(IP)和(d) 对Δlog(IP)拟合ARMA(1,0)后残差的样本ACF图。

序列\(\{Y_t\}\)平稳的条件是多项式

\[1 - \phi_1 x - \cdots - \phi_p x^p \qquad (12.37)\]

的所有根的绝对值都大于1。(关于多项式复根和复数绝对值的详细信息，参见附录A.21。)例如，当\(p = 1\)时，式(12.37)为

\[1 - \phi x\]

它有一个根\(1/\phi\)。我们知道，当\(|\phi| < 1\)时过程是平稳的，这当然等价于\(|1/\phi| > 1\)。

如果存在单位根，即有一个根的绝对值等于1，那么ARMA过程就是非平稳的，其表现非常类似于随机游走。毫不奇怪，这被称为单位根情形。爆炸性情形是某个根的绝对值小于1。

12.10.1 单位根检验是如何工作的？

关于单位根检验理论的完整讨论超出了本书的范围。这里只提及基本思想。更多信息请参见第12.14节。Dickey-Fuller检验基于AR(1)模型

\[ Y_t = \phi Y_{t-1} + \epsilon_t. \qquad (12.38) \]

原假设 \(H_0\) 是存在单位根，即 \(\phi = 1\)，备择假设 \(H_1\) 是平稳性，等价于 \(\phi < 1\)，假定 \(\phi > -1\)（这是合理的）。AR(1)模型(12.38)等价于

\[ \Delta Y_t = (\phi - 1)Y_{t-1} + \epsilon_t, \]

或

\[ \Delta Y_t = \pi Y_{t-1} + \epsilon_t, \qquad (12.39) \]

其中 \(\pi = \phi - 1\)。用 \(\pi\) 表示，\(H_0\) 为 \(\pi = 0\)，\(H_1\) 为 \(\pi < 0\)。

Dickey-Fuller检验将 \(\Delta Y_t\) 对 \(Y_{t-1}\) 回归并检验 \(H_0\)。由于在 \(H_0\) 下 \(Y_{t-1}\) 是非平稳的，\(\pi\) 的 t 统计量具有非标准分布，因此需要开发特殊的表格来计算 p 值。

增广 Dickey-Fuller 检验通过在模型(12.39)中加入时间趋势和 \(\Delta Y_t\) 的滞后项进行扩展。通常，时间趋势是线性的，因此扩展模型为

\[ \Delta Y_t = \beta_0 + \beta_1 t + \pi Y_{t-1} + \sum_{j=1}^p \gamma_j \Delta Y_{t-j} + \epsilon_t. \qquad (12.40) \]

假设仍然为 \(H_0: \pi = 0\) 和 \(H_1: \pi < 0\)。选择 \(p\) 有几种方法。adf.test() 函数中 \(p\) 的默认值为 trunc((length(y)-1)^(1/3))，其中 \(y\) 是输入序列（即我们记号中的 \(Y_t\)）。

import numpy as np
import pandas as pd
from statsmodels.tsa.stattools import kpss
from statsmodels.tsa.arima.model import ARIMA
import itertools

def auto_arima_python(y, max_p=5, max_q=5, ic='bic'):
    # Determine d using KPSS test
    def determine_d(series):
        d = 0
        while True:
            stat, pval, _, _ = kpss(series, regression='c')
            if pval > 0.05:
                return d
            else:
                d += 1
                series = np.diff(series)
    d = determine_d(y.values if isinstance(y, pd.Series) else y)

    best_aic = np.inf
    best_bic = np.inf
    best_order = None
    best_model = None
    for p, q in itertools.product(range(max_p+1), range(max_q+1)):
        if p == 0 and q == 0:
            continue
        try:
            model = ARIMA(y, order=(p, d, q))
            fitted = model.fit()
            if ic == 'aic':
                crit = fitted.aic
            else:
                crit = fitted.bic
            if crit < (best_aic if ic == 'aic' else best_bic):
                best_aic = fitted.aic
                best_bic = fitted.bic
                best_order = (p, d, q)
                best_model = fitted
        except:
            continue
    return best_model, best_order

12.12 预测

预测是指利用当前信息集（即时间序列的现在和过去值组成的集合）来预测时间序列的未来值。在某些情况下，信息集可以包含与该时间序列相关的其他变量，但本节中信息集仅包含被预测时间序列的过去和现在值。

12.12 预测

ARIMA模型常用于预测。考虑使用AR(1)过程进行预测。假设我们有数据 \(Y_1, \ldots, Y_n\) 以及估计值 \(\hat{\mu}\) 和 \(\hat{\phi}\)。我们知道

\[ Y_{n+1} = \mu + \phi (Y_n - \mu) + \epsilon_{n+1}. \tag{12.41} \]

由于 \(\epsilon_{n+1}\) 与过去和现在独立，根据第11.9.2节的结果11.1，对 \(\epsilon_{n+1}\) 的最佳预测是其期望值0。当然，我们明白 \(\epsilon_{n+1}\) 不为0，但0是我们对其值的最佳猜测。另一方面，我们知道或估计了(12.41)中所有其他量。因此，我们通过下式预测 \(Y_{n+1}\)：

\[ \hat{Y}_{n+1} = \hat{\mu} + \hat{\phi} (Y_n - \hat{\mu}). \]

同理，我们预测 \(Y_{n+2}\) 为

\[ \hat{Y}_{n+2} = \hat{\mu} + \hat{\phi} (\hat{Y}_{n+1} - \hat{\mu}) = \hat{\mu} + \hat{\phi} \{ \hat{\phi} (Y_n - \hat{\mu}) \}, \tag{12.42} \]

以此类推。注意，在(12.42)中我们使用的不是 \(Y_{n+1}\)（它在时间 \(n\) 是未知的），而是预测值 \(\hat{Y}_{n+1}\)。继续这样做，我们得到 \(k\) 步超前预测的通用公式：

\[ \hat{Y}_{n+k} = \hat{\mu} + \hat{\phi}^k (Y_n - \hat{\mu}). \tag{12.43} \]

如果 \(|\hat{\phi}| < 1\)（平稳序列的情形），则随着 \(k\) 的增加，预测值将几何级数地快速收敛到 \(\hat{\mu}\)。

公式(12.43)仅适用于AR(1)过程，但对其他AR(p)过程的预测是类似的。对于AR(2)过程，

\[ \hat{Y}_{n+1} = \hat{\mu} + \hat{\phi}_1 (Y_n - \hat{\mu}) + \hat{\phi}_2 (Y_{n-1} - \hat{\mu}), \] \[ \hat{Y}_{n+2} = \hat{\mu} + \hat{\phi}_1 (\hat{Y}_{n+1} - \hat{\mu}) + \hat{\phi}_2 (Y_n - \hat{\mu}), \]

等等。

对ARMA和ARIMA过程的预测与对AR过程的预测类似。考虑MA(1)过程 \(Y_t - \mu = \epsilon_t + \theta \epsilon_{t-1}\)。那么下一个观测值为

\[ Y_{n+1} = \mu + \epsilon_{n+1} + \theta \epsilon_n. \tag{12.44} \]

在(12.44)的右侧，我们将 \(\mu\) 和 \(\theta\) 替换为估计值，将 \(\epsilon_n\) 替换为残差 \(\hat{\epsilon}_n\)。此外，由于 \(\epsilon_{n+1}\) 与观测数据独立，故用其均值0代替。则预测值为

\[ \hat{Y}_{n+1} = \hat{\mu} + \hat{\theta} \hat{\epsilon}_n. \]

对 \(Y_{n+2} = \mu + \epsilon_{n+2} + \theta \epsilon_{n+1}\) 的两步超前预测是简单的 \(\hat{Y}_{n+2} = \hat{\mu}\)，因为 \(\epsilon_{n+1}\) 和 \(\epsilon_{n+2}\) 与观测数据独立。类似地，对所有 \(k > 2\)，有 \(\hat{Y}_{n+k} = \hat{\mu}\)。

为了预测ARMA(1,1)过程

\[Y_t - \mu = \phi (Y_{t-1} - \mu) + \epsilon_t + \theta \epsilon_{t-1},\]

我们使用

\[\hat{Y}_{n+1} = \hat{\mu} + \hat{\phi}(Y_n - \hat{\mu}) + \hat{\theta}\hat{\epsilon}_n\]

作为一步向前预测，并

\[\hat{Y}_{n+k} = \hat{\mu} + \hat{\phi}(\hat{Y}_{n+k-1} - \hat{\mu}), \quad k \ge 2\]

作为两步及以上的预测。

作为最后一个示例，假设\(Y_t\)是ARIMA(1,1,0)，因此\(\Delta Y_t\)是AR(1)。为了预测\(Y_{n+k}\)，\(k \ge 1\)，我们首先对\(\Delta Y_t\)过程拟合AR(1)模型，并预测\(\Delta Y_{n+k}\)，\(k \ge 1\)。设预测值记为\(\hat{\Delta}Y_{n+k}\)，\(k \ge 1\)。那么，由于

\[Y_{n+1} = Y_n + \Delta Y_{n+1},\]

\(Y_{n+1}\)的预测为

\[\hat{Y}_{n+1} = Y_n + \hat{\Delta}Y_{n+1},\]

类似地

\[\hat{Y}_{n+2} = \hat{Y}_{n+1} + \hat{\Delta}Y_{n+2} = Y_n + \hat{\Delta}Y_{n+1} + \hat{\Delta}Y_{n+2},\]

以此类推。

大多数时间序列软件包都提供自动化预测的函数。R中的predict()函数使用arima()拟合函数返回的“对象”进行预测。

12.12.1 预测误差与预测区间

在进行预测时，我们自然希望知道预测的不确定性。为此，首先计算预测误差的方差。然后，\((1-\alpha)100\%\)预测区间为预测值本身加减预测误差的标准差乘以\(z_{\alpha/2}\)（正态上分位数）。使用\(z_{\alpha/2}\)假设\(\epsilon_1, \epsilon_2, \ldots\)是高斯白噪声。如果残差具有厚尾特征，我们可能不愿做出高斯假设。避免这一假设的一种方法在12.12.2节中讨论。现代统计软件自动计算预测误差方差和预测区间，因此我们无需给出预测误差方差的一般公式。然而，为了理解一般原理，我们考察两个特例：一个平稳，另一个非平稳。

12.12 预测

平稳AR(1)预测误差

我们首先考虑预测平稳AR(1)过程时的误差。第一步预测的误差为：

\[ Y_{n+1} - \hat{Y}_{n+1} = \{\mu + \phi (Y_n - \mu) + \epsilon_{n+1}\} - \{\hat{\mu} + \hat{\phi}(Y_n - \hat{\mu})\} \] \[ = (\mu - \hat{\mu}) + (\phi - \hat{\phi})Y_n - (\phi\mu - \hat{\phi}\hat{\mu}) + \epsilon_{n+1} \tag{12.45} \] \[ \approx \epsilon_{n+1}. \tag{12.46} \]

其中(12.45)是精确误差，(12.46)是“大样本”近似。(12.46)的依据是：随着样本量增大，\(\hat{\mu} \to \mu\) 且 \(\hat{\phi} \to \phi\)，因此(12.45)中的前三项收敛于0，而最后一项保持不变。大样本近似简化了公式，并帮助我们聚焦于预测误差的主要成分。再次使用大样本近似，即用 \(\mu\) 替代 \(\hat{\mu}\)，用 \(\phi\) 替代 \(\hat{\phi}\)，两步超前预测的误差为：

\[ Y_{n+2} - \hat{Y}_{n+2} = \{\mu + \phi(Y_{n+1} - \mu) + \epsilon_{n+2}\} - \{\mu + \phi(\hat{Y}_{n+1} - \mu)\} \] \[ = \phi(Y_{n+1} - \hat{Y}_{n+1}) + \epsilon_{n+1} \] \[ = \phi\epsilon_{n+1} + \epsilon_{n+2}. \tag{12.47} \]

以此类推，我们发现 \(k\) 步超前预测误差为：

\[ Y_{n+k} - \hat{Y}_{n+k} \approx \{\mu + \phi(Y_{n+k-1} - \mu) + \epsilon_{n+k}\} - \{\mu + \phi(\hat{Y}_{n+k-1} - \mu)\} \] \[ = \phi^{k-1}\epsilon_{n+1} + \phi^{k-2}\epsilon_{n+2} + \cdots + \phi\epsilon_{n+k-1} + \epsilon_{n+k}. \tag{12.48} \]

根据有限几何级数求和公式，(12.48)右边部分的方差为：

\[ \left\{ \phi^{2(k-1)} + \phi^{2(k-2)} + \cdots + \phi^2 + 1 \right\} \sigma^2_\epsilon = \left( \frac{1 - \phi^{2k}}{1 - \phi^2} \right) \sigma^2_\epsilon \to \frac{\sigma^2_\epsilon}{1 - \phi^2} \quad \text{当 } k \to \infty. \tag{12.49} \]

这里重要的一点是，预测误差的方差不会随着 \(k \to \infty\) 而发散，而是收敛到 \(\gamma(0)\)，即由(12.7)给出的AR(1)过程的边际协方差。这是一个一般原则的例子：对任何平稳ARMA过程，预测误差的方差收敛到边际方差。

预测随机游走

对于随机游走过程 \(Y_{n+1} = \mu + Y_n + \epsilon_{n+1}\)，刚才为AR(1)过程推导的许多公式仍然成立，但其中 \(\phi = 1\)。一个例外是

(12.49)中的最后一个结果不成立，因为当\(\phi = 1\)时，几何级数的求和公式不适用。但仍然成立的一个结果是

\[Y_{n+k} - \hat{Y}_{n+k} = \epsilon_{n+1} + \epsilon_{n+2} + \cdots + \epsilon_{n+k-1} + \epsilon_{n+k}\]

因此，\(k\)步超前预测误差的方差为\(k\sigma_\epsilon^2\)，与平稳AR(1)情况不同，当\(k \to \infty\)时，预测误差方差发散至\(\infty\)。

ARIMA过程的预测

如前所述，在实践中我们不需要ARIMA过程预测误差方差的一般公式，因为统计软件可以计算方差。然而，值得重申一个一般原理：对于平稳ARMA过程，\(k\)步超前预测误差方差的方差随着\(k \to \infty\)收敛到一个有限值，但对于非平稳ARIMA过程，该方差发散至\(\infty\)。这一原理的结果是，对于非平稳过程，预测限随着\(k \to \infty\)而彼此发散，但对于平稳过程，预测限收敛到平行的水平线。

12.12.2 通过模拟计算预测限

模拟可用于计算预测限。通过模拟随机预测并找出它们的\(\alpha/2\)上、下样本分位数来完成。对于ARMA过程，通过递归生成一组到未来\(m\)个时间单位的随机预测：

\[\hat{Y}_{n+t} = \hat{\mu} + \hat{\phi}_1(\hat{Y}_{n+t-1} - \hat{\mu}) + \cdots + \hat{\phi}_p(\hat{Y}_{n+t-p} - \hat{\mu}) + \hat{\epsilon}_{n+t} + \hat{\theta}_1\hat{\epsilon}_{n+t-1} + \cdots + \hat{\theta}_q\hat{\epsilon}_{n+t-q}, \quad t = 1, \ldots, m \qquad (12.50)\]

12.12 预测

其中

  • \(\hat{\epsilon}_k\) 是第 \(k\) 个残差（若 \(k \le n\)）；
  • \(\{\hat{\epsilon}_k : k = n+1, \dots, n+m\}\) 是从残差中重抽样得到的集合。

因此，\(\hat{Y}_{n+1}\) 由 \(Y_{n+1-p}, \dots, Y_n, \hat{\epsilon}_{n+1-q}, \dots, \hat{\epsilon}_{n+1}\) 生成，然后 \(\hat{Y}_{n+2}\) 由 \(Y_{n+2-p}, \dots, Y_n, \hat{Y}_{n+1}, \hat{\epsilon}_{n+2-q}, \dots, \hat{\epsilon}_{n+2}\) 生成，接着 \(\hat{Y}_{n+3}\) 由 \(Y_{n+3-p}, \dots, Y_n, \hat{Y}_{n+1}, \hat{Y}_{n+2}, \hat{\epsilon}_{n+3-q}, \dots, \hat{\epsilon}_{n+3}\) 生成，以此类推。

通过这种方式生成大量（记作 \(B\) 组）随机预测集。它们之所以不同，是因为步骤二中生成的未来误差集是相互独立的。对于每个 \(t = 1, \dots, m\)，这 \(B\) 个随机 \(\hat{Y}_{n+h}\) 值的 \(\alpha/2\) 上分位数和下分位数即为 \(Y_{n+h}\) 的预测限。

要获得预测值（而非预测限），在步骤二中取 \(\hat{\epsilon}_k = 0\)，其中 \(k = n+1, \dots, n+m\)。这些预测值是非随机的，在给定数据条件下的条件预测，因此只需计算一次。

如果 \(Y_t = \Delta W_t\) 是对某个非平稳序列 \(\{W_1, \dots, W_n\}\) 而言的，那么 \(\{W_{n+1}, \dots\}\) 的随机预测可作为 \(\{W_n, \hat{Y}_{n+1}, \dots\}\) 的部分和得到。例如：

\[ \begin{aligned} \hat{W}_{n+1} &= W_n + \hat{Y}_{n+1},\\ \hat{W}_{n+2} &= \hat{W}_{n+1} + \hat{Y}_{n+2} = W_n + \hat{Y}_{n+1} + \hat{Y}_{n+2},\\ \hat{W}_{n+3} &= \hat{W}_{n+2} + \hat{Y}_{n+3} = W_n + \hat{Y}_{n+1} + \hat{Y}_{n+2} + \hat{Y}_{n+3}, \end{aligned} \]

以此类推。然后，随机生成的 \(\hat{W}_{n+h}\) 的上、下分位数可作为 \(W_{n+h}\) 的预测限。

12.13 偏自相关系数

偏自相关函数（PACF）可用于识别AR过程的阶数。对于平稳过程\(Y_t\)，第\(k\)个偏自相关系数记为\(\phi_{k,k}\)，是\(Y_t\)与\(Y_{t+k}\)在给定\(Y_{t+1},\ldots,Y_{t+k-1}\)的条件下的相关系数。当\(k=1\)时，集合\(Y_{t+1},\ldots,Y_{t+k-1}\)为空，因此偏自相关系数等于自相关系数，即\(\phi_{1,1} = \rho(1)\)。记\(\hat{\phi}_{k,k}\)为\(\phi_{k,k}\)的估计量。\(\hat{\phi}_{k,k}\)可通过拟合回归模型得到：

\[ Y_t = \phi_{0,k} + \phi_{1,k} Y_{t-1} + \cdots + \phi_{k,k} Y_{t-k} + \epsilon_{k,t}. \]

如果\(Y_t\)是AR(\(p\))过程，则当\(k > p\)时，\(\phi_{k,k} = 0\)。因此，时间序列可以用AR(\(p\))模型拟合的标志是，样本偏自相关函数（PACF）在滞后\(p\)之前非零，而在更大的滞后处接近于零。

例12.17. 宝马对数收益率的PACF 图12.23是使用R函数pacf()计算的宝马对数收益率的样本PACF。较大的\(\hat{\phi}_{1,1}\)和较小的\(\hat{\phi}_{k,k}\)（\(k = 2, \dots, 9\)）表明该时间序列可以用AR(1)模型拟合，与例12.4的结果一致。注意，对于某些\(k > 9\)，特别是\(k = 19\)，\(\hat{\phi}_{k,k}\)位于检验界值之外。这很可能是由于随机变异所致。□

当计算资源昂贵时，标准做法是使用样本ACF和PACF识别一个暂时的ARMA模型，拟合该模型，然后检查残差的ACF和PACF。如果残差ACF和PACF显示出某些拟合不足，则可以扩展模型。随着计算变得更便宜和更快，以及基于信息的模型选择准则的普及，这种做法已经改变。现在许多数据分析师倾向于从一组相对较大的模型开始，并使用AIC和BIC等选择准则进行比较。这可以通过R中的auto.arima()或其他软件包中的类似函数自动完成。
