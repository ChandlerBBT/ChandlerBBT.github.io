第21章 非参数回归与样条

21.1 引言

如第9章所述，回归分析估计给定预测变量下响应变量的条件期望。条件期望称为回归函数，它是基于预测变量对响应变量进行预测的最佳方式，因为它最小化期望平方预测误差。

回归有三种类型：线性回归、非线性参数回归和非参数回归。线性回归假设回归函数是参数的线性函数，并估计截距和斜率（回归系数）。非线性参数回归（在第11.2节中讨论过）不假设线性关系，但假设回归函数具有已知的参数形式，例如Nelson-Siegel模型。在本章中，我们研究非参数回归，其中回归函数的形式也是非线性的，但与非线性参数回归不同，它不是由模型指定，而是由数据确定。当我们知道或怀疑回归函数是弯曲的，但没有弯曲的模型时，使用非参数回归。

非参数回归有许多技术，但局部多项式回归和样条是最广泛使用的，这里只讨论这两种。局部多项式回归和样条通常效果良好，并且由于它们通常给出类似的估计，因此很难推荐其中一种。局部多项式估计可能更容易理解。样条在数学的许多领域都有应用，例如插值，因此熟悉它们是有价值的。此外，样条在复杂模型中作为组成部分也很有用。本章末尾的R实验给出了一个示例。

短期利率演化模型在金融中非常重要，例如，它们用于利率衍生品的定价。图21.1展示了R的Ecdat包中Capm数据集1的月度无风险收益率。该数据集在前几章中已多次用于不同目的。此处我们用它来阐述非参数回归。面板(a)和(b)分别是收益率和收益率变化的时间序列图。

短期利率变化的常见模型为： \[ \Delta r_t = \mu(r_{t-1}) + \sigma(r_{t-1})\epsilon_t, \] 其中 \(r_t\) 是时刻 \(t\) 的利率，\(\Delta r_t = r_t - r_{t-1}\)，\(\mu(\cdot)\) 是漂移函数，\(\sigma(\cdot)\) 是波动率函数（也称为扩散函数），\(\epsilon_t\) 是 \(N(0,1)\) 噪声。针对 \(\mu(\cdot)\) 和 \(\sigma(\cdot)\)，已提出了许多不同的参数模型，如 Merton (1973)、Vasicek (1977)、Cox, Ingersoll 和 Ross (1985)、Yau 和 Kohn (2003) 以及 Chan 等 (1992)。最简单的模型归功于 Merton (1973)，假设 \(\mu(\cdot)\) 和 \(\sigma(\cdot)\) 为常数。Chan 等 (1992) 假设 \(\mu(r) = \beta(r - \alpha)\) 且 \(\sigma(r) = \theta r^\gamma\)，其中 \(\alpha > 0\)，\(\beta < 0\)，\(\theta > 0\)，\(\gamma\) 是未知参数——该过程均值回归到 \(\alpha\)。Chan 等人的模型在第11.12.1节中作为非线性回归的示例使用。

这里采用的Yau和Kohn (2003)的方法是对\(\mu(\cdot)\)和\(\sigma(\cdot)\)同时进行非参数建模。这样做可以检验哪些参数化模型（如果有的话）拟合数据，并在没有参数模型拟合良好时提供非参数替代方案。

图21.1c和图21.1d中的实红曲线是通过非参数回归方法——局部线性回归（局部多项式回归的一种特例）得到的\(\mu(\cdot)\)和\(\sigma^2(\cdot)\)的估计。由(21.1)，\(E(\Delta r_t) = \mu(r_{t-1})\)，\(\text{Var}(\Delta r_t) = \sigma^2(r_{t-1})\)，因此通过将\(\Delta r_t\)对\(r_{t-1}\)回归得到\(\hat{\mu}(\cdot)\)，通过将\(\{\Delta r_t - \hat{\mu}(r_{t-1})\}^2\)对\(r_{t-1}\)回归得到\(\hat{\sigma}^2(\cdot)\)。后者是估计条件方差的一个例子；参见第14.2节。

21.2 局部多项式回归

局部多项式回归基于这样一个原理：一个光滑函数可以用低阶多项式进行局部近似。假设我们有一个样本 \((X_i,Y_i), i=1,\ldots,n\)，且对于一个光滑函数 \(\mu\)，有 \(E(Y|X=x)=\mu(x)\)。函数 \(\mu\) 将在网格点 \(x_1,\ldots,x_M\) 上进行估计。这些网格点可以是、也可以不是观察 \(Y\) 时的实际值 \(X_1,\ldots,X_n\)。估计是在网格 \(x_1,\ldots,x_M\) 上逐点进行的。为了估计 \(\mu\) 在点 \(x_\ell\) 处的值，我们仅使用 \(X_i\) 接近 \(x_\ell\) 的 \((X_i,Y_i)\) 拟合一个 \(p\) 次多项式。这是通过由核函数 \(K\) 确定的权重来完成的。\(K\) 是一个关于0对称的概率密度函数，且随着 \(|x|\) 的增加而递减，例如均值为0的正态密度。我们在第4.2节中已经看到核函数用于密度估计。

在 \(x_\ell\) 处的回归函数通过核加权最小二乘法进行估计，优化以下式子： \[ \sum_{i=1}^n \left[ Y_i - \left\{ \beta_0 + \beta_1(X_i-x_\ell) + \cdots + \beta_p(X_i-x_\ell)^p \right\} \right]^2 K\left( \frac{X_i-x_\ell}{h} \right) \tag{21.2} \] 然后 \(\hat\mu(x_\ell) = \hat\beta_0\)，因为回归模型 \(\beta_0 + \beta_1(x-x_\ell) + \cdots + \beta_p(x-x_\ell)^p\) 在 \(x=x_\ell\) 处等于 \(\beta_0\)。权重 \(K((X_i-x_\ell)/h)\) 随着 \(|X_i-x_\ell|\) 的增加而减小，因此只使用靠近 \(x_\ell\) 的数据。参数 \(h\) 称为带宽，它决定了用于估计的数据量；\(h\) 越大，使用的数据越多。

21.2 局部多项式回归

局部线性估计（其中 \(p=1\)）在图21.2中进行了说明。图中蓝色虚线曲线表示在两个点 \(x_{25} = 0.32\) 和 \(x_{75} = 0.72\) 处的核函数。在每个核函数上方，红色线段表示局部线性拟合，而大的“+”号位于 \(\{x, \hat{\mu}(x)\}\) 处。黑色曲线 \(\hat{\mu}\) 是通过在一个由75个 \(x_\ell\) 值组成的网格上计算局部拟合，并在该网格上绘制所有 \(\{x_\ell, \hat{\mu}(x_\ell)\}\) 而得到的。例如，图21.2中的曲线使用了R语言KernSmooth包中的locpoly()函数，并使用了401个等间距的 \(x\) 值网格（默认设置）。通常，网格就是观测到的 \(X\) 值，即 \(X_1, \ldots, X_n\)。

带宽 \(h\) 被称为“平滑参数”，因为它决定了 \(\hat{\mu}\) 的平滑程度。\(h\) 的值越大，曲线越平滑。\(h\) 的选择至关重要。如果 \(h\) 太大，则多项式逼近可能很差，导致 \(\mu(x)\) 的估计出现严重偏差。相反，如果 \(h\) 太小，则使用的数据过少，导致 \(\mu\) 的估计方差过大。带宽的良好选择应最小化估计量的均方误差，即方差加上平方偏差。平方偏差和方差都是未知的，必须进行估计，或者至少需要估计它们的和。自动带宽选择方法直接或间接估计并尝试最小化均方误差，这一直是研究的热点，并且有许多基于数据的带宽选择器可用。图21.2中的曲线使用了Ruppert、Sheather和Wand（1995）提出的流行直接插入（dpi）带宽选择器。dpi选择器估计 \(\hat{\mu}\) 的均方积分误差（MISE），即

\[\mathbb{E}\left[\int_{\min(X_i)}^{\max(X_i)} \left\{\mu(x) - \hat{\mu}(x)\right\}^2 dx\right], \tag{21.3}\]

并找到使估计MISE最小的带宽。

非参数回归估计量也被称为平滑器，因为它们平滑了数据中的噪声。使用过小的带宽会导致过拟合，即欠平滑。相反，带宽过大会导致欠拟合，即过平滑——关于核密度估计中欠平滑和过平滑的进一步讨论见第4.2节。

21.2.1 Lowess 和 Loess

Loess 和其早期版本 lowess 是局部多项式平滑器，带宽随空间变化，由参数 span 控制。span 是在每个点用于估计的数据比例。带宽，记为 \(h(x, \text{span})\)，在点 \(x\) 处的调整方法是，当 \(\text{span} \le 1\) 时，\(K\{(X_i - x)/h(x, \text{span})\}\) 在 \(\text{span} \times 100\%\) 的 \(X_i\) 上非零。

如果 \(\text{span} = 1\)，则在每个点使用所有数据进行估计，但距离 \(X_i\) 最远的数据权重较小。由于这些权重较小，对于小数据集，\(\text{span}=1\) 的 lowess（或 loess）平滑可能不够平滑。为解决此问题，span 可定义为大于 1 的值，通过 \(h(x, \text{span}) = \text{span} \times h(x, 1)\)。

21.3 线性平滑器

当 span 增加到超过 1 时，权重 \(K\{(X_i - x)/h(x, \text{span})\}\) 变得越来越均匀。当 \(\text{span} \to \infty\) 时，权重收敛到一个常数 \(K(0)\)，lowess（或 loess）拟合收敛到多项式回归拟合。

21.3 线性平滑器

局部多项式回归以及接下来要介绍的惩罚样条回归都是线性平滑器的例子。线性平滑器有一个 \(n \times n\) 的平滑矩阵 \(H\)，它不依赖于 \(Y\)，使得

\[ \hat{Y} = H Y, \tag{21.4} \]

其中 \(Y = (Y_1, \dots, Y_n)^T\) 是响应向量，\(\hat{Y} = (\hat{Y}_1, \dots, \hat{Y}_n)^T\) 是拟合值向量。方程 (21.4) 可以写成

\[ \hat{Y}_i = \sum_{j=1}^n H_{ij} Y_j, \quad i = 1, \dots, n. \tag{21.5} \]

平滑矩阵依赖于一个平滑参数，对于局部多项式回归，该参数就是带宽。我们用 \(\lambda\) 表示平滑参数，并将平滑矩阵记为 \(H(\lambda)\)。平滑矩阵类似于线性回归中的投影矩阵，本身也常被称为帽子矩阵。

21.3.1 平滑矩阵与有效自由度

在参数化模型中，参数的数量量化了模型拟合数据的能力。在非参数估计中，拟合（以及过拟合）的潜力可以通过拟合的有效参数个数或有效自由度来量化。从概念上讲，有效参数个数类似于第 20.7.6 节中的贝叶斯 \(p_D\)。

由 (21.5)，帽子对角线 \(H(\lambda)_{ii}\) 给出了 \(Y_i\) 的杠杆或自影响，因为它是计算 \(\hat{Y}_i\) 时赋予 \(Y_i\) 的权重。\(H(\lambda)_{ii}\) 的值大意味着过拟合的可能性高。有效参数个数是杠杆的总和：

\[ p_{\text{eff}} = \sum_{i=1}^n H(\lambda)_{ii} = \operatorname{tr}\{H(\lambda)\}. \tag{21.6} \]

如果 \(p_{\text{eff}}\) 太小（太大），则数据欠拟合（过拟合）。

残差均方和为

\[ \sum_{i=1}^n (Y_i - \hat{Y}_i)^2 = \| Y - \hat{Y} \|^2 = \| \{ I - H(\lambda) \} Y \|^2, \tag{21.7} \]

21.4 多项式样条

其中 \(I\) 是 \(n\times n\) 的单位矩阵。噪声方差估计为

\[\hat{\sigma}(\lambda)^2 = \frac{\|\{\mathbf{I} - \mathbf{H}(\lambda)\} \mathbf{Y}\|^2}{n - p_{\text{eff}}},\]

这与 (9.16) 式直接类似。

21.3.2 AIC、CV 与 GCV

对于线性回归模型，AIC 为

\[\text{AIC} = n\log(\hat{\sigma}^2) + 2(1 + p),\]

其中 \(1+p\) 是参数个数（截距加上 \(p\) 个斜率）。对于线性平滑器，AIC 用 \(p_{\text{eff}}\) 代替 \(p+1\)，因此

\[\text{AIC}(\lambda) = n\log\{\hat{\sigma}^2(\lambda)\} + 2 p_{\text{eff}}.\]

然后我们可以通过最小化 AIC 来选择 \(\lambda\)。

交叉验证（CV）统计量为

\[\text{CV}(\lambda) = \sum_{i=1}^n \{Y_i - \hat{Y}_{-i}(\lambda)\}^2,\]

其中 \(\hat{Y}_{-i}(\lambda)\) 是删除第 \(i\) 个观测值后计算的第 \(i\) 个拟合值，以防止过拟合。显然，应选择 \(\text{CV}(\lambda)\) 值较小的 \(\lambda\)。

广义交叉验证（GCV）统计量为

\[\text{GCV}(\lambda) = \frac{\|\mathbf{Y} - \hat{\mathbf{Y}}(\lambda)\|^2}{(n - p_{\text{eff}})^2}. \tag{21.9}\]

GCV\((\lambda)\) 的计算速度比 CV\((\lambda)\) 快，且 GCV\((\lambda)\) 是 CV\((\lambda)/n^2\) 的良好近似，因此最小化 GCV 是选择 \(\lambda\) 的另一种方式。

AIC 和 GCV 都可以快速计算，并且通常给出基本相同的平滑量。事实上，理论上已证明两个准则应给出相似的估计。因此，使用哪个并不重要，但在非参数回归中 GCV 比 AIC 更常用。

21.4 多项式样条

多项式样条在非参数回归以及数学许多其他领域中的应用，基于与局部多项式回归相同的原理——一个光滑函数可以局部地用低次多项式精确近似。\(p\) 次多项式样条通过拼接 \(p\) 次多项式构造而成，使它们在称为“结点”的指定位置连接。多项式被拼接起来，使得样条具有 \(p-1\) 阶连续导数。样条的 \(p\) 阶导数在结点之间为常数，并可在结点处跳跃。

21.4.1 具有一个结点的线性样条

我们从简单的开始——具有一个结点的线性样条。图21.5(a)展示了这样一个样条。该样条定义为

\[f(x)=\begin{cases} 0.5+0.2x, & x<2,\\ -0.5+0.7x, & x\ge2. \end{cases}\]

21.4.2 多节点线性样条

当定义具有多个节点的线性样条时，正部函数非常方便，因为它们自动将分段的线性函数连接起来，使得样条连续。例如，假设我们希望一个线性样条有 \(K\) 个节点 \(t_1 < \cdots < t_K\)，并满足当 \(x < t_1\) 时样条 \(s(x) = \beta_0 + \beta_1 x\)，且样条的一阶导数在每个节点 \(t_k\) 处跳跃 \(b_k\)（\(k = 1,\ldots,K\)）。那么，该样条可以由每个节点对应的线性正部函数构造：

\[ s(x) = \beta_0 + \beta_1 x + b_1 (x-t_1)_+ + b_2 (x-t_2)_+ + \cdots + b_K (x-t_K)_+ \]

由于正部函数是连续的，样条作为连续函数的和，因此其本身也是连续的。

21.4.3 二次样条

线性样条是连续的，但在节点处有“折点”，其一阶导数发生跳跃。如果我们希望得到一个没有这些折点的函数，就不能使用线性样条。二次样条是通过拼接二次多项式得到的函数。更精确地说，若 \(s(x)\) 在 \(t_1\) 左侧等于一个二次多项式，在 \(t_1\) 和 \(t_2\) 之间等于另一个二次多项式，依此类推，则 \(s(x)\) 是节点为 \(t_1 < \cdots < t_K\) 的二次样条。这些二次多项式被拼接在一起，使得样条连续且一阶导数也连续（以确保没有折点）。图 21.6a 展示了节点为 1 的二次样条。注意，函数在节点处没有折点，但在此处从凸变为凹。

给定次数 \(p\) 和节点 \(\kappa_1, \ldots, \kappa_K\)，多项式 \(1, x, \ldots, x^p\) 以及加函数 \((x - \kappa_1)_+^p, \ldots, (x - \kappa_K)_+^p\) 构成样条基。这意味着，任何以 \(\kappa_1, \ldots, \kappa_K\) 为节点的 \(p\) 次样条都是这些基函数的线性组合。多项式加函数基易于理解，但已知在节点数较多时数值不稳定。因此，数值计算中常使用其他基。B-样条基尤其流行。这里假定读者不会从头编写样条估计程序，而是使用样条软件。因此，本处不讨论 B-样条及其他基，但参见第21.6节以获得进一步阅读。

21.5 惩罚样条

由于具有 \(K\) 个节点的 \(p\) 次样条有 \(1 + p + K\) 个参数，普通最小二乘拟合通常会导致过拟合，除非 \(p\) 和 \(K\) 都保持较小，例如 \(1 + p + K \leq 6\)。（数字6并无特殊之处，仅用作经验法则；5到10之间的任何数字同样适用。）一个例子是例11.3中用作远期利率曲线的二次样条（只有一个节点，因此 \(1 + p + K = 4\)）。然而，同时取较小 \(p\) 和 \(K\) 的样条本质上是参数模型。为了获得非参数模型的灵活性，即较大的潜在 \(p_{\text{eff}}\) 取值范围，我们需要让 \(K\) 较大，并寻找其他避免过拟合的方法。惩罚最小二乘估计可以达到这个目的。

设 \(\mu(\mathbf{x}; \boldsymbol{\beta}) = \mathbf{B}(\mathbf{x})^T \boldsymbol{\beta}\) 是一个样条，其中 \(\boldsymbol{\beta}\) 是系数向量，\(\mathbf{B}(\mathbf{x}) = (B_1(\mathbf{x}), \ldots, B_{1+p+K}(\mathbf{x}))^T\) 是样条基。例如，若采用模型 (21.12)，则 \(\mathbf{B}(\mathbf{x}) = (1, x, \ldots, x^p, (x - \kappa_1)_+^p, \ldots, (x - \kappa_K)_+^p)\)。惩罚最小二乘估计通过最小化关于 \(\boldsymbol{\beta}\) 的惩罚平方和

\[ \sum_{i=1}^n \{Y_i - \mu(X_i; \boldsymbol{\beta})\}^2 + \lambda \boldsymbol{\beta}^T \mathbf{D} \boldsymbol{\beta} \tag{21.13} \]

得到，其中 \(\mathbf{D}\) 是半正定矩阵，\(\lambda > 0\) 是惩罚参数。一个常见的 \(\mathbf{D}\) 取法是其第 \((i, j)\) 个元素等于

\[ \int_a^b B_i^{(2)}(x) B_j^{(2)}(x) \, dx \tag{21.14} \]

21.5 惩罚样条

对于某些 \(a \[ \lambda \beta^T D \beta = \lambda \int_a^b \left\{ \mu^{(2)}(x; \beta) \right\}^2 dx, \tag{21.15} \]

由于 \(\mu^{(2)}(x)\) 是 \(\mu\) 在 \(x\) 处的曲率，这种 \(D\) 的选择会惩罚波动剧烈的函数，并且如果 \(\lambda\) 选择得当，可以防止过拟合。如果 \(\lambda = 0\)，则没有惩罚，有效参数个数为 \(1 + p + K\)。使用这个 \(D\)，当 \(\lambda \to \infty\) 时，任何曲率都会受到无限惩罚，因此估计量收敛到线性多项式拟合，有效参数个数收敛到 2。介于 2 和 \(1 + p + K\) 之间的任何 \(p_{\text{eff}}\) 值都可以通过介于 0 和 \(\infty\) 之间的某个 \(\lambda\) 值实现。

令 \(X\) 为 \(n \times (1 + p + K)\) 矩阵，其第 \((i, j)\) 元素为 \(B_j(X_i)\)，并令 \(Y = (Y_1, \dots, Y_n)^T\)。惩罚最小二乘估计为

\[ \hat{\beta}(\lambda) = \left( X^T X + \lambda D \right)^{-1} X^T Y, \tag{21.16} \]

这是通过将 (21.13) 的梯度设为零并求解得到的。拟合值为

\[ \hat{Y}(\lambda) = X \hat{\beta}(\lambda) = \left\{ X (X^T X + \lambda D)^{-1} X^T \right\} Y = H(\lambda) Y, \tag{21.17} \]

其中 \(H(\lambda) = \left\{ X (X^T X + \lambda D)^{-1} X^T \right\}\) 是光滑矩阵。

21.5.1 三次光滑样条

一种非常广泛使用的非参数回归估计量是三次光滑样条。该估计量在 \(\{X_1, \dots, X_n\}\) 的每个唯一值处使用一个节点，并使用 (21.15) 中的二阶导数罚项。使用这么多节点并非必要，三次光滑样条的一种变体也使用罚项 (21.15)，但使用更少的节点。节点可以是等间距的，也可以是 \(\{X_1, \dots, X_n\}\) 的选定分位数。

R 中的函数 smooth.spline() 在参数 all.knots = TRUE 或 \(n < 50\) 时拟合三次光滑样条。如果 all.knots = FALSE 且 \(n > 49\)，则它使用少于完整节点集的节点。

21.5.2 选择惩罚量

惩罚参数 \(\lambda\) 决定光滑量，可以通过 AIC 或 GCV 选择。另一种流行的选择 \(\lambda\) 的方法是 REML（受限最大似然）。REML 基于所谓的混合模型，其中一些样条系数是随机变量。混合模型和 REML 的描述超出了本书的范围，感兴趣的读者可参考 21.6 节中的参考文献。
