第16章 投资组合选择

16.1 期望收益率与风险的权衡

我们应如何配置财富？投资组合理论基于两个原则回答了这一问题：

  • 我们希望最大化期望收益率；
  • 我们希望最小化风险，本章中将风险定义为收益率的标准差，尽管我们最终可能关注的是发生巨大损失的概率。

这两个目标在一定程度上相互冲突，因为风险较高的资产通常具有较高的期望收益率——投资者承担风险需要相应的回报。风险资产的期望收益率与无风险利率之间的差额称为风险溢价。如果没有风险溢价，很少有投资者愿意投资于风险资产。

尽管如此，在期望收益率与风险之间仍存在最优的折中方案。本章将展示如何在给定风险上限的条件下最大化期望收益率，或在给定期望收益率下限的条件下最小化风险。我们讨论的一个关键概念是通过分散投资组合来降低风险。

16.2 一个风险资产与一个无风险资产

我们从只有一个风险资产的简单示例开始，该风险资产可以是一个投资组合，例如一只共同基金。假设其期望收益率为0.15，收益率标准差为0.25。同时假设存在一个无风险资产，例如90天期国库券，无风险利率为6%，因此无风险资产的收益率为6%（即0.06）。该无风险资产收益率的标准差为

根据定义，“无风险”资产收益率的方差为零。这里的比率和收益率都按年计算，尽管唯一的要求是它们采用同一时间单位。

我们面临的问题是构建一个将持有一个时间周期的投资组合，该周期称为持有期，可以是一天、一个月、一个季度、一年、十年等。在持有期结束时，我们可能希望调整投资组合，因此目前我们只考察一个时间周期内的收益率。假设我们将财富的比例 \(w\) 投资于风险资产，剩余比例 \(1-w\) 投资于无风险资产。那么期望收益率为

\[ E(R)= w(0.15)+(1 -w)(0.06) = 0.06+0.09w, \tag{16.1} \]

收益率的方差为

\[ \sigma_R^2 = w^2 (0.25)^2 +(1-w)^2 (0)^2 = w^2(0.25)^2, \]

收益率的标准差为

\[ \sigma_R = 0.25|w|. \tag{16.2} \]

如后文所述，如果风险资产被卖空，则 \(w\) 为负，因此在(16.2)中我们使用 \(|w|\) 而非 \(w\)。

为了决定将财富的多少比例 \(w\) 投资于风险资产，投资者要么选择期望收益率 \(E(R)\)，要么选择愿意承受的风险程度 \(\sigma_R\)。一旦选择了 \(E(R)\) 或 \(\sigma_R\)，就可以确定 \(w\)。

虽然 \(\sigma\) 是风险的一种度量，但更直接的风险度量是实际的货币损失。在下一个例子中，我们将选择 \(w\) 以控制损失的最大规模。

例16.1. 寻找达到目标风险价值的 \(w\)

假设一家公司计划投资1,000,000美元，其资本储备能够覆盖150,000美元的损失，但无法覆盖更多。因此，公司希望确保一旦发生损失，损失不超过15%，即收益率 \(R\) 大于 \(-0.15\)。假设 \(R\) 服从正态分布。那么，要使 \(R\) 以概率1大于 \(-0.15\)，唯一的方法是将全部资金投资于无风险资产。公司或许可以更现实一些，只要求 \(P(R < -0.15)\) 很小，例如0.01。因此，公司应当找到 \(w\) 的值，使得

\[ P(R < -0.15) = \Phi\left( \frac{-0.15 - (0.06+0.09w)}{0.25w} \right) = 0.01. \]

解为

\[ w = \frac{-0.21}{0.25\Phi^{-1}(0.01) + 0.9} = 0.4264. \]

其中 \(\Phi^{-1}(0.01)\) 的值由 qnorm(0.01) 计算得出，为 \(-2.33\)。

16.2 一种风险资产与一种无风险资产

在第19章中，150,000美元称为风险价值（VaR），而\(1-0.01 = 0.99\)称为置信系数。本示例所做的是找到在0.99置信水平下VaR为150,000美元的投资组合。

我们在第5章中看到，股票收益率的分布通常具有比正态分布更厚的尾部。在第19章中，将在更现实的假设下估计VaR，例如收益率服从t分布。\(\square\)

更一般地，如果风险资产和无风险资产的期望收益率分别为\(\mu_1\)和\(\mu_f\)，风险资产的标准差为\(\sigma_1\)，则投资组合的期望收益率为\(w\mu_1 + (1 - w)\mu_f\)，而投资组合收益率的标准差为\(|w|\sigma_1\)。

这个模型虽然简单，但并非像乍看起来那样无用。正如后面所讨论的，寻找最优投资组合可以通过两个步骤实现：

  1. 找到风险资产的“最优”投资组合，称为“切点组合”；
  2. 找到无风险资产与切点组合的适当混合。

因此我们现在知道如何进行第二步。我们仍需要学习的是如何找到切点组合。

16.2.1 估计\(E(R)\)和\(\sigma_R\)

无风险利率\(\mu_f\)的值是已知的，因为国库券利率在提供金融信息的来源中公布。

对于\(E(R)\)和\(\sigma_R\)的值，我们应该使用什么？如果假设资产收益率是平稳的，我们可以取过去收益率的时间序列，并使用样本均值和标准差。平稳性假设是否现实总是存在争议。如果我们认为\(E(R)\)和\(\sigma_R\)将与过去不同，我们可以根据我们的观点主观地向上或向下调整这些估计，但我们必须承受如果我们的观点被证明是不正确的后果。此外，样本均值和标准差并不特别准确，可以用来自因子模型（如CAPM或Fama-French模型）的估计替代；参见第17、18和20章。

另一个问题是使用多长的时间序列，即应该回溯多长时间的数据。一个长序列，例如10年或20年，将给出变化小得多的估计。然而，如果序列不是平稳的而是具有缓慢漂移的参数，那么较短的序列（可能1年或2年）将更能代表未来。几乎每个收益率时间序列在足够短的时间段内都是近似平稳的。

即使时间序列是平稳的，它也可能表现出波动率聚集。在这种情况下，人们可以使用GARCH模型来估计持有期内收益率的条件标准差。参见第14章。

16.3 两种风险资产

16.3.1 风险与期望回报

混合风险资产的数学原理在只有两种风险资产的情况下最为简单易懂。我们由此开始。

假设两种风险资产的收益率分别为 \(R_1\) 和 \(R_2\)，我们将其分别按比例 \(w\) 和 \(1-w\) 混合。投资组合的收益率为 \(R_P = w R_1 + (1-w) R_2\)。投资组合的期望收益率为 \(E(R_P) = w\mu_1 + (1-w)\mu_2\)。令 \(\rho_{12}\) 为两种风险资产收益率之间的相关系数。投资组合收益率的方差为

\[ \sigma^2_{R_P} = w^2\sigma_1^2 + (1-w)^2\sigma_2^2 + 2w(1-w)\rho_{12}\sigma_1\sigma_2 \tag{16.3} \]

注意 \(\rho_{12}\sigma_1\sigma_2 = \sigma_{R_1,R_2}\)。

例16.2. 两种风险资产投资组合的期望收益率和方差

假设 \(\mu_1 = 0.14\)，\(\mu_2 = 0.08\)，\(\sigma_1 = 0.2\)，\(\sigma_2 = 0.15\)，且 \(\rho_{12} = 0\)。则

\[ E(R_P) = 0.08 + 0.06w \]

由于此例中 \(\rho_{12} = 0\)，有

\[ \sigma^2_{R_P} = (0.2)^2 w^2 + (0.15)^2 (1-w)^2 \]

通过微分法，易得最小风险的投资组合权重为 \(w = 0.045 / 0.125 = 0.36\)。对于该投资组合，\(E(R_P) = 0.08 + (0.06)(0.36) = 0.1016\)，\(\sigma_{R_P} = \sqrt{(0.2)^2(0.36)^2 + (0.15)^2(0.64)^2} = 0.12\)。

图16.1中略显抛物线的曲线1是当 \(0 \le w \le 1\) 时 \((\sigma_R, E(R))\) 的轨迹。该轨迹最左侧的点达到风险最小值，称为最小方差投资组合。轨迹上期望收益率不低于最小方差投资组合的点构成有效前沿。有效前沿上的投资组合称为有效投资组合，更确切地说，是均值-方差有效投资组合2。标记为 \(R_1\) 和 \(R_2\) 的点分别对应 \(w=1\) 和 \(w=0\)。图中其他特征将在第16.4节中解释。□

第16章 投资组合选择

风险资产与无风险资产的组合是一条连接点F与有效前沿上某一点的直线，例如紫色虚线。注意，连接F与标记为T的点的蓝色实线位于连接F与典型组合的紫色虚线之上。这意味着对于任意\(\sigma_{R_P}\)值，蓝色实线给出的期望收益率都比紫色虚线更高。每条线的斜率称为夏普比率，得名于威廉·夏普，我们将在第17章再次见到他。

若\(E(R_P)\)和\(\sigma_{R_P}\)分别为一个投资组合的期望收益率和标准差，\(\mu_f\)为无风险利率，则

\[ \frac{E(R_P) - \mu_f}{\sigma_{R_P}} \tag{16.4} \]

是该投资组合的夏普比率。夏普比率可以看作是一种“收益-风险”比率。它是用超额期望收益率3量化的收益与用标准差度量的风险之比。

斜率更大的直线在给定风险水平下给出更高的期望收益率，因此夏普比率越大越好，无论投资者愿意接受何种风险水平。有效前沿上的点T是夏普比率最高的投资组合。它是用于与无风险资产混合的最优组合。该组合称为切点组合，因为其直线与有效前沿相切。

结果16.1 最优或有效的投资组合将切点组合与无风险资产混合。每个有效组合具有两个性质：

  • 对于相同或更低的风险，其期望收益率高于任何其他组合；
  • 对于相同或更高的期望收益率，其风险低于任何其他组合。

因此，我们只能通过接受更差（更低）的期望收益率来改进（降低）有效组合的风险，只能通过接受更差（更高）的风险来改进（提高）有效组合的期望收益率。

注意，所有有效组合都使用相同的两种风险资产配置，即切点组合。只有分配给切点组合的比例和分配给无风险资产的比例不同。

鉴于切点组合的重要性，读者可能会问“如何找到它？”同样，设\(\mu_1\)、\(\mu_2\)和\(\mu_f\)分别为两种风险资产的期望收益率和无风险资产的收益率。设\(\sigma_1\)和\(\sigma_2\)为两种风险资产收益率的标准差，\(\rho_{12}\)为风险资产收益率之间的相关系数。

  1. 此处“超额”指超过无风险利率的部分。 ↩

16.4 将两种风险资产与无风险资产组合

令 \(V_1 = \mu_1 - \mu_f\) 和 \(V_2 = \mu_2 - \mu_f\)，即超额期望收益率。则切点组合对第一种风险资产分配的权重为

\[ w_T = \frac{V_1 \sigma_2^2 - V_2 \rho_{12} \sigma_1 \sigma_2}{V_1 \sigma_2^2 + V_2 \sigma_1^2 - (V_1 + V_2) \rho_{12} \sigma_1 \sigma_2} \qquad (16.5) \]

对第二种风险资产的权重为 \(1 - w_T\)。

令 \(R_T\)、\(E(R_T)\) 和 \(\sigma_T\) 分别为切点组合的收益率、期望收益率和收益率标准差。则 \(E(R_T)\) 和 \(\sigma_T\) 可通过先利用 (16.5) 求出 \(w_T\)，再利用以下公式得到：

\[ E(R_T) = w_T \mu_1 + (1 - w_T) \mu_2 \]

和

\[ \sigma_T = \sqrt{w_T^2 \sigma_1^2 + (1 - w_T)^2 \sigma_2^2 + 2 w_T (1 - w_T) \rho_{12} \sigma_1 \sigma_2}. \]

例 16.3. 两种风险资产的切点组合

如前假设 \(\mu_1 = 0.14\)、\(\mu_2 = 0.08\)、\(\sigma_1 = 0.2\)、\(\sigma_2 = 0.15\)、\(\rho_{12} = 0\)。同时假设 \(\mu_f = 0.06\)。则 \(V_1 = 0.14 - 0.06 = 0.08\)，\(V_2 = 0.08 - 0.06 = 0.02\)。将这些值代入公式 (16.5)，得到 \(w_T = 0.693\)，\(1 - w_T = 0.307\)。因此，

\[ E(R_T) = (0.693)(0.14) + (0.307)(0.08) = 0.122, \]

且

\[ \sigma_T = \sqrt{(0.693)^2 (0.2)^2 + (0.307)^2 (0.15)^2} = 0.146. \]

16.4.2 将切点组合与无风险资产组合

令 \(R_p\) 为将比例 \(\omega\) 的资金投资于切点组合、比例 \(1 - \omega\) 投资于无风险资产的组合的收益率。则 \(R_p = \omega R_T + (1 - \omega) \mu_f = \mu_f + \omega (R_T - R_f)\)，因此

\[ E(R_p) = \mu_f + \omega \{E(R_T) - \mu_f\}, \quad \sigma_{R_p} = \omega \sigma_T. \]

16.4.3 \(\rho_{12}\)的影响

两种风险资产之间的正相关会提高风险。当正相关时，两种资产倾向于同向变动，从而增加投资组合的波动性。相反，负相关是有利的，因为它降低了风险。如果资产负相关，一种资产的负收益往往伴随着另一种资产的正收益，从而投资组合的波动性下降。图16.2展示了当\(\mu_1 = 0.14\)，\(\mu_2 = 0.09\)，\(\sigma_1 = 0.2\)，\(\sigma_2 = 0.15\)，\(\mu_f = 0.03\)时的有效前沿和切点组合。\(\rho_{12}\)的取值从0.5变化到-0.7。注意，切点组合的夏普比率随着\(\rho_{12}\)的减小而增大。这意味着，当\(\rho_{12}\)较小时，相比于\(\rho_{12}\)较大的情形，有效前沿在给定期望收益率下承担的风险更小。

16.5 卖空

卖空通常，有效组合中的某些权重为负。资产上的负权重意味着该资产被卖空。卖空是一种在股票价格下跌时获利的方式。要卖空一只股票，你不需要拥有它就可以卖出。股票必须从经纪人或其他客户那里借入。在未来某个时点，你买入股票并将其归还给借出方，从而平仓空头头寸。

假设一只股票售价为每股 25 美元，你卖空了 100 股。这给你带来了 2500 美元。如果股票跌至每股 17 美元，你可以用 1700 美元买入 100 股，并平仓你的空头头寸。你赚了 800 美元（忽略交易成本），因为股票下跌了 8 个点。如果股票上涨，你就会遭受损失。

现在假设你有 100 美元，并且有两种风险资产。用你的钱，你可以买入价值 150 美元的风险资产 1，同时卖空价值 50 美元的风险资产 2。净成本正好是 100 美元。如果 \(R_1\) 和 \(R_2\) 是风险资产 1 和 2 的收益率，那么你的投资组合收益率为：

\[2R_1 + (-1)R_2\]

你的投资组合权重为 \(w_1 = 3/2\) 和 \(w_2 = -1/2\)。因此，你希望风险资产 1 价格上涨，风险资产 2 价格下跌。这里我们再次忽略了交易成本。

16.6 具有\(N\)种风险资产的风险有效投资组合

如果某人卖空某只股票，则称其在该股票上持有空头头寸，而持有该股票称为多头头寸。

在本节中，我们使用二次规划来求解具有任意数量资产的有效投资组合。二次规划的一个优点是可以施加诸如限制卖空等约束。在分配向量\(\mathbf{w}\)无约束的情况下，可以用拉格朗日乘子法推导出切点组合的解析公式，但这种方法无法推广到有约束的\(\mathbf{w}\)。

假设我们有\(N\)种风险资产，第\(i\)种风险资产的收益率为\(R_i\)，其期望值为\(\mu_i\)。定义 \[ \mathbf{R} = \begin{pmatrix} R_1 \\ \vdots \\ R_N \end{pmatrix} \] 为收益率的随机向量， \[ \mathbb{E}(\mathbf{R}) = \boldsymbol{\mu} = \begin{pmatrix} \mu_1 \\ \vdots \\ \mu_N \end{pmatrix}, \] 且\(\boldsymbol{\Sigma}\)为\(\mathbf{R}\)的协方差矩阵。

令 \[ \mathbf{w} = \begin{pmatrix} w_1 \\ \vdots \\ w_N \end{pmatrix} \] 为投资组合权重向量，满足\(w_1 + \cdots + w_N = \mathbf{1}^\top \mathbf{w} = 1\)，其中 \[ \mathbf{1} = \begin{pmatrix} 1 \\ \vdots \\ 1 \end{pmatrix} \] 为\(N\)个1组成的列向量。该投资组合的期望收益率为 \[ \sum_{i=1}^N w_i \mu_i = \mathbf{w}^\top \boldsymbol{\mu}. \qquad (16.6) \] 假设期望收益率有一个目标值\(\mu_P\)。当\(N=2\)时，目标期望收益率仅由一个投资组合实现，其\(w_1\)满足 \[ \mu_P = w_1 \mu_1 + w_2 \mu_2 = \mu_2 + w_1 (\mu_1 - \mu_2). \] 对于\(N \geq 3\)，

16.6 含N个风险资产的风险有效投资组合

会有无穷多个投资组合达到目标μ_P。其中方差最小的称为“有效”投资组合。我们的目标是找到有效投资组合。

权重为w的投资组合收益的方差为

\[ \mathbf{w}^T \Sigma \mathbf{w}. \tag{16.7} \]

因此，给定目标μ_P，有效投资组合在以下约束下最小化(16.7)：

\[ \mathbf{w}^T \boldsymbol{\mu} = \mu_P \tag{16.8} \]

和

\[ \mathbf{w}^T \mathbf{1} = 1. \tag{16.9} \]

二次规划用于在满足线性约束的条件下最小化二次目标函数。在投资组合优化的应用中，目标函数是投资组合收益的方差。目标函数是N个变量的函数，例如N个资产的权重，用N×1向量x表示。假设要最小化的二次目标函数为

\[ \frac{1}{2} \mathbf{x}^T D \mathbf{x} - \mathbf{d}^T \mathbf{x}, \tag{16.10} \]

其中D是N×N矩阵，d是N×1向量。系数1/2并非必要，此处使用是为了与R的符号保持一致。对x有两种类型的线性约束：不等式约束和等式约束。线性不等式约束为

\[ A_{\text{neq}}^T \mathbf{x} \geq \mathbf{b}_{\text{neq}}, \tag{16.11} \]

其中A_neq是m×N矩阵，b_neq是m×1向量，m是不等式约束的个数。等式约束为

\[ A_{\text{eq}}^T \mathbf{x} = \mathbf{b}_{\text{eq}}, \tag{16.12} \]

其中A_eq是n×N矩阵，b_eq是n×1向量，n是等式约束的个数。二次规划即在满足线性不等式约束(16.11)和线性等式约束(16.12)的条件下最小化二次目标函数(16.10)。

为了应用二次规划求解有效投资组合，我们令x = w，D = 2Σ，d等于N×1的零向量，使得(16.10)变为w^T Σ w，即投资组合的收益方差。有两个等式约束：一个要求权重之和为1，另一个要求投资组合收益等于指定的目标μ_P。因此，我们定义

\[ A_{\text{eq}}^T = \begin{pmatrix} \mathbf{1}^T \\ \boldsymbol{\mu}^T \end{pmatrix} \]

且 \( \mathbf{b}_{eq} = \begin{pmatrix} \mu_P \end{pmatrix} \), 从而(16.12)变为 \[ \begin{pmatrix} \mathbf{1}^T \mathbf{w} \\ \boldsymbol{\mu}^T \mathbf{w} \end{pmatrix} = \begin{pmatrix} 1 \\ \mu_P \end{pmatrix}, \] 这与约束条件(16.8)和(16.9)相同。到目前为止，尚未使用不等式约束。

投资者经常希望施加额外的不等式约束。如果投资者不能或不愿卖空，则可以使用约束 \( \mathbf{w} \ge \mathbf{0} \). 这里\(\mathbf{0}\)是一个由\(N\)个零组成的向量。在这种情况下，\(\mathbf{A}_{neq}\)是\(N \times N\)单位矩阵，\(\mathbf{b}_{neq} = \mathbf{0}\).

为了避免投资组合过度集中于一只或几只股票，投资者可能希望对投资组合进行约束，使得任一\(w_i\)不超过一个界限\(\lambda\)，例如\(\lambda = 1/4\)意味着投资组合中任何单只股票的比例不能超过1/4。在这种情况下，\(\mathbf{w} \le \lambda \mathbf{1}\)，或等价地\(-\mathbf{w} \ge -\lambda \mathbf{1}\)，因此\(\mathbf{A}_{neq}\)是\(N \times N\)单位矩阵的负数，\(\mathbf{b}_{neq} = -\lambda \mathbf{1}\)。可以将这些约束与禁止卖空的约束结合起来。

为了找到有效前沿，我们使用一组\(\mu_P\)的网格值，并找到相应的有效投资组合。对于每个投资组合，可以计算\(\sigma_P^2\)，它是目标函数的最小化值。然后通过寻找具有最小\(\sigma_P^2\)值的投资组合来找到最小方差投资组合。有效前沿是具有高于最小方差投资组合期望收益的有效投资组合的集合。还可以计算有效前沿上每个投资组合的夏普比率，切点组合是最大化夏普比率的那个投资组合。

例16.10 全球资产配置问题：收缩估计与禁止卖空

在例16.9中，我们看到禁止卖空可以提高估计的切点组合的夏普比率，但改进幅度有限。进一步的改进需要对收益率的均值向量或协方差矩阵进行更准确的估计。

本例研究将10个估计均值相互收缩可能带来的改进。具体而言，若 \(Y_i\) 是第 \(i\) 个国家的样本均值，\(Y = (\sum_{i=1}^{10} Y_i)/10\) 是总均值（均值的均值），\(\alpha\) 是介于0和1之间的调节参数，则第 \(i\) 个国家的估计平均收益率为

\[\hat{\mu}_i = \alpha Y_i + (1 - \alpha) Y. \tag{16.13}\]

收缩的目的在于降低估计量的方差，虽然方差减少会带来一些偏差。由于它是均值的……

10个均值中，Y的变异性远小于Y1,...,Y10中的任何一个。因此，\(\mathrm{Var}(\hat{\mu}_i)\) 随着 \(\alpha\) 向0减小而减小。但是， \[E(\hat{\mu}_i) = \alpha \mu_i + \frac{1-\alpha}{10} \sum_{i=1}^{10} \mu_i \tag{16.14}\] 因此，对于任意 \(\alpha \neq 1\)，\(\hat{\mu}_i\) 是有偏的，除非在 \(\mu_1 = \cdots = \mu_{10}\) 这种非常可能的情况下。参数 \(\alpha\) 控制着偏差-方差的权衡。在这个例子中，我们使用 \(\alpha = 1/2\) 进行说明，并且不允许卖空。

图16.6比较了使用收缩和不使用收缩的表现。图(a)包含我们在图16.5(b)中看到的箱线图，其中 \(\alpha = 1\)。图(b)是使用 \(\alpha = 1/2\) 估计切点组合时的箱线图。与图(a)相比，图(b)中的实际夏普比率更接近表示真实切点组合夏普比率的虚线；使用收缩和不使用收缩的实际夏普比率均值分别为0.317和0.318。这些值应与真实（但未知）切点组合的夏普比率0.34进行比较。此外，图(b)中的估计夏普比率更小且更接近真实夏普比率，因此乐观偏差较小——收缩在两个方面有所帮助。使用收缩和不使用收缩的估计夏普比率均值分别为0.390和0.404。

16.8 效用

还有其他方法可以改进均值向量的估计，同时也可以改进协方差矩阵的估计，例如使用第18章的因子模型或第20章的贝叶斯估计。此外，我们不必只关注切点组合，也可以估计最小方差组合。无论估计的焦点是什么，都可以使用自助法来比较改进最优组合估计的各种策略。

经济学家通常不通过收益率的均值和方差来建模经济决策，而是使用效用函数。一笔金额 \(X\) 的货币的效用记为 \(U(X)\)，其中效用函数 \(U\) 通常具有以下性质：

  1. \(U(0) = 0\)；
  2. \(U\) 严格递增；
  3. 一阶导数 \(U'(X)\) 严格递减。

假设1并非必需，但它是合理的，表示0美元的效用为0。假设2仅仅表示钱越多越好。假设3意味着我们拥有的钱越多，对额外一美元的估值就越低，这被称为风险厌恶。假设3意味着我们会拒绝一个以等概率支付 \(\pm\Delta\) 的赌局。事实上，假设3意味着我们会拒绝任何收益关于0对称分布的赌局，因为接受赌局会降低我们财富的期望效用。数学上，假设3意味着 \(U\) 严格凹。如果二阶导数 \(U''\) 存在，则假设3等价于对所有 \(X\) 有 \(U''(X) < 0\)。

假设理性人会做出投资决策以最大化

\[ E\{U(X)\} = E[U\{X_0(1+R)\}] \tag{16.15} \]

其中 \(X\) 是该人的最终财富，\(X_0\) 是该人的初始财富，\(R\) 是投资的收益率。在经济学中，这几乎是理性人定义的一部分，该定义的另一部分是理性人将使用贝叶斯法则更新概率（见第20章）。假设每个个体都有自己的效用函数，两个不同的理性人可能因为效用函数不同而做出不同的决策。

均值-方差有效组合与最大化期望效用的组合有多大区别？在收益率服从正态分布的情况下，可以回答这个问题。

结果 16.2 若所有组合的收益率均服从正态分布，且 \(U\) 满足假设 3，则最大化期望效用的组合位于有效前沿上。

因此，如果选择最大化期望效用的组合，那么就会选出一个均值-方差有效组合。具体选择有效前沿上的哪个组合取决于个人的效用函数。

结果 16.2 的证明

该结果可通过证明以下事实来证明：若 \(R_1\) 和 \(R_2\) 服从正态分布，具有相同的均值，标准差分别为 \(\sigma_1\) 和 \(\sigma_2\) 且 \(\sigma_1 < \sigma_2\)，则 \(E\{U(R_1)\} > E\{U(R_2)\}\)。我们将证明这可由 Jensen 不等式推出，Jensen 不等式指出：若 \(U\) 是凹函数，\(X\) 是任意随机变量，则 \(E\{U(X)\} \leq U\{E(X)\}\)。若 \(U\) 严格凸且 \(X\) 非退化，则不等式严格成立。4

令 \(X = R_1 + e\)，其中 \(e\) 独立于 \(R_1\) 且服从均值为 0、方差为 \(\sigma_2^2 - \sigma_1^2\) 的正态分布。则 \(X\) 与 \(R_2\) 同分布，\(e\) 非退化，且利用迭代期望定律以及 Jensen 不等式和假设 3，我们有

\[E\{U(R_2)\} = E\{U(X)\} = E[E\{U(X)|R_1\}] < E[U\{E(X|R_1)\}] = E\{U(R_1)\},\tag{16.16}\]

因为 \(E(X|R_1) = R_1\)。□

结果 16.2 中收益率服从正态分布的假设可以放宽为更现实的假设，即资产收益率向量服从多元尺度混合分布，例如多元 \(t\) 分布。要证明这一推广，需要对混合变量进行条件化，使得收益率具有条件多元正态分布。那么 (16.16) 对混合变量的所有值条件成立，因此无条件成立。

一个常见的效用函数类别是

\[U(x;\lambda) = 1 - \exp(-\lambda x),\tag{16.17}\]

其中 \(\lambda > 0\) 决定风险厌恶程度。注意 \(U'(x;\lambda) = \lambda \exp(-\lambda x)\)，\(U''(x;\lambda) = -\lambda^2 \exp(-\lambda x)\)（关于 \(x\) 求导）。

  1. Jensen 不等式通常对凸函数表述且不等式方向相反。若 \(U\) 是凹函数，则 \(-U\) 是凸函数，因此两种形式的 Jensen 不等式等价。若存在常数 \(a\) 使得 \(P(X = a) = 1\)，则随机变量 \(X\) 是退化的，否则是非退化的。 ↩

因此，对于所有\(x\)，\(U''(x;\lambda) < 0\)，从而满足假设3；容易看出假设1和2也成立。当\(x \to \infty\)时，\(U(x;\lambda) \to 1\)。

用正常数乘以效用函数不会影响最大化效用的决策，并且可以对效用函数进行标准化以使其更具可比性。图16.7绘制了当\(\lambda=0.25,0.5,1,2,5\)时，\(\tilde{U}(x;\lambda)=U(x;\lambda)/U(0;\lambda)\)。由于对所有\(\lambda\)有\(\tilde{U}(1;\lambda)=1\)，这些效用函数被标准化，使得对应收益率为0的效用始终为1。换句话说，收益率为0对于所有风险厌恶程度具有相同的效用。

向效用函数添加常数不影响最优决策，因此可以使用稍简单的效用函数\(-\exp(-\lambda x)\)替代\(U(x;\lambda)\)或\(\tilde{U}(x;\lambda)\)。

当效用函数由(16.17)给出且\(R\)服从正态分布时，根据对数正态分布的性质（见附录A.9.4），(16.15)变为 \[1-\exp\left[-\lambda X_0\{1+E(R)\}+(\lambda X_0)^2\operatorname{var}(R)\right]\quad (16.18)\] 对于给定的\(\lambda\)和\(X_0\)值，期望效用通过最大化下式达到最大： \[E(R)-\frac{(\lambda X_0)\operatorname{var}(R)}{2}\quad (16.19)\] 因此，使用第16.6节的记号，选择投资组合的配置向量\(\mathbf{w}\)以最大化

最大化以下目标函数：

\[\mathbf{w}^T \boldsymbol{\mu} - \frac{\lambda X_0}{2} \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w} \qquad (16.20)\]

约束条件为 \(\mathbf{w}^T \mathbf{1} = 1\)。

在线性约束下最大化 (16.20) 是一个二次规划问题。当 \(\lambda \to 0\) 时，期望收益率和收益率标准差趋于无穷。反之，当 \(\lambda \to \infty\) 时，解收敛到最小方差组合。因此，当 \(\lambda\) 从 \(\infty\) 变化到 0 时，我们可以从右到左找到有效前沿上的所有组合。这一行为在下一个例子中加以说明，见图 16.8。

    b = matrix(1.0)
    sol = solvers.qp(P, q, A=A, b=b)
    w = np.array(sol['x']).flatten()
    mu_vect[i] = w @ mean_vect.flatten()
    sd_vect[i] = np.sqrt(w @ cov_mat @ w)
    w_matrix[i, :] = w
    ExUtil_vect[i] = sol['primal objective']

然后，将期望收益率和收益率标准差相对于 \(\lambda\) 作图，再通过绘制期望收益率对收益率标准差得到有效前沿。图见 16.8。\(\Box\)

16.10.3 寻找可能的期望收益率集合

在第16.6节中，当我们通过二次规划寻找有效前沿时，需要设定一个投资组合期望收益率的可能取值网格。当权重向量w除了元素之和为1之外没有其他约束时，任何期望收益率都是可行的5。我们在例16.7中看到，如果对所有\(i\)施加约束\(0 \le w_i \le 1\)从而禁止卖空，那么可行的投资组合期望收益率介于单个资产的最小和最大期望收益率之间。

当对\(w_i\)施加更复杂的约束时，可行的投资组合期望收益率集合可以通过线性规划找到。在本节中，我们使用与第16.10.1节相同的数据。我们将对所有\(i\)施加约束\(w_i \le B_1\)和\(-B_2 \le w_i\)。

函数solveLP()（来自linprog包）对\(N\)维变量x最小化（或最大化）目标函数\(\mathbf{c}^T\mathbf{x}\)，满足\(\mathbf{A}\mathbf{x} \le \mathbf{b}\)和\(\mathbf{x} \ge 0\)。这里\(\mathbf{c}\)是\(N\times 1\)常数向量，\(\mathbf{A}\)是\(N\times k\)常数矩阵，\(\mathbf{b}\)是\(k\times 1\)常数向量，其中\(N\)和\(k\)是整数。此外，\(0\)是\(k\)维零向量。

由于\(\mathbf{x} \ge 0\)，除非我们禁止卖空，否则我们不能直接将w设为x。当允许卖空时，我们可以令w等于\(\mathbf{x}_1 - \mathbf{x}_2\)，且\(\mathbf{x}^T = (\mathbf{x}_1^T, \mathbf{x}_2^T)\)。那么约束条件为\(\mathbf{x}_1\)的每个元素最多为\(B_1\)，\(\mathbf{x}_2\)的每个元素最多为\(B_2\)。目标函数\(\mathbf{w}^T\boldsymbol{\mu}\)等于\((\boldsymbol{\mu}^T, -\boldsymbol{\mu}^T)\mathbf{x}\)。

  1. “可行的”意味着存在一个向量\( \mathbf{w} \)可以达到该期望收益率。 ↩
