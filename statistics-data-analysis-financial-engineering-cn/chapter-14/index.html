第14章 GARCH模型

14.1 引言

正如前几章所见，金融市场数据常常表现出波动率聚集（volatility clustering）现象，即时间序列交替出现高波动率和低波动率的时期；例如，请参见图14.1。事实上，对于经济和金融数据，时变波动率比恒定波动率更为常见，而精确建模时变波动率在金融工程中具有重要意义。

我们在第12章中看到，ARMA模型用于建模给定过去条件下过程的条件期望，但在ARMA模型中，给定过去条件下的条件方差是恒定的。这对股票收益率建模意味着什么？假设我们注意到最近日收益率异常波动。我们可能预期明天的收益率也比平常更易变。然而，ARMA模型无法捕捉这种行为，因为它的条件方差是恒定的。因此，如果我们要建模非恒定波动率，就需要更好的时间序列模型。在本章中，我们将研究GARCH时间序列模型，这些模型由于具有随机变化的波动率而在计量经济学和金融学中被广泛使用。

ARCH是自回归条件异方差（Auto-Regressive Conditional Heteroskedasticity）的缩写。在ARCH模型中，条件方差的结构与AR模型中的条件期望结构非常相似。我们首先研究一阶ARCH(1)模型，它是最简单的GARCH模型，类似于AR(1)模型。然后，我们考察类似于AR(p)模型的ARCH(p)模型，以及GARCH（广义ARCH）模型，后者对条件方差的建模方式类似于ARMA模型对条件期望的建模。最后，我们考虑几种多元GARCH过程。

14.2 估计条件均值与条件方差

在研究GARCH模型之前，我们先学习一些关于非恒定条件方差建模的一般原理。考虑具有恒定条件方差 \(\operatorname{Var}(Y_t \mid X_{1,t},\ldots,X_{p,t}) = \sigma^2\) 的回归建模。那么 \(Y_t\) 关于 \(X_{1,t},\ldots,X_{p,t}\) 的回归的一般形式为

\[Y_t = f(X_{1,t},\ldots,X_{p,t}) + \epsilon_t, \qquad (14.1)\]

其中 \(\epsilon_t\) 与 \(X_{1,t},\ldots,X_{p,t}\) 独立，期望为0，并具有恒定条件方差 \(\sigma^2_\epsilon\)。函数 \(f(\cdot)\) 是给定 \(X_{1,t},\ldots,X_{p,t}\) 时 \(Y_t\) 的条件期望。此外，\(Y_t\) 的条件方差为 \(\sigma^2_\epsilon\)。

方程(14.1)可以修改为允许条件异方差性。令 \(\sigma^2(X_{1,t},\ldots,X_{p,t})\) 为给定 \(X_{1,t},\ldots,X_{p,t}\) 时 \(Y_t\) 的条件方差。那么模型

\[Y_t = f(X_{1,t},\ldots,X_{p,t}) + \epsilon_t \, \sigma(X_{1,t},\ldots,X_{p,t}), \qquad (14.2)\]

其中，\(\epsilon_t\)的条件（给定\(X_{1,t},\ldots,X_{p,t}\)）均值为0，条件方差为1，从而给出了\(Y_t\)正确的条件均值和条件方差。函数\(\sigma(X_{1,t},\ldots,X_{p,t})\)应为非负，因为它是标准差。如果函数\(\sigma(\cdot)\)是线性的，则其系数必须施加约束以确保非负性。这种约束实施起来很繁琐，因此通常改用非线性非负函数。条件方差模型通常被称为方差函数模型。本章的GARCH模型是一类重要的方差函数模型。

假设目前\(\epsilon_1,\epsilon_2,\ldots\)是单位方差的高斯白噪声。之后我们将允许噪声是独立同分布的白噪声，可能具有非正态分布，例如标准化\(t\)分布。则

\[E(\epsilon_t|\epsilon_{t-1},\ldots)=0\]， \[\operatorname{Var}(\epsilon_t|\epsilon_{t-1},\ldots)=1。\tag{14.3}\]

性质(14.3)称为条件同方差性。

过程\(a_t\)在模型

\[a_t = \epsilon_t \sqrt{\omega + \alpha a_{t-1}^2} \tag{14.4}\]

下是ARCH(1)过程，这是(14.2)的一个特例，其中\(f\)等于0，\(\sigma\)等于\(\sqrt{\omega + \alpha a_{t-1}^2}\)。我们需要\(\omega>0\)且\(\alpha\geq 0\)，以保证对所有\(t\)有\(\omega+\alpha a_{t-1}^2 > 0\)。还需要\(\alpha<1\)，以使\(\{a_t\}\)平稳且具有有限方差。

方程(14.4)可以写为

\[a_t^2 = \epsilon_t^2 (\omega + \alpha a_{t-1}^2)，\]

这类似于AR(1)，但是针对\(a_t^2\)而不是\(a_t\)，且具有均值为1的乘法噪声而不是均值为0的加法噪声。事实上，ARCH(1)模型导致\(a_t^2\)的ACF与AR(1)的ACF相同，我们从下面的计算中将会看到。

定义

\[\sigma_t^2 = \operatorname{Var}(a_t | a_{t-1}, \ldots)\]

为给定过去值时\(a_t\)的条件方差。由于\(\epsilon_t\)独立于\(a_{t-1}\)且\(E(\epsilon_t^2)=\operatorname{Var}(\epsilon_t)=1\)，我们有

\[E(a_t | a_{t-1}, \ldots)=0，\tag{14.5}\]

以及

\[ \sigma_t^2 = E\left\{ (\omega + \alpha a_{t-1}^2) \epsilon_t^2 \mid a_{t-1}, a_{t-2}, \ldots \right\} = (\omega + \alpha a_{t-1}^2) E\left\{ \epsilon_t^2 \mid a_{t-1}, a_{t-2}, \ldots \right\} = \omega + \alpha a_{t-1}^2. \tag{14.6} \]

方程 (14.6) 对于理解GARCH过程的工作原理至关重要。如果 \(a_{t-1}\) 具有异常大的绝对值，那么 \(\sigma_t\) 比平时更大，因此 \(a_t\) 也预计具有异常大的幅度。这种波动性会传播，因为当 \(a_t\) 具有大幅度时，会使 \(\sigma_{t+1}^2\) 变大，从而 \(a_{t+1}\) 倾向于具有大幅度，依此类推。类似地，如果 \(a_{t-1}^2\) 异常小，那么 \(\sigma_t^2\) 也小，且 \(a_t^2\) 也预计小，以此类推。由于这种性质，\(a_t\) 中的异常波动性倾向于持续，尽管不是永远。条件方差倾向于回归到无条件方差，前提是 \(\alpha < 1\)，这样过程是平稳的且具有有限方差。

\(a_t\) 的无条件（即边际）方差记作 \(\gamma_a(0)\)，通过对 (14.6) 取期望得到，对于平稳模型有：

\[ \gamma_a(0) = \omega + \alpha \gamma_a(0). \]

若 \(\alpha < 1\)，该方程具有正解：

\[ \gamma_a(0) = \omega/(1 - \alpha). \]

若 \(\alpha = 1\)，则 \(\gamma_a(0)\) 无穷大，但 \(a_t\) 仍然平稳，称为积分GARCH（I-GARCH）模型。

利用 (14.5) 的直接计算表明 \(a_t\) 的ACF为：

\[ \rho_a(h) = 0 \quad \text{若 } h \neq 0. \]

实际上，任何在给定过去条件下当前观测的条件期望为常数的过程都是不相关过程。在初级统计课程中，常提到独立性意味着零相关，但反之不成立。一个过程，如GARCH过程，其条件均值恒定但条件方差非常数，就是不相关但相依过程的一个例子。条件方差对过去的依赖性导致过程相依，而条件均值对过去的独立性则是过程不相关的原因。

尽管 \(a_t\) 是不相关过程，但 \(a_t^2\) 过程具有更有趣的ACF。若 \(\alpha < 1\)，则

\[ \rho_{a^2}(h) = \alpha^{|h|}, \quad \forall h. \]

若 \(\alpha \geq 1\)，则 \(a_t^2\) 要么非平稳，要么方差无穷，因此没有ACF。对于ARCH(1)模型，\(a_t^2\) 的ACF呈现几何衰减。

14.4 AR(1)+ARCH(1) 模型

该过程类似于AR(1)过程ACF中的几何衰减。为完成类比，定义 \(\eta_t = a_t^2 - \sigma_t^2\)，并注意到 \(\{\eta_t\}\) 是均值为零的弱白噪声过程，但不是独立同分布的白噪声过程。将 \(\eta_t\) 加到(14.6)的两边并简化，我们得到

\[ \sigma_t^2 + \eta_t = a_t^2 = \omega + \alpha a_{t-1}^2 + \eta_t, \tag{14.7} \]

这是将 \(\{a_t^2\}\) 直接表示为AR(1)过程。

14.4 AR(1)+ARCH(1) 模型

如我们所见，AR(1)过程具有非恒定的条件均值但恒定的条件方差，而ARCH(1)过程则正好相反。如果数据的条件均值和条件方差都依赖于过去，那么我们可以将这两个模型结合起来。事实上，我们可以将任何ARMA模型与第14.6节中的任何GARCH模型结合起来。在本节中，我们将AR(1)模型与ARCH(1)模型结合起来。

令 \(a_t\) 为ARCH(1)过程，使得 \(a_t = \epsilon_t \sqrt{\omega + \alpha a_{t-1}^2}\)，其中 \(\epsilon_t\) 是独立同分布的 \(N(0,1)\)，并假设

\[ y_t - \mu = \phi (y_{t-1} - \mu) + a_t. \]

过程 \(y_t\) 是AR(1)过程，只是噪声项 \(a_t\) 不是独立同分布的白噪声，而是ARCH(1)过程，它仅是弱白噪声。

因为 \(a_t\) 是不相关过程，它具有与独立白噪声相同的ACF，因此，在平稳情形下，\(y_t\) 具有与独立白噪声的AR(1)过程相同的ACF：

\[ \rho_y(h) = \phi^{|h|} \quad \forall h. \]

此外，\(a_t^2\) 具有ARCH(1)的ACF：

\[ \rho_{a^2}(h) = \alpha^{|h|} \quad \forall h. \]

在平稳情形下，如果还满足一些额外假设，\(y_t^2\) 的ACF也以几何速率随 \(|h|\) 衰减，但精确表达式更为复杂（参见Palma和Zevallos，2004）。为使 \(y_t\) 平稳且具有有限方差，我们需要假设 \(|\phi| < 1\) 且 \(\alpha < 1\)。当然，为保证条件方差过程 \(\sigma_t^2\) 的正定性，还需假设 \(\omega > 0\) 和 \(\alpha \geq 0\)。过程 \(y_t\) 的条件均值和条件方差（给定过去）都是非恒定的，因此可以对多种时间序列进行建模。

例14.1. 模拟的ARCH(1)过程和AR(1)+ARCH(1)过程

图14.2展示了一个模拟的ARCH(1)过程。面板(a)是独立同分布的白噪声过程\(\epsilon_t\)，(b)是条件标准差过程\(\sigma_t = \sqrt{1 + 0.55 a_{t-1}^2}\)，(c)是ARCH(1)过程\(a_t = \sigma_t \epsilon_t\)。如前所述，ARCH(1)过程可作为AR(1)过程的噪声项，面板(d)展示了这一过程。AR(1)的参数为\(\mu = 0.1\)和\(\phi = 0.8\)。\(a_t\)的无条件方差为\(\gamma_a(0) = 1/(1 - 0.55) = 2.22\)，因此无条件标准差为\(\sqrt{2.22} = 1.49\)。面板(e)–(h)分别是ARCH和AR+ARCH过程及其平方过程的样本ACF图。注意，对于ARCH序列，过程本身不相关，但平方序列存在自相关。对于AR(1)+ARCH(1)序列，过程及其平方过程的ACF（面板(g)和(h)）均显示自相关。虽然真实ACF呈精确的几何衰减，但面板(f)–(h)中的样本ACF仅是近似；类似地，真实ACF中不存在负值，但样本ACF因抽样误差可能产生负值。所有过程均从0开始，模拟生成10,200个观测值，前10,000个观测值作为预热期被舍弃。□

14.5 ARCH(p)模型

如前，令\(\epsilon_t\)为单位方差的高斯白噪声。则\(a_t\)为ARCH(p)过程，若

\[a_t = \sigma_t \epsilon_t,\]

其中

\[\sigma_t = \sqrt{\omega + \sum_{i=1}^{p} \alpha_i a_{t-i}^2}\]

是给定该过程过去值\(a_{t-1}, a_{t-2}, \ldots\)时\(a_t\)的条件标准差。与ARCH(1)过程类似，ARCH(p)过程不相关，具有常数均值（条件与非条件）和常数无条件方差，但其条件方差非恒定。事实上，\(a_t^2\)的ACF与AR(p)过程的ACF具有相同的结构；参见第14.9节。

14.6 ARIMA(p_M, d, q_M)+GARCH(p_V, q_V)模型

ARCH(p)模型的一个缺陷是条件标准差过程具有高频振荡性，高波动性以短脉冲形式出现。这种行为可在图14.2b中看到。GARCH模型允许更广泛的行为，尤其是更持久的波动性。GARCH(p,q)模型为

\[a_t = \sigma_t \epsilon_t,\]

其中

\[\sigma_t = \sqrt{\omega + \sum_{i=1}^{p} \alpha_i a_{t-i}^2 + \sum_{j=1}^{q} \beta_j \sigma_{t-j}^2}. \qquad (14.8)\]

由于\(\sigma_t\)过程的过去值被反馈到当前值（带有非负系数\(\beta_j\)），条件标准差可以表现出比ARCH过程更持久的高或低波动性时期。在平稳情况下，过程\(a_t\)不相关，具有常数无条件均值和方差，且\(a_t^2\)的ACF类似于ARMA过程（参见第14.9节）。GARCH模型包含ARCH模型作为特例，我们使用术语“GARCH”来指代ARCH和GARCH模型。

一个非常一般的时间序列模型让\(a_t\)为GARCH(p_V, q_V)，并将\(a_t\)作为ARIMA(p_M, d, q_M)模型中的噪声项。p和q的下标区分条件方差（V）或GARCH参数与条件均值（M）或ARIMA参数。我们将这样的过程称为ARIMA(p_M, d, q_M)+GARCH(p_V, q_V)模型。

图14.3展示了来自GARCH(1,1)过程和AR(1)+GARCH(1,1)过程的500次观测的模拟结果。GARCH参数为 \(\omega=1\), \(\alpha=0.08\), \(\beta=0.9\)。较大的\(\beta\)值使得\(\sigma_t\)与\(\sigma_{t-1}\)高度相关，并使条件标准差过程具有相对长期的持久性，至少与ARCH模型下的行为相比如此。特别要注意，条件标准差比图14.2中ARCH(1)过程的“突发性”更小。

14.6.1 ARIMA(\(p_M\), d, \(q_M\))+GARCH(\(p_V\), \(q_V\)) 模型的残差

当将ARIMA(\(p_M\), d, \(q_M\))+GARCH(\(p_V\), \(q_V\))模型拟合到时间序列\(Y_t\)时，有两种类型的残差。普通残差，记为 \(\hat{a}_t\)，是\(Y_t\)与其条件期望之间的差值。正如符号所示，\(\hat{a}_t\)估计 \(a_t\)。标准化残差，记为 \(\hat{\epsilon}_t\)，是普通残差 \(\hat{a}_t\) 除以其估计的条件标准差 \(\hat{\sigma}_t\)。标准化残差估计 \(\epsilon_t\)。标准化残差应用于模型检查。如果模型拟合良好，则 \(\hat{\epsilon}_t\) 和 \(\hat{\epsilon}_t^2\) 都不应表现出序列相关性。此外，如果假定 \(\epsilon_t\) 服从正态分布，则可以通过标准化残差 \(\hat{\epsilon}_t\) 的正态图来检查该假设。\(\hat{a}_t\) 是ARIMA过程的残差，用于第12.12节中的预测方法。

14.7 GARCH过程具有厚尾

研究者很早就注意到股票收益率具有“厚尾”或“易出现异常值”的概率分布，我们在前面章节中也已亲眼目睹。异常值的一个原因可能是条件方差不是常数，当方差很大时就会出现异常值，如同5.5节的正态混合例子所示。事实上，即使\(\{\epsilon_t\}\)是高斯分布，GARCH过程也表现出厚尾。因此，当我们使用GARCH模型时，可以同时建模条件异方差性和金融市场数据的厚尾分布。不过，许多金融时间序列的尾部比具有高斯\(\{\epsilon_t\}\)的GARCH过程所隐含的更厚。为处理这类数据，可以假设\(\{\epsilon_t\}\)不是高斯白噪声，而是具有厚尾分布的独立同分布白噪声过程。

本章中看到的GARCH模型与ARMA模型之间的相似性并非巧合。如果\(a_t\)是一个GARCH过程，那么\(a_t^2\)是一个ARMA过程，但是具有弱白噪声，而非独立同分布白噪声。为证明这一点，我们从GARCH(1,1)模型开始，其中\(a_t = \sigma_t \epsilon_t\)。这里\(\epsilon_t\)是独立同分布白噪声，且

\[ E(a_t^2 | \mathcal{F}_{t-1}) = \sigma_t^2 = \omega + \alpha a_{t-1}^2 + \beta \sigma_{t-1}^2, \tag{14.9} \]

其中\(\mathcal{F}_{t-1}\)是时刻\(t-1\)的信息集。定义\(\eta_t = a_t^2 - \sigma_t^2\)。由于\(E(\eta_t | \mathcal{F}_{t-1}) = E(a_t^2 | \mathcal{F}_{t-1}) - \sigma_t^2 = 0\)（由(A.33)），\(\eta_t\)是一个不相关的过程，即弱白噪声过程。\(a_t\)的条件异方差性被传递给\(\eta_t\)，因此\(\eta_t\)不是独立同分布的白噪声。

简单的代数运算表明

\[ \sigma_t^2 = \omega + (\alpha + \beta) a_{t-1}^2 - \beta \eta_{t-1} \tag{14.10} \]

14.10 GARCH(1,1)过程

因此

\[a_t^2 = \sigma_t^2 + \eta_t = \omega + (\alpha + \beta) a_{t-1}^2 - \beta \eta_{t-1} + \eta_t. \tag{14.11}\]

假设 \(\alpha + \beta < 1\)。令 \(\upsilon = \omega / \{1 - (\alpha + \beta)\}\)，则

\[a_t^2 - \upsilon = (\alpha + \beta)(a_{t-1}^2 - \upsilon) - \beta \eta_{t-1} + \eta_t. \tag{14.12}\]

从(14.12)可以看出，\(a_t^2\) 是一个 ARMA(1,1) 过程。使用 (12.25) 的记号，均值为 \(\mu = \upsilon\)，AR(1) 系数为 \(\phi = \alpha + \beta\)，MA(1) 系数为 \(\theta = -\beta\)。

对于一般情形，假设 \(\sigma_t\) 遵循 (14.8)，使得

\[\sigma_t^2 = \omega + \sum_{i=1}^p \alpha_i a_{t-i}^2 + \sum_{j=1}^q \beta_j \sigma_{t-j}^2. \tag{14.13}\]

为简化记号，若 \(q > p\)，则定义 \(\alpha_i = 0\)，\(i = p+1,\dots,q\)；类似地，若 \(p > q\)，则定义 \(\beta_j = 0\)，\(j = q+1,\dots,p\)。令

\[\upsilon = \omega / \left\{1 - \sum_{i=1}^{\max(p,q)} (\alpha_i + \beta_i)\right\}.\]

类似于 GARCH(1,1) 情形的直接代数计算表明

\[a_t^2 - \upsilon = \sum_{i=1}^{\max(p,q)} (\alpha_i + \beta_i)(a_{t-i}^2 - \upsilon) - \sum_{j=1}^q \beta_j \eta_{t-j} + \eta_t, \tag{14.14}\]

因此 \(a_t^2\) 是一个 ARMA(\max(p,q), q) 过程，均值为 \(\mu = \upsilon\)，AR 系数为 \(\phi_i = \alpha_i + \beta_i\)，MA 系数为 \(\theta_j = -\beta_j\)。作为这些计算的副产品，我们得到了 \(a_t\) 平稳的必要条件：

\[\sum_{i=1}^{\max(p,q)} (\alpha_i + \beta_i) < 1. \tag{14.15}\]

14.10 GARCH(1,1)过程

GARCH(1,1) 是最常用的 GARCH 过程，因此值得详细研究。若 \(a_t\) 服从 GARCH(1,1)，则如我们刚刚所见，\(a_t^2\) 是 ARMA(1,1) 过程。因此，\(a_t^2\) 的 ACF 可以由公式 (12.31) 和 (12.32) 得到。经过一些代数推导，可得

\[\rho_{a^2}(1) = \frac{\alpha(1 - \alpha\beta - \beta^2)}{1 - 2\alpha\beta - \beta^2} \tag{14.16}\]

以及

\[\rho_{a^2}(h) = (\alpha + \beta)^{h-1} \rho_{a^2}(1), \quad h \ge 2. \tag{14.17}\]

这些公式同样适用于AR(1)+GARCH(1,1)模型，在平稳条件下，若某些附加假设成立，\(y_t^2\)的ACF也以几何速率随\(h \geq 2\)衰减，然而精确表达式更为复杂（参见Palma与Zevallos, 2004）。

由(14.16)，存在无穷多组\((\alpha,\beta)\)具有相同的\(\rho_{a^2}(1)\)值。由(14.17)，\(\alpha+\beta\)的值越大，\(\rho_{a^2}(\cdot)\)在第一个滞后之后的衰减越慢。图14.6展示了这一行为，图中给出了三个GARCH(1,1)过程的\(a_t^2\)的ACF，其一阶自相关均为0.5。实线曲线具有最高的\(\alpha+\beta\)值，ACF衰减非常缓慢。虚线曲线是纯ARCH(1)过程，衰减最快。

在例14.2中，对BMW日对数收益率拟合了AR(1)+GARCH(1,1)模型。GARCH参数估计值为\(\hat{\alpha}=0.10\)和\(\hat{\beta}=0.86\)。由(14.16)，该过程的\(\hat{\rho}_{a^2}(1)=0.197\)，较大的\(\hat{\beta}\)值暗示衰减缓慢。图14.7绘制了（来自AR(1)模型的）平方残差的样本ACF。在该图中，我们看到一阶自相关略低于0.2，一阶滞后之后ACF缓慢衰减，完全符合预期。

GARCH(1,1)模型能够分别拟合一阶自相关及其后的衰减速率，这一能力在实践中非常重要。这似乎是GARCH(1,1)模型能拟合众多金融时间序列的主要原因。

14.11 APARCH模型

在某些金融时间序列中，大幅负收益率似乎比相同幅度的正收益率更能增加波动性。这被称为杠杆效应。标准GARCH模型（即公式(14.8)给出的模型）无法建模杠杆效应，因为它们将\(\sigma_t\)建模为过去\(a_t^2\)值的函数——而不考虑\(a_t\)的过去值是正还是负。问题在于平方函数\(x^2\)关于\(x\)是对称的。解决方案是用一类灵活的非负函数（包含非对称函数）替代平方函数。APARCH（非对称幂ARCH）模型实现了这一点。它们通过建模\(\sigma_t^\delta\)（其中\(\delta>0\)是另一个参数）提供了比GARCH模型更大的灵活性。

APARCH\((p,q)\)模型的条件标准差为： \[ \sigma_t^\delta = \omega + \sum_{i=1}^p \alpha_i (|a_{t-i}| - \gamma_i a_{t-i})^\delta + \sum_{j=1}^q \beta_j \sigma_{t-j}^\delta, \qquad (14.18) \] 其中\(\delta>0\)且\(-1 < \gamma_i < 1\), \(i=1,\ldots,p\)。注意，当\(\delta=2\)且\(\gamma_1=\cdots=\gamma_p=0\)时，即为标准GARCH模型。

\(a_{t-i}\)对\(\sigma_t\)的影响通过函数\(g_{\gamma_i}\)来实现，其中\(g_\gamma(x) = |x| - \gamma x\)。图14.8展示了不同\(\gamma\)值下的\(g_\gamma(x)\)。当\(\gamma>0\)时，对于任何\(x>0\)，有\(g_\gamma(-x) > g_\gamma(x)\)，因此存在杠杆效应。如果\(\gamma<0\)，则杠杆效应与预期方向相反——即\(a_t\)的正向过去值比相同幅度的负向过去值更能增加波动性。

14.12 带ARMA+GARCH误差的线性回归

在使用时间序列回归时，我们常常观察到自相关的残差。为此，我们在第13.3.3节中引入了带ARMA扰动项的线性回归。所考虑的模型为：

\[ Y_t = \beta_0 + \beta_1 X_{t,1} + \cdots + \beta_p X_{t,p} + e_t, \tag{14.19} \]

其中

\[ (1 - \phi_1 B - \cdots - \phi_p B^p)(e_t - \mu) = (1 + \theta_1 B + \cdots + \theta_q B^q) a_t, \tag{14.20} \]

且\(\{a_t\}\)为独立同分布白噪声。该模型足以处理序列相关误差，但不能处理残差中常见的波动率聚集。

一种解决方案是将噪声建模为ARMA+GARCH过程。因此，我们现假定\(\{a_t\}\)不再是独立同分布白噪声，而是一个GARCH过程，使得

\[ a_t = \sigma_t \epsilon_t, \tag{14.21} \]

其中

\[ \sigma_t = \sqrt{\omega + \sum_{i=1}^{p} \alpha_i a_{t-i}^2 + \sum_{j=1}^{q} \beta_j \sigma_{t-j}^2}, \tag{14.22} \]

且\(\{\epsilon_t\}\)为独立同分布白噪声。由(14.19)-(14.22)给出的模型即为带ARMA+GARCH扰动项的线性回归模型。

一些软件，包括R包rugarch中的ugarchfit()函数，可以一步拟合带ARMA+GARCH扰动项的线性回归模型。另一种解决方案是通过第13.3.2节中的HAC估计量来调整或修正回归系数的估计协方差矩阵，使用R包sandwich中的NeweyWest()函数。然而，如果没有此类软件可用，则可采用以下三步估计方法：

  1. 用普通最小二乘法估计(14.19)中的参数；
  2. 对普通最小二乘残差拟合模型(14.20)-(14.22)；
  3. 用步骤2中的条件方差倒数作为权重，通过加权最小二乘法重新估计(14.19)中的参数。

14.13 ARMA+GARCH过程的预测

这些结果与例9.9中未加权拟合的结果之间没有显著差异。在这种情况下，对残差使用GARCH模型的主要原因在于，如果模型用于预测，它可以提供更准确的预测区间；参见第14.13节。□

ARMA+GARCH过程的预测在某些方面与ARMA过程的预测类似——点估计，例如条件均值的预测，是相同的，因为GARCH过程是弱白噪声。ARMA+GARCH过程与ARMA过程预测的不同之处在于预测区间的行为。在高波动时期，使用ARMA+GARCH模型的预测区间会变宽，以考虑更高的不确定性。类似地，在低波动时期，预测区间会变窄。使用没有条件异方差的ARMA模型的预测区间无法以这种方式调整。

为了说明，我们将比较高斯白噪声过程的预测和具有高斯新息项的GARCH(1,1)过程的预测。

14.13 ARMA+GARCH过程的预测

两个模型的条件均值均为ARMA(0,0)，因此它们的预测等于边际均值，记为\(\mu\)。对于高斯白噪声，预测区间为\(\mu \pm z_{\alpha/2} \sigma\)，其中\(\sigma\)是边际标准差。对于GARCH(1,1)过程\(\{Y_t\}\)，在时间原点\(n\)处向前\(h\)步的预测区间为\(\mu \pm z_{\alpha/2} \sigma_{n+h|n}\)，其中\(\sigma_{n+h|n}\)是基于时刻\(n\)可用信息对\(Y_{n+h}\)的条件标准差。随着\(h\)的增加，\(\sigma_{n+h|n}\)收敛到\(\sigma\)，因此对于长提前期，两个模型的预测区间相似。然而，对于较短的提前期，预测限可能差异很大。

14.14 多元GARCH过程

金融资产收益率随时间推移倾向于共同运动，其各自波动率在资产和市场之间也是如此。对时变条件协方差矩阵（即波动率矩阵）进行建模，在许多金融应用中至关重要，包括资产定价、对冲、投资组合选择和风险管理。

多元波动率建模面临重大挑战。首先是维数灾难：对于一个 \(d\) 维过程，有 \(d(d+1)/2\) 个方差和协方差（例如，当 \(d=9\) 时为45个），所有这些都可能随时间变化。此外，与收益率不同，所有这些方差和协方差都是不可观测的，即潜变量。许多波动率矩阵演化的参数化方法使用大量参数，以至于当 \(d>10\) 时估计变得不可行。除了经验充分性（即模型对数据的拟合优度）外，估计的简便性和可行性也是重要的考虑因素。

类似于单变量GARCH模型中的正定性约束，一个定义良好的多元波动率矩阵过程必须在每个时间点正定，且基于模型的预测也应如此。从实践角度来看，应用中经常需要波动率矩阵的逆矩阵。此外，投资组合收益率（资产收益率的线性组合）的条件方差估计必须为正；幸运的是，正定性保证了这一点。

14.14.2 基本设定

令 \(\mathbf{Y}_t = (Y_{1,t}, \dots, Y_{d,t})' \) 表示一个 \(d\) 维向量过程，并令 \(\mathcal{F}_t\) 表示由 \(\mathbf{Y}_t, \mathbf{Y}_{t-1}, \dots \) 生成的 \(t\) 时刻的信息集。我们可以将过程分解为

\[ \mathbf{Y}_t = \boldsymbol{\mu}_t + \mathbf{a}_t, \qquad (14.23) \]

其中 \(\boldsymbol{\mu}_t = E(\mathbf{Y}_t | \mathcal{F}_{t-1})\) 是 \(t\) 时刻的条件均值向量，\(\{\mathbf{a}_t\}\) 是均值为零的弱白噪声新息向量过程，其无条件协方差矩阵为 \(\boldsymbol{\Sigma}_a = \text{Cov}(\mathbf{a}_t)\)。令

\[ \boldsymbol{\Sigma}_t = \text{Cov}(\mathbf{a}_t | \mathcal{F}_{t-1}) = \text{Cov}(\mathbf{Y}_t | \mathcal{F}_{t-1}) \qquad (14.24) \]

表示 \(t\) 时刻的条件协方差矩阵或波动率矩阵。多元时间序列建模主要关注 \(\boldsymbol{\mu}_t\) 和 \(\boldsymbol{\Sigma}_t\)（即条件均值和条件协方差矩阵）随时间的变化。对于平稳过程，即使条件均值和条件协方差矩阵可能随时间变化，无条件均值和无条件协方差矩阵却是常数。

在本节中，我们假设 \(\boldsymbol{\mu}_t\) 遵循平稳的VAR(\(p\)) 模型，即 \(\boldsymbol{\mu}_t = \boldsymbol{\mu} + \sum_{\ell=1}^p \boldsymbol{\Phi}_\ell (\mathbf{Y}_{t-\ell} - \boldsymbol{\mu})\)，其中 \(p\) 是非负整数，\(\boldsymbol{\mu}\) 是 \(d \times 1\) 无条件均值向量，\(\boldsymbol{\Phi}_\ell\) 是 \(d \times d\) 系数矩阵。回顾一下，图14.13中考虑的残差序列来自 \(p=1\) 的VAR模型。

新息过程与波动率过程之间的关系由下式定义：

\[ \mathbf{a}_t = \boldsymbol{\Sigma}_t^{1/2} \boldsymbol{\epsilon}_t, \qquad \boldsymbol{\epsilon}_t \stackrel{\text{iid}}{\sim} F(\mathbf{0}, \mathbf{I}_d), \qquad (14.25) \]

其中 \(\boldsymbol{\Sigma}_t^{1/2}\) 是 \(\boldsymbol{\Sigma}_t\) 的对称矩阵平方根，满足 \(\boldsymbol{\Sigma}_t^{1/2} \boldsymbol{\Sigma}_t^{1/2} = \boldsymbol{\Sigma}_t\)。独立同分布的白噪声 \(\boldsymbol{\epsilon}_t\) 是来自多元分布 \(F\) 的标准化新息，其均值为零，协方差矩阵为单位阵。下面详述的模型描述了波动率矩阵 \(\boldsymbol{\Sigma}_t\) 的动态演变。

14.14.3 指数加权移动平均（EWMA）模型

单变量波动率模型最简单的矩阵推广是指数加权移动平均（EWMA）模型。该模型由一个参数 \(\lambda \in (0,1)\) 索引，并通过递归公式定义：

\[ \boldsymbol{\Sigma}_t = (1-\lambda) \mathbf{a}_{t-1} \mathbf{a}_{t-1}' + \lambda \boldsymbol{\Sigma}_{t-1} = (1-\lambda) \sum_{\ell=1}^{\infty} \lambda^{\ell-1} \mathbf{a}_{t-\ell} \mathbf{a}_{t-\ell}' \qquad (14.26) \]

当 (14.26) 中的递归从一个正定矩阵初始化时，序列保持正定。这个单参数模型无论维数如何都易于估计，较大的 \(\lambda\) 值表示波动率过程具有较高的持续性。然而，在实际中这种动态可能过于严格，因为各分量的演化都具有相同的折扣因子（即持续性参数）\(\lambda\)。

14.14.4 正交GARCH模型

为了减少参数数量和参数约束，一些因子和正交模型被提出，它们通过对方差矩阵的元素施加共同的动态结构。Alexander (2001)的正交GARCH模型因其简洁性而成为最流行的模型之一。该模型假设新息\(\boldsymbol{a}_t\)可以通过线性变换\(\mathbf{U}\)分解为正交分量\(\boldsymbol{z}_t\)。这通常结合主成分分析（PCA，见第18.2节）进行如下操作。令\(\mathbf{O}\)为特征向量矩阵，\(\boldsymbol{\Lambda}\)为\(\boldsymbol{\Sigma}_a\)的对应特征值对角矩阵。然后取\(\mathbf{U} = \boldsymbol{\Lambda}^{-1/2}\mathbf{O}'\)，并令

14.4.5 动态正交成分（DOC）模型

为了在估计类似上述 O-GARCH 模型的线性变换后正确应用单变量建模，得到的成分过程不仅必须在当前时刻正交，而且条件相关性也必须为零。此外，平方成分的滞后交叉相关性也必须为零。在 Matteson 和 Tsay (2011) 中，如果时间序列 \(s_t\) 的成分满足这些条件，则称它们为波动率中的动态正交成分（DOCs）。

令 \(s_t = (s_{1,t}, \ldots, s_{d,t})'\) 表示 DOCs 的向量时间序列。不失一般性，假设 \(s_t\) 经过标准化使得 \(E(s_{i,t}) = 0\) 且 \(\text{Var}(s_{i,t}) = 1\)，其中 \(i = 1, \ldots, d\)。使用以下定义的 Ljung-Box 型统计量来检验 DOCs 在波动率中是否存在。检验中包含滞后零意味着平稳 DOCs 的成对乘积过程 \(s_{i,t} s_{j,t}\) 具有零序列相关性，因为由 Cauchy-Schwarz 不等式可得：

\[ |\text{Cov}(s_{i,t} s_{j,t}, s_{i,t-h} s_{j,t-h})| \leq \text{Var}(s_{i,t} s_{j,t}) = E(s_i^2 s_j^2), \tag{14.27} \]

且根据 DOC 模型的假设，有 \(E(s_{i,t} s_{j,t}) = E(s_{i,t-h} s_{j,t-h}) = 0\)。令 \(\rho_{s_i^2, s_j^2}(h) = \text{Corr}(s_{i,t}^2, s_{j,t-h}^2)\)。检验波动率中 DOCs 是否存在的联合滞后-K 原假设和备择假设为：

\[ H_0 : \rho_{s_i^2, s_j^2}(h) = 0 \quad \text{对所有 } i \neq j, \, h = 0, \ldots, K \] \[ H_A : \rho_{s_i^2, s_j^2}(h) \neq 0 \quad \text{对某些 } i \neq j, \, h = 0, \ldots, K. \]

相应的 Ljung-Box 型检验统计量为：

\[ Q_d^0(s^2; K) = n \sum_{i < j} \rho_{s_i^2, s_j^2}(0)^2 + n(n+2) \sum_{h=1}^K \sum_{i \neq j} \frac{\rho_{s_i^2, s_j^2}(h)^2}{n-h}. \tag{14.28} \]

在 \(H_0\) 下，\(Q_d^0(s^2; K)\) 渐近服从自由度为 \(d(d-1)/2 + K d(d-1)\) 的卡方分布。当 \(Q_d^0\) 值较大时拒绝原假设。当 \(H_0\) 被拒绝时，必须寻求备选建模方法。

如图 14.13 所示，VAR(1) 残差的波动率 DOCs 假设被拒绝。检验统计量为 \(Q_2^0(\hat{a}^2, 5) = 356.926\)，p 值接近零。O-GARCH 模型中使用的成分也拒绝了波动率 DOCs 假设，检验统计量为 \(Q_2^0(z^2, 5) = 135.492\)，p 值接近零。从无相关的主成分 \(z_t\) 出发，Matteson 和 Tsay (2011) 提出估计一个正交矩阵 \(W\)，使得成分 \(s_t = W z_t\) 尽可能接近波动率中的 DOCs。这通过最小化 Ljung-Box 型检验统计量 (14.28) 的加权版本相对于分离矩阵 \(W\) 来实现。对于估计的成分 \(s_t\)，波动率 DOCs 的原假设被接受，其中 \(Q_2^0(s^2, 5) = 7.845\)，p 值约为 0.727。

14.14 多元GARCH过程

在识别出DOCs后，对每个过程考虑单变量波动率模型 \(v^2_{i,t} = \mathrm{Var}(s_{i,t}|\mathcal{F}_{t-1})\)。例如，拟合了以下模型：

\[ a_t = M s_t = M V_t^{1/2} \epsilon_t, \] \[ V_t = \mathrm{diag}\{v^2_{1,t}, \dots, v^2_{d,t}\}, \quad \epsilon_{it} \overset{\text{iid}}{\sim} t_{\nu_i}(0,1), \] \[ v^2_{i,t} = \omega_i + \alpha_i s^2_{i,t-1} + \beta_i v^2_{i,t-1}, \] \[ \Sigma_t = M V_t M', \]

其中 \(t_{\nu_i}(0,1)\) 表示标准化Student-t分布，尾指数为 \(\nu_i\)。若所有分量的 \(v^2_{i,t}>0\)，则每个 \(\Sigma_t\) 为正定矩阵。基本动机是，经验上 \(a_t\) 的动态通常可以通过DOCs的可逆线性组合 \(a_t = M s_t\) 很好地近似，其中按定义 \(M = U^{-1}W'\)。

总结而言，\(U\) 通过PCA进行估计以对 \(a_t\) 去相关，\(W\) 通过最小化上述（14.28）的加权版本（对较低滞后赋予更大权重）进行估计。将矩阵 \(U\) 和 \(W\) 结合以估计DOCs \(s_t\)，然后可以适当地应用单变量波动率建模。该方法允许用 \(d\) 个单变量波动率模型对一个 \(d\) 维多元波动率过程进行建模，同时大大减少了参数数量和估计的计算成本，并保持了足够的经验性能。

# 定义 DOC-GARCH 函数 (简化版)
def doc_garch(X, L=4., c=2.25):
    # 1. PCA 去相关
    pca = PCA(whiten=True)
    s = pca.fit_transform(X)  # 去相关成分 (T x d)
    # 2. 对每个成分拟合 GARCH(1,1) 并使用 Student-t 分布
    d = X.shape[1]
    v2 = np.zeros_like(s)
    params = []
    for i in range(d):
        model = arch_model(s[:, i], vol='Garch', p=1, q=1, dist='studentsT')
        res = model.fit(disp='off')
        params.append(res.params)
        v2[:, i] = res.conditional_volatility ** 2
    # 3. 重构协方差矩阵
    M = pca.components_.T  # d x d
    Sigma_pca = np.zeros((d, d, T))
    for t in range(T):
        Sigma_pca[:, :, t] = M @ np.diag(v2[t, :]) @ M.T
    return type('DOC', (object,), {'Sigma_pca': Sigma_pca, 'v2': v2, 's': s})()

14.14.6 动态条件相关（DCC）模型

为了允许时变相关性（这是许多金融应用中常见的特征），研究者提出了单变量波动率模型的非线性组合。Tse 和 Tsui (2002) 以及 Engle (2002) 将 Bollerslev (1990) 的常数相关模型推广，允许这种动态条件相关（DCC）。

与GARCH(1,1)模型类似，Engle (2002) 提出的DCC模型的一阶形式可由以下方程表示：

\[\sigma_{i,t}^2 = \omega_i + \alpha_i a_{i,t-1}^2 + \beta_i \sigma_{i,t-1}^2,\]

\[D_t = \mathrm{diag}\{\sigma_{1,t}, \dots, \sigma_{d,t}\},\]

\[\boldsymbol{\varepsilon}_t = D_t^{-1} \mathbf{a}_t,\]

\[Q_t = (1-\lambda) \boldsymbol{\varepsilon}_{t-1} \boldsymbol{\varepsilon}_{t-1}' + \lambda Q_{t-1},\]

\[R_t = \mathrm{diag}\{Q_t\}^{-1/2} Q_t \mathrm{diag}\{Q_t\}^{-1/2},\]

\[\Sigma_t = D_t R_t D_t.\]

其主要思想是先用单变量波动率模型对每个序列的条件方差\(\sigma_{i,t}^2\)建模，从这些模型中估计scaled innovations \(\boldsymbol{\varepsilon}_t\)（不要与standardized innovations \(\boldsymbol{\epsilon}_t\)混淆），然后专注于对条件相关矩阵\(R_t\)建模。接着在每个时间点\(t\)将它们结合起来估计波动率矩阵\(\Sigma_t\)。递归关系\(Q_t\)是对scaled innovations \(\boldsymbol{\varepsilon}_t\)应用的EWMA模型。它由一个参数\(\lambda \in (0,1)\)控制。矩阵\(Q_t\)需要重新缩放以形成正确的相关矩阵\(R_t\)，其对角线上所有元素均为1。

DCC模型参数可以通过拟极大似然方法分两阶段一致估计。首先，对每个序列拟合单变量GARCH(1,1)模型以估计\(\sigma_{i,t}^2\)。然后，给定\(\boldsymbol{\varepsilon}_t\)，通过最大化仅依赖于相关性的拟似然成分来估计\(\lambda\)。这样做是合理的，因为平方残差不依赖于相关性参数。

在上述形式中，方差分量仅依赖于各自的滞后收益率，而不是联合收益率。此外，每个条件相关性的动态过程被约束为具有相同的持久性参数，类似于EWMA模型。对条件相关矩阵\(R_t\)进行显式参数化并具有灵活的动态性，在高维下的估计难度与\(\Sigma_t\)本身相当。图14.17展示了使用拟极大似然估计对\(\hat{\mathbf{a}}_t\)拟合的DCC模型。估计的条件标准差显示在(a)和(d)中，条件协方差和条件相关分别显示在(b)和(c)中。以下R命令用于估计并生成图14.17。

14.16.2 GARCH-in-均值（GARCH-M）模型

GARCH-in-均值或GARCH-M模型的形式为：

\[ Y_t = \mu + \delta \sigma_t + a_t \] \[ a_t = \sigma_t \epsilon_t \] \[ \sigma_t^2 = \omega + \alpha a_{t-1}^2 + \beta \sigma_{t-1}^2 \]

其中 \(\epsilon_t \stackrel{\text{iid}}{\sim} (0,1)\)。GARCH-M模型直接将波动率作为回归变量。参数 \(\delta\) 代表风险溢价，即对额外风险的补偿。现代投资组合理论表明，波动率增加导致风险增加，需要更高的预期收益。波动率作为收益的统计显著预测变量的存在，是历史收益序列中序列相关性的主要原因之一。数据集 GPRO.csv() 包含 GoPro 股票从2014年6月26日至2015年1月28日的调整后每日收盘价。
