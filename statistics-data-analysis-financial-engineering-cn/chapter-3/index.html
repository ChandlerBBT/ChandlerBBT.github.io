第3章 固定收益证券

3.1 引言

公司通过出售股票和债券为其运营融资。拥有股票份额意味着部分拥有公司所有权。股东共享公司的利润和亏损。持有债券则不同。当你购买债券时，你是在向公司提供贷款，尽管债券与贷款不同，它是可交易的。除非公司违约，否则公司有义务按约定偿还本金并支付利息。债券持有人获得固定的收入流，因此债券被称为“固定收益”证券。

债券看似无风险，甚至有些乏味，但事实并非如此。许多债券是长期的，例如5年、10年、20年甚至30年。即使公司保持偿付能力，或者你购买的是美国国债（违约几乎不可能），你的债券收入仅在持有至到期时才有保障。如果你在到期前出售债券，你的收益将取决于债券价格的变化。债券价格与利率反向变动，因此利率下降会导致债券“上涨”，债券价格上升。长期债券对利率变化的敏感度高于短期债券。你的债券利率是固定的，但市场利率会波动，因此你的债券市场价值也会波动。例如，如果你购买了一张支付5%利息的债券，而利率上升到6%，那么你的债券就不如提供6%利息的新债券。因此，你的债券价格将下跌。如果出售债券，你可能会亏损。

债券的利率取决于其到期期限。例如，2001年3月28日，1期限为3个月的国库券利率为4.23%。2年期、10年期和30年期国债的收益率分别为4.41%、5.01%和5.46%。利率的期限结构描述了利率如何随到期期限变化。

3.2 零息债券

零息债券，也称为纯贴现债券，有时俗称“零券”，在到期前不支付本金或利息。零息债券具有票面价值或面值，即在到期时支付给债券持有人的金额。零息债券以低于面值的价格出售，因此它是一种贴现债券。

例如，考虑一只20年期、面值1000美元、年复利6%的零息债券。市场价格为1000美元在年利率6%下按年折现的现值。即市场价格为

\[\$1,000 / (1.06)^{20} = \$311.80\]

如果年利率为6%但每半年复利一次，则价格为

\[\$1,000 / (1.03)^{40} = \$306.56\]

如果年利率为6%连续复利，则价格为

\[\$1,000 / \exp\{(0.06)(20)\} = \$301.19\]

3.2.1 价格与收益率随利率波动

为具体起见，假设半年复利。假设你以306.56美元买入该零息债券，6个月后利率上升至7%。市场价格现在变为

\[\$1,000 / (1.035)^{39} = \$261.41\]

因此你的投资价值将下跌(306.56美元 - 261.41美元) = 45.15美元。如果你持有债券20年，你仍将获得1000美元，但如果你现在出售，你将损失45.15美元。这相当于收益率

\[(\$261.41 - \$306.56) / \$306.56 = -0.147\]

即-14.7%。

年化收益率 = \(\left(\frac{381.74 - 306.56}{306.56}\right) = 24.5\%\) 每半年，即每年 \(49.05\%\)。而利率仅从 \(6\%\) 变为 \(7\%\)！2 注意，利率上升而债券价格下降。这是一个普遍现象：债券价格与利率总是呈反向变动。

如果利率在六个月后降至 \(5\%\)，那么你的债券价值为

\[\frac{\$1,000}{(1.025)^{39}} = \$381.74.\]

年化收益率为

\[\left(\frac{381.74 - 306.56}{306.56}\right) = 49.05\%.\]

如果利率保持不变仍为 \(6\%\)，那么债券价格为

\[\frac{\$1,000}{(1.03)^{39}} = \$315.75.\]

年化收益率为

\[\left(\frac{315.75 - 306.56}{306.56}\right) = 6\%.\]

因此，如果利率不变，通过在到期前出售债券，你可以获得 \(6\%\) 的年化收益率，与利率相同。但如果利率发生变化，只有持有债券至到期，才能保证 \(6\%\) 的年化收益率。

通用公式

零息债券的价格由下式给出

\[\text{PRICE} = \text{PAR}(1+r)^{-T},\]

其中 \(T\) 是距到期的时间（年），\(r\) 是年利率（按年复利）。如果采用半年复利，则价格为

\[\text{PRICE} = \text{PAR}(1+r/2)^{-2T}. \qquad (3.1)\]

3.3 附息债券

附息债券定期支付利息。附息债券在发行时通常以面值或接近面值出售。到期时，持有人获得等于债券面值的本金支付以及最后一次利息支付。

例如，考虑一个20年期附息债券，面值1,000美元，年票面利率6%，每半年付息一次，即6%按半年复利计算。每次利息支付为30美元。因此，债券持有人每6个月收到40笔30美元的支付，外加20年后的一笔1,000美元本金支付。我们可以验证，按6%年利率（半年利率3%）贴现后，所有付款的现值等于1,000美元：\[\sum_{t=1}^{40} (1.03)^t + \frac{1000}{(1.03)^{40}} = 1000.\]

6个月后，如果利率不变，那么债券（包括现在到期的第一次利息支付）的价值为：\[\sum_{t=0}^{39} (1.03)^t + \frac{1000}{(1.03)^{39}} = (1.03)\left(\sum_{t=1}^{40} (1.03)^t + \frac{1000}{(1.03)^{40}}\right) = 1030,\] 这正是预期中的半年复利6%年收益率。如果利率上升至7%，则6个月后债券（加上到期利息）的价值仅为：\[\sum_{t=0}^{39} (1.035)^t + \frac{1000}{(1.035)^{39}} = (1.035)\left(\sum_{t=1}^{40} (1.035)^t + \frac{1000}{(1.035)^{40}}\right) = 924.49.\] 年收益率为\[\left(\frac{924.49-1000}{1000}\right) = -15.1\%.\] 如果利率在6个月后降至5%，则投资价值为：\[\sum_{t=0}^{39} (1.025)^t + \frac{1000}{(1.025)^{39}} = (1.025)\left(\sum_{t=1}^{40} (1.025)^t + \frac{1000}{(1.025)^{40}}\right) = 1,153.70,\tag{3.2}\] 年收益率为\[\left(\frac{1153.7-1000}{1000}\right) = 30.72\%.\]

3.3.1 通用公式

让我们推导一些有用的公式。假设一个面值为PAR的债券在T年后到期，每半年支付票息C，每半年的收益率（利率）为r，那么该债券发行时的价值为

\[\sum_{t=1}^{2T} \frac{C}{(1+r)^t} + \frac{PAR}{(1+r)^{2T}} = \frac{C}{r}\left\{1-(1+r)^{-2T}\right\} + \frac{PAR}{(1+r)^{2T}} = \frac{C}{r} + \left\{PAR - \frac{C}{r}\right\}(1+r)^{-2T}. \quad (3.3)\]

公式(3.3)的推导

有限几何级数的求和公式为

\[\sum_{i=0}^T r^i = \frac{1-r^{T+1}}{1-r}, \quad (3.4)\]

3.4 到期收益率

假定\(r \neq 1\)。因此，

\[\sum_{t=1}^{2T} \frac{C}{(1+r)^t} = \frac{C}{1+r} \sum_{t=0}^{2T-1} \left(\frac{1}{1+r}\right)^t = \frac{C\left\{1-(1+r)^{-2T}\right\}}{(1+r)\left\{1-(1+r)^{-1}\right\}} = \frac{C}{r}\left\{1-(1+r)^{-2T}\right\}. \quad (3.5)\]

推导的剩余部分是直接的代数运算。

3.4 到期收益率

假设一个\(T=30\)且\(C=40\)的债券售价为1200美元，高于面值200美元。如果该债券按面值出售，那么利率将为每半年0.04（每年0.08）。这个每半年4%的利率称为票息率。但该债券并未按面值出售。如果你以1200美元购买该债券，你每年获得的利息将低于8%。利率低于8%有两个原因。首先，票息支付为40美元，对于1200美元的投资而言，每半年为40/1200=3.333%（每年6.67%）；每年6.67%称为当期收益率。其次，到期时你只能收回1000美元，而非全部1200美元的投资。每年6.67%的当期收益率虽然低于每年8%的票息率，但它高估了收益率，因为未考虑本金的损失。

到期收益率（yield to maturity），常简称为收益率，是平均回报率，包括因债券以高于（或低于）面值购买而导致的资本损失（或收益）。对于该债券，到期收益率是满足下式的\(r\)值：

\[ 1200 = \frac{40}{r} + \left(1000 - \frac{40}{r}\right)(1+r)^{-60}. \tag{3.6} \]

式(3.6)的右边是式(3.3)取\(C=40\)，\(T=30\)，\(\text{PAR}=1000\)的结果。数值求解方程(3.6)很容易。第3.10.1节中的R程序完成以下工作：

  • 在一个网格上计算每个\(r\)值对应的债券价格；
  • 绘制债券价格与\(r\)的关系图（非必需，但查看图形很有趣）；
  • 插值找到使得债券价值等于$1,200的\(r\)值。

求解可得到期收益率为0.0324，即每半年3.24%。图3.1展示了债券价格与收益率（\(r\)）的关系图，显示\(r=0.0324\)对应债券价格$1,200。

到期收益率0.0324小于当期收益率0.0333，而当期收益率又小于票面利率40/1000 = 0.04。（三者均为半年期利率。）每当债券以高于面值的价格出售时，如本例所示，

如果债券价格高于面值，则息票率大于当期收益率，因为债券溢价发行；当期收益率大于到期收益率，因为到期收益率考虑了到期日只收回面值而非全部投资时的资本损失。概括来说：

价格 > 面值 ⇒ 息票率 > 当期收益率 > 到期收益率。

如果债券折价发行，则一切反向。例如，如果债券价格仅为900美元，则到期收益率为0.0448（如前所述，该值可通过插值确定），当期收益率为40/900 = 0.0444，息票率仍为40/1000 = 0.04。一般而言，

价格 < 面值 ⇒ 息票率 < 当期收益率 < 到期收益率。

3.4.1 到期收益率的通用方法

附息债券的到期收益率（基于半年计息）是使得下式成立的 r 值：

\[ \text{PRICE} = \frac{C}{r} + \left\{\text{PAR} - \frac{C}{r}\right\}(1+r)^{-2T}. \tag{3.7} \]

其中 PRICE 是债券的市场价格，PAR 是面值，C 是半年付息金额，T 是距到期日的年数（假定为1/2的整数倍）。

对于零息债券，C=0，则(3.7)变为

\[ \text{PRICE} = \text{PAR}(1+r)^{-2T}. \tag{3.8} \]

3.4.2 即期利率

到期期限为 n 年的零息债券的到期收益率称为 n 年期即期利率，记为 \(y_n\)。用 n 年期即期利率折现从现在起 n 年后的支付，因此，如果 \(y_n\) 是年即期利率，则从现在起 n 年后支付的1美元的净现值为 \(1/(1+y_n)^n\) 美元；如果 \(y_n\) 是半年期利率，则为 \(1/(1+y_n)^{2n}\) 美元。

附息债券是一组零息债券的捆绑，每个利息支付对应一个零息债券，最后一个本金支付对应一个零息债券。这些组成零息债券有不同的到期日，因此有不同的即期利率。因此，附息债券的到期收益率是这一捆绑中零息债券即期利率的复杂“平均”。

例3.1. 使用即期利率求附息债券的价格和到期收益率

考虑一个简单的1年期附息债券，每半年付息40美元，面值1000美元。假设半年即期利率为2.5%/半年，1年期即期利率为3%/半年。将附息债券视为两个零息债券的组合，一个期限为\(T=1/2\)，面值为40美元，另一个期限为\(T=1\)，面值为1040美元。债券的价格等于这两个零息债券的价格之和。两次应用(3.8)得到这些零息债券的价格并求和，我们得到该零息债券的价格：

\[ \frac{40}{1.025} + \frac{1040}{(1.03)^2} = 1019.32. \]

该附息债券的到期收益率是满足下式的\(y\)值：

\[ \frac{40}{1+y} + \frac{1040}{(1+y)^2} = 1019.32. \]

解为\(y=0.0299\)/半年。因此，年化到期收益率为\(2 \times 0.0299 = 5.98\%\)/年。□

一般公式

在本节中，我们将推导出推广例3.1的公式。假设一个附息债券每半年支付票息\(C\)，面值为\(PAR\)，距到期还有\(T\)年。令\(y_1, y_2, \dots, y_{2T}\)分别为期限1/2, 1, 3/2, ..., \(T\)年的零息债券的半年即期利率。那么该附息债券的到期收益率（以半年为基准）是满足下式的\(y\)：

\[ \frac{C}{1+y_1} + \frac{C}{(1+y_2)^2} + \cdots + \frac{C}{(1+y_{2T-1})^{2T-1}} + \frac{PAR+C}{(1+y_n)^{2T}} = \frac{C}{1+y} + \frac{C}{(1+y)^2} + \cdots + \frac{C}{(1+y)^{2T-1}} + \frac{PAR+C}{(1+y)^{2T}}. \tag{3.9} \]

方程(3.9)的左边是附息债券的价格，到期收益率是使(3.9)右边等于该价格的\(y\)值。

求解(3.9)的方法将在第3.10节的R实验中探讨。

3.5.2 描述期限结构

所有期限最多为\(n\)年的期限结构可以通过以下任意一种方式来描述：

  • 期限为1年、2年、…、\(n\)年的零息债券价格在此分别记为\(P(1), P(2), \ldots, P(n)\)；
  • 期限为1年、2年、…、\(n\)年的即期利率（零息债券的到期收益率）分别记为\(y_1, \ldots, y_n\)；

• 远期利率 \(r_1, \ldots, r_n\)，其中 \(r_i\) 是现在可以锁定的第 \(i\) 个未来年度的借款利率（\(i = 1\) 代表下一年，以此类推）。

如本节所述，每组 \(\{P(1), \ldots, P(n)\}\)、\(\{y_1, \ldots, y_n\}\) 和 \(\{r_1, \ldots, r_n\}\) 均可由其他任一组计算得出。例如，后面的公式 (3.11) 给出了用 \(\{r_1, \ldots, r_n\}\) 表示的 \(\{P(1), \ldots, P(n)\}\)；后面的公式 (3.12) 和 (3.13) 分别给出了用 \(\{P(1), \ldots, P(n)\}\) 或 \(\{r_1, \ldots, r_n\}\) 表示的 \(\{y_1, \ldots, y_n\}\)。

期限结构可以通过将当前时间到债券到期日之间的时间间隔分解为若干短的时间段来描述，每个时间段内的利率恒定，但不同时间段之间的利率可以变化。例如，一个3年期贷款可以被视为三个连续的1年期贷款，或六个连续的半年期贷款，依此类推。

3.6 连续复利

现在假设采用连续复利，远期利率为 \(r_1, \ldots, r_n\)。使用连续复利简化了远期利率、到期收益率和零息债券价格之间的关系。

若 \(P(n)\) 是面值为1000美元的 \(n\) 年期零息债券的价格，则

\[ P(n) = \frac{1000}{\exp(r_1 + r_2 + \cdots + r_n)}. \tag{3.18} \]

因此，

\[ \frac{P(n-1)}{P(n)} = \frac{\exp(r_1 + \cdots + r_n)}{\exp(r_1 + \cdots + r_{n-1})} = \exp(r_n), \tag{3.19} \]

且

\[ \log\left( \frac{P(n-1)}{P(n)} \right) = r_n. \tag{3.20} \]

\(n\) 年期零息债券的到期收益率满足方程

\[ P(n) = \frac{1000}{\exp(n y_n)}, \]

且易得

\[ y_n = (r_1 + \cdots + r_n)/n. \tag{3.21} \]

因此，由 \(\{y_1, \ldots, y_n\}\) 可容易地得到 \(\{r_1, \ldots, r_n\}\)，关系为

\[ r_1 = y_1, \]

且对 \(n > 1\)，有

\[ r_n = n y_n - (n-1) y_{n-1}. \]

3.7 连续远期利率

到目前为止，我们一直假设远期利率逐年变化，但在每年内保持不变。当然，这一假设是不现实的，仅仅是为了简化远期利率的引入而做出的。远期利率应被建模为随时间连续变化的函数。

为了以现实的方式指定期限结构，我们假设存在一个称为远期利率函数的函数 \(r(t)\)，使得到期日为 \(T\)、面值等于1的零息债券的当前价格由下式给出：

\[ D(T) = \exp\left\{-\int_0^T r(t) \, dt\right\}. \tag{3.22} \]

其中 \(D(T)\) 称为贴现函数，任何零息债券的价格通过将其面值与贴现函数相乘得到贴现；即，

\[ P(T) = PAR \times D(T), \tag{3.23} \]

其中 \(P(T)\) 是到期日为 \(T\)、面值为 \(PAR\) 的零息债券的价格。同样，

\[ \log P(T) = \log(PAR) - \int_0^T r(t) \, dt, \]

因此对所有 \(T\) 有

\[ -\frac{d}{dT} \log P(T) = r(T). \tag{3.24} \]

公式(3.22)是公式(3.18)的推广。为了理解这一点，假设 \(r(t)\) 是一个分段常数函数 \(r(t) = r_k\)，其中 \(k-1 < t \le k\)。对于这个分段常数 \(r\)，对于任意整数 \(T\)，我们有

\[ \int_0^T r(t) \, dt = r_1 + r_2 + \dots + r_T, \]

因此

\[ \exp\left\{-\int_0^T r(t) \, dt\right\} = \exp\left\{-(r_1+\dots+r_T)\right\}, \]

于是在特殊情况下(3.18)与(3.22)一致。然而，(3.22)是一个更通用的公式，因为它适用于非整数 \(T\) 和任意 \(r(t)\)，而不仅仅是分段常数函数。

到期日为 \(T\) 的零息债券的到期收益率定义为

将(3.25)右端视为\(r(t)\)在区间\(0 \leq t \leq T\)上的平均值，我们看到(3.25)是(3.21)的类似形式。由(3.22)和(3.25)可知，折现函数可从到期收益率获得：

\[D(T)=\exp\{-Ty_T\},\]

因此，到期时间为\(T\)的零息债券的价格等于存在一个等于\(y_T\)的恒定远期利率时的价格。由(3.26)可得：

\[y_T = -\log\{D(T)\}/T.\]

公式(3.22)中的折现函数\(D(T)\)和远期利率函数\(r(t)\)依赖于当前时刻，该时刻在公式中被设为零。然而，我们可能对折现函数和远期利率函数随时间的变化感兴趣。在这种情况下，我们定义\(D(s,T)\)为在时刻\(s\)、面值为1美元、到期时间为\(T\)的零息债券的价格。类似地，如果时刻\(s\)的远期利率曲线为\(r(s,t),\, t \ge s\)，则

\[ D(s,T)=\exp\left\{-\int_{s}^{T} r(s,t) dt\right\}. \tag{3.28} \] 在时间\(s\)，到期时间\(T>s\)的债券的收益率为 \[ y(s,T)=(T-s)^{-1}\int_{s}^{T} r(s,u) du. \] 由于(3.22)中的\(r(t)\)和\(D(t)\)在新记号中分别是\(r(0,t)\)和\(D(0,t)\)，因此(3.22)是(3.28)当\(s=0\)时的特例。类似地，\(y_T\)在新记号中等于\(y(0,T)\)。不过，在本章剩余部分，我们假设\(s=0\)，回到\(r(t)\)和\(D(t)\)的简单记号。

正如我们所看到的，债券是有风险的，因为债券价格对利率敏感。这个问题被称为利率风险。本节描述了一种量化利率风险的传统方法。

3.8 价格对收益率的敏感性

利用式(3.26)，我们可以近似得到如果收益率发生微小变化，零息债券价格如何变化。假设\(y_T\)变为\(y_T+\delta\)，其中收益率的变化\(\delta\)很小。那么\(D(T)\)的变化近似为\(\delta\)乘以 \[ \frac{d}{dy_T}\exp\{-T y_T\}\approx -T\exp\{-T y_T\} = -T D(T). \tag{3.29} \] 因此，由式(3.23)，对于到期期限为\(T\)的零息债券， \[ \frac{\text{债券价格变化}}{\text{债券价格}} \approx -T \times \text{收益率变化}. \tag{3.30} \] 在这个等式中，“\(\approx\)”表示随着\(\delta\to 0\)，右边与左边的比值趋近于1。

式(3.30)值得审视。右侧的负号表明了我们已知的事实：债券价格与利率反向变动。此外，债券价格的相对变化（即等式的左侧）与\(T\)成正比，这量化了长期债券比短期债券具有更高利率风险的原则。

3.8.1 附息债券的久期

回顾一下，附息债券可以被视为一系列不同期限的零息债券的组合。附息债券的久期（我们用\(DUR\)表示）是这些期限的加权平均，权重为...

与各现金流（票息支付和到期面值）的净现值成比例。

现在假设所有收益率都变化一个常数 \(\delta\)，即所有期限 \(T\) 的 \(y_T\) 变为 \(y_T + \delta\)。这个限制性假设是定义久期所必需的。基于这一假设，式 (3.30) 适用于每个现金流，并用这些权重进行加权平均，从而得到对于附息债券：

\[ \frac{\text{债券价格变化}}{\text{债券价格}} \approx -\text{DUR} \times \delta. \qquad (3.31) \]

(3.31) 的推导细节留作练习（练习 15）。久期分析利用 (3.31) 来近似收益率变化对债券价格的影响。

我们可以将 (3.31) 重写为：

\[ \text{DUR} \approx -\frac{1}{\text{价格}} \times \frac{\text{价格变化}}{\text{收益率变化}} \qquad (3.32) \]

并将 (3.32) 用作久期的定义。注意，“债券价格”已被替换为“价格”。原因在于，(3.32) 不仅可以定义债券的久期，还可以定义那些价格依赖于收益率的衍生证券（例如债券看涨期权）的久期。当这一定义推广到衍生品时，久期与基础证券的到期期限无关，而仅仅是价格对收益率敏感性的度量。Tuckman (2002) 给出了一个例子：一个 10 年期附息债券的久期为 7.79 年，而该债券的一个看涨期权的久期为 120.82 年。这些久期表明，看涨期权的风险远高于债券，因为它对收益率变化的敏感度是债券的 15.5 倍（= 129.82 / 7.79）。

遗憾的是，(3.31) 背后关于所有收益率等量变化的假设并不现实，因此久期分析正逐渐不受欢迎，风险价值正在取代久期分析成为评估利率风险的方法4。风险价值及其他风险度量将在第 19 章中讨论。

bondvalue = function(c, T, r, par)
{
  # Computes bv = bond values (current prices) corresponding
  # to all values of yield to maturity in the
  # input vector r
  #
  # INPUT
  # c = coupon payment (semiannual)
  # T = time to maturity (in years)
  # r = vector of yields to maturity (semiannual rates)
  # par = par value
  #
  bv = c/r + (par - c/r)*(1 + r)^(-2*T)

}
def bondvalue(c, T, r, par):
    # Computes bv = bond values (current prices) corresponding
    # to all values of yield to maturity in the input vector r
    #
    # INPUT
    # c = coupon payment (semiannual)
    # T = time to maturity (in years)
    # r = vector of yields to maturity (semiannual rates)
    # par = par value
    #
    bv = c / r + (par - c / r) * (1 + r) ** (-2 * T)
    return bv
