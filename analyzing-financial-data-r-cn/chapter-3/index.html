第3章 投资组合收益率

金融学中的一个关键教训是分散投资（diversification），这意味着我们不应将所有鸡蛋放在一个篮子里。从投资组合管理的角度来看，这意味着我们的投资组合中应该拥有多只股票。在上一章中，我们讨论了单个证券的收益率。在本章中，我们将扩展收益率的计算，以处理投资组合中包含多个证券的情况。

在一个投资组合中，有些股票的价值会上升，而另一些则会下降。最终，我们应该关心的是投资组合的总价值随时间增长。换句话说，我们应该抵制这样一种诱惑，即认为我们选择的投资组合中每只证券都应该是赢家。

投资组合收益率是组合中各证券收益率的加权平均值。每只证券的权重等于该证券投资额占投资组合总价值的百分比。下面，我们将在第3.1节中逐步展示投资组合收益率的计算方法。

对于某些应用，使用矩阵代数（matrix algebra）进行投资组合计算更为简便，尤其是当投资组合包含大量证券时。例如，在第7章讨论投资组合优化时，我们就会采用这种方法。因此，我们在第3.2节中展示如何使用矩阵代数实现投资组合收益率的计算。

在第3.3节中，我们将展示如何构建基准投资组合收益率（benchmark portfolio returns），即一个假设投资组合的收益率，用于与我们自己的投资组合表现进行比较。基准投资组合构建的两种常见方法是使用等权重投资组合（equal-weighted portfolio）和市值加权投资组合（value-weighted portfolio）。我们假设每季度再平衡（rebalancing）来实施这两种方法，这是实践中构建指数时的常用做法。两种加权方法的区别在于再平衡日投资组合价值的重新分配方式。等权重投资组合在每个再平衡日将投资组合的总价值平均分配给所有证券；相比之下，市值加权投资组合则根据证券的市值（即市值较大的公司获得更大的份额）在每个再平衡日重新分配总价值。在这两种方法中，每个再平衡日的权重

3.1 投资组合收益率：长方法

在第2章中，我们计算了单个证券的收益率。现在我们将此扩展到计算由两个或多个证券构成的投资组合的收益率。从数学上讲，投资组合收益率等于投资组合中各个证券收益率的加权平均，权重计算为相对于投资组合总投资额投资于该证券的百分比。具体而言，在一个n资产投资组合中，我们可以按如下方式计算总收益率。首先，计算总投资为

\[ I = \sum_{i=1}^n I_i = I_1 + I_2 + \cdots + I_n, \qquad (3.1) \]

其中 \(I_i\) 是投资于证券 \(i\) 的金额。下一步，投资于每个证券的权重为

\[ w_i = I_i / I, \qquad (3.2) \]

因此，我们有

\[ \sum_{i=1}^n w_i = 1, \qquad (3.3) \]

也就是说，投资于所有资产的百分比之和必须等于投资总额的100%。这保证了投资组合价值中的每一美元都分配给了某种资产。注意，如果投资组合中有现金，我们仍然可以为现金分配权重以及收益率。例如，如果现金投资于货币市场共同基金，我们可以使用该基金的收益率。由于所有资产都分配了权重，这些权重的总和仍必须等于1（即等于投资组合的100%）。

最后，我们可以计算投资组合收益率：

\[ R_p = \sum_{i=1}^n w_i R_i \]

3.1 投资组合收益率：长方法

\[ = w_1 R_1 + w_2 R_2 + \dots + w_n R_n \qquad (3.4) \]

3.2 使用矩阵代数的投资组合收益率

随着投资组合中证券数量的增加，使用式(3.4)会变得更加繁琐。因此，我们必须借助矩阵代数来简化计算。为此，我们需要两个向量。向量是按行或列组织的一系列数字。第一个向量是权重向量，用于存储上述例子中三只股票的权重。第二个向量是收益率向量，用于存储上述例子中三只股票的收益率。

在数学上，我们可以将权重列向量定义为

\[ \mathbf{w} = \begin{pmatrix} w_1 \\ w_2 \\ \vdots \\ w_n \end{pmatrix} \qquad (3.5) \]

将收益率列向量定义为

\[ \mathbf{R} = \begin{pmatrix} R_1 \\ R_2 \\ \vdots \\ R_n \end{pmatrix} \qquad (3.6) \]

因此，投资组合收益率 \(R_p\) 等于

\[ R_p = \mathbf{w}^\top \mathbf{R}, \qquad (3.7) \]

其中 \(\top\) 是矩阵转置运算符。在此例中，转置运算符将权重列向量转换为行向量。这一步是必要的，以便我们可以将两个向量进行矩阵乘法。这是矩阵代数中的一个技术细节。上述最后一步的关键在于，我们需要对每只股票将权重与收益率相乘，然后加总这些值。

步骤1：创建权重行向量

在上面的方程中，由于我们对权重列向量进行了转置（\(\mathbf{w}^\top\)），实际上我们使用的是权重行向量。要创建权重行向量，我们使用 c() 函数，然后使用 matrix() 将其转换为矩阵。matrix() 的第二个参数告诉 R 我们将所有元素放在一行中。

1 > (weight<− c(w.amzn , w.goog , w.aapl))
2 [1] 0.5 0.3 0.2
3 > (mat.weight<− matrix(weight , 1))
4 [,1] [,2] [,3]
5 [1,] 0.5 0.3 0.2

1 > (port.ret2<- as.numeric(mat.weight %*% mat.returns))
2 [1] 3.32018

1 > (port.ret2<- mat.weight %*% mat.returns)

为了确定我们的投资表现，我们将投资的收益率与基准收益率进行比较。选择基准超出了本书的范围，但一般来说，基准应表现出与标的投资相同的风险和收益特征。通常，使用一些可比证券的投资组合作为基准。有些基准是现成的（例如，大盘股的 S&P 500 ETF）。另外，也可以构建基准。在本节中，我们通过一个示例演示如何构建等权重（EW）投资组合和市值加权（VW）投资组合：2018年12月31日投资1000美元于AMZN、GOOG和AAPL，并于2019年12月31日清算持仓。

由于我们正在比较股票的表现与基准的表现，重要的是我们构建的基准收益率是现实的。为了...

3.3 构建基准投资组合收益率 81

例如，相对于不可交易的标普500指数，如果我们想要一个大盘股的基准，我们可能会选择SPDR S&P 500 ETF (SPY)。后者试图模拟标普500指数的收益率，但你可以登录经纪账户购买SPY。这样，当我们比较股票与基准的表现时，投资者实际上可以将资金投入任一证券。

如果我们必须从头开始构建基准，很难找到两家完全相同的公司。因此，合适的比较可能是两只或多只股票的组合。为了正确比较标的股票和基准的表现，我们需要确保基准收益率的计算方式从投资角度来看也是现实的。否则，得出的比较可能存在偏差。例如，共同基金招募说明书有时会报告不扣除费用和手续费的基准。即使我们认为报告的基准是合适的，我们也必须对投资者在尝试复制基准收益率时将产生的费用和手续费进行调整。忽略费用和手续费会导致基准收益率被高估，因为我们可能使用的是标的基金的净收益率（扣除费用和手续费后的净值）。因此，与基准的比较会偏向于显示标的基金表现不佳。

如果我们必须构建自己的基准，例如由可比公司股票组成的基准，那么我们将需要考虑如何现实地创建基准投资组合的收益率。一个关键因素是再平衡假设。在再平衡方面，我们考虑两个问题：再平衡权重和再平衡频率。对于再平衡权重，两种常见的方法是等权重或市值加权。等权重假设在再平衡日期，投资组合价值平均分配给组合中的每只股票。市值加权假设投资组合价值根据股票市值在组合中分配。

对于再平衡频率，我们可以假设只在开始时重新分配权重，然后持有组合（买入并持有）。持有期越长，这个假设就越不现实。典型的再平衡频率是每季度一次，这意味着在每个季度末，我们将根据再平衡权重假设重新分配投资组合价值。例如，标普500指数使用季度再平衡。在每个季度的中间，投资可以自由上下波动，这意味着季度内任意一天的权重可能与季度初的权重不同。

有时，也可以使用其他规则作为日历周期再平衡的替代或补充。这些规则会在季度末之前触发再平衡。例如，可以对每只股票的权重设定一个区间，如果股票的权重超出上限或低于下限，则触发再平衡。这些区间可能因股票而异，因为区间的大小基于诸如股票波动性以及股票与组合中其他股票的相关性等因素。在下面的示例中，我们忽略这些附加规则，只关注日历周期再平衡。

3.3.3 等权重投资组合

等权重投资组合（equal-weighted portfolio，EW）将投资资本均匀分配给组合中的各证券。例如，如果投资 100 美元且组合包含两只股票，则每只股票获得 50 美元投资；如果投资 100 美元但组合包含四只股票，则每只股票获得 25 美元投资。换言之，每只证券的权重为 \(1/n\)，其中 \(n\) 是组合中的证券数量。可以看出，等权重投资组合对小公司和大公司赋予相同权重。等权重指数的例子包括标普 500 等权重指数（S&P 500 Equal Weight Index，由标普公司为满足市场对标普 500 指数“官方”等权重版本的需求而创建），以及 MSCI 为发达市场、新兴市场和所有国家提供的等权重指数。这两个指数每季度再平衡，这意味着在季度之间，成分股权重会根据其表现浮动。在下面的例子中，我们假设投资发生在 2018 年 12 月 31 日，并希望了解截至 2019 年 12 月 31 日的累计收益率。

3.4 时间加权收益率

其中 \(V_t\) 是投资组合在时间 \(t\) 的市场价值，\(C_t\) 是投资组合在时间 \(t\) 的现金流入或流出。例如，如果投资组合收到资金，\(C_t\) 为正现金流。反之，如果从投资组合中提取资金，\(C_t\) 为负现金流。对于每个计算市场价值以及发生现金流入或流出的日期，我们计算持有期收益率。然后，我们将投资期间内的各个持有期收益率连接起来。换句话说，

\[ HPR_t = \frac{V_t + C_t - V_{t-1}}{V_{t-1}}, \qquad (3.8) \] \[ TWRR = \prod_{t=1}^{T} (1 + HPR_t) = (1 + HPR_1) \times (1 + HPR_2) \times \cdots \times (1 + HPR_T). \qquad (3.9) \]

资金加权收益率（Money-Weighted Rate of Return, MWRR）是投资组合的内部收益率（Internal Rate of Return, IRR）。IRR是使项目净现值（即现金流现值减去投资成本）等于零的折现率。类似地，MWRR是使投资组合现金流的现值等于初始值的收益率。即：

\[ 0 = \sum_{t=0}^{T} \frac{C_t}{(1 + \text{MWRR})^t} = C_0 + \frac{C_1}{(1 + \text{MWRR})} + \frac{C_2}{(1 + \text{MWRR})^2} + \cdots + \frac{C_T}{(1 + \text{MWRR})^T}, \tag{3.10} \]

其中 \(C_t\) 是时间 \(t\) 的现金流。我们需要决定对现金流入和流出赋予何种符号。例如，如果将流入投资组合的现金视为投资者的投资，那么 \(C_0\) 为负，随后的现金流入也为负，但现金流出（即从投资组合向投资者的分配）为正。根据笛卡尔符号法则，符号变化的次数可以有多少个不同的 MWRR（即根）。因此，为了至少有一个

对于 MWRR，我们至少需要一个符号变化（即，一个为负现金流量的投资金额，以及随后为正现金流量的分配）。然而，符号变化越多（即，流入和流出的混合），可能存在的 MWRR（即，根）就越多。

我们也可以重写上述方程，令 \(a = 1/(1 + MWRR)\)，使得

\[0 = C_0 + a C_1 + a^2 C_2 + \cdots + a^n C_n. \tag{3.11}\]

上述方程表明，MWRR 的公式是一个 \(T\) 次多项式，必须通过数值方法求解（例如，通过试错法或迭代法）。

在许多 IRR 计算中，假设现金流量以固定的时间间隔（例如，每月或每年）发生。然而，在投资组合的背景下，现金流量可能在任何时间发生。因此，我们需要创建两个函数来使 MWRR 计算工作。首先，我们创建一个计算现金流量净现值（NPV）的函数，该函数根据现金流量发生的日期计算现值。其次，我们创建使用此 NPV 函数的 MWRR 函数。

步骤 1：创建现值函数 我们创建的现值函数接受在不同日期发生的每一笔现金流量，然后将其折现到现在。该函数接受三个参数：现金流量向量、这些现金流量发生的日期向量以及折现率 \(r\)。第 2 行创建一个向量，表示距估值日期的年数。第 3 行为每个现金流量日期计算折现因子。第 4 行通过将现金流量的未来值乘以现值因子来计算每笔现金流量的现值。第 5 行计算所有现金流量流的现值。

1 >pv <- function(cf, dates, r) {
2 +  t <- as.numeric((dates - dates[1]) / 365)
3 +  pv_factor <- (1 + r)^(-t)
4 +  pv <- cf * pv_factor
5 +  value <- sum(pv)
6 +  return(value)
7 +}

步骤 2：创建 MWRR 函数 mwrr() 函数接受三个参数：现金流量向量、这些现金流量发生的日期向量以及一个关于收益率应为多少的猜测。由于我们通过迭代来寻找解，因此一个良好的猜测（即最后一个参数）有助于得到正确的解。在存在多个可能根的情况下，选择正确的猜测有助于我们得到正确的 MWRR。第 2 行告诉 R 在每次迭代中猜测值改变多少。第 3 行确定在我们认为解可接受之前的误差容限。其思路是我们通过试错法来得到收益率。容差 \(tol\) 的值决定了我们在停止迭代前误差应达到的程度。第 4 行是传递给函数的当前收益率的初始值。

循环继续。第5行将迭代计数器设为0。我们将使用这个计数器来观察R经过多少次迭代达到解。

mwrr <- function(cf, dates, guess) {
  delta.x <- 0.01
  tol <- 0.0000001
  cur.x <- guess
  iter <- 0
  for (i in 1:1000) {
    fx <- pv(cf, dates, cur.x)
    cur.x.delta <- cur.x - delta.x
    fx.delta <- pv(cf, dates, cur.x.delta)
    dx <- (fx - fx.delta) / delta.x
    cur.x <- cur.x - (fx / dx)
    iter <- iter + 1
    cat("At iteration", iter, "MWRR equals", cur.x, "\n")
    if (abs(fx) < tol) break
  }
}

在for循环中，我们当前运行1000次迭代。这通常足够了。第7行使用pv()根据我们的收益率猜测值cur.x计算现金流现值。第8行从当前猜测值中减去增量delta.x，然后在第9行使用新猜测值重新运行pv()。第7行和第9行的现值之差在第10行计算。第11行对收益率猜测值进行调整，只要fx中的净现值大于容限（第14行），我们就重复计算。如果必须重复循环，第12行将迭代次数加1。这样，当第13行报告答案时，我们也能知道正确的迭代次数。

计算收益率是大多数投资应用的核心。我们仅在前一章和本章中触及了基本的收益率概念。关于这些及其他收益率计算的优秀讨论见文献[1]。我们还讨论了来自S&P和MSCI的等权重指数。有关这些指数的更多信息可参考以下文档：S&P 500 Equal Weight Index Methodology, August 2016 和 MSCI Equal Weight Indexes Methodology, September 2014。

我们讨论了如何对单个投资组合进行价值加权，并使用了投资于每只证券的金额来计算权重。然而，当现成的指数（例如标准普尔500指数）宣称自己是市值加权时，这些指数并不意味着指数中的每只证券都投入了特定的金额。因此，要进行价值加权，这些指数需要衡量指数各成分的总价值。自然地，可以使用每只股票的市值，即股票价格乘以总流通股数。然而，部分流通股实际上无法交易。这是因为公司的一些股份由无法交易或不打算出售其股份的投资者持有，例如内部人士或出于战略目的而不交易这些股份的长期持有者。因此，这些股份通常从流通股中剔除，以得到股票的自由流通量。使用自由流通量被认为能提高可投资性，因为计算中仅包含可以买卖的股份。主要指数提供商（如S&P和FTSE Russell）在其指数中采用自由流通市值调整方法。
