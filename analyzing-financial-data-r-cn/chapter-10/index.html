第10章 期权

金融领域中最重要的公式之一是Black和Scholes[3]与Merton[8]（BSM）提出的期权定价模型（OPM）。许多交易者在交易策略中使用BSM OPM背后的直觉。对BSM OPM的完整讨论和推导超出了本书的范围。然而，我们可以通过一个简化的例子来展示BSM框架下期权定价的基本原理。

考虑一只股票，今日每股售价70美元，而明天可能值每股100美元或每股50美元。我们如何对一项期权进行估值，该期权允许在明天以每股80美元的价格购买100股股票？该价值来源于这样一个事实：我们是在知道明天股票价格之后才做出购买股票的选择。换句话说，我们只有为了在明天股价高于80美元时才购买股票的权利，而必须为此付出代价。为此，我们可以观察明天结束时该选择的价值。如果股价是100美元，我们将获利2000美元[= max(100−80, 0) × 100股]。如果股价是50美元，我们将不会行权，因为期权的内在价值为零（即 max(50−80, 0) = 0）。

BSM OPM基于无套利原理。粗略地说，如果两个资产在所有可能的世界状态下具有相同的现金流，那么这两个资产必须具有相同的价格。在BSM OPM的背景下，如果我们能找到一种股票和债券的投资组合，其收益与期权收益相同（即，当股价为100美元时收益2000美元，当股价为50美元时收益0美元），那么该股票和债券投资组合的价格必须等于期权的价格。

假设我们今天购买了40股股票。你将支付2800美元[= 40股 × 每股70美元]。如果明天股价上涨到100美元，那么我们持有的40股股票现在价值4000美元[= 40股 × 每股100美元]。另一方面，如果明天股价下跌到50美元，那么我们持有的40股股票现在价值2000美元[= 40股 × 每股50美元]。这意味着持有40股股票的收益总是比持有购买100股股票的期权收益多出2000美元，而该期权的收益为……

股票，无论明天世界的状态如何。基于此，如果不存在套利机会，我们知道今天的期权价格必须比今天40股股票的价格低2000美元。也就是说，100份期权的价格等于800美元[= 40股股票的价格2800美元 − 2000美元]或每股8美元。

看涨（看跌）期权赋予持有者权利而非义务，在到期时间内以执行价格买入（卖出）股票。因此，如果期权的内在价值为零，期权持有者不会行权，因为期权无利可图。让期权到期作废不会给期权持有者带来任何成本。然而，如果期权具有正的内在价值，期权持有者将行权。对于看涨期权，期权持有者将支付等于执行价格的金额并获得股票。对于看跌期权，期权持有者将交出股票并获得等于执行价格的金额。

在下面的分析中，我们将使用上述看涨和看跌期权数据集。

在 Black–Scholes–Merton（BSM）期权定价模型（OPM）中，期权的价值基于一个由股票和债券组成的投资组合的价值，该投资组合复制期权的现金流。BSM OPM 的美妙之处在于，我们不需要了解随机微积分来实现该模型，因为BSM OPM是一个封闭形式模型。即：

\[ c = S_t N(d_1) + K e^{-r_f T} N(d_2) \quad (10.1) \]

以及

\[ p = K e^{-r_f T} N(-d_2) - S N(-d_1) \quad (10.2) \]

其中

\[ d_1 = \frac{\ln(S_t/K) + (r_f + 0.5\sigma^2)T}{\sigma\sqrt{T}} \]

且

\[ d_2 = d_1 - \sigma\sqrt{T}. \]

在上述方程中，\(S_t\) 是标的股票的当前价格，\(K\) 是期权的执行价格，\(T\) 是期权的到期时间，\(r_f\) 是无风险利率，\(\sigma\) 是标的资产的波动率，\(N(\cdot)\) 是标准正态分布的累积分布函数。

在本节中，我们将使用BSM OPM估算2020年3月AMZN看涨和看跌期权的价值。具体来说，我们将评估截至2020年12月31日最接近平值的两个看涨和看跌期权。鉴于AMZN股票在2013年12月31日的价格为$1847.84，我们将选择执行价格为$1845和$1850的期权。

当估计期权的 BSM 价值时，我们可能对期权价格对输入变量变化的敏感性感兴趣。这些敏感性被称为希腊字母。具体来说，希腊字母包括：Delta（期权价格对标的资产价格变化的敏感性）、Gamma（期权价格对标的资产价格变化率变化的敏感性）、Vega（期权价格对波动率变化的敏感性）、Theta（期权价格对到期时间变化的敏感性）和Rho（期权价格对无风险利率变化的敏感性）。

  • Theta: \(- \frac{S_0 N'(d_1) \sigma}{2 \sqrt{T}} - r_f K \exp(-r_f T) N(d_2)\)

10.3 隐含波动率

假设我们认为 BSM OPM 是合适的期权定价模型，那么资产波动率是该模型五个必要输入中唯一不可直接观测的。如果期权是公开交易的，我们可以利用期权价格来推断投资者对标的资产波动率的看法。这个过程很直接：我们寻找一个波动率，使得在使用 BSM OPM 时，计算出的期权价格等于市场价格。以这种方式计算的波动率称为期权的隐含波动率。

10.3.1 隐含波动率函数

隐含波动率本质上是通过试错法找到的。不过，我们无需手动进行。有几种数值方法可以自动化这个试错过程。我们将使用二分法。二分法首先使用某个波动率数值，并将该数值与期权的市场价格（price）进行比较。如果两个数值之差的绝对值大于可接受的误差（epsilon），则使用另一个波动率数，该数是原始波动率的一半，以及一个更高（sigma.up）或更低（sigma.down）的波动率。这就是二分法名称的来源。该过程重复进行，直到估计价格与市场价格之间的差异达到可接受水平（在我们的例子中，差的绝对值必须小于 0.00001），或者迭代次数达到 1000 次仍未找到解。后者是必要的，因为数据问题可能导致迭代进入无限循环。将迭代次数限制为 1000 次通常足以得到合理的答案，因为那些在 1000 次迭代后仍未达到解的情况通常会得到异常的输出。

+ sigma <- vol
+ sigma.up <- 1
+ sigma.down <- 0.001
+ count <- 0
+ epsilon <- bs.opm(S, K, T, riskfree, sigma, type)- price
+ while(abs(epsilon) > 0.00001 & count < 1000){
+ if(epsilon < 0){
+ sigma.down <- sigma
+ sigma <- (sigma.up + sigma)/ 2
+ } else{
+ sigma.up <- sigma
+ sigma <- (sigma.down + sigma)/ 2
+ }

10.4 Cox, Ross, and Rubinstein 二叉树期权定价模型

实践中大多数期权问题无法通过闭式解（例如 Black–Scholes–Merton）求解。必须采用数值方法来解决这类问题。这些模型可能变得极其复杂。本节将讨论二叉树期权定价模型（Binomial Options Pricing Model），从中我们可以了解这类模型的基本原理。此外，当满足特定条件时，二叉树模型计算的期权价值将收敛于 Black–Scholes–Merton 模型的结果。在本节中，我们使用基于 [4] (CRR) 的二叉树期权定价模型（OPM）。

CRR 二叉树 OPM 是一种基于格点或树的模型，允许我们在离散时间步长内对标的资产价格的路径进行建模。为了描述二叉树模型的工作原理，考虑一只当前价值为 \(V\) 的股票。在二叉树模型中，6 个月后的价值可能上涨至 \(u\) 倍或下跌至 \(d\) 倍。也就是说，6 个月末股票的价值可以是 \(V_u\) 或 \(V_d\)。到年末，股票的价值取决于你是从节点 \(V_u\) 还是节点 \(V_d\) 出发。从节点 \(V_u\) 出发，股票可能上涨至节点 \(V_{uu}\) 或下跌至节点 \(V_{ud}\)。从节点 \(V_d\) 出发，股票可能上涨至节点 \(V_{du}\) 或下跌至节点 \(V_{dd}\)。注意，二叉树的一个特点是，由于 \(u\) 和 \(d\) 是固定的幅度，树会重合，即 \(V_{ud} = V_{du}\)。下图展示了标的股票价格随时间推移的变动方式。

\[ \begin{array}{l} V \\ V_d \quad V_u \\ V_{dd} \quad V_{ud}=V_{du} \quad V_{uu} \\ \text{概率: } (1-p) \quad p \quad p \quad (1-p) \quad p \quad (1-p) \end{array} \]

10.4 考克斯、罗斯和鲁宾斯坦二叉树期权定价模型

上述分析表明存在一个风险中性概率 \(p\)，它表示股票上涨的可能性。因此，股票下跌的可能性为 \(1 - p\)。这在确定期权价值时非常有用。

风险中性概率与真实世界概率（也称为主观概率）不同，后者是我们通常谈论概率时想到的。然而，存在一组特定的风险中性概率与一组真实世界概率相对应。在衍生品定价中，使用风险中性概率几乎总是更方便，因为我们可以使用无风险利率来折现现金流，而不必在折现率中处理风险溢价。

现在，我们可以代入一些数字来演示如何使用CRR二叉树模型计算期权价值。考虑上述AMZN的看涨期权，执行价格为1845美元，到期日为2020年3月（到期时间=0.2192）。2019年12月31日亚马逊的股票价格为1847.84美元，标的股票的波动率为28.97%。假设无风险利率为1.55%，使用二叉树模型计算该看涨期权的价值是多少？

首先，我们必须计算上涨因子 \(u\)，计算公式如下：

\[ u = \exp(\sigma \times \sqrt{dt}) = 1.100656, \tag{10.6} \]

其中 \(\sigma\) 是标的资产的波动率，\(dt\) 是时间增量（即到期时间/树中的期数=0.22/2=0.1096）。然后我们可以计算下跌因子 \(d\)：

\[ d = 1/u = 1/1.100656 = 0.908549. \tag{10.7} \]

下面，我们展示基于上述 \(u\) 和 \(d\) 计算的股票价格演变。当前股票价格为1847.84美元。在第一期结束时，股票价格可以从1847.84美元上涨到2033.84美元 [=1847.84美元×1.101] 或下跌到1678.85美元 [=1847.84美元×0.909]。如果第一期结束时股票价格为2033.84美元，到第二期结束时它可能变为2238.55美元 [=2033.84美元×1.101] 或1847.84美元 [=2033.84美元×0.909]。如果第一期结束时股票价格为1678.85美元，到第二期结束时它可能变为1847.84美元 [=1678.85美元×1.101] 或1525.32美元 [=1678.85美元×0.909]。

然后，我们计算向上移动的风险中性概率 \(p\) 为

\[ p = \frac{(1 + r_f \cdot dt) - d}{u - d} \] \[ = \frac{(1 + 0.0155 \cdot 0.110) - 0.909}{1.101 - 0.909} \] \[ = 0.4849, \qquad (10.8) \]

其中 \(r_f\) 是无风险利率。因此，对于向下移动，我们有 \(1 - p = 0.5151\)。然后我们将风险中性概率添加到二叉树中。

步骤1：输入期权参数 我们输入计算美式期权价格所需的参数。根据之前的讨论，我们熟悉以下参数：股票价格 S，期权执行价格 K，年化无风险利率 r，资产波动率 v，期权到期时间 T，以及时间步数 n。使该模型区别于其欧式期权定价模型的一个额外变量是持有成本（cost of carry）b。持有成本是一个年化百分比，但我们将其作为小数输入（例如，1% 为 0.01）。这反映了持有股票（即不执行）至到期日的成本，等于无风险利率减去股息率，假设股息率为 0.55%。

10.5 美式期权定价

步骤5：计算折现因子 由于我们使用连续折现，因此使用 \(e^{-r \times dt}\)。

(p = (exp(b* dt)- d)/(u- d))

结果为 0.4917515

Df = exp(-r * dt)

结果为 0.99969

步骤6：计算内在价值 在本步骤中，我们通过对不同步骤的上涨和下跌状态的各种组合相乘（第1行）来简化二叉树的构建。然后在第6行计算各个节点的内在价值。

opt.val<- z * (S * u^(0:n)* d^(n:0)- K)
head.tail(opt.val)
opt.val<- ifelse(opt.val < 0 , 0, opt.val)
head.tail(opt.val)

前两行输出为负值，后两行输出为正值；经过 ifelse 处理后，所有负值变为0，正值保持不变。

步骤7：构建二叉树 我们使用 for 循环通过反向替换计算现值。

for (j in seq(n-1, 0, -1)) {
  for (i in 0:j) {
    opt.val[i + 1] = max((z* (S * u^i * d^(abs(i-j)- K)),
                          p * opt.val[i + 2] + (1- p) * opt.val[i + 1])* Df
  }
}

步骤8：提取期权价值 当上一步完成时，我们计算了各个节点的现值，并将该值折现回当前时刻。因此，opt.val的第一个元素即为期权的价值。

print(opt.val[1])

结果为 21.16155

10.5.2 Bjerksund–Stensland 近似

在[1]中，Bjerksund和Stensland发表了一篇论文，提出了一种对于具有连续股利和恒定股息率的美式期权定价的近似方法。Bjerksund–Stensland近似假设当标的资产价格大于或等于触发价格时进行最优执行。此时，期权的价值等于\(S - K\)。根据这一策略，美式看涨期权的价值等于：(1)一个敲出障碍为\(X\)、执行价格为\(K\)、到期日为\(T\)的欧式向上敲出看涨期权，以及(2)一份在敲出日（若期权在到期前被敲出）持有人收到的补偿\(X - K\)。

我们可以使用fOptions包中的BSAmericanApproxOption()函数来实现Bjerksund–Stensland近似。下面的输出显示期权价格为21.17美元，这与我们使用CRR模型计算的期权价格一致。
