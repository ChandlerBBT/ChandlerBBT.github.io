关键财务报表比率汇总

资产收益率（Return on Assets, ROA）

= [净利润（Net Income）+ (1 - 税率（Tax Rate）) × 利息费用（Interest Expense）+ 非控制性权益收益（Noncontrolling Interest in Earnings）] / 平均总资产（Average Total Assets）
（第201页）

资产收益率利润率（Profit Margin for ROA）

= [净利润 + (1 - 税率) × 利息费用 + 非控制性权益收益] / 销售收入（Sales）
（第206页）

总资产周转率（Total Assets Turnover）

= 销售收入 / 平均总资产
（第206页）

普通股权益收益率（Return on Common Equity, ROCE）

= (净利润 - 非控制性权益收益 - 优先股股利（Preferred Stock Dividends）) / 平均普通股权益（Average Common Shareholders’ Equity）
（第207页）

普通股权益收益率利润率（Profit Margin for ROCE）

= (净利润 - 非控制性权益收益 - 优先股股利) / 销售收入
（第213页）

资本结构杠杆（Capital Structure Leverage）

= 平均总资产 / 平均普通股权益
（第213页）

普通股权益收益率分解

ROCE = 经营资产收益率（Operating ROA）+ (杠杆（Leverage）× 利差（Spread）)
（第288页）

经营资产收益率（Operating ROA）

= 净利润 / 平均净经营资产（Average Net Operating Assets）
（第288页）

杠杆（Leverage）

= 平均融资义务（Average Financing Obligations） / 平均普通股权益
（第292页）

利差（Spread）

= 经营资产收益率 - 净借款利率（Net Borrowing Rate）
（第292页）

净借款利率（Net Borrowing Rate）

= 净融资费用（税后）（Net Financing Expense, After Tax） / 平均融资义务
（第292页）

短期流动性风险

流动比率= 流动资产 / 流动负债（第295页）
速动比率=（现金及现金等价物 + 短期投资 + 应收账款）/ 流动负债（第296页）
经营活动现金流量对流动负债比率= 经营活动现金流量 / 平均流动负债（第297页）
应收账款周转率= 销售额 / 平均应收账款（第297页）
存货周转率= 销货成本 / 平均存货（第297页）
应付账款周转率= 采购额 / 平均应付账款（第297页）

长期偿付能力风险

负债对资产比率= 总负债 / 总资产（第301页）
负债对股东权益比率= 总负债 / 总股东权益（第301页）
长期债务对长期资本比率= 长期债务 /（长期债务 + 总股东权益）（第301页）
长期债务对股东权益比率= 长期债务 / 总股东权益（第301页）
利息保障倍数（净利润基础）=（净利润 + 利息费用 + 所得税费用 + 归属于非控股权益的净利润）/ 利息费用（第303页）
经营活动现金流量对总负债比率= 经营活动现金流量 / 平均总负债（第304页）
