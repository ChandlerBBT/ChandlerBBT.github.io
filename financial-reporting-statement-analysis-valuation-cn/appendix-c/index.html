随附本书的财务报表分析软件包（Financial Statement Analysis Package, FSAP）是一系列用户友好、适应性强的Excel®模板。用户可利用FSAP手工录入某家公司的财务报表数据，继而开展财务报表分析、预测及估值。FSAP包含五张工作底表：数据表、分析表、预测表、预测开发表及估值表。
